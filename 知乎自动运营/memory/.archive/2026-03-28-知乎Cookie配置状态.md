## 📋 用户配置指南

### 配置知乎 Cookie 的方法

**方法 A：使用日常 Chrome（推荐）**

```bash
# 1. 以调试模式启动 Chrome
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --remote-debugging-port=9222 \
  --remote-allow-origins=*

# 2. 在 Chrome 中访问 https://www.zhihu.com 并登录

# 3. 提取 Cookie
node ~/.openclaw/workspace/scripts/extract-zhihu-cookies-playwright.js

# 4. 验证 Cookie
node ~/.openclaw/workspace/scripts/fetch-zhihu-hot-topics.js
```

**方法 B：使用自动化登录脚本**

```bash
# 打开浏览器并等待登录
node ~/.openclaw/workspace/scripts/zhihu-login-save-cookies.js --headful

# 然后在浏览器中手动登录，脚本会自动保存 Cookie
```
