### 4. 创建批量发布脚本

**文件**: `🛠️自动化系统/scripts/batch-publish-articles.js`

**核心功能**:
1. 读取标准化元数据
2. 加载对应的 Markdown 内容
3. 批量发布到知乎（专栏文章或回答）
4. 记录发布结果

**使用方法**:
```bash
# 模拟发布（不实际发布）
node scripts/batch-publish-articles.js --dry-run --limit 5

# 实际发布
node scripts/batch-publish-articles.js --limit 5
```

**脚本特点**:
- 支持两种类型：专栏文章和回答
- 自动加载 Cookie
- 自动读取 Markdown 内容
- 自动生成发布报告
- 支持批量发布和单篇发布
- 包含详细的日志输出
