## 🎯 使用建议

### 记录原则

1. **真实性**：真实记录每次心跳的执行情况，不虚构
2. **详细性**：详细记录完成的工作，不简化
3. **完整性**：记录所有相关内容，不遗漏
4. **规范性**：按照模板记录，保持格式一致

### 记录时机

- **每次心跳执行完成后**：立即记录，避免遗漏
- **遇到问题时**：记录问题和解决方案
- **发现新方法时**：记录方法和应用场景
- **总结经验时**：记录经验和应用场景

### 回顾频率

- **每周回顾一次**：回顾本周的心跳执行记录
- **每月总结一次**：总结本月的心跳执行记录
- **遇到问题时回顾**：回顾相关的心跳执行记录
