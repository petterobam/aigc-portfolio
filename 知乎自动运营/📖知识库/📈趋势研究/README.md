## 🔍 研究方向

### 1. 大模型进展（周更）

**追踪内容**：
- GPT/Claude/Gemini/国产模型新版本发布
- 能力提升点（上下文长度、推理能力、多模态）
- 定价策略变化

**关注渠道**：
- 各大模型公司官方 Blog
- HuggingFace 模型榜单
- Twitter/X 上的 AI 研究者动态

**输出内容**：
- 新模型能力对比表
- 对开发者的实际影响
- 可以结合的选题方向

---

### 2. AIGC 应用（周更）

**追踪内容**：
- 新工具发布（AutoGPT、LangChain 等）
- 使用场景拓展（写作、编程、设计）
- 企业落地案例

**关注渠道**：
- Product Hunt
- GitHub Trending
- AI 工具推荐社区

**输出内容**：
- 工具横评
- 实战使用经验
- "AI + X" 新场景分析

---

### 3. 知乎平台（月更）

**追踪内容**：
- 算法变化（推荐机制、流量分配）
- 创作激励政策（盐粒、专栏分成）
- 付费专栏规则变化

**关注渠道**：
- 知乎创作中心公告
- 知乎官方创作群
- 创作者社区讨论

**输出内容**：
- 算法变化对内容策略的影响
- 新的变现机会
- 避坑指南（违规风险）

---

### 4. 知识付费市场（月更）

**追踪内容**：
- 用户需求变化（想学什么、愿意付费什么）
- 爆款课程方向（定价、内容结构）
- 竞品专栏动态

**关注渠道**：
- 知乎热门专栏
- 得到、极客时间等平台
- 用户评论和反馈

**输出内容**：
- 用户需求画像
- 爆款课程结构拆解
- 定价策略分析

## 📊 研究产出

### 输出格式

每个研究笔记应包含：

1. **信息来源**（时间、渠道）
2. **核心事实**（发生了什么）
3. **影响分析**（对我意味着什么）
4. **行动建议**（应该怎么做）

### 示例模板

```markdown
# 2026-03-28 - OpenAI 发布 GPT-5

## 📌 信息来源
- 时间：2026-03-28
- 渠道：OpenAI 官方 Blog

## 💡 核心事实
- 上下文长度从 128K 扩展到 1M tokens
- 推理能力显著提升，编程能力接近资深工程师
- 定价策略调整（详见价格表）

## 🔍 影响分析
- 1M 上下文让长文档处理成为可能，可以写"如何用 AI 处理长文档"
- 编程能力提升，"AI + 编程"赛道竞争加剧
- 定价变化可能影响用户选择

## 🎯 行动建议
- 立即选题：《1M 上下文意味着什么？AI 长文档处理实战》
- 优先级：高（热点 + 实用）
- 变现路径：付费专栏《AI 编程实战》
```
