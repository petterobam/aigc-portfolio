## 🔧 OpenClaw 技巧

**用途**：深入研究 OpenClaw 的功能特性和高阶用法，为「OpenClaw 实战系列」提供内容素材。

### 研究重点方向

| 方向 | 内容 | 优先级 |
|------|------|---------|
| 核心功能 | 各模块功能全解，初学者友好 | 🔴 高 |
| Workflow 搭建 | 从需求到落地的完整搭建流程 | 🔴 高 |
| Prompt 工程 | 在 OpenClaw 中写出高质量 Prompt | 🔴 高 |
| 工具集成 | 与 API/外部工具/数据源集成 | 🟡 中 |
| 高阶技巧 | 批处理、自动化、性能优化 | 🟡 中 |
| 避坑指南 | 常见错误和解决方案 | 🟢 低 |

### 研究方法
1. 实际操作，记录每个功能的使用细节
2. 参考官方文档，确保信息准确
3. 对比其他同类工具，发现差异化价值
4. 收集社区常见问题，整理成 FAQ 内容

## 📊 竞品分析

**用途**：研究知乎同赛道头部创作者的内容策略，学习爆款内容结构，避免走弯路。

### 分析维度

| 维度 | 关注点 |
|------|-------|
| 选题策略 | 高频选题方向、话题角度、选题时机 |
| 内容结构 | 开篇方式、论证逻辑、结尾设计 |
| 数据表现 | 赞同/收藏/评论/关注数分布 |
| 变现方式 | 付费专栏定价、内容结构、转化话术 |
| 更新节奏 | 发布频率、内容长度、系列化策略 |

### 重点研究对象
- 知乎 AI 技术方向 10w+ 关注的创作者
- 知识付费专栏销量排名前列的技术专栏
- 单篇赞同数 1000+ 的 AI 技术文章

### 值得收录的内容
- ✅ 对创作有直接参考价值的研究结论
