## ❓ 常见问题与解决方案

### 一、知乎登录问题

#### 问题 1：Cookie 过期
**症状**：`check-zhihu-login.js` 提示"未登录"
**原因**：Cookie 已过期（知乎 Cookie 有效期约 7-14 天）
**解决方案**：
1. 重新运行 `zhihu-login-persistent.js`
2. 手动完成登录
3. 验证 Cookie 有效性

#### 问题 2：登录验证码失败
**症状**：登录时验证码无法通过
**原因**：验证码识别失败或验证码过期
**解决方案**：
1. 手动输入验证码
2. 刷新验证码
3. 重新开始登录流程

#### 问题 3：账号被限制
**症状**：无法登录或登录后账号异常
**原因**：账号被知乎限制（如发布违规内容）
**解决方案**：
1. 检查账号状态
2. 联系知乎客服
3. 申请账号恢复

### 二、批量发布问题

#### 问题 1：文件不匹配
**症状**：`batch-publish-articles-v2.js` 提示"找不到 Markdown 文件"
**原因**：标准化元数据文件名和实际 Markdown 文件名不匹配
**解决方案**：
1. 运行 `fix-md-file-mapping.js`
2. 手动检查文件名
3. 更新元数据文件

#### 问题 2：发布失败
**症状**：文章发布时提示错误
**原因**：网络问题、知乎反爬、内容违规
**解决方案**：
1. 检查网络连接
2. 增加随机延迟
3. 检查内容是否符合知乎规范
4. 手动发布失败的文章

#### 问题 3：文章格式错误
**症状**：发布后的文章格式不正确
**原因**：Markdown 转换失败
**解决方案**：
1. 检查 Markdown 格式
2. 使用工具转换 Markdown 到 HTML
3. 手动编辑文章格式

### 三、专栏上线问题

#### 问题 1：专栏审核失败
**症状**：知乎审核拒绝专栏上线
**原因**：内容不符合知乎规范、版权问题
**解决方案**：
1. 修改内容
2. 删除侵权内容
3. 重新提交审核

#### 问题 2：专栏定价问题
**症状**：知乎限制专栏定价
**原因**：定价过高或过低，不符合知乎定价策略
**解决方案**：
1. 调整定价
2. 参考同类专栏定价
3. 联系知乎客服

#### 问题 3：专栏订阅量低
**症状**：上线后订阅量远低于预期
**原因**：推广不足、内容质量不高、定价不合理
**解决方案**：
1. 加强推广
2. 优化内容
3. 调整定价
4. 收集用户反馈

### 四、数据追踪问题

#### 问题 1：数据采集失败
**症状**：无法获取专栏数据
**原因**：Cookie 过期、知乎 API 变更
**解决方案**：
1. 重新登录知乎
2. 更新数据采集脚本
3. 手动记录数据

#### 问题 2：数据分析不准确
**症状**：数据分析结果与实际情况不符
**原因**：数据采集不完整、分析方法错误
**解决方案**：
1. 检查数据采集
2. 验证数据分析方法
3. 手动计算关键指标
