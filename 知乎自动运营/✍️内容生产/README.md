## 🔄 内容生产流程

```
选题池（挑选）→ 文章草稿（写作）→ 自查打磨 → 发布包（准备发布）→ 📤待发布
```

### 第一步：选题
- 从 `选题池/` 中挑选优先级最高的选题
- 选题来源：知乎热榜、AIGC 行业动态、OpenClaw 新特性、用户评论需求
- 选题评估维度：搜索量、竞争度、与账号定位匹配度、变现潜力

### 第二步：写作草稿
- 在 `文章草稿/` 新建文件，命名格式：`{日期}-{选题关键词}.md`
- 先写结构大纲，再填充内容
- 遵循黄金写作公式：**痛点场景 → 原理讲解 → 实操步骤 → 避坑总结**

### 第三步：自查打磨
完成初稿后，必须逐项自查：
- [ ] 标题是否符合爆款公式（见 `📚方法论/🎯爆款公式/`）
- [ ] 是否有明确的实用价值（读者看完能立刻用上）
- [ ] 结构是否清晰（善用标题层级、分隔线）
- [ ] 是否有图/代码/示例（技术内容必须有）
- [ ] 字数是否达标（回答 800-2000 字，专栏 1500-3000 字）
- [ ] 有无重复段落、语义重复的句子
- [ ] 结尾是否有互动引导（提问、引导关注专栏）

### 第四步：打包发布
- 在 `发布包/` 整理完整发布物料：
  - 正文 Markdown 文件
  - 配图（流程图、架构图、截图等）
  - 摘要（100 字内，用于专栏封面描述）
  - 标签建议（3-5 个知乎话题标签）
- 转移到 `📤待发布/` 等待自动发布

## 📌 写作注意事项

1. **不要重复段落**：初稿完成后全文搜索重复表达，坚决删除
2. **不要空洞开头**：第一段直接切入主题，不要写"随着 AI 的发展……"
3. **代码要可运行**：所有代码示例必须经过验证，不能贴伪代码
4. **图要有意义**：不要用纯装饰图，每张图都要配说明文字
5. **结论前置**：知乎用户习惯先看结论，再决定是否深读
