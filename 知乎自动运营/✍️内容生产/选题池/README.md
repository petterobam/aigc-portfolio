## 🎯 选题策略

### 选题原则

1. **内容分布**（遵循 40:30:20:10 黄金比例）
   - 实战教程（40%）：手把手教，立竿见影
   - 原理深度（30%）：有深度有门槛，建立权威
   - 热点解读（20%）：追热点蹭流量
   - 复盘总结（10%）：真实感强，建立信任

2. **选题优先级**
   - 🔥 高优先级：热榜相关 + 强变现潜力，立即创作
   - ⭐ 中优先级：搜索量大 + 持续流量，本周创作
   - 📌 储备：有价值但不紧急，策略性创作

3. **变现路径设计**
   - 每个选题都要明确变现路径
   - 高优先级选题优先设计付费产品
   - 中优先级选题补充产品矩阵

### 选题评估维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 搜索量 | 30% | 知乎搜索热度、百度指数 |
| 竞争度 | 20% | 同类内容的数量和质量 |
| 定位匹配度 | 25% | 与账号定位的契合度 |
| 变现潜力 | 25% | 能否转化为付费产品 |
