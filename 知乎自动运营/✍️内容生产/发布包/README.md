# 📤 发布包

> **定位**：待发布内容的准备和打包区，内容经过检查后移至此处，准备发布。

---

## 🎯 使用说明

**发布包是内容的"准发布"状态**：
- 内容从 `✍️内容生产/文章草稿/` 创作完成
- 经过自查和优化后，移至此处
- 准备发布（手动或自动化）到知乎平台

## ✅ 发布检查清单

### 基础检查
- [ ] 标题是否吸引人？
- [ ] 开场是否抓住注意力？
- [ ] 结构是否清晰（标题层级）？
- [ ] 字数是否合适？
- [ ] 是否有配图（至少 4 张）？

### 内容检查
- [ ] 是否有明确的价值承诺？
- [ ] 是否有实用的代码/配置示例？
- [ ] 是否有避坑指南？
- [ ] 是否讲清楚了原理？
- [ ] 是否有实际案例？

### 互动检查
- [ ] 是否引导评论？
- [ ] 是否引导关注？
- [ ] 是否引导点赞/收藏？

### 格式检查
- [ ] Markdown 格式是否正确？
- [ ] 图片是否清晰？
- [ ] 代码是否可运行？
- [ ] 链接是否有效？

## 📋 发布流程

### 1. 内容完成
- 文章创作完成（`✍️内容生产/文章草稿/`）
- 自查和优化完成
- 质量检查通过

### 2. 移动到发布包
- 从 `文章草稿/` 移动到 `发布包/待发布队列/`
- 按优先级编号（001、002、003...）

### 3. 发布前检查
- 使用发布检查清单逐项检查
- 修复发现的问题

### 4. 发布
- 手动发布：复制内容到知乎发布
- 自动发布：使用自动化系统发布（待实现）

### 5. 归档
- 发布完成后，移动到 `📤已发布/`
- 更新发布记录
