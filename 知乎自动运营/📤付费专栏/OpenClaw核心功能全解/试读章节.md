## 1. OpenClaw 是什么？能做什么？

### 1.1 什么是 OpenClaw？

OpenClaw 是一个**基于 AI Agent 的开源工作流自动化平台**，让你用自然语言描述任务，AI 自动帮你完成。

**核心定位**：
- 🤖 AI Agent 平台：智能体自动执行任务
- 🔧 工作流引擎：串行、并行、条件、循环等复杂流程
- 🧩 可扩展架构：通过 Skill 扩展功能
- 🌐 跨平台支持：macOS、Linux、Docker、云服务

**一句话总结**：
> OpenClaw = AI Agent + Workflow Automation + Extensible Skills

---

## 2. OpenClaw 核心概念

在开始使用 OpenClaw 之前，需要理解以下几个核心概念：

### 2.1 Agent（智能体）

**Agent 是什么？**
Agent 是 OpenClaw 的执行单元，它是一个 AI 智能体，负责理解和执行你的指令。

**Agent 的特点**：
- 🧠 **理解力**：用自然语言描述任务，Agent 理解并执行
- 🔄 **记忆能力**：Agent 可以记住上下文，支持多轮对话
- 🛠️ **工具调用**：Agent 可以调用各种工具完成任务
- 🤝 **协作能力**：多个 Agent 可以协同完成复杂任务

**Agent 示例**：
```
你: 帮我查一下今天的天气
Agent: 正在获取天气信息... [调用 weather Skill]
     今天北京晴，气温 18°C，微风。
```

---

### 2.2 Skill（技能）

**Skill 是什么？**
Skill 是 OpenClaw 的功能单元，类似于 App Store 里的 App，每个 Skill 提供特定的功能。

**Skill 的特点**：
- 🔧 **即插即用**：安装后直接使用，无需配置
- 🌐 **跨平台**：一个 Skill 可以在多个平台上运行
- 📦 **模块化**：通过组合多个 Skill 完成复杂任务
- 🔓 **开源**：所有 Skill 都是开源的，可以自定义修改

**常用 Skill**：
- `web-search`：网络搜索
- `weather`：天气查询
- `memory`：记忆管理
- `image`：图像分析
- `feishu`：飞书集成
- `github`：GitHub 操作

**Skill 示例**：
```yaml
# 使用 web-search Skill 搜索信息
你: 搜索 "OpenClaw 教程"
Agent: [调用 web-search Skill]
     找到 5 条相关结果：
     1. OpenClaw 快速入门 - 官方文档
     2. OpenClaw 核心功能全解 - 知乎专栏
     3. OpenClaw 最佳实践 - GitHub
     ...
```

---

### 2.3 Workflow（工作流）

**Workflow 是什么？**
Workflow 是 OpenClaw 的任务编排引擎，用于定义任务的执行流程。

**Workflow 的特点**：
- 🔗 **流程定义**：用自然语言或代码定义任务的执行顺序
- 🔄 **多种模式**：支持串行、并行、条件、循环等执行模式
- ⚡ **异步执行**：支持异步任务和并行处理
- 🔁 **可复用**：Workflow 可以复用和组合

**Workflow 示例**：
```yaml
# 定义一个邮件自动化工作流
workflow:
  - step: 检查邮件
    tool: gmail
    action: list
  - step: 分类邮件
    tool: ai
    action: classify
  - step: 回复草稿
    tool: gmail
    action: reply
```

---

### 2.4 Session（会话）

**Session 是什么？**
Session 是 OpenClaw 的会话管理单元，每个 Session 是一个独立的执行环境。

**Session 的特点**：
- 🔐 **隔离性**：每个 Session 是独立的，互不干扰
- 🧠 **记忆能力**：Session 可以记住上下文，支持多轮对话
- 🔄 **持久化**：Session 可以保存和恢复
- 🌐 **多实例**：可以同时运行多个 Session

**Session 示例**：
```
Session 1: [知乎运营] 自动发布文章
Session 2: [番茄小说] 自动创作故事
Session 3: [个人助手] 日程管理
```

## 3. 安装和配置

### 3.1 系统要求

**支持的操作系统**：
- macOS（推荐 macOS 12+）
- Linux（推荐 Ubuntu 20.04+）
- Docker（全平台支持）

**硬件要求**：
- CPU：至少 2 核
- 内存：至少 4GB
- 硬盘：至少 10GB 可用空间

**网络要求**：
- 需要能够访问 OpenAI API（如果使用 OpenAI 模型）
- 需要能够访问 GitHub（安装 Skill）

---

### 3.2 安装 OpenClaw

#### 方法 1：使用 Homebrew（macOS 推荐）

```bash
# 安装 OpenClaw
brew install openclaw

# 验证安装
openclaw --version
```

#### 方法 2：使用 npm（跨平台）

```bash
# 安装 OpenClaw
npm install -g openclaw

# 验证安装
openclaw --version
```

#### 方法 3：使用 Docker（全平台）

```bash
# 拉取镜像
docker pull openclaw/openclaw:latest

# 运行容器
docker run -it openclaw/openclaw
```

---

### 3.3 初始化配置

```bash
# 初始化配置
openclaw init

# 按照提示完成配置
# 1. 设置 API Key（OpenAI、Anthropic 等）
# 2. 选择默认模型
# 3. 设置工作目录
# 4. 配置日志级别
```

**配置文件示例**（`~/.openclaw/config.yaml`）：
```yaml
# API 配置
apis:
  openai:
    apiKey: sk-xxx...
    model: gpt-4
  anthropic:
    apiKey: sk-ant-xxx...
    model: claude-3-opus

# 默认模型
defaultModel: gpt-4

# 工作目录
workspace: ~/.openclaw/workspace

# 日志级别
logLevel: info
```

---

### 3.4 安装常用 Skill

```bash

# 安装 weather Skill
openclaw skill install weather

# 安装 memory Skill
```

## 4. 第一个 Workflow：自动化发邮件

现在，让我们用 OpenClaw 搭建第一个自动化 Workflow：**自动化发邮件**。

### 4.1 需求分析

**场景**：每天早上 9 点，自动发送邮件提醒团队成员今日任务

**需求拆解**：
1. 📅 获取今日任务（从 Trello 或 Google Sheets）
2. 📝 生成邮件内容（用 AI 总结任务）
3. 📧 发送邮件（通过 Gmail API）
4. 📊 记录发送日志（保存到本地文件）

---

### 4.2 创建 Workflow

#### 步骤 1：获取今日任务

```yaml
# workflow/daily-task-reminder.yaml
version: "1.0"
name: "每日任务提醒"
description: "每天早上 9 点自动发送邮件提醒团队成员今日任务"

steps:
  - name: "获取今日任务"
    tool: "trello"
    action: "list"
    params:
      list: "今日任务"
```

---

#### 步骤 2：生成邮件内容

```yaml
  - name: "生成邮件内容"
    tool: "ai"
    action: "generate"
    params:
      prompt: "请根据以下任务列表，生成一封邮件提醒：
               {tasks}
               要求：简洁明了，突出重点"
```

---

#### 步骤 3：发送邮件

```yaml
  - name: "发送邮件"
    tool: "gmail"
    action: "send"
    params:
      to: "team@example.com"
      subject: "📅 今日任务提醒"
      body: "{content}"
```

---

#### 步骤 4：记录发送日志

```yaml
  - name: "记录发送日志"
    tool: "file"
    action: "write"
    params:
      path: "logs/email-sent.log"
      content: "{timestamp}: 发送给 {to}，共 {taskCount} 个任务"
```

---

### 4.3 完整 Workflow 文件

```yaml
# workflow/daily-task-reminder.yaml
version: "1.0"
name: "每日任务提醒"
description: "每天早上 9 点自动发送邮件提醒团队成员今日任务"

steps:
  - name: "获取今日任务"
    tool: "trello"
    action: "list"
    params:
      list: "今日任务"
    output: "tasks"

  - name: "生成邮件内容"
    tool: "ai"
    action: "generate"
    params:
      prompt: "请根据以下任务列表，生成一封邮件提醒：
               {tasks}
               要求：简洁明了，突出重点"
    output: "content"

  - name: "发送邮件"
    tool: "gmail"
    action: "send"
    params:
      to: "team@example.com"
      subject: "📅 今日任务提醒"
      body: "{content}"
    output: "emailId"

  - name: "记录发送日志"
    tool: "file"
    action: "write"
    params:
      path: "logs/email-sent.log"
      content: "{timestamp}: 发送给 {to}，共 {taskCount} 个任务"
    output: "logPath"
```

---

### 4.4 运行 Workflow

```bash
# 运行 Workflow
openclaw workflow run workflow/daily-task-reminder.yaml

# 查看执行日志
openclaw workflow logs daily-task-reminder

# 查看执行状态
openclaw workflow status daily-task-reminder
```

---

### 4.5 定时执行

```bash

openclaw workflow schedule daily-task-reminder \
  --cron "0 9 * * *"

# 查看定时任务

```

## 5. 实战案例：自动化日程提醒

让我们再做一个实战案例：**自动化日程提醒**。

**场景**：每天早上 8 点，自动提醒今天的重要日程

**Workflow 设计**：
```yaml
# workflow/daily-schedule-reminder.yaml
version: "1.0"
name: "每日日程提醒"
description: "每天早上 8 点自动提醒今天的重要日程"

steps:
  - name: "获取今日日程"
    tool: "calendar"
    action: "list"
    params:
      date: "today"
    output: "events"

  - name: "筛选重要日程"
    tool: "ai"
    action: "filter"
    params:
      prompt: "请从以下日程中筛选出重要事件：
               {events}
               标准：重要会议、截止日期、重要约会"
    output: "importantEvents"

  - name: "生成提醒消息"
    tool: "ai"
    action: "generate"
    params:
      prompt: "请根据以下重要日程，生成一条提醒消息：
               {importantEvents}
               要求：简洁友好，突出时间"
    output: "message"

  - name: "发送提醒"
    tool: "feishu"
    action: "send"
    params:
      channel: "日程提醒"
      text: "{message}"
```

**运行 Workflow**：
```bash
# 运行 Workflow
openclaw workflow run workflow/daily-schedule-reminder.yaml

# 设置定时任务
openclaw workflow schedule daily-schedule-reminder \
  --cron "0 8 * * *"
```
