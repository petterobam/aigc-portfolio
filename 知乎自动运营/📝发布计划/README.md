## 📋 发布前检查清单

发布前必须逐项确认：

- [ ] 标题是否精准命中目标读者搜索意图？
- [ ] 开头前 100 字是否有足够的吸引力？
- [ ] 内容是否有清晰的结构（结论先行）？
- [ ] 是否包含代码示例 / 流程图 / 截图等辅助说明？
- [ ] 是否有明确的收藏点（可复用的模板、清单、总结）？
- [ ] 结尾是否有互动引导（提问或引导关注专栏）？
- [ ] 标签是否选择准确（最多 5 个，优先选高热度话题）？
- [ ] 是否关联到相关专栏？
- [ ] 格式预览是否正常（排版、代码块、图片）？

---

## 📊 发布节奏原则

1. **稳定优先于爆发**：宁可每天一篇中等质量，也不要爆发式更新后断更
2. **热点必须快**：知乎热榜问题有时效性，发现热点立即响应
3. **系列化坚持**：OpenClaw 系列、AIGC 系列要保持固定更新节奏，培养读者预期
4. **质量不妥协**：每篇内容发布前必须通过检查清单
