### 3. 技术内容公式（3个）

#### 3.1 代码示例公式

**公式**: 完整代码 + 详细注释 + 运行结果 + 避坑说明

**应用场景**: 适用于所有包含代码示例的技术内容

**要素说明**:
- **完整代码**: 代码可以直接运行，不依赖外部环境
- **详细注释**: 每个关键步骤都有注释，便于理解
- **运行结果**: 展示代码运行后的输出，验证效果
- **避坑说明**: 常见问题和解决方案，避免踩坑

**应用示例**:
```python
# 完整代码：简单的 Attention 实现
import torch
import torch.nn.functional as F
import math

def attention(Q, K, V):
    """
    计算 Self-Attention

    参数:
        Q: Query 矩阵，形状 [batch_size, seq_len, d_k]
        K: Key 矩阵，形状 [batch_size, seq_len, d_k]
        V: Value 矩阵，形状 [batch_size, seq_len, d_k]

    返回:
        注意力输出，形状 [batch_size, seq_len, d_k]
    """
    d_k = Q.size(-1)  # 获取维度

    # 计算 Q 和 K 的点积
    scores = torch.matmul(Q, K.transpose(-2, -1))

    # 除以 sqrt(d_k)，防止梯度消失
    scores = scores / math.sqrt(d_k)

    # Softmax 归一化
    attn_weights = F.softmax(scores, dim=-1)

    # 与 V 相乘
    output = torch.matmul(attn_weights, V)

    return output

# 运行示例
Q = torch.randn(2, 5, 8)  # batch_size=2, seq_len=5, d_k=8
K = torch.randn(2, 5, 8)
V = torch.randn(2, 5, 8)

output = attention(Q, K, V)
print(f"输出形状: {output.shape}")  # 输出形状: torch.Size([2, 5, 8])

# 避坑说明：d_k 不能为 0，否则会除零错误
# 解决：确保 d_k > 0
```
