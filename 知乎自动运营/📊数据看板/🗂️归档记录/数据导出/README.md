# 📊 数据导出

> 导出知乎技术内容运营的数据，用于外部分析和展示

---

## 🎯 数据导出目标

### 核心目标

1. **数据导出**：导出运营数据，供外部分析使用
2. **数据可视化**：导出数据供可视化工具使用
3. **数据分享**：导出数据供团队分享
4. **数据备份**：导出数据作为备份

---

## 📋 导出类型

### 导出分类

| 导出类型 | 频率 | 格式 | 用途 |
|---------|------|------|------|
| 文章数据导出 | 每周 | CSV、JSON | 数据分析 |
| 账号增长导出 | 每周 | CSV、JSON | 数据分析 |
| 变现数据导出 | 每周 | CSV、JSON | 数据分析 |
| 周复盘报告导出 | 每周 | Markdown、PDF | 报告分享 |
| 月度总结导出 | 每月 | Markdown、PDF | 报告分享 |

---

## 📋 导出流程

### 文章数据导出流程

1. **数据收集**
   - 从知乎创作中心收集文章数据
   - 从 `📈核心数据/文章数据追踪.md` 收集数据
   - 整合数据到统一格式

2. **数据验证**
   - 验证数据完整性
   - 验证数据准确性
   - 记录数据异常

3. **数据导出**
   - 导出为 CSV 格式
   - 导出为 JSON 格式
   - 保存到导出目录

4. **验证导出**
   - 验证导出文件完整性
   - 验证导出文件可读性
   - 记录导出元数据

### 账号增长导出流程

1. **数据收集**
   - 从知乎创作中心收集账号增长数据
   - 从 `📈核心数据/账号增长数据.md` 收集数据
   - 整合数据到统一格式

2. **数据验证**
   - 验证数据完整性
   - 验证数据准确性
   - 记录数据异常

3. **数据导出**
   - 导出为 CSV 格式
   - 导出为 JSON 格式
   - 保存到导出目录

4. **验证导出**
   - 验证导出文件完整性
   - 验证导出文件可读性
   - 记录导出元数据

### 周复盘报告导出流程

1. **报告生成**
   - 在 `📊周复盘/` 目录生成周复盘报告
   - 验证报告完整性

2. **报告导出**
   - 导出为 Markdown 格式
   - 导出为 PDF 格式
   - 保存到导出目录

3. **验证导出**
   - 验证导出文件完整性
   - 验证导出文件可读性
   - 记录导出元数据

## 📋 导出命名规范

### 文件命名规范

```
{数据类型}-{日期}.{扩展名}

示例：
- 文章数据-20260331.csv
- 文章数据-20260331.json
- 账号增长-20260331.csv
- 变现数据-20260331.csv
- 周复盘报告-Week1.md
- 周复盘报告-Week1.pdf
- 月度总结-2026-03.md
- 月度总结-2026-03.pdf
```

## 📋 导出数据格式

### CSV 格式

**文章数据导出（CSV）**

```csv
文章ID,标题,发布日期,赞同数,收藏数,评论数,阅读量,标签1,标签2,字数,代码片段数
ZH-ART-202603-001,文章标题,2026-03-28,500,250,80,10000,标签1,标签2,10000,10
```

**账号增长导出（CSV）**

```csv
日期,粉丝数,新增粉丝,增长率
2026-03-01,700,30,0.045
2026-03-02,710,10,0.014
```

**变现数据导出（CSV）**

```csv
专栏ID,专栏名称,价格,订阅数,收入
COL-001,付费专栏名称,299,15,4485
```

### JSON 格式

**文章数据导出（JSON）**

```json
{
  "exportDate": "2026-03-31",
  "exportTime": "2026-03-31 23:59:59",
  "exportBy": "知乎技术分享与知识付费运营 AI",
  "articles": [
    {
      "articleId": "ZH-ART-202603-001",
      "title": "文章标题",
      "publishDate": "2026-03-28",
      "likes": 500,
      "favorites": 250,
      "comments": 80,
      "views": 10000,
      "tags": ["标签1", "标签2"],
      "wordCount": 10000,
      "codeSnippets": 10
    }
  ]
}
```
