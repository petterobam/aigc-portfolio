## 📈 数据分析流程

### 每日数据检查

1. **检查发布文章的数据**
   - 赞同数、收藏数、评论数
   - 阅读量、完读率
   - 关注转化数

2. **记录数据**
   - 更新 `📊数据看板/📈核心数据/文章数据追踪.md`
   - 记录异常数据

3. **快速分析**
   - 识别增长亮点
   - 发现潜在问题

### 每周数据复盘

1. **汇总本周数据**
   - 发布文章数、阅读量、互动数
   - 粉丝增长数据
   - 收入数据（如果有）

2. **深度分析**
   - 对比上周数据（环比分析）
   - 分析爆款文章特征
   - 识别内容策略优化点

3. **生成报告**
   - 创建周复盘报告：`📊数据看板/📊周复盘/Week{N}-周复盘报告.md`
   - 提出下周优化建议

### 每月数据总结

1. **月度数据汇总**
   - 月度发布统计
   - 月度增长统计
   - 月度收入统计

2. **策略调整**
   - 根据月度数据调整内容策略
   - 优化选题池优先级
   - 调整发布节奏

## 🔍 数据分析维度

### 文章维度

| 维度 | 说明 | 用途 |
|------|------|------|
| 字数 | 文章字数 | 分析最佳字数范围 |
| 代码片段数 | 代码片段数量 | 分析代码与数据关系 |
| 标签选择 | 选择的标签 | 分析标签热度 |
| 发布时间 | 发布时间点 | 分析最佳发布时间 |
| 内容类型 | 实战/原理/热点/复盘 | 分析内容类型效果 |
| 技术深度 | 深度评分 | 分析深度与数据关系 |

### 时间维度

| 维度 | 说明 | 用途 |

| 日数据 | 每日发布数据 | 追踪单日表现 |

| 月数据 | 每月汇总数据 | 识别月趋势 |
| 长期数据 | 历史数据汇总 | 识别长期趋势 |

### 竞品维度

## 🎯 数据驱动的优化

### 内容优化

1. **高赞同文章特征分析**
   - 总结高赞同文章的共同特征
   - 固化到方法论：`📚方法论/🎯爆款公式/`

2. **低表现文章问题分析**
   - 识别低表现文章的问题
   - 形成避坑指南

3. **数据反馈到选题池**
   - 根据数据调整选题优先级
   - 淘汰低价值选题

### 发布优化

1. **最佳发布时间分析**
   - 分析不同发布时间的数据表现
   - 优化发布时间策略

2. **发布频率优化**
   - 分析发布频率与数据的关系
   - 调整发布节奏

### 变现优化

1. **转化率分析**
   - 分析免费内容到付费内容的转化率
   - 优化转化路径设计

2. **客单价分析**
   - 分析不同付费产品的客单价
   - 优化定价策略
