## 🔄 发布流程

### 1. 选择文章

- 在高优先级内容发布完后，选择中优先级文章
- 选择得分较高的文章（80分以上）
- 或者根据当前热点选择合适的文章

### 2. 验证文章

- 检查文章内容是否完整
- 检查评估报告是否完整
- 检查是否有优化建议未完成

### 3. 发布文章

- 使用自动化发布脚本发布
- 或者手动发布到知乎专栏

### 4. 归档文章

- 发布后移动到 `📤已发布/` 目录
- 记录发布时间和文章 URL

### 5. 跟踪数据

- 发布后 1 天、3 天、7 天跟踪数据表现
- 记录赞同数、收藏数、评论数、关注转化
- 对比预期表现，分析差异原因

### 数据指标

| 指标 | 目标值 | 记录方式 |

| 赞同数 | 300+ | 知乎创作中心 - 数据分析 |
| 收藏数 | 100+ | 知乎创作中心 - 数据分析 |

## 💡 使用指南

### 查看待发布内容

1. 查看本目录中的 JSON 文件
2. 查看配套的评估报告（evaluation-report.json）
3. 了解文章的得分和优化建议

### 发布文章

1. 在高优先级内容发布完后，选择中优先级文章

3. 使用自动化发布脚本发布

1. 发布后 1 天、3 天、7 天跟踪数据表现
2. 记录赞同数、收藏数、评论数、关注转化
3. 对比预期表现，分析差异原因
4. 将数据记录到 `📊数据分析/` 或 `📊数据看板/` 目录
