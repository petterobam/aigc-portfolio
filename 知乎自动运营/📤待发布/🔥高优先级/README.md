## 🎯 发布流程

### 1. 选择文章

- 选择得分最高的文章（100分优先）
- 或者根据当前热点选择合适的文章

### 2. 验证文章

- 检查文章内容是否完整
- 检查评估报告是否完整
- 检查是否有优化建议未完成

### 3. 发布文章

- 使用自动化发布脚本发布
- 或者手动发布到知乎专栏

### 4. 归档文章

- 发布后移动到 `📤已发布/` 目录
- 记录发布时间和文章 URL

### 5. 跟踪数据

- 发布后 1 天、3 天、7 天跟踪数据表现
- 记录赞同数、收藏数、评论数、关注转化
- 对比预期表现，分析差异原因
