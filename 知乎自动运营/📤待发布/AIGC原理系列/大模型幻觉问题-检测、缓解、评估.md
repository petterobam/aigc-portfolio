# 大模型幻觉问题：检测、缓解、评估

## 核心结论

> 大模型的幻觉问题不是 bug，而是特征。理解它的本质，才能设计有效的检测、缓解和评估方案。幻觉的本质是模型在不确定性条件下"合理地编造"。

---

## 1. 为什么幻觉问题如此重要？

### 1.1 幻觉的定义

**幻觉（Hallucination）**：大模型生成的内容与事实不符、逻辑混乱或凭空捏造的现象。

**三大特征**：
- **事实性幻觉**：生成的事实与真实世界不符（如"奥巴马出生于 1980 年"）
- **逻辑性幻觉**：推理过程中出现逻辑矛盾（如"因为 A，所以 B，但 A 和 B 无关"）
- **一致性幻觉**：前后生成内容自相矛盾（如先说"我喜欢苹果"，后说"我讨厌苹果"）

### 1.2 为什么难以解决？

**根本原因**：
1. **训练数据的噪声**：网络数据本身就包含错误信息
2. **概率生成的本质**：模型预测下一个 token，而不是判断真伪
3. **知识截止**：模型无法获取训练后的事件
4. **过度自信**：即使不知道，模型也会"自信地编造"

**数学直觉**：
```
P(x | context) = softmax(xW_context / √d)
```
模型输出的是概率分布，而非事实判断。高概率 ≠ 事实正确。

## 2. 幻觉的根本原因

### 2.1 数据层面

**问题 1：训练数据包含噪声**

- 网络文本本身存在错误
- 训练数据未经过严格清洗
- 对立信息同时存在（模型不知道哪个是对的）

**问题 2：数据分布偏差**

- 训练数据不能覆盖真实世界的所有场景
- 少数群体的信息缺失
- 过拟合高频模式

## 3. 幻觉检测方法

### 3.1 自动检测方法

#### 方法 1：Self-Consistency（自洽性检测）

**原理**：对同一个问题生成多个答案，检查一致性

```python
import openai
from collections import Counter

def self_consistency_detection(query, n_samples=5):
    """生成多个答案，检查一致性"""

    # 生成多个答案
    answers = []
    for _ in range(n_samples):
        response = openai.ChatCompletion.create(
            model="gpt-4",
            messages=[{"role": "user", "content": query}],
            temperature=0.7  # 允许一定的随机性
        )
        answers.append(response.choices[0].message.content)

    # 统计答案分布
    answer_counts = Counter(answers)

    # 如果多数答案一致，则可信
    most_common_answer, count = answer_counts.most_common(1)[0]
    confidence = count / n_samples

    # 检测到幻觉的阈值
    if confidence < 0.6:
        return {
            "is_hallucination": True,
            "confidence": confidence,
            "most_common_answer": most_common_answer,
            "all_answers": list(answer_counts.keys()),
            "explanation": f"答案不一致，置信度 {confidence:.2%}"
        }
    else:
        return {
            "is_hallucination": False,
            "confidence": confidence,
            "answer": most_common_answer,
            "explanation": f"答案一致，置信度 {confidence:.2%}"
        }

# 示例
query = "地球的半径是多少？"
result = self_consistency_detection(query, n_samples=5)
print(result)
```

**效果分析**：
- 优点：简单易行，不需要额外模型
- 缺点：需要多次调用，成本高；对一致但错误的答案无效

#### 方法 2：Fact-Checking（事实核查）

**原理**：使用外部知识库验证生成内容

```python
import requests
import json

class FactChecker:
    def __init__(self, api_key):
        self.api_key = api_key
        self.knowledge_base_url = "https://api.wikipedia.org/api/rest_v1/"

    def check_fact(self, statement):
        """检查事实是否正确"""

        # 1. 提取关键实体
        entities = self.extract_entities(statement)
        print(f"提取的实体: {entities}")

        # 2. 对每个实体进行事实核查
        results = []
        for entity in entities:
            fact = self.verify_entity(entity)
            results.append(fact)

        # 3. 综合判断
        all_correct = all(r["is_correct"] for r in results)

        return {
            "is_hallucination": not all_correct,
            "entity_checks": results,
            "confidence": sum(r["confidence"] for r in results) / len(results)
        }

    def extract_entities(self, statement):
        """提取关键实体（简化版）"""
        # 实际应用中应使用 NER 模型
        import re
        # 简单提取人名和地名
        entities = re.findall(r'([A-Z][a-z]+(?:\s+[A-Z][a-z]+)*)', statement)
        return entities

    def verify_entity(self, entity):
        """验证实体信息（使用维基百科 API）"""
        try:
            # 搜索实体
            search_url = f"{self.knowledge_base_url}page/summary/{entity}"
            response = requests.get(search_url, timeout=5)

            if response.status_code == 200:
                data = response.json()
                return {
                    "entity": entity,
                    "is_correct": True,
                    "confidence": 0.9,
                    "source": "wikipedia",
                    "verified_info": data.get("extract", "")
                }
            else:
                return {
                    "entity": entity,
                    "is_correct": False,
                    "confidence": 0.3,
                    "source": "not_found",
                    "verified_info": "未找到相关信息"
                }
        except Exception as e:
            return {
                "entity": entity,
                "is_correct": False,
                "confidence": 0.2,
                "source": "error",
                "verified_info": str(e)
            }

# 示例
fact_checker = FactChecker(api_key="your_api_key")
statement = "奥巴马出生于 1980 年"
result = fact_checker.check_fact(statement)
print(json.dumps(result, indent=2, ensure_ascii=False))
```

**效果分析**：
- 优点：准确性高，可验证事实
- 缺点：依赖外部知识库，需要实体提取

#### 方法 3：Uncertainty Estimation（不确定性估计）

**原理**：估计模型的不确定性，高不确定性可能意味着幻觉

```python
import torch
import torch.nn.functional as F

def uncertainty_estimation(model, input_text, n_samples=10):
    """估计模型的不确定性"""

    # 生成多个样本（通过调整温度）
    logits_list = []
    for _ in range(n_samples):
        with torch.no_grad():
            outputs = model.generate(
                input_text,
                max_length=100,
                temperature=0.8 + 0.4 * (torch.rand(1).item() - 0.5),  # 随机温度
                return_dict=True,
                output_scores=True
            )
            logits_list.append(outputs.scores)

    # 计算 logits 的方差（不确定性）
    logits_tensor = torch.stack(logits_list)
    uncertainty = torch.var(logits_tensor, dim=0).mean().item()

    # 归一化不确定性
    normalized_uncertainty = min(uncertainty / 2.0, 1.0)

    return {
        "is_hallucination": normalized_uncertainty > 0.5,
        "uncertainty": normalized_uncertainty,
        "explanation": f"不确定性 {normalized_uncertainty:.2%}，可能存在幻觉"
    }

# 示例
result = uncertainty_estimation(model, "巴黎的首都是哪里？")
print(result)
```

**效果分析**：
- 优点：不需要外部资源，可实时检测
- 缺点：准确性有限，不确定性高不代表一定幻觉

#### 方法 4：Entailment Check（蕴含检查）

**原理**：检查生成内容是否与上下文或事实一致

```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification

class EntailmentChecker:
    def __init__(self):
        # 使用蕴含模型
        self.tokenizer = AutoTokenizer.from_pretrained("roberta-large-mnli")
        self.model = AutoModelForSequenceClassification.from_pretrained("roberta-large-mnli")
        self.model.eval()

    def check_entailment(self, premise, hypothesis):
        """检查 hypothesis 是否蕴含于 premise"""

        # 编码
        inputs = self.tokenizer(
            premise,
            hypothesis,
            return_tensors="pt",
            truncation=True,
            max_length=512
        )

        # 预测
        with torch.no_grad():
            outputs = self.model(**inputs)
            logits = outputs.logits
            probs = torch.softmax(logits, dim=-1)

        # 标签：0=矛盾，1=蕴含，2=中立
        label = torch.argmax(probs).item()
        confidence = probs[0][label].item()

        if label == 0:
            return {
                "is_hallucination": True,
                "relation": "矛盾",
                "confidence": confidence,
                "explanation": "生成内容与上下文矛盾"
            }
        elif label == 1:
            return {
                "is_hallucination": False,
                "relation": "蕴含",
                "confidence": confidence,
                "explanation": "生成内容与上下文一致"
            }
        else:
            return {
                "is_hallucination": None,
                "relation": "中立",
                "confidence": confidence,
                "explanation": "无法确定是否一致"
            }

# 示例
checker = EntailmentChecker()
premise = "地球是太阳系中第三颗行星"
hypothesis = "地球是太阳系中最大的行星"
result = checker.check_entailment(premise, hypothesis)
print(result)
```

**效果分析**：
- 优点：可检测逻辑矛盾
- 缺点：需要预训练的蕴含模型

    from sentence_transformers import SentenceTransformer

#### 策略 2：Constrained Decoding（约束解码）

**原理**：在解码时约束模型只能生成事实正确的内容

```python
import torch

def constrained_decoding(model, query, facts, max_length=100):
    """约束解码，只生成符合事实的内容"""

    # 1. 将事实编码为约束
    constraints = encode_facts_as_constraints(facts)

    # 2. 逐 token 生成，检查约束
    input_ids = model.tokenizer.encode(query, return_tensors="pt")

    for _ in range(max_length):
        # 生成候选 tokens
        with torch.no_grad():
            outputs = model(input_ids)
            logits = outputs.logits[:, -1, :]

        # 2.1 约束：只选择符合事实的 tokens
        constrained_logits = apply_constraints(logits, constraints)

        # 2.2 采样
        next_token = torch.multinomial(
            torch.softmax(constrained_logits, dim=-1),
            num_samples=1
        )

        # 2.3 检查是否应该停止
        if next_token == model.tokenizer.eos_token_id:
            break

        input_ids = torch.cat([input_ids, next_token], dim=-1)

    # 解码
    response = model.tokenizer.decode(input_ids[0], skip_special_tokens=True)

    return response

def encode_facts_as_constraints(facts):
    """将事实编码为约束"""

    constraints = []

    for fact in facts:
        # 例如：约束模型不能生成与事实矛盾的内容
        constraint = {
            "type": "not_contain",
            "text": fact["contradiction"]
        }
        constraints.append(constraint)

    return constraints
```

### 4.3 推理层面

#### 策略 1：Retrieval-Augmented Generation（RAG）

**原理**：从外部知识库检索相关信息，增强生成

```python
class RAGSystem:
    def __init__(self, vector_db, llm):
        self.vector_db = vector_db
        self.llm = llm

    def generate_with_rag(self, query, k=5):
        """使用 RAG 生成答案"""

        # 1. 检索相关文档
        retrieved_docs = self.vector_db.search(query, k=k)

        # 2. 构建增强的 prompt
        prompt = f"""
        基于以下文档回答问题。如果文档中没有相关信息，请明确说明。

        文档：
        {chr(10).join([doc['content'] for doc in retrieved_docs])}

        问题：{query}

        回答时请引用文档编号，格式如 [doc1], [doc2]。
        如果无法从文档中找到答案，请说明"文档中未提及相关信息"。
        """

        # 3. 生成答案
        response = self.llm.generate(prompt)

        return {
            "answer": response,
            "retrieved_docs": retrieved_docs,
            "has_source": len(retrieved_docs) > 0
        }

# 示例

#### 策略 2：Chain-of-Verification（验证链）

def chain_of_verification(model, query):
    """验证链：生成 → 验证 → 修正"""

    # 步骤 1：生成初始答案
    initial_answer = model.generate(query)

    # 步骤 2：分解为多个可验证的断言
    assertions = decompose_into_assertions(initial_answer)

    # 步骤 3：验证每个断言
    verified_assertions = []
    for assertion in assertions:
        verification = verify_assertion(assertion)
        verified_assertions.append(verification)

    # 步骤 4：修正答案
    corrected_answer = correct_answer(
        initial_answer,
        verified_assertions
    )

    return {
        "initial_answer": initial_answer,
        "assertions": assertions,
        "verified_assertions": verified_assertions,
        "corrected_answer": corrected_answer
    }

def verify_assertion(assertion):
    """验证断言"""

    # 方法 1：使用事实核查
    fact_check_result = fact_checker.check_fact(assertion)

    # 方法 2：使用蕴含检查
    if "context" in assertion:
        entailment_result = entailment_checker.check_entailment(
            assertion["context"],
            assertion["statement"]
        )
    else:
        entailment_result = None

    # 综合判断
    is_correct = (
        fact_check_result.get("is_hallucination") == False or
        (entailment_result and entailment_result.get("is_hallucination") == False)
    )

    return {
        "assertion": assertion,
        "is_correct": is_correct,
        "fact_check": fact_check_result,
        "entailment": entailment_result
    }
```

class HumanInTheLoopSystem:

        self.auto_detector = auto_detector
        self.human_reviewer = human_reviewer

        answer = self.model.generate(query)

        # 2. 自动检测
        detection_result = self.auto_detector.detect(answer)

## 5. 幻觉评估方法

### 5.1 自动化指标

#### 指标 1：FactScore（事实准确率）

**原理**：将答案分解为原子事实，逐个验证

```python
def fact_score(answer, knowledge_base):
    """计算 FactScore"""

    # 1. 分解为原子事实
    atomic_facts = decompose_into_atomic_facts(answer)

    # 2. 验证每个事实
    correct_facts = 0
    for fact in atomic_facts:
        if verify_fact(fact, knowledge_base):
            correct_facts += 1

    # 3. 计算分数
    score = correct_facts / len(atomic_facts) if atomic_facts else 0

    return {
        "fact_score": score,
        "total_facts": len(atomic_facts),
        "correct_facts": correct_facts,
        "incorrect_facts": len(atomic_facts) - correct_facts
    }

def decompose_into_atomic_facts(answer):
    """分解为原子事实"""
    # 简化版：按句号分割
    sentences = answer.split("。")
    atomic_facts = [s.strip() + "。" for s in sentences if s.strip()]
    return atomic_facts
```

#### 指标 2：Entailment Score（蕴含分数）

**原理**：检查生成内容是否与上下文一致

```python
def entailment_score(generation, reference):
    """计算蕴含分数"""

    # 1. 使用蕴含模型检查
    result = entailment_checker.check_entailment(reference, generation)

    # 2. 映射为分数
    if result["relation"] == "蕴含":
        score = 1.0
    elif result["relation"] == "矛盾":
        score = 0.0
    else:
        score = 0.5

    return {
        "entailment_score": score,
        "relation": result["relation"],
        "confidence": result["confidence"]
    }
```

#### 指标 3：Hallucination Rate（幻觉率）

**原理**：统计幻觉的数量

```python
def hallucination_rate(generation):
    """计算幻觉率"""

    # 1. 使用多种检测方法
    detection_results = [
        self_consistency_detection(generation),
        fact_checker.check_fact(generation),
        uncertainty_estimation(model, generation)
    ]

    # 2. 综合判断
    hallucination_votes = sum(
        1 for r in detection_results
        if r.get("is_hallucination")
    )

    # 3. 计算幻觉率
    rate = hallucination_votes / len(detection_results)

    return {
        "hallucination_rate": rate,
        "detection_results": detection_results
    }
```

- 多个文档可能矛盾
- 员工提问可能超出知识库范围

import torch
from sentence_transformers import SentenceTransformer
import requests

class EnterpriseRAGSystem:
    def __init__(self, vector_db_path, llm_model):
        # 1. 向量数据库
        self.vector_db = VectorDB(vector_db_path)

        # 2. 嵌入模型
        self.embedder = SentenceTransformer('all-MiniLM-L6-v2')

        # 3. 大语言模型
        self.llm = llm_model

        # 4. 幻觉检测器
        self.hallucination_detector = HallucinationDetector()

    def query(self, question, k=5, verify=True):
        """查询企业知识库"""

        # 步骤 1：检索相关文档
        relevant_docs = self.vector_db.search(question, k=k)

        if not relevant_docs:
            return {
                "answer": "抱歉，知识库中没有找到相关信息。",
                "has_source": False,
                "is_hallucinated": False
            }

        # 步骤 2：生成答案
        answer = self._generate_answer(question, relevant_docs)

        # 步骤 3：检测幻觉
        if verify:
            detection_result = self.hallucination_detector.detect(
                answer,
                context=question,
                sources=[doc["content"] for doc in relevant_docs]
            )

            # 步骤 4：如果检测到幻觉，重新生成
            if detection_result["is_hallucination"]:
                corrected_answer = self._correct_hallucination(
                    question,
                    answer,
                    relevant_docs,
                    detection_result
                )
                return {
                    "answer": corrected_answer,
                    "original_answer": answer,
                    "has_source": True,
                    "is_hallucinated": True,
                    "correction": detection_result
                }

        return {
            "answer": answer,
            "has_source": True,
            "is_hallucinated": False,
            "sources": relevant_docs
        }

    def _generate_answer(self, question, docs):
        """生成答案"""

        # 构建增强的 prompt
        context = "\n\n".join([
            f"文档{i+1}：{doc['content']}"
            for i, doc in enumerate(docs)
        ])

        prompt = f"""
        你是一个企业知识库问答助手。请基于以下文档回答问题。

        {context}

        问题：{question}

        回答要求：
        1. 只使用文档中的信息，不要编造
        2. 如果文档中没有答案，明确说明
        3. 回答时引用文档编号，如 [文档1]
        4. 回答简洁准确

        回答：
        """

        answer = self.llm.generate(prompt)
        return answer

    def _correct_hallucination(self, question, answer, docs, detection_result):

        # 方法 1：使用验证链

        # 方法 2：约束解码

            facts=[doc["content"] for doc in docs]

        # 方法 3：自我反思
        reflected_answer = self_reflection(self.llm, question)

        # 选择最佳答案
        answers = [verified_answer, constrained_answer, reflected_answer]
        best_answer = self._select_best_answer(answers, docs)

        return best_answer

    def _select_best_answer(self, answers, docs):

        scores = []
        for answer in answers:
            # 检测幻觉
            detection = self.hallucination_detector.detect(
                answer,
                sources=[doc["content"] for doc in docs]
            )

            # 计算分数
            if detection["is_hallucination"]:
                score = 0
            else:
                score = detection["confidence"]

            scores.append(score)

        # 选择分数最高的答案
        best_index = scores.index(max(scores))
        return answers[best_index]

class HallucinationDetector:
    """幻觉检测器"""

    def __init__(self):
        self.embedder = SentenceTransformer('all-MiniLM-L6-v2')

    def detect(self, answer, context=None, sources=None):
        """检测幻觉"""

        detection_results = []

        # 方法 1：自洽性检测
        consistency_result = self._self_consistency_check(answer)
        detection_results.append(consistency_result)

        # 方法 2：来源一致性检测
        if sources:
            source_result = self._source_consistency_check(answer, sources)
            detection_results.append(source_result)

        # 方法 3：蕴含检查
        if context:
            entailment_result = self._entailment_check(answer, context)
            detection_results.append(entailment_result)

        # 综合判断
        hallucination_votes = sum(
            1 for r in detection_results
            if r["is_hallucination"]
        )
        is_hallucinated = hallucination_votes > len(detection_results) / 2

        return {
            "is_hallucination": is_hallucinated,
            "confidence": sum(r["confidence"] for r in detection_results) / len(detection_results),
            "detection_results": detection_results
        }

    def _self_consistency_check(self, answer, n_samples=3):
        """自洽性检测"""
        # 简化版：检查答案中的关键信息是否一致
        # 实际应用中应生成多个答案并比较

        # 检查是否有矛盾陈述
        contradictions = self._find_contradictions(answer)

        return {
            "method": "self_consistency",
            "is_hallucinated": len(contradictions) > 0,
            "confidence": 1.0 if len(contradictions) == 0 else 0.3,
            "details": contradictions
        }

    def _source_consistency_check(self, answer, sources):
        """来源一致性检测"""

        # 1. 将答案和来源编码为向量
        answer_embedding = self.embedder.encode([answer])
        source_embeddings = self.embedder.encode([s["content"] for s in sources])

        # 2. 计算相似度
        from sklearn.metrics.pairwise import cosine_similarity
        similarities = cosine_similarity(answer_embedding, source_embeddings)[0]

        # 3. 判断
        max_similarity = max(similarities)

        return {
            "method": "source_consistency",
            "is_hallucinated": max_similarity < 0.6,
            "confidence": max_similarity,
            "similarities": similarities.tolist()
        }

    def _entailment_check(self, answer, context):
        """蕴含检查"""

        # 使用蕴含模型
        entailment_checker = EntailmentChecker()
        result = entailment_checker.check_entailment(context, answer)

        return {
            "method": "entailment",
            "is_hallucinated": result.get("is_hallucination", False),
            "confidence": result.get("confidence", 0.5),
            "details": result
        }

    def _find_contradictions(self, answer):
        """查找矛盾陈述"""

        contradictions = []

        # 简化版：查找"但"、"然而"等转折词
        import re
        sentences = re.split(r'[。！？]', answer)

        for i in range(len(sentences) - 1):
            if "但" in sentences[i+1] or "然而" in sentences[i+1]:
                contradictions.append({
                    "sentence1": sentences[i],
                    "sentence2": sentences[i+1]
                })

        return contradictions
