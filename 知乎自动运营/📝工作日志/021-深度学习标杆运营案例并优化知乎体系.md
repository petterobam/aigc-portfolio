#### 1.2 核心运营方法论

**有效内容公式**:
```
有效内容 = 实用价值（收藏）+ 情绪价值（点赞）+ 互动设计（评论）
```

**养成系内容公式**:
```
真实经历 + 数据复盘 + 情绪起伏 + 成长轨迹
```

**内容类型分布**:
- 养成系教程（40%）: 真实踩坑+解决方案
- 趋势妆容（30%）: 帕斯蒂尔风、韩系、巴恩风
- 实用干货（20%）: 空瓶记、产品测评
- 数据复盘（10%）: 周复盘、月复盘、策略调整

#### 1.3 数据驱动的运营策略

**数据追踪机制**:
- **发布后1h**: 记录初始数据
- **发布后24h**: 记录首日数据
- **每日检查**: 更新数据追踪看板
- **每周复盘**: 撰写周复盘报告
- **每月分析**: 深度数据分析

**关键指标**:
| 指标 | 公式 | 健康值 |
|-----|------|-------|
| 点赞率 | 点赞/浏览×100% | 3%+ |
| 收藏率 | 收藏/浏览×100% | 2%+ |
| 评论率 | 评论/浏览×100% | 1%+ |
| 互动率 | (点赞+收藏+评论)/浏览×100% | 5%+ |

**CES评分优化**:
```
CES = 点赞(1分) + 收藏(1分) + 评论(4分) + 关注(8分) + 分享(4分)
```

**黄金2小时策略**: 发布后前2小时是关键期
- 第一时间转发到私域群聊
- 主动回复每一条评论
- 在评论区补充干货内容

```

### 4.2 合并内容
**规则**:
1. 合并后的内容保留主要内容的编号
2. 被合并的内容记录在对照表中

**示例**:
```
合并: 第5篇 + 第6篇 → 第5篇
记录: 第6篇内容已合并到第5篇
```

### 4.3 拆分内容
**规则**:
1. 拆分后的内容重新编号
2. 原内容保留原编号

**示例**:
```
拆分: 第7篇 → 第7篇 + 第8篇
记录: 第8篇从第7篇拆分
