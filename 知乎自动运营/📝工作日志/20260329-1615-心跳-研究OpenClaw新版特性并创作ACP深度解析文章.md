## ⚡ 执行步骤

### 1. 研究 OpenClaw 新版特性

**发现的核心特性**：

1. **ACP (Agent Control Protocol)** - 新的 Agent 控制协议
   - 标准化 Agent 通信接口
   - 支持 WebSocket 和 Stdio 两种传输方式
   - 支持会话管理、工具调用、消息路由

2. **agents 命令** - 管理独立 agents（工作区 + 认证 + 路由）
   - `openclaw agents add` - 添加新 Agent
   - `openclaw agents list` - 列出所有 Agent
   - `openclaw agents delete` - 删除 Agent
   - `openclaw agents bind` - 添加路由绑定

3. **models 命令** - 模型发现、扫描和配置
   - `openclaw models list` - 列出模型
   - `openclaw models set` - 设置默认模型
   - `openclaw models scan` - 扫描 OpenRouter 免费模型

4. **acp client** - 交互式 ACP 客户端
   - `openclaw acp client` - 启动交互式客户端
   - 支持本地和远程 ACP 服务器

5. **openclaw agent** - 直接通过 Gateway 运行 agent
   - `openclaw agent --message "xxx"` - 运行单次对话
   - `openclaw agent --session session:key` - 指定会话

**版本信息**：
- 当前版本：OpenClaw 2026.3.24 (cff6dc9)
- 可用更新：npm 2026.3.28
