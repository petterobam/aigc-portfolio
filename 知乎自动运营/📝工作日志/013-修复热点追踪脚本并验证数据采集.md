**问题**: 使用了错误的选择器 `.list-identifier`

**修复方案**:
```javascript
// 修改前
const paperElements = document.querySelectorAll('.list-identifier');

// 修改后
const paperElements = document.querySelectorAll('dd');

// 查找对应的 dt 元素获取链接
const dtElement = element.previousElementSibling;
let link = '';
if (dtElement) {
  const linkEl = dtElement.querySelector('a[href*="abs"]');
  if (linkEl) {
    link = linkEl.href;
  }
}
```
