## 🎯 关键发现

### 1. 平台定位清晰
- **字节跳动旗下**: 免费阅读 + 广告变现
- **下沉市场**: 70%+用户来自三四线城市及乡镇
- **用户画像**: 18-45岁,女性60%,碎片化阅读,偏好快节奏爽点密集的内容

### 2. 核心机制明确
- **推荐算法**: 完读率40%(最重要) + 留存率30% + 互动率20% + 点击率10%
- **流量池**: 测试100-500人 → 正式1,000-10,000/天 → 热门10,000-100,000/天 → 榜单100,000+/天
- **变现机制**: 广告分成0.01-0.05元/次阅读,签约分成70%-80%,保底1,000-8,000元/月
- **签约标准**: 5篇作品 + 2篇完读率>30% + 500粉丝

### 3. 最佳参数明确
- **字数范围**: 15,000-25,000字(短篇最佳)
- **章节长度**: 2,000-2,500字/章
- **标题长度**: ≤25字(点击率最优)
- **简介长度**: 100-150字(读者有耐心看完)
- **标签数量**: 最多3个
- **封面尺寸**: 800×1200像素(竖屏)

### 4. 关键指标明确
- **完读率**: ≥40%(正式推荐池),≥60%(热门推荐),≥80%(榜单推荐)
- **第1章留存率**: ≥70%
- **互动率**: ≥5%
- **点击率**: ≥10%
