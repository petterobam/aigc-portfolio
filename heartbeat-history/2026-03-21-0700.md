## 🔧 技术细节

### Python 脚本：convert_md_to_text.py

**功能**：将 Markdown 文件转换为纯文本，去除 Markdown 格式标签

**去除的格式**：
- 标题标记（# 标题）
- 粗体标记（**粗体**）
- 斜体标记（*斜体*）
- 代码标记（`代码`）
- 代码块标记（```代码```）
- 链接标记（[链接](URL)）
- 图片标记（![alt](URL)）
- 分割线标记（---、***）
- 引用标记（> 引用）
- 列表标记（- 列表、* 列表、1. 列表）

**去除多余的空行**：保留一个空行作为段落分隔
