## 经验总结

### 成功经验
1. **先分析，后开发**：通过正则表达式分析 HTML 文件，快速找到正确的选择器
2. **快速迭代**：从 V1 → V2 → V3，每个版本都解决了特定问题

4. **文档沉淀**：及时更新 MEMORY.md、task-list.md、heartbeat-history

### 遇到的问题
1. **选择器错误**：原始脚本选择器不匹配，导致抓取失败
2. **API 调用问题**：`page.evaluate is not a function` - 使用 `browser.pages()[0]` 修复
3. **浏览器锁定**：`ProcessSingleton` 错误 - 删除 `SingletonLock` 文件修复

### 解决方案
1. **结构分析工具**：创建 `analyze-story-list-structure.js`，用正则表达式分析 HTML
2. **API 调用修复**：使用 `browser.pages()[0]` 获取页面对象
3. **锁定文件清理**：删除 `data/chrome-user-data/SingletonLock` 等文件
