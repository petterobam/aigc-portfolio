## 📝 读书笔记规范

### 笔记模板
每篇读书笔记应包含：
- **经典名称**与具体章节/段落
- **原文摘录**（重要段落）
- **语境说明**（成文背景、在整部经典中的位置）
- **概念解析**（关键术语的含义）
- **个人理解**与感悟
- **疑问记录**（待深入研究的问题）
- **跨宗教联想**（是否让你想到其他宗教的类似表述）

### 命名规范
- 格式：`[经典名]-[章节]-[笔记类型].md`
- 示例：`心经-全文-深度读书笔记.md`
- 示例：`第一章-道可道非常道.md`
- 综合类：`般若经系思想脉络综述.md`
- 注意：宗教前缀不再需要（已通过目录层级区分）

### 目录扩展规范
- 新增宗教传统：在根目录创建 `[emoji][宗教名]/` 目录 + README.md
- 新增经典/流派：在宗教目录下创建子目录 + README.md
- 每个目录必须包含 README.md 作为导航索引
