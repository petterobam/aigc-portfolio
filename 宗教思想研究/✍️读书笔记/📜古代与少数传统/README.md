## 🔗 研究价值

这些传统虽在当代信众规模上不及主流宗教，但对世界宗教史具有深远影响：
- **诺斯替主义**：早期基督教最重要的“异端”对手，深刻影响了正统教义的界定
- **琐罗亚斯德教**：善恶二元论、末日审判等概念对犹太教-基督教-伊斯兰教传统的源头性影响
- **神道教**：理解日本文明的精神根基，宗教与政治关系的典型案例
