## 一、《阿维斯陀》整体结构

现存《阿维斯陀》仅为原典的约四分之一。据9世纪帕拉维文献《丹卡尔特》（Dēnkart）记载，完整版包含21卷（nask），萨珊王朝时期完成编纂，亚历山大征服波斯时大量原稿被毁。现存文本是从残篇中重新汇编的。

### 五大组成部分

| 分卷 | 名称 | 性质 | 核心内容 |
|------|------|------|---------|
| **Yasna** | 「祭祀」 | 礼仪文本（72章） | 包含《伽陀》（第28-34、43-51、53章），是祭祀仪式中祭司诵念的核心文本。涉及豪麻（haoma）献祭的准备与执行 |
| **Visperad** | 「全体裁判者」 | 补充礼仪 | 对Yasna的扩充，增加对各存在类别之"主"（ratus）的 invocation 和供奉，用于重大节日 |
| **Vendidad** | 「反迪弗之法」（Vidēvdāt） | 律法典籍（22章） | 2章序言（讲述律法如何授予人类）+ 18章规则。涵盖洁净律法、创世叙事、原始人类伊玛（Yima）的故事。是琐罗亚斯德教民事与仪式律法的主要来源 |
| **Yashts** | 「赞歌」（21篇） | 神话赞美诗 | 分致21位神圣存在（yazatas）：密特拉（Mithra）、阿娜希塔（Anahita）、韦雷斯拉格纳（Verethraghna）等。富含伊朗古代神话材料 |
| **Khūrda Avesta** | 「小阿维斯陀」 | 日常祈祷集 | 日常祷告、特定场合的短篇文本。包括《阿舍姆·沃胡》（Ashem Vohu）、《耶恩格·哈塔姆·迈蒂》（Yenghē Hātām）等经典短祷 |

### 文本年代层次

《阿维斯陀》的语言本身揭示了其年代分层：

- **最古老层**：《伽陀》（伽陀阿维斯陀语，约公元前1500-1200年）——先知本人的语言
- **次古老层**：部分Yashts、Yasna的非伽陀部分（后期阿维斯陀语，约公元前1000-600年）
- **较晚层**：Vendidad、Visperad、Khūrda Avesta（后期阿维斯陀语，但成文整理可能晚至萨珊时期）

这种分层类似于《圣经》中摩西五经的JEDP底本学说——一个宗教传统的经典往往是多层累积的产物。

## 二、Yasna（祭祀）——仪式与神学的交汇

### 2.1 结构与功能

Yasna（意为"祭祀""崇拜"）是琐罗亚斯德教最核心的礼仪文本，共72章。它不是供阅读的经卷，而是**仪式中活的声音**——祭司在Yasna仪式中逐章诵念，同时执行相应的祭仪动作。

Yasna仪式的核心是**豪麻献祭**（haoma ceremony）：将豪麻植物（一种可能有致幻特性的植物，学界长期争议其物种归属）的茎研磨、挤压，获得汁液，混合牛奶和其他成分，供奉给阿胡拉·马兹达。

### 2.2 《伽陀》在Yasna中的嵌入

《伽陀》17首颂歌嵌入在Yasna的第28-34章（小伽陀）、43-51章（大伽陀）和第53章中。这意味着：**先知本人的诗歌被包裹在一层更晚的礼仪框架中**。

这种结构极具宗教意义：
- 先知的个人启示（伽陀）被共同体**仪式化**——从个人体验转化为集体实践
- 伽陀的诵念不是独立的学术活动，而是嵌在祭祀行动中——**言说即行动**
- 这与佛教的诵经传统形成有趣的对照：佛经中也常将核心教义包裹在叙事和仪轨框架中

### 2.3 非伽陀部分的Yasna

Yasna的非伽陀部分（第1-27章、35-42章、52章、54-72章）包含：
- 对各种神圣存在的invocation
- 豪麻献祭的具体步骤
- 对阿胡拉·马兹达和七大圣灵（Amesha Spentas）的赞美
- 标准化的仪式用语

**思考**：Yasna的非伽陀部分代表了一种"次生神学"——后人在先知启示的基础上，发展出系统的仪式体系。这几乎是所有宗教的共同模式：创始人的个人体验 → 共同体的制度化实践。佛陀的觉悟变成寺院的早晚课，耶稣的比喻变成弥撒，琐罗亚斯德的颂歌变成Yasna仪式。

## 三、Vendidad（文迪达德）——洁净律法与宇宙叙事

### 3.1 "反迪弗之法"

Vendidad（Vidēvdāt）意为"反迪弗（daevas）之法"——"迪弗"是琐罗亚斯德教中的恶灵，对应印度教中的"提婆"（devas，天神）。这里有一个极为重要的印欧宗教分裂证据：

- **印度传统**：devas（天神）vs asuras（阿修罗，恶魔）
- **伊朗传统**：daevas（恶魔）vs ahuras（阿胡拉，善神）

同一印欧宗教遗产在两支后裔中发生了**善恶倒转**。这暗示着在印度-伊朗共同体分裂之前，这两个群体可能已经存在宗教分歧。

### 3.2 核心内容

**创世叙事**（Vendidad第1-2章）：
- 阿胡拉·马兹达创造了十六个完美的国度（airyanem vaejah为首），安格拉·曼纽逐一加以破坏
- 这是一种"对抗性创世"叙事——善的创造总是被恶所反创，但善的创造先于恶的破坏

**伊玛（Yima）的故事**：
- 伊玛是神话中的原始王者（对应印度教的阎摩Yama）
- 他最初被赋予"牧者"之职，管理世间万物
- 因骄傲而犯罪（声称自己是创造者），被阿胡拉·马兹达剥夺王权
- 阿胡拉·马兹达命他建造一个"瓦尔"（var，地下避难所），以躲避即将到来的大洪水/严冬
- 这个故事与《圣经》的诺亚方舟有明显的结构相似性——一个被选中的领袖建造避难所以保存物种

**洁净律法**（第3-19章）：
- 大量关于身体洁净、死尸处理、疾病隔离、月经禁忌的详细规定
- 这些律法的基础是**物质世界的洁净等同于精神世界的善**——物质与精神不是二元对立，而是同一战场
- 死尸被视为最严重的污染源（nasu），因为死亡是安格拉·曼纽的最大"作品"
- 由此发展出独特的"天葬"（dokhmeshini）习俗——将死者置于"寂静之塔"（dakhma）供秃鹫啄食，避免尸体污染大地

### 3.3 洁净律法的深层神学

Vendidad的洁净律法不是单纯的卫生法规。其深层逻辑是：

> **善 = 整洁、有序、光明、生命**
> **恶 = 污秽、混乱、黑暗、死亡**

宇宙是一个善恶交战的战场，**日常生活的每一个行为都参与这场战争**。洗手、处理死尸、隔离病人——这些看似琐碎的行为，实际上是宇宙战争的微观参与。

这种"将日常生活宇宙化"的思维模式，与犹太教的洁净律法（kashrut）高度相似。两者都将日常伦理提升到宇宙战争的层面——这不太可能是巧合。

## 四、Yashts（耶什特）——神话宝库与神殿

### 4.1 21篇赞歌

Yashts是《阿维斯陀》中最富神话色彩的文本，21篇赞歌分别献给：

| 神圣存在 | 职能 | 对应印度神 |
|----------|------|-----------|
| **密特拉（Mithra）** | 光之神、誓言之守护者、正义之保护者 | 密多罗（Mitra） |
| **阿娜希塔（Anahita）** | 水之女神、丰饶之母 | 萨拉斯瓦蒂（Sarasvatī） |
| **韦雷斯拉格纳（Verethraghna）** | 胜利之神 | 因陀罗（Indra）功能的替代 |
| **蒂什特里亚（Tishtrya）** | 星辰之神（天狼星） | — |
| **德鲁瓦斯帕（Drvaspa）** | 牲畜守护神 | — |
| **拉姆（Ram）** | 安宁之神 | — |
| ……等21位 | | |

### 4.2 密特拉 Yasht（Yasht 10）——最具影响力的文本

密特拉Yasht描绘了一位：
- **全观之神**：无所不见的天界光明之神
- **誓言之守护者**：监视一切契约的执行，伪誓者必受惩罚
- **正义之保护者**：在今生和来世都护佑正直之人
- **审判者**：与拉什努（Rashnu）和斯劳沙（Sraosha）一同审判死者灵魂

**密特拉的传播史是世界宗教史上的奇迹之一**：
1. 琐罗亚斯德教中的密特拉 →
2. 罗马帝国时期的密特拉教（Mithraism，军人宗教） →
3. 对早期基督教的影响（12月25日本是密特拉的生日——"不可征服的太阳"之节）

### 4.3 前琐罗亚斯德宗教的回声

Yashts的重要性在于：**它们保存了前琐罗亚斯德时期伊朗宗教的大量材料**。先知琐罗亚斯德试图进行宗教改革——将纷繁的多神教统合为以阿胡拉·马兹达为核心的一神（或有限多神）体系。但Yashts表明，古老的众神从未被完全消除，而是被"收编"为阿胡拉·马兹达麾下的"亚扎塔"（yazatas，值得崇拜者）。

这与佛教在印度的发展模式极为相似：佛陀并未否定印度教众神，而是将他们"降级"为护法神。宗教改革往往是**收编而非消灭**。

## 五、Khūrda Avesta（小阿维斯陀）——日常祈祷

### 5.1 核心短祷

**阿舍姆·沃胡（Ashem Vohu）**——琐罗亚斯德教最短也最核心的祈祷：

> *ashem vohu vahishtem astī*
> *īti astī vahishtem ashem*
> *ashāi vahishtāi ashem*
>
> （真理/善念是最好的，
> 它是可欲的，是至上的，
> 真理属于最善的真理）

这四行短祷几乎出现在每一个琐罗亚斯德教仪式中，地位类似于佛教的"南无阿弥陀佛"或基督教的"主祷文"。

**耶恩格·哈塔姆·迈蒂（Yenghē Hātām）**——包容性祈祷：

> 我们赞美所有那些，——无论男女——
> 因他们的善思、善言、善行
> 而使我们离真理更近的人。

### 5.2 日常祈祷的神学意义

Khūrda Avesta揭示了一个重要维度：琐罗亚斯德教不是仅属于祭司的宗教。普通信徒通过日常祈祷——**在每天的五次祈祷时间（gah）中诵念短祷**——参与到宇宙善恶之战中。

"善思、善言、善行"（Humata, Hukhta, Hvarshta）这一伦理三原则，通过日常祈祷被内化为每个信徒的生活方式。这与伊斯兰教每日五次礼拜（salat）的结构高度相似。

## 六、帕拉维文献——阿维斯陀的诠释传统

《阿维斯陀》本身（用阿维斯陀语写成）在萨珊时期被翻译为帕拉维语（中古波斯语），并加上了大量注释。这些帕拉维文献本身也成为重要的神学来源：

| 文献 | 性质 | 核心价值 |
|------|------|---------|
| **Bundahishn**（原始创造） | 宇宙论 | 最完整的琐罗亚斯德教创世叙事，详述善恶二神的对抗 |
| **Dēnkart**（宗教之行为） | 百科全书 | 9世纪编纂，总结了21卷完整《阿维斯陀》的内容 |
| **Mēnōk-i Khrat**（智慧之灵） | 理性神学 | 基于理性的教义摘要 |
| **Artāy Virāf之书** | 灵界游记 | 维拉夫的灵魂游历天堂与地狱的叙事——但丁《神曲》的远祖 |
| **Shkand-Gumānīk Vichār**（疑惑的最终消散） | 护教论 | 针对摩尼教、基督教、犹太教、伊斯兰教的辩驳 |

**Artāy Virāf之书**特别值得关注：一位祭司的灵魂离开肉体，由神圣向导带领穿越天堂、炼狱和地狱，目睹善恶各有其报。这个叙事结构——灵魂出窍、天使引路、三界游历、善恶审判——在但丁的《神曲》（约1314年）中得到登峰造极的表达。但丁是否直接接触过波斯材料尚有争议，但结构上的相似性令人深思。

## 七、文本演变的神学张力——一个核心发现

研读《阿维斯陀》各分卷，我发现了一个贯穿始终的神学张力：

### 先知的愿景 vs 共同体的实践

| 维度 | 伽陀（先知本人） | 后期阿维斯陀 |
|------|-----------------|-------------|
| **神论** | 阿胡拉·马兹达为最高神，其他神或被忽视或被收编为"实体" | 众神回归，形成以阿胡拉·马兹达为首的多神体系 |
| **善恶关系** | 两种精神（Spenta Mainyu vs Angra Mainyu）都是某种更高存在的"子" | 善恶二神几乎成为对等的两个最高原则——趋向绝对二元论 |
| **伦理** | 个人内在的道德选择是核心 | 外在的洁净律法、仪式规定变得同等重要 |
| **语言** | 诗性的、多义的、充满个人情感的 | 规范化的、仪式化的、教条化的 |

这种张力几乎是**宗教史的普遍模式**：

- 佛陀的"体验缘起" → 部派佛教的阿毗达磨论辩
- 耶稣的"上帝之国就在你们中间" → 基督教会的教义信条
- 穆罕默德的"顺从真主" → 伊斯兰法学的沙里亚体系
- 老子的"道可道非常道" → 道教的科仪法术

**先知带来的是活的体验，共同体需要的是稳定的制度**。这两者之间的张力，构成了宗教传统内在的创造性动力，也是其最大的危险所在。

### 祖尔万主义（Zurvanism）——被定为异端的解决方案

当善（奥尔马兹达/阿胡拉·马兹达）和恶（阿赫里曼/安格拉·曼纽）成为对等的最高原则时，一个不可避免的问题出现了：**谁创造了他们？**

祖尔万主义的回答是：**时间（Zurvān）** 是善恶之父。这是一个试图超越二元论的方案——在善恶之上设定一个更高的本原。但这个方案被正统琐罗亚斯德教判定为异端。

祖尔万主义与以下传统有惊人的结构相似性：
- 诺斯替主义的"普累若玛"——在善恶造物主之上有一个更高的神性领域
- 卡巴拉的"无限"（Ein Sof）——超越善恶对立的终极神圣
- 道教的"道"——在阴阳之上，"道生一，一生二"

**思考**：几乎所有深刻触及善恶二元问题的宗教传统，都产生了某种"超越二元"的思辨。这似乎是人类理性面对"恶的起源"问题时的一种自然运动——如果善是最高的，恶从何而来？如果善恶同源，那个"源"是什么？

## 八、跨宗教比较思考

### 8.1 《阿维斯陀》与《吠陀》——印欧双胞胎

《阿维斯陀》和印度《吠陀》是印欧宗教遗产的两个分支，共享大量词汇和概念：

| 阿维斯陀概念 | 吠陀对应 | 共同含义 |
|-------------|---------|---------|
| Ahura（阿胡拉） | Asura（阿修罗） | 神圣存在/主宰 |
| Daeva（迪弗） | Deva（提婆） | 神圣存在 |
| Asha（阿夏） | Rita（梨陀） | 宇宙秩序/真理 |
| Yama/Yima（伊玛） | Yama（阎摩） | 死者之王/原始王者 |
| Mithra（密特拉） | Mitra（密多罗） | 契约/友谊之神 |
| Haoma（豪麻） | Soma（苏摩） | 祭祀用神圣植物 |

但善恶完全倒转：伊朗的asura（ahura）是善神，印度的是恶魔；伊朗的deva（daeva）是恶魔，印度的是天神。这暗示着**印度-伊朗分裂不仅是政治性的，更是宗教性的**——可能是一场关于祭祀实践和神学优先性的根本分歧。

### 8.2 洁净律法的跨宗教谱系

琐罗亚斯德教的洁净律法 → 犹太教的洁净律法（kashrut）→ 伊斯兰教的清真律法（halal）

这个传播链条的时间顺序和地理接触（犹太人在巴比伦流亡期间接触波斯文明）使得影响关系几乎是确定的。琐罗亚斯德教是这一"将日常饮食与生活细节宇宙化"的宗教实践的源头。

### 8.3 灵魂审判的跨宗教传播

琐罗亚斯德教的灵魂审判结构：
1. 死后灵魂走向**钦瓦特桥**（Chinvat Bridge）
2. 如果生平善多于恶，桥变宽阔，灵魂走向"善思之屋"（天堂）
3. 如果恶多于善，桥变如刀刃般锋利，灵魂坠入"谎言之屋"（地狱）
4. 最终末日（Frashokereti），善完全战胜恶，一切被更新

→ 犹太教的死后审判（但以理书12:2）
→ 基督教的最后审判（马太福音25章）
→ 伊斯兰教的天桥（Sirat）审判

**钦瓦特桥 → 伊斯兰教的绥拉特桥（As-Sirāt）**：结构几乎完全一致——善人顺利通过，恶人坠入火狱。

## 九、个人研究反思

### 9.1 文本完整性的问题

《阿维斯陀》的残缺状态（仅存约四分之一）给研究带来巨大困难。我们通过《丹卡尔特》的摘要知道原典曾有21卷，但大部分内容已永久丧失。这意味着我们对琐罗亚斯德教的理解**可能严重偏向幸存文献所强调的方面**。

类比：如果基督教《新约》仅存《启示录》和部分保罗书信，我们对基督教的理解会多么不同。

### 9.2 从先知到祭司阶层的转化

研读这些文本让我更深刻地理解了一个问题：**宗教传统的真正塑造者往往不是先知本人，而是其后继者**。

琐罗亚斯德的《伽陀》充满个人体验的直接性和紧迫感——一个人类面对终极实在时的敬畏与困惑。但《阿维斯陀》的其他部分——Yashts的标准化赞美、Vendidad的繁琐律法、Khūrda Avesta的日常祈祷——反映的是一个**已经制度化的祭司阶层**在维护和扩展其宗教体系。

这不是贬低后期文本的价值。制度化的宗教提供了先知体验所无法提供的东西：**连续性、可传授性、社会凝聚力**。但代价是——活的火焰往往变成了被精心维护的火坛。火仍在燃烧，但再也不是那场最初的大火了。

### 9.3 为什么琐罗亚斯德教如此重要

这次研读让我更加确信：琐罗亚斯德教对理解世界宗教格局的**关键性**远超其现存信徒人数（全球约12-15万人，主要在印度帕西人社区和伊朗）所暗示的。

它是连接：
- **印欧古老遗产**（与吠陀共享的底层）与
- **亚伯拉罕诸教**（犹太教、基督教、伊斯兰教深受其影响）

的**关键桥梁**。不理解琐罗亚斯德教，就无法理解天使、魔鬼、末世审判、天堂地狱、身体复活这些概念如何从伊朗高原传播到地中海世界，最终塑造了整个西方宗教传统。
