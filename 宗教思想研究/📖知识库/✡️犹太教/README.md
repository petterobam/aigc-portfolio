# ✡️ 犹太教

> **定位**：系统研读犹太教经典文献，梳理从圣经时代到现代犹太思想的发展脉络，理解犹太教作为亚伯拉罕诸宗教之源的核心思想与独特贡献。

---

## 📌 学科概览

犹太教是世界上最古老的一神教传统，其历史跨越三千余年。犹太教不仅是一种宗教信仰，更是一整套涵盖律法、伦理、哲学、神秘主义的文明体系。犹太教的经典诠释传统（从米德拉什到塔木德）深刻影响了基督教和伊斯兰教的形成，其独特的律法思维和伦理反思至今仍具有重要的学术价值。

### 核心特征

| 特征 | 说明 |
|------|------|
| 严格一神论 | 强调上帝的唯一性与不可描绘性（Shema：שְׁמַע יִשְׂרָאֵל） |
| 盟约神学 | 上帝与以色列民族之间的契约关系是信仰核心 |
| 律法中心 | Torah（托拉）既是启示也是生活指南，613 条诫命（mitzvot） |
| 诠释传统 | 口传律法与书面律法并重，注重多层次文本解读 |
| 伦理一神论 | 宗教实践与伦理行为不可分割 |
| 历史意识 | 强调集体记忆与历史中的神圣意义 |

## 📚 核心典籍

### 第一层：圣经文献（塔纳赫 תַּנַ"ךְ）

| 典籍 | 希伯来名 | 内容概要 | 重要性 |
|------|----------|----------|--------|
| **托拉（摩西五经）** | תּוֹרָה | 创世记、出埃及记、利未记、民数记、申命记 | ★★★★★ |
| **先知书（Nevi'im）** | נְבִיאִים | 前先知书（约书亚记—列王纪）、后先知书（以赛亚—玛拉基） | ★★★★★ |
| **圣文集（Ketuvim）** | כְּתוּבִים | 诗篇、箴言、约伯记、五卷轴、但以理书、历代志等 | ★★★★☆ |

### 第二层：拉比文献

| 典籍 | 成书时期 | 内容概要 | 重要性 |
|------|----------|----------|--------|
| **密西拿（Mishnah）** | 约公元 200 年 | 口传律法的首次系统编纂，六大部（Sedarim） | ★★★★★ |
| **巴比伦塔木德** | 约公元 500 年 | 对密西拿的详尽讨论与扩展，犹太律法最高权威 | ★★★★★ |
| **耶路撒冷塔木德** | 约公元 400 年 | 巴勒斯坦地区拉比对密西拿的讨论 | ★★★★☆ |
| **米德拉什文献** | 公元前后数世纪 | 经文诠释文集（哈拉卡与阿加达） | ★★★★☆ |
| **托塞夫塔（Tosefta）** | 约公元 300 年 | 密西拿的补充材料 | ★★★☆☆ |

### 第三层：中世纪经典

| 典籍 | 作者 | 内容概要 | 重要性 |
|------|------|----------|--------|

| **迷途指津** | 迈蒙尼德 | 犹太哲学巅峰之作，调和理性与信仰 | ★★★★★ |
| **库扎里（Kuzari）** | 耶胡达·哈列维（约 1075-1141） | 以对话体为犹太教辩护 | ★★★★☆ |
| **舒尔汗·阿鲁赫** | 约瑟夫·卡罗（1488-1575） | 犹太律法权威法典 | ★★★★☆ |

### 第四层：卡巴拉与神秘主义

| 典籍 | 成书时期 | 内容概要 | 重要性 |
|------|----------|----------|--------|
| **创造之书（Sefer Yetzirah）** | 公元 2-6 世纪 | 最早的卡巴拉文献，论述创世与希伯来字母 | ★★★★☆ |

## 📖 推荐阅读顺序

### 入门路径（适合零基础）

```
第一阶段：概览认知
├── 1. 《犹太教》（尼古拉斯·德兰格）——学术入门概览
├── 2. 创世记 + 出埃及记 ——叙事性最强，适合入门
└── 3. 诗篇（精选）——感受犹太灵性传统

第二阶段：核心经典
├── 4. 申命记 ——犹太律法与盟约神学的核心
├── 5. 以赛亚书 ——先知传统的代表
├── 6. 密西拿·阿沃特篇（Pirkei Avot）——拉比伦理格言
└── 7. 拉什对创世记的注释（精选）——体验犹太经典诠释

第三阶段：深入研读
├── 8. 巴比伦塔木德（选读一个 tractate，如 Berakhot）
├── 9. 迈蒙尼德《迷途指津》（精选章节）
├── 10. 佐哈尔（精选段落）
└── 11. 马丁·布伯《我与你》
```

## 🔬 研究重点

### 重点方向一：托拉诠释学（Hermeneutics）

- **核心问题**：犹太传统如何理解"神圣文本"？多层次诠释方法（PaRDeS）的内在逻辑是什么？
- **关键概念**：Peshat（字面义）、Remez（暗示义）、Derash（诠释义）、Sod（奥秘义）
- **研究内容**：
  - 希列的七条诠释规则与以实玛利拉比的十三条规则
  - 米德拉什方法论：哈拉卡（律法性）vs 阿加达（叙事性）诠释
  - 中世纪注释传统（拉什 vs 伊本·以斯拉 vs 纳赫马尼德斯）
  - 现代圣经批评与传统诠释的对话
- **优先级**：🔴 高

### 重点方向二：拉比文学与塔木德研究

- **核心问题**：口传律法的权威性从何而来？塔木德的辩证思维模式有何独特之处？
- **研究内容**：
  - 密西拿的结构与编纂原则（犹大·哈纳西的方法论）
  - 塔木德的 sugya（论题单元）分析方法
  - 哈拉卡决议过程：从塔木德争论到法律裁决
  - 阿加达文学的叙事策略与神学意涵
- **优先级**：🔴 高

### 重点方向三：卡巴拉神秘主义

- **核心问题**：犹太神秘主义如何理解神性结构？"十个质点"（Sefirot）体系的哲学意义是什么？
- **研究内容**：
  - 早期犹太神秘主义（Merkavah / Hekhalot 文学）
  - 佐哈尔的宇宙论与神性结构
  - 卢里亚卡巴拉：Tzimtzum（收缩）、Shevirat ha-Kelim（器皿破碎）、Tikkun（修复）
  - 哈西德主义对卡巴拉的大众化转化
  - 卡巴拉对基督教神秘主义和西方密教传统的影响
- **优先级**：🟡 中

### 重点方向四：犹太伦理与律法哲学

- **核心问题**：犹太律法体系（Halakhah）如何在变迁的历史中保持延续与更新？
- **研究内容**：
  - 613 条诫命的分类与体系
  - 犹太伦理（Musar）传统的发展
  - 律法与伦理的关系：是否存在"超越律法的伦理"（lifnim mishurat ha-din）？
  - 现代犹太伦理议题（生命伦理、社会正义、环境伦理）
- **优先级**：🟡 中

### 重点方向五：现代犹太思想与后大屠杀神学

- **核心问题**：大屠杀（Shoah）如何重塑犹太神学？现代犹太思想如何回应世俗化挑战？
- **研究内容**：
  - 犹太启蒙运动（Haskalah）与宗教改革
  - 布伯的对话哲学与宗教存在主义
  - 罗森茨维格的"新思维"
  - 列维纳斯的伦理形而上学与塔木德解读
  - 后大屠杀神学的多元回应
- **优先级**：🟢 中低（在掌握传统文献后展开）

## 🔑 核心术语表

| 术语 | 希伯来文 | 含义 |
|------|----------|------|
| Torah | תּוֹרָה | 律法/教导，狭义指摩西五经，广义指全部犹太教导 |
| Halakhah | הֲלָכָה | 犹太律法，字面意为"行走的道路" |
| Aggadah | אַגָּדָה | 非律法性的叙事、传说、伦理教导 |
| Midrash | מִדְרָשׁ | 经文诠释，字面意为"探究" |
| Talmud | תַּלְמוּד | 学习/教导，密西拿及其讨论（革马拉）的合集 |
| Sefirot | סְפִירוֹת | 卡巴拉中神性的十个属性/质点 |
| Tikkun | תִּיקּוּן | 修复，卡巴拉中宇宙修复的核心概念 |
| Shekhinah | שְׁכִינָה | 神的内在性/临在，常被理解为神性的女性面向 |
| Teshuvah | תְּשׁוּבָה | 悔改/回归，字面意为"转向" |
| Tzedakah | צְדָקָה | 正义/慈善，犹太伦理核心概念 |
