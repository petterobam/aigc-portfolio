# 📖 知识库

> **定位**：外部输入中心 —— 研读宗教原典、收集学术研究、追踪宗教学前沿，为方法论和专题研究提供原始素材。

---

## 🧠 知识库 vs 方法论

| 维度 | 📖 知识库 | 🔬 方法论 |
|------|----------|----------|
| **性质** | 外部输入 | 内部输出 |
| **内容** | 原典研读、学术资料、文献综述 | 研究方法、分析框架、验证工具 |
| **目标** | 积累素材、理解传统 | 构建视角、产出洞见 |
| **更新驱动** | 阅读与收集 | 反思与迭代 |

**认知闭环**：

```
知识库（输入）→ 方法论（方法）→ 专题研究（产出）→ 反馈更新知识库
```

知识库为方法论提供**原始材料**，方法论为知识库提供**筛选标准**，两者互为表里。

## ☸️ 佛教

**用途**：系统研读佛教经典，了解原始佛教、部派佛教、大乘佛教、密宗等思想体系。

### 核心典籍图谱

| 传统/学派 | 核心典籍 |
|-----------|---------|
| **原始佛教** | 巴利三藏（经藏、律藏、论藏）、阿含经（杂阿含、中阿含、长阿含、增一阿含） |
| **大乘经典** | 般若经系（《心经》《金刚经》《大般若经》）、《法华经》、《华严经》、《维摩诘经》、《楞严经》 |
| **中观学派** | 龙树《中论》《大智度论》、提婆《百论》、月称《入中论》 |
| **唯识学派** | 世亲《唯识三十颂》、《成唯识论》、无著《摄大乘论》《瑜伽师地论》 |
| **禅宗** | 《六祖坛经》、《碧岩录》、《无门关》、《景德传灯录》 |
| **净土** | 《阿弥陀经》、《无量寿经》、《观无量寿经》 |
| **密教** | 《大日经》、《金刚顶经》、宗喀巴《菩提道次第广论》 |

### 重点研究方向

| 方向 | 内容 | 优先级 |
|------|------|--------|
| 缘起性空 | 从原始缘起到中观空性的理论演进 | ⭐⭐⭐ |
| 心识理论 | 唯识学的八识体系与心所法 | ⭐⭐⭐ |
| 禅修体系 | 止观双运、四禅八定、默照与话头 | ⭐⭐ |
| 佛教伦理 | 戒律体系、菩萨道、慈悲思想 | ⭐⭐ |
| 佛教与现代 | 人间佛教、正念运动、佛教心理学 | ⭐ |

## 🔍 知识入库标准

### ✅ 值得收录

- 宗教原典及权威译本
- 重要学者的学术专著和论文
- 经典注疏和诠释传统
- 具有方法论创新的比较宗教学研究
- 一手田野调查和民族志资料
- 宗教学权威期刊的前沿论文

### ❌ 不需要收录

- 未经学术检验的通俗读物
- 带有明显传教目的的宣传材料
- 缺乏文献支撑的个人臆测
- 重复已有收录内容的冗余资料
- 与研究方向无关的边缘材料
