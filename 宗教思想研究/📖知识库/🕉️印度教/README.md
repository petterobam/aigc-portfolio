# 🕉️ 印度教

> **定位**：系统研读印度教经典与哲学传统，涵盖吠陀时代至近现代印度教改革运动，梳理印度教多元的神学与哲学体系。

## 📖 推荐阅读顺序

### 第一阶段：核心入门
1. **《薄伽梵歌》** — 印度教哲学的精华浓缩，三条道路（知识瑜伽、行动瑜伽、虔信瑜伽）
2. **主要奥义书选读** — 从《伊沙奥义书》《卡塔奥义书》《蒙查卡奥义书》入手
3. **《瑜伽经》** — 帕坦伽利的冥想与心灵修炼体系

### 第二阶段：史诗与神话
4. **《摩诃婆罗多》**（节选） — 重点阅读《薄伽梵歌》的完整语境、尸毗王故事等
5. **《罗摩衍那》**（节选） — 达磨观念的叙事展开
6. **《薄伽梵往世书》**（节选） — 黑天（Kṛṣṇa）信仰的核心文本

### 第三阶段：哲学深入
7. **商羯罗《梵经注》** — 不二论（Advaita Vedānta）的经典阐释
8. **罗摩努阇《吉祥注》** — 限定不二论（Viśiṣṭādvaita）的体系
9. **摩陀婆《梵经注》** — 二元论（Dvaita Vedānta）的立场
10. **《数论颂》** — 理解印度二元论哲学的基础

### 第四阶段：专题拓展
11. **《梨俱吠陀》**（选译） — 印度最古老的宗教文献
12. **《摩奴法典》** — 印度教社会伦理与法律传统
13. **近现代改革** — 罗摩克里希纳、辨喜、甘地、奥罗宾多的著作
