# 🏛️ 儒教 / 儒学

> **定位**：系统研读儒家经典与儒学思想，探讨儒教的哲学内涵、宗教性质及其对中华文明的深远影响。

## 🎯 研究定位

儒教/儒学在世界宗教版图中具有特殊地位——它兼具哲学、宗教、政治、伦理多重面向。「儒教是否是宗教」本身就是一个重要的学术议题。本目录以开放态度研究儒学思想体系，既关注其哲学深度，也关注其宗教性维度。

## 📚 核心典籍体系

### 第一层：经典根基（五经）

| 经典 | 内容 | 研究重点 |
|------|------|---------|
| 《周易》 | 卜筮之书→哲学经典，包含卦辞、爻辞、十翼 | 阴阳变易哲学、天人感应、象数义理之争 |
| 《诗经》 | 最早的诗歌总集（风、雅、颂） | 礼乐文化、情感表达、政治讽谏 |
| 《尚书》 | 上古政治文献 | 天命思想、德治理念、华夏政治哲学源头 |
| 《礼记》 | 礼制与哲学论文集 | 含《大学》《中庸》，礼乐秩序、人伦规范 |
| 《春秋》 | 鲁国编年史（含三传：左传、公羊传、穀梁传） | 微言大义、春秋笔法、历史哲学 |

### 第二层：四书（理学核心）

| 经典 | 作者/编者 | 研究重点 |
|------|----------|---------|
| 《论语》 | 孔子弟子记录 | 仁学、礼学、君子人格、为政以德 |
| 《孟子》 | 孟子 | 性善论、仁政、浩然之气、民本思想 |
| 《大学》 | 曾子（传统说法） | 三纲领（明明德、亲民、止于至善）、八条目 |
| 《中庸》 | 子思（传统说法） | 天命之谓性、中和、诚、慎独 |

### 第三层：先秦儒家

| 典籍/人物 | 核心思想 |
|-----------|---------|
| 《荀子》 | 性恶论、隆礼重法、天论（天行有常） |
| 《孝经》 | 孝道伦理、以孝治天下 |
| 《礼记》其他篇章 | 礼运大同、学记、乐记 |

### 第四层：汉唐经学

| 人物/著作 | 核心贡献 |
|-----------|---------|
| 董仲舒《春秋繁露》 | 天人感应、三纲五常、大一统理论 |
| 郑玄经注 | 遍注群经，古文经学集大成 |
| 韩愈《原道》 | 道统说，排佛老，儒学复兴先声 |
| 李翱《复性书》 | 心性论，宋明理学的先导 |

### 第五层：宋明理学（新儒学第一期）

| 人物/著作 | 核心思想 |
|-----------|---------|
| 周敦颐《太极图说》《通书》 | 太极生两仪，诚学，理学奠基 |
| 张载《正蒙》 | 气本论、"为天地立心"、民胞物与 |
| 程颢/程颐 | 天理论、格物致知、仁者以天地万物为一体 |
| 朱熹《四书章句集注》 | 理气论、格物穷理、理学集大成 |
| 陆九渊 | 心即理、发明本心、简易工夫 |
| 王阳明《传习录》 | 心即理、致良知、知行合一、四句教 |

### 第六层：清代与近现代

| 人物/著作 | 核心贡献 |
|-----------|---------|
| 戴震《孟子字义疏证》 | 批判理学禁欲，情感哲学 |

| 熊十力《新唯识论》 | 新儒家，体用不二，融合佛儒 |
| 牟宗三《心体与性体》 | 道德的形而上学，良知坎陷说 |
| 杜维明 | 儒学第三期发展，文明对话 |

## 📖 推荐阅读顺序

### 入门路径

1. 《论语》（杨伯峻《论语译注》）
2. 《孟子》（杨伯峻《孟子译注》）
3. 《大学》《中庸》
4. 冯友兰《中国哲学简史》（整体把握脉络）

### 进阶路径

1. 朱熹《四书章句集注》
2. 王阳明《传习录》
3. 《周易》（含程颐《伊川易传》）
4. 《礼记·礼运》《礼记·学记》

### 深入路径

1. 《荀子》选读
2. 张载《正蒙》/ 二程语录
3. 戴震《孟子字义疏证》
4. 牟宗三《心体与性体》选读

## 🔑 关键概念索引

| 概念 | 含义 | 关联 |
|------|------|------|
| 仁 (rén) | 爱人，儒学核心德目 | 基督教 agape、佛教慈悲 |
| 礼 (lǐ) | 社会规范与仪式秩序 | 伊斯兰 Sharia、犹太 Halakha |
| 天命 (tiānmìng) | 上天赋予的使命与道德法则 | 基督教天意、印度教 dharma |
| 道 (dào) | 天道、人道——宇宙与人生的根本法则 | 道教的道、佛教的法 |
| 诚 (chéng) | 真实无妄，天道与人德的统一 | 佛教真如、基督教真理 |
| 格物致知 | 研究事物以获取知识 | 佛教观照、现象学方法 |
| 知行合一 | 知与行本为一体（王阳明） | 佛教修证一如、实践哲学 |
| 良知 (liángzhī) | 先天的道德知觉（王阳明） | 佛教佛性、基督教良心 |
