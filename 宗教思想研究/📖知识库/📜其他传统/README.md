# 📜 其他传统

> **定位**：收录世界宗教版图中除六大主流宗教之外的重要信仰传统，关注其独特的哲学贡献与文化价值。

## 🎯 研究范围

本目录涵盖以下宗教与思想传统：

### 1. 锡克教（Sikhism）
- **核心典籍**：
  - 《古鲁·格兰特·萨希卜》（Guru Granth Sahib）—— 锡克教圣典
  - 《达萨姆·格兰特》（Dasam Granth）
  - 贾南·萨基（Janam Sakhis）—— 古鲁那纳克传记文献
- **重点方向**：一神论神学、种姓制度批判、锡克教伦理、宗教融合主义

### 2. 耆那教（Jainism）
- **核心典籍**：
  - 《谛义证得经》（Tattvartha Sutra）—— 乌摩斯伐蒂著
  - 阿含经典（Agamas）—— 白衣派经典
  - 《根本经》（Mulasutras）
  - 空衣派文献：《六原理论》（Shatkhandagama）
- **重点方向**：非暴力（Ahimsa）哲学、相对主义（Anekantavada）、苦行传统、耆那教逻辑学

### 3. 巴哈伊教（Bahá'í Faith）
- **核心典籍**：
  - 巴哈欧拉《至圣书》（Kitáb-i-Aqdas）
  - 巴哈欧拉《确信之书》（Kitáb-i-Íqán）
  - 巴哈欧拉《隐言经》（Hidden Words）
  - 阿博都·巴哈《已答之问》
- **重点方向**：宗教统一论、渐进启示、世界秩序构想、现代宗教运动

### 4. 琐罗亚斯德教 / 拜火教（Zoroastrianism）
- **核心典籍**：
  - 《阿维斯塔》（Avesta）—— 包括《迦萨》（Gathas）、《亚斯纳》（Yasna）
  - 《本达希申》（Bundahishn）—— 中古波斯语宇宙论文献
  - 《丁卡德》（Denkard）—— 宗教百科全书
- **重点方向**：善恶二元论、末世论起源、对亚伯拉罕宗教的影响、帕西社区

### 5. 原住民宗教传统
- **涵盖范围**：
  - 非洲传统宗教（约鲁巴、伏都等）
  - 美洲原住民灵性传统
  - 大洋洲原住民信仰（澳大利亚梦时代等）
  - 萨满教传统（西伯利亚、蒙古、韩国等）
  - 神道教（日本）
- **重点方向**：万物有灵论、祖先崇拜、口传传统、仪式与治愈、宗教生态学

### 6. 摩尼教（Manichaeism）
- **性质**：已衰落的世界性宗教，历史研究价值极高
- **核心文献**：
  - 科隆摩尼古卷（Cologne Mani Codex）
  - 敦煌摩尼教文献
  - 吐鲁番摩尼教残卷
- **重点方向**：光暗二元论、宗教融合主义典范、丝绸之路传播史

## 📚 推荐阅读顺序

### 第一阶段：概论入门
1. 休斯顿·史密斯《人的宗教》—— 世界宗教全景概览
2. 尼尼安·斯马特《世界宗教》—— 学术性通识教材
3. 牟钟鉴、张践《中国宗教通史》—— 中国视角的宗教全景

### 第二阶段：印度系宗教拓展
1. 《谛义证得经》—— 耆那教哲学入门
2. 《古鲁·格兰特·萨希卜》选读 —— 锡克教精神核心
3. 保罗·邓达斯《耆那教》—— 学术导论

### 第三阶段：波斯传统与新兴宗教
1. 《迦萨》（查拉图斯特拉的颂歌）
2. 玛丽·博伊斯《琐罗亚斯德教》
3. 巴哈欧拉《确信之书》
4. 彼得·史密斯《巴哈伊信仰简介》

### 第四阶段：原住民与消亡传统
1. 米尔恰·伊利亚德《萨满教：古老的入迷术》
2. 杰森·贝杜恩《摩尼教》
3. 格雷厄姆·哈维《万物有灵论》

## ⚠️ 研究注意事项

1. **避免"他者化"**：研究小众传统时，应以内部视角理解其自洽逻辑，而非以主流宗教为标尺评判
2. **活态传统优先**：区分仍有活跃信众的传统（锡克教、耆那教等）与历史传统（摩尼教），研究方法有所不同
3. **田野与文本结合**：原住民传统尤其依赖口传和仪式，不能仅靠文献研究
4. **术语敏感性**：注意"原始宗教""异教"等词汇的殖民主义色彩，使用学术中性术语
