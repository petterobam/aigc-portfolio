# 📚 道教核心典籍阅读清单

> **创建日期**：2026-04-16
> **目的**：为系统研读道教思想提供清晰的典籍路线图

---

## 📖 阅读路径总览

### 路径一：基础入门路径（适合初学者）
```
1. 《道德经》（多注本对读）
2. 《庄子》内七篇
3. 葛兆光《道教与中国文化》（入门读物）
4. 卿希泰《中国道教史》第一卷（历史脉络）
```

### 路径二：哲学深研路径（老庄哲学）
```
第一阶段：老子思想（2-3个月）
├── 《道德经》（河上公注）
├── 《道德经》（王弼注）
├── 《道德经》（陈鼓应注）
├── 《道德经》（帛书/竹简版）
└── 老子思想综合研究

第二阶段：庄子思想（3-4个月）
├── 《庄子》内七篇（郭象注）
├── 《庄子》内七篇（陈鼓应注）
├── 《庄子》外杂篇选读
├── 《庄子》与佛教互动研究
└── 庄子哲学综合研究

第三阶段：先秦道家（1-2个月）
├── 《列子》
├── 《文子》
├── 《管子》心术篇等
└── 黄老思想研究
```

### 路径三：道教炼养路径（内丹学）
```
第一阶段：丹道理论基础（1-2个月）
├── 《周易参同契》
├── 《抱朴子内篇》
└── 丹道历史与理论概览

第二阶段：内丹核心经典（3-4个月）
├── 《悟真篇》
├── 《钟吕传道集》
├── 《灵宝毕法》
├── 《性命圭旨》
└── 内丹思想综合研究

第三阶段：上清经系（2-3个月）
├── 《黄庭经》
├── 上清经选读
└── 存思法研究
```

### 路径四：制度史路径（道教史与科仪）
```
1. 《太平经》选读
2. 《度人经》
3. 《清静经》
4. 道教科仪文献选读
5. 卿希泰《中国道教史》全四卷
```

### 路径五：主题性阅读（按研究兴趣）
```
主题：生死观
├── 《道德经》（生死相关章）
├── 《庄子》（生死篇）
├── 《列子》（力命篇）
├── 《悟真篇》（生死与丹道）
└── 《太平经》（承负说）

主题：修身养性
├── 《道德经》（养生相关章）
├── 《庄子》（养生篇、人间世）
├── 《黄庭经》
├── 《清静经》
├── 《太乙金华宗旨》
└── 《性命圭旨》
```
