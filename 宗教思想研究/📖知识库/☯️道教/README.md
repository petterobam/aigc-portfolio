# ☯️ 道教

> **定位**：系统研读道教经典与思想体系，涵盖老庄哲学、黄老之学、道教炼养术、科仪制度及其对中国文化的深远影响。

---

## 📌 学科概述

道教是中国本土生长的宗教传统，其思想根基可追溯至先秦道家哲学（老子、庄子），而作为制度化宗教则形成于东汉末年。道教融合了哲学思辨、宇宙论、身体修炼、仪式实践和民间信仰，构成了一个极为庞杂而丰富的思想体系。

**研究范围**：
- 道家哲学（先秦至魏晋）
- 道教教团与制度史（天师道、上清派、灵宝派、全真道、正一道）
- 道教炼养学（内丹、外丹、导引、存思）
- 道教科仪与神学
- 道教与中国文化（文学、艺术、医学、科技）

## 📖 推荐阅读顺序

### 入门阶段（1-3 个月）

```
1. 《道德经》 ← 多个注本对读：河上公注、王弼注、现代陈鼓应注
2. 《庄子》内七篇 ← 先读郭象注，再读现代陈鼓应《庄子今注今译》
3. 卿希泰主编《中国道教史》第一卷 ← 把握历史脉络
4. 入门读物：葛兆光《道教与中国文化》
```

### 进阶阶段（3-6 个月）

```
1. 《庄子》外杂篇
2. 《列子》
3. 《太平经》选读 ← 王明《太平经合校》
4. 《抱朴子内篇》 ← 王明校释本
5. 《周易参同契》 ← 朱熹《周易参同契考异》、现代注本
6. 任继愈主编《中国道教史》 ← 深化历史认识
```

### 深入阶段（6-12 个月）

```
1. 《黄庭经》及上清经系
2. 《度人经》及灵宝经系
3. 《悟真篇》 ← 翁葆光注、现代萧天石注
4. 《钟吕传道集》
5. 《道藏》相关专题选读
6. 施舟人（Kristofer Schipper）《道体论》
7. 柳存仁道教研究论集
```

### 专题研究阶段

```
1. 内丹学专题：《悟真篇》→《钟吕传道集》→《灵宝毕法》→《性命圭旨》
2. 老庄哲学专题：深入各家注疏传统
3. 科仪专题：《道藏》仪式文献 + 田野调查资料
4. 道教史专题：按断代深入（汉魏→隋唐→宋元→明清）
```

## 🔑 关键概念索引

- **道**（Dao）：终极实在、万物之源、不可言说的本体
- **德**（De）：道之显现、万物之本性
- **无为**（Wuwei）：顺应自然、不强为、非不作为
- **自然**（Ziran）：自己如此、本然状态
- **气**（Qi）：构成万物的基本质料、生命能量
- **阴阳**：宇宙间两种对立互补的基本力量
- **精气神**：内丹修炼的三宝
- **坐忘**：庄子提出的去除心知、与道合一的修行法
- **存思**：上清派核心修炼法，观想体内神灵
- **内丹**：以身体为鼎炉，炼精化气、炼气化神、炼神还虚
- **承负**：早期道教因果报应理论，祖先与后代间的善恶承续

1. **基本信息**：书名、作者、成书年代、版本（底本与参校本）
