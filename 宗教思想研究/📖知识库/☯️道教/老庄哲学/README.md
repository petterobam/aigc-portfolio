## 📝 研究方法论笔记

### 经典多版本对读法
- **选本**：通行本（王弼本）为底本，帛书本（甲乙）、郭店楚简本、河上公注本为参校
- **关注差异**：异文往往揭示不同时代的思想取向
- **注疏传统**：王弼本代表玄学诠释，河上公本代表养生/治国诠释，两者对照可见哲学与宗教的张力
- **记录格式**：每章设"版本对照"板块，列出关键异文并注释意义差异

### 佛道比较哲学方法
- 先分别梳理各自概念体系，避免"格义"式简单比附
- 在本体论、认识论、实践论三个层面分别比较
- 区分"表层相似"与"深层结构共鸣"
- 注意历史性的思想互动（如重玄学对般若思想的吸收）
