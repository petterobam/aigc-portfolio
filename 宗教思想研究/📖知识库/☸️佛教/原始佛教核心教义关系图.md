## 一、五大教义全景


## 二、三法印（Tilakkhaṇa）——实相的三个标记

三法印是佛教对一切存在现象的根本判断，是整个教义体系的世界观基础。

| 法印 | 巴利 | 含义 | 出处 |
|------|------|------|------|
| **无常** | anicca | 一切现象都在生灭变化，无一刹那停留 | 《无我相经》核心论证起点 |
| **苦** | dukkha | 一切现象都不圆满、不可依赖、终将导致不满足 | 《转法轮经》苦谛；贯穿全部经典 |
| **无我** | anattā | 一切现象中没有一个永恒不变、可主宰的核心 | 《无我相经》主题；《大念处经》法念处 |

### 三法印的逻辑链

```
无常（anicca）
  │  → 一切都在变灭，无法保持
  ↓
苦（dukkha）  
  │  → 无法保持的东西不可依赖，执着必生苦
  ↓
无我（anattā）
  │  → 变灭的、不满足的东西，不可能是一个永恒可主宰的"我"
  ↓
结论：不应执着（na abhinivisitabba）
```

**经典依据**（《无我相经》§6）：

> *「诸比库，你们认为如何？色是常还是无常？」——「无常，尊者。」*
> *「无常的是苦还是乐？」——「是苦，尊者。」*
> *「无常、苦、变易之法，适合视为『这是我的，我即是此，这是我的自我』吗？」——「不适合，尊者。」*

## 三、四圣谛（Ariyasaccāni）——诊断框架

四圣谛是佛陀对人类生存境况的完整诊断，采用"病理学"模型：**症状 → 病因 → 康复可能 → 治疗方案**。

```
┌──────────────────────────────────────────────────────┐
│                    四圣谛                               │
│                                                        │
│   ┌─────────┐    ┌─────────┐                          │
│   │ 苦谛    │    │ 集谛    │  ← 苦的因果关系            │
│   │ Dukkha  │───→│Samudaya │     (症状 → 病因)          │
│   │ 果（症状）│    │ 因（病因）│                          │
│   └─────────┘    └────┬────┘                          │
│                        │                               │
│   ┌─────────┐    ┌────┴────┐                          │
│   │ 道谛    │    │ 灭谛    │  ← 解脱的因果关系           │
│   │ Magga   │───→│Nirodha  │     (治疗方案 → 康复)       │
│   │ 因（治疗）│    │ 果（康复）│                          │
│   └─────────┘    └─────────┘                          │
│                                                        │
└──────────────────────────────────────────────────────┘
```

### 三转十二行相（《转法轮经》核心结构）

对每一谛进行三重观照，共十二个认知行为：

| 谛 | 第一转：知（Sacca-ñāṇa） | 第二转：修（Kicca-ñāṇa） | 第三转：证（Kata-ñāṇa） |
|---|---|---|---|
| **苦谛** | 「此是苦，应彻底了知」 | 「苦应被了知」 | 「苦已被了知」 |
| **集谛** | 「此是苦之集，即渴爱」 | 「集应被断除」 | 「集已被断除」 |
| **灭谛** | 「此是苦之灭，即渴爱灭」 | 「灭应被证得」 | 「灭已被证得」 |
| **道谛** | 「此是导致苦灭之道」 | 「道应被修习」 | 「道已被修习」 |

**关键洞察**：四圣谛不只是四个命题，而是**十二个修行动作**——知→修→证，每个谛都要经历完整的认知循环。

## 四、五蕴（Pañcakkhandhā）——存在分析

五蕴是佛教对「人是什么」的回答，将看似统一的「自我」分解为五个不断变化的过程。

```
┌────────────────────────────────────────────────────────┐
│                      五蕴（Khandhā）                     │
│                                                          │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐            │
│  │ 色 rūpa  │   │ 受 vedanā│   │ 想 saññā │            │
│  │ 物质身体  │   │ 感受体验  │   │ 认知标记  │            │
│  │ (所依)    │   │ (评价)    │   │ (辨识)    │            │
│  └──────────┘   └──────────┘   └──────────┘            │
│                                                          │
│  ┌──────────┐   ┌──────────────────────┐               │
│  │ 行saṅkhārā│   │ 识 viññāṇa          │               │
│  │ 意志行为  │   │ 觉知/分别            │               │
│  │ (造业)    │   │ (了别)               │               │
│  └──────────┘   └──────────────────────┘               │
│                                                          │
│  五蕴 = 「我」的完整解构                                 │
│  无一蕴是常、是乐、是可主宰的 → 无一蕴是我              │
└────────────────────────────────────────────────────────┘
```

### 五蕴与三法印的关系（《无我相经》论证结构）

| 蕴 | 巴利 | 无常证据 | 苦的证据 | 无我结论 |
|---|---|---|---|---|
| 色 | rūpa | 身体会老病死 | 无法主宰身体状态 | anattā |
| 受 | vedanā | 感受生灭不停 | 乐受变苦受，苦受不可控 | anattā |
| 想 | saññā | 认知随时改变 | 错觉、偏见不可免 | anattā |
| 行 | saṅkhārā | 意图念念生灭 | 善恶不定，不能完全控制 | anattā |
| 识 | viññāṇa | 觉知随境迁流 | 不可自主其生灭 | anattā |

**十一维度穷举**（对每一蕴）：

> 过去、未来、现在；内的、外的；粗的、细的；劣的、胜的；远的、近的——一切色（受/想/行/识），都应如理观见：「这不是我的，我非此，此非我的自我。」

### 五蕴与四念处的对应

《大念处经》的四念处修行实际上是以五蕴为观察对象：

| 念处 | 巴利 | 对应的蕴 | 观察内容 |
|------|------|---------|---------|
| 身念处 | kāyānupassanā | 色（rūpa） | 呼吸、姿势、三十二身分、四大、墓园九想 |
| 受念处 | vedanānupassanā | 受（vedanā） | 乐受、苦受、不苦不乐受（× 肉体/非肉体） |
| 心念处 | cittānupassanā | 识（viññāṇa）+ 想（saññā） | 贪/离贪心、嗔/离嗔心、痴/离痴心等16种状态 |
| 法念处 | dhammānupassanā | 行（saṅkhārā）+ 全体五蕴 | 五盖、五取蕴、六处、七觉支、四圣谛 |

## 五、八正道（Aṭṭhaṅgika Magga）——解脱路径

八正道是四圣谛中道谛的具体展开，是通向苦灭的实践道路。

```
                    八正道（三学分组）
                    
    ┌──────────────慧学（Paññā）──────────────┐
    │                                          │
    │  ① 正见（Sammā-diṭṭhi）                │
    │     → 四圣谛的如实了知                   │
    │  ② 正思惟（Sammā-saṅkappa）             │
    │     → 出离、无嗔、无害的意向             │
    │                                          │
    ├──────────────戒学（Sīla）────────────────┤
    │                                          │
    │  ③ 正语（Sammā-vācā）                   │
    │     → 不妄语、不两舌、不恶口、不绮语     │
    │  ④ 正业（Sammā-kammanta）               │
    │     → 不杀生、不偷盗、不邪淫             │
    │  ⑤ 正命（Sammā-ājīva）                  │
    │     → 以正当方式谋生                     │
    │                                          │
    ├──────────────定学（Samādhi）─────────────┤
    │                                          │
    │  ⑥ 正精进（Sammā-vāyāma）               │
    │     → 四正勤：已生恶令断、未生恶令不生、  │
    │       未生善令生、已生善令增长            │
    │  ⑦ 正念（Sammā-sati）                   │
    │     → 四念处：身、受、心、法              │
    │  ⑧ 正定（Sammā-samādhi）                │
    │     → 四禅：初禅至四禅                   │
    │                                          │
    └──────────────────────────────────────────┘
```

### 八正道的螺旋递进

八正道不是线性步骤，而是螺旋递进的——每一支都含摄其他支：

```
正见 → 知道什么是对的方向
  ↓
正思惟 → 内心朝那个方向调整
  ↓
正语/正业/正命 → 外在行为与方向一致
  ↓
正精进 → 持续不懈地努力
  ↓
正念 → 时时觉察是否偏离方向
  ↓
正定 → 心完全安住于正确方向
  ↓
（更深层的）正见 → 修行带来更深层的理解
  ↓ （循环）
```

### 八正道与中道的关系（《转法轮经》）

佛陀在初转法轮时，先拒绝两种极端，然后指出中道就是八正道：

| 极端 | 巴利 | 问题 | 八正道的对治 |
|------|------|------|-------------|
| **感官放纵** | kāmasukhallikānuyoga | 沉溺欲乐，增长渴爱 | 戒学（正语、正业、正命）约束行为 |
| **自我折磨** | attakilamathānuyoga | 苦行无益，身心俱疲 | 定学（正精进、正念、正定）培养善巧 |

## 六、十二缘起（Paṭiccasamuppāda）——苦的机制

十二缘起是四圣谛中集谛的详细展开，回答「苦是如何产生的」这一核心问题。

```
                    十二缘起（顺观——苦的生起）

    ┌─ ① 无明（avijjā）──→ 根本无知：不知四圣谛
    │
    ├─ ② 行（saṅkhārā）──→ 无明驱动下的意志行为
    │
    ├─ ③ 识（viññāṇa）──→ 行业牵引的意识投生
    │
    ├─ ④ 名色（nāmarūpa）→ 心身组合体（精神+物质）
    │
    ├─ ⑤ 六处（saḷāyatana）→ 眼耳鼻舌身意六种感官
    │
    ├─ ⑥ 触（phassa）──→ 感官与对象的接触
    │
    ├─ ⑦ 受（vedanā）──→ 接触产生的感受（苦/乐/舍）
    │
    ├─ ⑧ 爱（taṇhā）──→ 对感受的渴求执着 ← 苦的直接原因
    │
    ├─ ⑨ 取（upādāna）──→ 渴爱强化为坚固的执取
    │
    ├─ ⑩ 有（bhava）──→ 执取导致的再生力量
    │
    ├─ ⑪ 生（jāti）──→ 再度出生
    │
    └─ ⑫ 老死（jarāmaraṇa）→ 必然伴随的忧悲苦恼
    
    ──────── 苦（dukkha）────────
```

### 十二缘起的三个时间维度

```
┌───────────┐     ┌───────────┐     ┌───────────┐
│  过去世    │     │  现在世    │     │  未来世    │
│           │     │           │     │           │
│ ① 无明    │     │ ⑥ 触      │     │ ⑩ 有      │
│ ② 行      │ ──→ │ ⑦ 受      │ ──→ │ ⑪ 生      │
│           │     │ ⑧ 爱      │     │ ⑫ 老死    │
│  (过去因)  │     │ ⑨ 取      │     │ (未来果)   │
│           │     │           │     │           │
│           │     │ ③ 识      │     │           │
│           │     │ ④ 名色    │     │           │
│           │     │ ⑤ 六处    │     │           │
│           │     │ (现在果/因)│     │           │
└───────────┘     └───────────┘     └───────────┘
```

### 十二缘起与五蕴的交叉

| 缘起支 | 对应蕴 | 关系 |
|--------|--------|------|
| ② 行（saṅkhārā） | 行蕴 | 同名概念：意志造作 |
| ③ 识（viññāṇa） | 识蕴 | 同名概念：分别觉知 |
| ④ 名色（nāmarūpa） | 全体五蕴 | 名=受想行识，色=色蕴 |
| ⑦ 受（vedanā） | 受蕴 | 同名概念：感受体验 |

**关键洞察**：十二缘起中的多个环节直接对应五蕴——这意味着缘起过程就是五蕴的相续运作。五蕴是**结构分析**（人是什么），缘起是**过程分析**（苦怎么来），二者是从不同角度描述同一现象。

### 7.2 教义功能分类

| 教义 | 功能 | 回答的问题 | 性质 |
|------|------|-----------|------|
| **三法印** | 世界观 | 存在的本质是什么？ | 实相描述 |
| **五蕴** | 存在分析 | 「我」到底是什么？ | 结构解构 |
| **四圣谛** | 诊断框架 | 人的根本问题是什么？能否解决？ | 总纲 |
| **十二缘起** | 因果分析 | 苦是如何产生的？如何止息？ | 机制揭示 |
| **八正道** | 实践路径 | 具体应该怎么做？ | 修行方法 |

### 7.3 修行中的教义关系

从实践角度看，五大教义构成一个完整的修行闭环：

```
1. 正见（慧学）──→ 理解三法印 + 四圣谛
                    ↓
2. 如理作意 ──────→ 在五蕴上观察无常、苦、无我（四念处）
                    ↓
3. 见到缘起 ──────→ 理解苦的因果链（十二缘起）
                    ↓
4. 断除渴爱 ──────→ 切断缘起链的关键环节（爱 → 取 → 有）
                    ↓
5. 修习八正道 ────→ 完整的戒定慧实践
                    ↓
6. 证得灭谛 ──────→ 苦的彻底止息（涅槃 nibbāna）
                    ↓
7. 确证三法印 ────→ 修行验证了最初的洞见
                    ↓
                  （回到1，更深层的正见）
```

## 九、核心术语速查

| 巴利文 | 汉译 | 英译 | 简释 |
|--------|------|------|------|
| anicca | 无常 | impermanence | 一切现象刹那生灭 |
| dukkha | 苦 | suffering / unsatisfactoriness | 不圆满、不可靠 |
