# 📚 佛教核心典籍阅读清单

> **创建日期**：2026-04-16
> **目的**：为系统研读佛教思想提供清晰的典籍路线图

---

## 📖 阅读路径总览

### 路径一：基础入门路径（适合初学者）
```
1. 《佛陀的启示》（罗睺罗·化普乐）
2. 《心经》+ 两种注解
3. 《杂阿含经》选读（四谛、八正道、十二因缘）
4. 《金刚经》+ 一种注解
5. 《六祖坛经》
```

### 路径二：系统研读路径（适合深入学习）
```
第一阶段：原始佛教基础（2-3个月）
├── 《杂阿含经》核心篇章
├── 《转法轮经》+ 巴利对照
└── 《大念处经》

第二阶段：般若空观（2-3个月）
├── 《心经》+ 深度研读
├── 《金刚经》+ 历代注疏对比
├── 《八千颂般若经》选段
└── 《般若经系思想脉络综述》

第三阶段：大乘核心经典（3-4个月）
├── 《法华经》
├── 《维摩诘经》
├── 《华严经·入法界品》
└── 《楞伽经》

第四阶段：学派思想研究（3-4个月）
├── 《中论》+ 青目释
├── 《唯识三十颂》+ 窥基述记
├── 《解深密经》
└── 《大般涅槃经》

第五阶段：禅宗与密教（3-4个月）
├── 《碧岩录》选读
├── 《无门关》选读
├── 《临济录》
└── 《菩提道次第广论》选读
```

### 路径三：主题性阅读（按研究兴趣）
```
主题：生死观
├── 《杂阿含经·生死篇》
├── 《金刚经》
├── 《法华经》
├── 《大般涅槃经》
└── 《中论·观业品》

主题：修行方法
├── 《大念处经》
├── 《六祖坛经》
├── 《碧岩录》
├── 《菩提道次第广论》
└── 《清净道论》

主题：空性思想
├── 《心经》
├── 《金刚经》
├── 《八千颂般若经》
├── 《中论》
└── 《入中论》
```
