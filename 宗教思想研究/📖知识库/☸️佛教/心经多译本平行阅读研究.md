### 略本与广本

这是理解心经版本学的关键：

- **略本**（T0250、T0251）：只有"正宗分"，即从"观自在菩萨行深般若波罗蜜多时"到结尾咒语。无"如是我闻"等序分，无结尾流通分。玄奘译本即此。
- **广本**（T0253-T0257）：具备完整的"序分-正宗分-流通分"三段结构。序分交代佛陀在王舍城灵鹫山，与千二百五十比丘及无量菩萨俱，入三昧，舍利弗请问观自在菩萨……正宗分与略本基本相同。流通分则有世尊从三昧起，赞善哉等情节。

这个差异极为重要：**略本是否是广本的节录？还是广本是略本的扩展？** 学术界至今仍有讨论。略本更接近"般若精要"的提炼，广本则将其纳入标准的佛经叙事框架。

## 二、核心段落平行对照

### 1. 开篇（起观）

| 译者 | 经文 |
|------|------|
| **鸠摩罗什** | 观世音菩萨，行深般若波罗蜜时，照见五阴空，度一切苦厄。 |
| **玄奘** | 观自在菩萨，行深般若波罗蜜多时，照见五蕴皆空，度一切苦厄。 |

**对比分析**：

- **菩萨名号**：罗什译"观世音"，玄奘译"观自在"。梵文原词 *Avalokiteśvara*，*avalokita*（观）+ *īśvara*（自在），玄奘更贴近字面。但"观世音"源自 *avalokita-svara*（观世间音声）的另一种解读，罗什沿用了更早的译法。这两个译名折射了不同的理解路径：一个强调"听众生呼唤而救度"（大悲），一个强调"观照万法而自在"（般若智慧）。
- **"五阴"与"五蕴"**：罗什用"阴"（*skandha*），玄奘用"蕴"。"蕴"是更标准的译法，意为"积聚"。
- **"多"字**：玄奘在"般若波罗蜜"后加了"多"（*pāramitā* 完整音译为"波罗蜜多"），罗什省略，这在当时是常见做法。
- **"皆空"**：玄奘加了"皆"字，强调五蕴全部皆空，无一例外，语气更确定。

### 2. 色空不二（核心中的核心）

| 译者 | 经文 |
|------|------|
| **鸠摩罗什** | 舍利弗，色不异空，空不异色，色即是空，空即是色。 |
| **玄奘** | 舍利弗，色不异空，空不异色，色即是空，空即是色，受想行识，亦复如是。 |

**对比分析**：

- 两者此段几乎完全一致，但玄奘在四句之后加了"受想行识，亦复如是"，将色、受、想、行、识五蕴一并列出。罗什以"色"代表五蕴全体，玄奘则明确补全。
- 这四句是整个心经的哲学核心：**色（物质）不异于空，空不异于色；色就是空，空就是色。** 这不是在否定物质世界的存在，而是在揭示：存在的本质（色）和空性（śūnyatā）不是两件事。"不异"是否定二元对立，"即是"更进一步——不只是"不异"，根本就是同一件事。

### 3. 般若观照（诸法空相）

| 译者 | 经文 |
|------|------|
| **鸠摩罗什** | 是诸法空相，不生不灭，不垢不净，不增不减。 |
| **玄奘** | 是诸法空相，不生不灭，不垢不净，不增不减。是故空中无色，无受想行识，无眼耳鼻舌身意，无色声香味触法，无眼界乃至无意识界，无无明亦无无明尽，乃至无老死亦无老死尽，无苦集灭道，无智亦无得。 |

**对比分析**：

- 玄奘在此展开了一个宏大的"否定系列"，用"无"字贯穿十八界（六根、六尘、六识）、十二因缘（无明到老死）、四谛（苦集灭道），最后归结到"无智亦无得"。
- 罗什译本在"不增不减"之后直接跳到"以无所得故"，中间省略了这一大段。这不是疏漏——可能反映了不同的梵文底本。
- **玄奘这段"无"字诀的结构精妙**：
  - 先破五蕴（无色受想行识）→ 破十二处（无眼耳鼻舌身意，无色声香味触法）→ 破十八界（无眼界乃至无意识界）→ 破十二因缘（无无明乃至无老死尽）→ 破四谛（无苦集灭道）→ 破一切证悟（无智亦无得）。
  - 这是从存在论→认识论→缘起论→解脱论的层层破斥，最终连"证悟"本身也要放下。

### 4. 菩提萨埵与涅槃

| 译者 | 经文 |
|------|------|
| **鸠摩罗什** | 菩萨依般若波罗蜜故，心无罣碍；无罣碍故，无有恐怖，离一切颠倒梦想苦恼，究竟涅槃。 |
| **玄奘** | 菩提萨埵，依般若波罗蜜多故，心无罣碍；无罣碍故，无有恐怖，远离颠倒梦想，究竟涅槃。三世诸佛，依般若波罗蜜多故，得阿耨多罗三藐三菩提。 |

**对比分析**：

- 罗什"菩萨"vs 玄奘"菩提萨埵"——后者是 *bodhisattva* 的完整音译，"觉悟有情"。
- 罗什将"苦恼"直接并入"颠倒梦想"之后，玄奘则分开处理，先"远离颠倒梦想"，再"究竟涅槃"。
- 玄奘加了"三世诸佛"句：不只是菩萨依般若，**一切佛也依般若而成佛**。这把般若波罗蜜多从菩萨道提升到了佛道的根本。

### 5. 咒语

| 译者 | 经文 |
|------|------|
| **鸠摩罗什** | 即说咒曰：竭帝竭帝，波罗竭帝，波罗僧竭帝，菩提僧莎呵。 |
| **玄奘** | 即说咒曰：揭谛揭谛，波罗揭谛，波罗僧揭谛，菩提萨婆诃。 |

**梵文对照**：*Gate gate pāragate pārasaṃgate bodhi svāhā*

- **gate**（去）→ 罗什"竭帝"，玄奘"揭谛"
- **pāragate**（到彼岸去）→ "波罗竭帝"/"波罗揭谛"
- **pārasaṃgate**（完全到彼岸去）→ "波罗僧竭帝"/"波罗僧揭谛"
- **bodhi**（觉悟）→ "菩提"
- **svāhā**（成就/圆满）→ "僧莎呵"/"萨婆诃"

这是心经最著名的一句——也是唯一一句梵文直译不翻的部分。"去吧，去吧，到彼岸去吧，完全到彼岸去吧，觉悟啊，圆满啊。" 整部心经260字的哲学思辨，最终浓缩为一句最简单的呼唤：**去吧。** 从此岸到彼岸。从无明到觉悟。从执着到自在。

## 三、翻译风格比较

### 鸠摩罗什（344-413）

- **风格**：简洁流畅，意译为主，注重可读性
- **语言特点**：用字较古，句式较短，有龟兹语影响的痕迹
- **译经哲学**："依实出腑，务存本旨"——忠于经义但不拘泥于字面
- **历史地位**：中国佛教四大译师之一，所译经文流传极广

### 玄奘（602-664）

- **风格**：精确严密，新译代表，梵汉对应严格
- **语言特点**：术语标准化（"蕴"取代"阴"，"观自在"取代"观世音"），句式完整
- **译经哲学**："新译"运动核心——严格对应梵文原文，避免旧译的歧义
- **历史地位**：四大译师之首，心经260字本成为东亚佛教最流通版本

### 核心差异总结

| 维度 | 鸠摩罗什 | 玄奘 |
|------|---------|------|
| 翻译策略 | 意译为主 | 直译为主 |
| 术语选择 | 沿用旧译（五阴、观世音） | 创立新译（五蕴、观自在） |
| 文本长度 | 更简略 | 更完整 |
| 用字风格 | 质朴古雅 | 精密严整 |
| 哲学侧重 | 强调实践（度苦厄） | 强调理论体系（完整的"无"字系列） |

## 四、平行阅读的方法论价值

### 为什么要读多个译本？

1. **突破单一理解的局限**：每个译本都是译者对梵文原典的一次解读，比较阅读相当于同时接触多个理解维度。
2. **术语演变的窗口**：从罗什到玄奘的术语变化，映射了中国佛教理解般若思想的深化过程。
3. **文本批判的工具**：不同译本所依梵文底本可能不同，对比可辅助还原最古老的文本形态。
4. **哲学思辨的触发**：每次对比都会产生新的疑问——而这些疑问正是深入理解的起点。

### 平行阅读的操作框架

```
第一步：选一个核心段落
第二步：列出所有译本的对应段落
第三步：逐字/逐句对比差异
第四步：追溯梵文原词（利用佛津38部辞典）
第五步：分析差异背后的原因（底本不同？译者风格？哲学立场？）
第六步：形成自己的理解
```
