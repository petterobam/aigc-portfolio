## 一、概况

佛津平台收录 **38 部佛学辞典**，总计超过 **71.2 万条** 词条，覆盖中文、巴利文、梵文、藏文、英文、满文、蒙文等多种语言，是目前所见最全面的在线佛学辞典聚合平台。

### API 接口

| 接口 | 用途 |
|------|------|
| `GET /api/dictionary/sources` | 获取全部辞典列表（含词条数、语言、描述） |
| `GET /api/dictionary/search?q={词条}&limit={数量}` | 跨辞典搜索词条 |
| `GET /api/dictionary/entry/{id}` | 获取单条词条详情 |

## 三、核心使用场景

### 1. 术语溯源（梵/巴利 → 中文）
查询一个中文明语的梵文或巴利文原义。例如搜"般若"，可同时获得：
- DPD 的巴利语源分析（paññā）
- Monier-Williams 的梵文释义（prajñā）
- 丁福保、佛光大辭典的中文详释

### 2. 跨辞典对照
同一词条在多部辞典中的释义可交叉比对，例如"般若"在10+部辞典中有不同角度的释义。

### 3. 宗派专业术语查询
研究唯识学时查法相辞典，研究禅宗时查祖庭事苑，研究戒律时查南山律学辞典。

### 4. 原典语言研究
梵文、巴利文、藏文辞典为直接阅读原典提供支撑。

## 四、API 使用建议

### 搜索技巧
- `q` 参数支持中文、梵文罗马化、巴利文
- 结果按 `source_id` 分组，可通过 `source_code` 过滤特定辞典
- `total` 字段显示总结果数，支持分页

### 查询示例
```
# 查"空"的相关词条
GET /api/dictionary/search?q=空&limit=20

# 查巴利语 panna
GET /api/dictionary/search?q=panna&limit=10

# 查梵文 shunyata
GET /api/dictionary/search?q=śūnyatā&limit=10
```

## 五、辞典价值评估

### ⭐ 五星级（研读必备）
- **佛光大辭典** — 最全面的现代中文佛学百科

- **DPD 数字巴利辞典** — 巴利文研究的瑞士军刀
