# 佛津（fojin.app）平台使用指南

> **版本**：2026-04-16
> **定位**：为宗教思想研究提供佛津平台的核心功能使用路径和最佳实践

---

## 一、平台概况

佛津（FoJin）是全球佛教古籍数字资源聚合平台，核心数据：
- **503个数据源**，覆盖27个国家和地区
- **38部佛学辞典**，71.2万+词条（中/梵/巴利/藏/英）
- **113,716个实体** + **28,048条关系**的知识图谱
- 支持汉传、藏传、南传、梵文等多种传统

## 二、核心功能与使用路径

### 2.1 典籍搜索

**入口**：`https://fojin.app/search` 或 API `https://fojin.app/api/search?q=关键词`

**常用检索路径**：

| 研究方向 | 检索关键词示例 | 说明 |
|---------|--------------|------|
| 般若经系 | `般若波罗蜜`、`金刚般若`、`大般若经` | 检索所有般若类经典 |
| 法华经 | `妙法莲华经` | 各译本自动关联 |
| 净土经典 | `阿弥陀经`、`无量寿经` | 净土三经检索 |
| 禅宗典籍 | `六祖坛经`、`碧岩录` | 禅宗核心文献 |
| 律藏 | `四分律`、`梵网经` | 戒律文献 |

**API参数**：
- `q`：搜索关键词
- `lang`：语言过滤（zh/lzh/bo/sa/pi/en）
- `limit`：返回数量

**关键特性**：
- 搜索结果自动标注 `related_translations`（关联异译），直接获得同经不同译本列表
- 每条结果包含 `taisho_id`（大正藏编号）和 `cbeta_url`（CBETA在线阅读链接）

### 2.2 平行阅读（多译本对照）

**方法一：利用搜索API的 `related_translations`**
1. 搜索某经，获得主译本ID
2. 在返回结果中查看 `related_translations` 数组
3. 获取所有异译本的ID和经号
4. 分别访问各CBETA链接获取文本

**方法二：浏览器直接使用**
- 访问 `https://fojin.app/read/Txxxx`（T后面加大正藏编号）
- 前端界面支持选择多个译本并排显示

**最佳实践**：
- 先用API获取完整译本列表，再用浏览器逐段对照
- 比较维度：术语差异、句子结构、哲学侧重、底本差异
- 参见《心经多译本平行阅读研究》作为范例

### 2.3 佛学辞典

**入口**：`https://fojin.app/dictionary` 或 API `https://fojin.app/api/dictionary?q=术语`

**38部辞典分类**（按使用频率排序）：

| 类别 | 代表辞典 | 用途 |
|------|---------|------|
| 🌟 综合辞典 | 《佛光大辞典》《丁福保佛学辞典》 | 通用查询首选 |
| 🔤 梵巴辞典 | 《梵汉大辞典》《巴英辞典》(PTS) | 梵文/巴利文原义考证 |
| 📚 专科辞典 | 《天台宗辞典》《禅宗辞典》 | 宗派专门术语 |
| 🏔️ 藏传辞典 | 《藏汉大辞典》《The Tibetan & Himalayan Library》 | 藏传佛教术语 |
| 📖 经典辞典 | 《一切经音义》《翻译名义集》 | 古代音义和译名 |

### 2.4 知识图谱

**入口**：`https://fojin.app/kg`

**实体类型**：人物、寺院、宗派、经典、地名等
**关系类型**：9种（师承、翻译、隶属、地理等共28,048条）

**使用场景**：
- 追溯某译师的完整译经目录
- 查找某宗派的师承关系网络
- 定位某经典相关的所有人物和事件

## 三、经典研读工作流

```
1. 典籍搜索 → 获取主译本和所有异译
2. 平行阅读 → 多译本逐段对照，记录差异
3. 辞典查询 → 关键术语追溯梵文原义
4. 知识图谱 → 查询译者背景、宗派脉络
5. CBETA深读 → 访问完整文本和注疏
6. 产出笔记 → 整理研究心得和对比分析
```
