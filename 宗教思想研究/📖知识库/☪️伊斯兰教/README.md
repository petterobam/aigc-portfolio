# ☪️ 伊斯兰教

> **定位**：系统研读伊斯兰教经典与学术文献，涵盖逊尼派、什叶派、苏菲派等主要传统，构建对伊斯兰思想体系的全面认知。

---

## 📌 研究范围

伊斯兰教（Islam）是世界三大亚伯拉罕宗教之一，由先知穆罕默德于公元 7 世纪在阿拉伯半岛创立。"伊斯兰"意为"顺从（真主的意志）"，其核心信条为"万物非主，唯有真主；穆罕默德是真主的使者"（清真言）。伊斯兰教在神学、律法、哲学、神秘主义等方面发展出极为丰富的思想传统。

本目录聚焦于：
- 古兰经学与经注学（塔夫西尔）
- 圣训学（哈迪斯学）及其鉴别方法
- 伊斯兰教法学（菲格赫）四大法学派
- 伊斯兰神学（凯拉姆学）
- 伊斯兰哲学（法尔萨法）
- 苏菲主义（塔萨乌夫）灵性传统

## 📚 核心典籍

### 第一层：根本经典
| 典籍 | 阿拉伯文名 | 说明 |
|------|-----------|------|
| 古兰经 | القرآن الكريم | 伊斯兰教最高经典，穆斯林信仰的根基，共 114 章（苏拉） |
| 布哈里圣训实录 | صحيح البخاري | 逊尼派最权威的圣训集，伊玛目布哈里编纂 |
| 穆斯林圣训实录 | صحيح مسلم | 逊尼派第二权威圣训集，伊玛目穆斯林编纂 |

### 第二层：经注与教法
| 典籍 | 作者 | 说明 |
|------|------|------|
| 《泰伯里经注》 | 泰伯里（al-Tabari） | 最早也是最详尽的古兰经注之一，传述式经注代表 |
| 《凯沙夫经注》 | 扎马赫沙里（al-Zamakhshari） | 修辞学经注的典范，注重语言分析 |
| 《伊本·凯西尔经注》 | 伊本·凯西尔 | 以经注经、以训注经的方法论代表 |
| 《法学通则》 | 各法学派 | 哈奈菲、马立基、沙斐仪、罕百里四大法学派基本著作 |

### 第三层：神学与哲学
| 典籍 | 作者 | 说明 |
|------|------|------|
| 《圣学复苏》 | 安萨里（al-Ghazali） | 整合教法、神学与苏菲灵性的里程碑巨著，共 40 卷 |
| 《迷途指津》 | 安萨里 | 安萨里思想自传，记述其精神危机与觉悟 |
| 《哲学家的矛盾》 | 安萨里 | 对亚里士多德-伊斯兰哲学传统的系统批判 |
| 《矛盾的矛盾》 | 伊本·鲁世德（Averroes） | 回应安萨里，捍卫理性哲学的经典 |
| 《治疗论》 | 伊本·西那（Avicenna） | 伊斯兰哲学百科全书式著作 |

### 第四层：苏菲主义
| 典籍 | 作者 | 说明 |
|------|------|------|
| 《麦加的启示》 | 伊本·阿拉比（Ibn Arabi） | 苏菲形而上学的百科全书，"存在单一论"体系 |
| 《智慧珍宝》 | 伊本·阿拉比 | 以先知为线索阐述神秘智慧 |
| 《玛斯纳维》 | 鲁米（Rumi） | "波斯语的古兰经"，苏菲诗歌最高成就 |
| 《鸟的会议》 | 阿塔尔（Attar） | 苏菲寓言诗杰作，描述灵魂归向真主的旅程 |
| 《觉悟之光》 | 苏赫拉瓦迪（Suhrawardi） | 照明学派（伊什拉格）奠基之作 |

## 📖 推荐阅读顺序

### 路线一：经典入门路线
```
古兰经（马坚译本 / 仝道章译本）
  → 圣训选读（四十段圣训 / 布哈里圣训选）
  → 安萨里《迷途指津》
  → 安萨里《圣学复苏》（选读）
  → 伊本·阿拉比《智慧珍宝》
  → 鲁米《玛斯纳维》（选读）
```

### 路线二：神学-哲学路线
```
古兰经 + 经注选读
  → 艾什尔里派神学基础
  → 安萨里《哲学家的矛盾》
  → 伊本·西那《治疗论》（形而上学部分）
  → 伊本·鲁世德《矛盾的矛盾》
  → 穆拉·萨德拉《四个理智旅程》
```

### 路线三：教法-制度路线
```
古兰经 + 圣训基础
  → 伊斯兰教法原理（乌苏勒·菲格赫）
  → 四大法学派比较
  → 沙提比《教法意图论》
  → 现代伊斯兰教法改革
```

### 路线四：苏菲灵性路线
```
古兰经（灵性主题经文）
  → 哈桑·巴士里与早期禁欲主义
  → 朱奈德与"清醒的苏菲"
  → 安萨里《圣学复苏》（灵性修行卷）
  → 伊本·阿拉比《麦加的启示》（选读）
  → 鲁米《玛斯纳维》
  → 阿塔尔《鸟的会议》
```
