# ✝️ 基督教

> **定位**：系统研读基督教经典与神学传统，涵盖圣经文本、教父著作、经院哲学、宗教改革及现代神学，建立基督教思想的完整知识图谱。

---

## 📌 研究范围

基督教作为世界最大宗教（约 24 亿信徒），拥有两千年的深厚神学传统。本目录聚焦于：

- **圣经文本**：旧约与新约的文本研究、版本考据、诠释传统
- **教父神学**：使徒教父、希腊教父、拉丁教父的思想体系
- **经院哲学**：中世纪神学的理性化与系统化
- **宗教改革**：路德、加尔文、茨温利的神学革新
- **现代神学**：自由主义神学、新正统、解放神学、过程神学等
- **神秘主义**：沙漠教父、莱茵兰神秘主义、东正教灵修传统

## 📖 推荐阅读顺序

### 路线一：圣经文本入手（推荐初学者）

```
创世记 → 出埃及记 → 诗篇（选读）→ 以赛亚书
  → 马可福音 → 马太福音 → 路加福音 → 约翰福音
  → 使徒行传 → 罗马书 → 哥林多前书 → 加拉太书
  → 希伯来书 → 启示录
```

### 路线二：神学思想史路线

```
新约核心书卷 → 使徒教父文献（十二使徒遗训、伊格那丢书信）
  → 爱任纽《反异端》→ 奥利金《论原理》
  → 奥古斯丁《忏悔录》→《上帝之城》
  → 安瑟伦《宣讲》→ 阿奎那《神学大全》选读
  → 路德核心著作 → 加尔文《基督教要义》
  → 施莱尔马赫《论宗教》→ 巴特《罗马书注释》
```

### 路线三：神秘主义专线

```
约翰福音 → 保罗书信中的神秘经验段落
  → 伪狄奥尼修斯《神秘神学》→ 沙漠教父语录
  → 埃克哈特大师讲道集 → 十架约翰《心灵的暗夜》
  → 《不知之云》→ 朱利安《神圣之爱的启示》
  → 东正教《慕善集》
```

## 🔑 关键术语表

| 术语 | 希腊/拉丁文 | 含义 |
|------|-----------|------|
| 三位一体 | Trinitas | 一神三位格：圣父、圣子、圣灵 |
| 道成肉身 | Incarnatio | 神的道（Logos）取了人性 |
| 因信称义 | Sola Fide | 唯独借信心被神宣告为义 |
| 神化/成神 | Theosis | 东正教核心救赎观：人分有神性 |

| 位格 | Hypostasis | 三一论中的独立存在方式 |
| 恩典 | Gratia/Charis | 神无条件的恩惠与能力 |
| 末世论 | Eschatologia | 关于终末事件的神学 |
