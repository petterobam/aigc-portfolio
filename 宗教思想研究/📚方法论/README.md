## 📝 文本分析方法论

### 核心文本分析框架

```
① 语境还原（了解成文背景、作者意图、受众群体）
    ↓
② 结构分析（章节脉络、论证逻辑、文体特征）
    ↓
③ 概念拆解（核心术语的定义、演变、多义性）
    ↓
④ 义理阐释（思想主旨、哲学含义、修行指导）
    ↓
⑤ 传承对比（不同注疏传统的理解差异）
    ↓
⑥ 当代反思（文本对今日的启示与价值）
```

### 文本分析核心原则

1. **忠于原典，先理解再评价** —— 不带预设地进入文本世界
2. **了解翻译问题** —— 梵文、巴利文、希伯来文、阿拉伯文、希腊文原文的重要性
3. **注意区分字面意义和象征意义** —— 宗教文本往往具有多层次含义
4. **关注互文关系** —— 同一传统内部经典之间的引用和对话
5. **避免以现代视角强行解读古代文本** —— 尊重历史距离感

> 📂 详见 [📝文本分析/README.md](📝文本分析/README.md)

## 🔄 比较研究方法论

### 比较宗教学核心方法

- **主题比较法**：选取特定主题（如爱、苦、解脱），对比不同宗教的理解
- **结构比较法**：对比不同宗教的组织结构、仪式体系、修行路径
- **历史比较法**：追溯不同宗教间的历史互动和相互影响
- **现象学方法**：搁置判断，如其所是地理解宗教体验

### 比较研究注意事项

- ⚠️ 避免简单类比（表面相似 ≠ 本质相同）
- ⚠️ 尊重差异性（不要强行统一）
- ⚠️ 注意翻译陷阱（不同语言的概念不能简单对等）
- ⚠️ 保持平衡，不以某一宗教为标准评判其他宗教

> 📂 详见 [🔄比较研究/README.md](🔄比较研究/README.md)
