# 数据目录

本目录用于存储宗教思想研究过程中产生的各种数据文件，包括：

- **JSON数据**：结构化数据，便于程序处理
- **Markdown报告**：数据分析结果的可读报告
- **HTML文件**：网页抓取的原始HTML文件
- **PNG图片**：网页截图

## 目录结构

```
数据/
├── README.md           # 本文件
├── fojin-sources-*.md  # 佛津平台数据源调研报告
├── fojin-sources-*.json # 佛津平台数据源JSON数据
├── fojin-sources-*.html # 佛津平台数据源HTML文件
└── fojin-sources-*.png  # 佛津平台数据源截图
```

## 数据清理策略

为了避免数据目录膨胀，建议：

1. **保留最新版本**：保留最近的数据文件，旧版本可以归档到`🗂️归档记录/`目录
2. **定期清理**：每月检查一次数据目录，清理不再需要的文件
3. **文件命名规范**：使用时间戳（ISO 8601格式）命名文件，便于管理

## 数据使用建议

1. **JSON数据**：可用于程序化处理和批量分析
2. **Markdown报告**：供人工阅读和参考
3. **HTML文件**：保留原始网页内容，便于追溯
4. **PNG图片**：保存网页截图，便于检查和验证

## 数据安全

- 本目录可能包含敏感信息，请注意保护
- 不要将本目录内容提交到公开的代码仓库
- 定期备份重要数据文件
