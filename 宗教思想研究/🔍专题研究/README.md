# 🔍 专题研究

> **定位**：跨宗教主题性深度研究中心 —— 以特定主题为切入点，横向比较不同宗教传统的理解和回答。

## 🎯 专题研究的价值

专题研究是知识库（输入）和方法论（方法）的交汇产出。通过主题性比较，可以：
- 发现不同宗教传统之间的共通关怀和独特视角
- 深化对每个宗教核心思想的理解
- 建立跨宗教对话的桥梁
- 形成个人的宗教哲学思考

## 📋 专题研究清单

### 生死观
涵盖：轮回（佛教/印度教）、复活（基督教/伊斯兰教）、来世（犹太教）、道教的生死观
核心问题：死后会怎样？生命的意义是什么？如何面对死亡？

### 伦理道德
涵盖：五戒（佛教）、十诫（基督教/犹太教）、道德经的自然伦理、伊斯兰律法伦理、dharma（印度教）
核心问题：什么是善？如何判断对错？道德的基础是什么？

### 冥想与修行
涵盖：禅定/正念（佛教）、瑜伽（印度教）、基督教默观祈祷、苏菲dhikr、犹太冥想、道教内丹
核心问题：如何通过修行转化自我？不同修行方法的异同？

### 宇宙观
涵盖：缘起论（佛教）、创世记（犹太-基督教）、安拉创世（伊斯兰教）、梵天创世（印度教）、道生万物（道教）
核心问题：世界从何而来？宇宙的本质是什么？人在宇宙中的位置？

### 救赎与解脱
涵盖：涅槃/解脱（佛教）、moksha（印度教）、救赎/恩典（基督教）、天堂/火狱（伊斯兰教）、olam ha-ba（犹太教）、得道（道教）
核心问题：人的终极归宿是什么？如何达到终极目标？谁可以被拯救/解脱？

## 📝 专题研究模板

每篇专题研究应包含：
1. 主题定义与核心问题
2. 各宗教传统的立场概述
3. 关键概念对比分析
4. 共通点与差异点总结
5. 个人反思与洞见
6. 参考文献
