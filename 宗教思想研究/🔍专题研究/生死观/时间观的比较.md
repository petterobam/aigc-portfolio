# 时间观的比较：轮回·复活·循环·线性

## 引言：时间——宗教思想的隐秘地基

> *"时间是什么？没人问我，我倒知道；若要我解释，我便不知道了。"*——奥古斯丁《忏悔录》卷十一

当我们讨论各宗教的生死观时，最容易注意到的差异是：有的宗教相信轮回（佛教、印度教），有的相信复活（基督教、伊斯兰教），有的强调回归（道教），还有的发展出了复杂的来世论（犹太教）。但这些表层差异的根源，指向一个更深层的问题——

**时间是什么？**

时间观是宗教思想的隐秘地基。一个传统如何理解时间的本质——是线性的还是循环的，是有始有终的还是无始无终的，是一次性的还是反复的——直接决定了它如何回答：

- 人只活一次，还是反复出现？
- 历史有没有方向？有没有终点？
- 终极解脱/救赎是在时间之内完成，还是从时间中出离？
- 过去、现在、未来的关系是什么？

本文将从时间观的角度，深入分析六大宗教生死观的哲学根源。这不是一个简单的"循环vs线性"二元分类——每个传统对时间的理解都比这个二分法远为复杂和精微。

## 一、佛教：无始轮回与出离时间

### 1.1 时间作为苦的结构

佛教的时间观可以用一个词来概括：**轮回（saṃsāra）**。

轮回不是"灵魂转世"这么简单。在佛教的理解中，轮回是一个**无始无终的生死流转过程**，其本质是苦（duḥkha）。时间不是中性的容器，而是苦的结构本身——因为一切都在生灭变化中，无常（anicca）是时间的基本法则。

《法句经》第277-279偈：

> *"诸行无常……诸行是苦……诸法无我。"*

这三偈揭示了佛教对时间的根本理解：一切有为法（saṃskṛta dharma）都在刹那生灭中，时间就是这生灭的连续。因为无常，所以是苦；因为无我，所以没有谁在时间里"拥有"任何东西。

### 1.2 十二缘起：时间的内在机制

佛教不满足于"时间是循环的"这个笼统说法。十二缘起（pratītyasamutpāda）提供了一个精密的**时间因果机制**：

> 无明→行→识→名色→六入→触→受→爱→取→有→生→老死

这个链条不是一次性事件，而是**每一刻都在运作**的因果律。无明驱动渴爱，渴爱驱动执取，执取导致再生，再生导致老死——然后循环重演。时间就是这个链条的持续运转。

关键在于：**佛教的轮回没有第一因**。无始以来，众生就在轮回中流转。《相应部》中佛陀说：

> *"诸比丘，轮回无始。众生为无明所盖、渴爱所缚，流转轮回，起点不可知。"*

"无始"（anamatagga）意味着时间没有创世点，没有起点——它是一个无尽的因果流。

### 1.3 涅槃：时间的终结

佛教最独特的时间观特征是：**终极目标不是在时间之内实现的，而是从时间中出离**。

涅槃（nirvāṇa）的字面意思是"吹灭"——吹灭的是什么？是贪、嗔、痴的火焰，也就是驱动轮回因果链的动力。当无明被智慧斩断，十二缘起的链条就断了，轮回停止了。

这意味着：**涅槃不是时间中的某个状态，而是时间的终结**。不是"永远活着"，而是"不再生灭"。不是"去到一个更好的地方"，而是"从流转中出离"。

这里有一个深刻的哲学洞见：在佛教看来，时间本身就是问题。只要你在时间中（即只要有生灭），你就在苦中。解脱不是"到时间中去一个更好的位置"，而是"从时间中解脱出来"。

### 1.4 佛教时间观的核心特征

| 维度 | 内容 |
|------|------|
| **时间的基本性质** | 无始无终的因果流转（轮回） |
| **时间的驱动力** | 无明（avidyā）+ 渴爱（tṛṣṇā） |
| **时间的结构** | 过去→现在→未来的因果相续（无恒常主体） |
| **时间的方向** | 没有终极方向——除非通过修行截断因果链 |
| **时间与苦的关系** | 时间 = 苦的结构（无常故苦） |
| **终极目标** | 从时间中出离（涅槃 = 轮回的止息） |

## 二、印度教：宇宙大循环与灵魂的永恒性

### 2.1 梵的周期性展开

印度教的时间观与佛教共享"轮回"（saṃsāra）的基本框架，但在宇宙论层面远为宏大。

印度教的时间不是简单的循环，而是一个**嵌套的、分层的周期系统**：

- **劫（Kalpa）**：一个宇宙的完整周期，约43.2亿年
- **四个尤伽（Yuga）**：每个劫分为四个时代——圆满时（Kṛta Yuga）、三分时（Tretā Yuga）、二分时（Dvāpara Yuga）、争斗时（Kali Yuga），品质依次递减
- **曼万陀罗（Manvantara）**：每个劫有14个人类始祖周期
- **大劫（Mahākalpa）**：无数劫构成更大周期

《摩诃波罗多》和《往世书》描述了宇宙的**创世—维持—毁灭—再创世**的无限循环。梵天（Brahmā）的一天就是一个劫，他的寿命是100年——也就是说，宇宙要经历无数次的创生与毁灭。

### 2.2 Ātman：时间之外的真实自我

印度教时间观中最关键的哲学元素是**Ātman（我）**。

与佛教的"无我"截然相反，印度教认为个体的真实自我（Ātman）是**超越时间的**——不生不灭，永恒存在。《薄伽梵歌》2.20：

> *"灵魂（dehī）从来没有诞生，也没有死亡。它不是出生后才存在，也不是出生后才不再存在。它无始、永恒、常在、原始。身体杀死时，它并不被杀死。"*

这意味着：轮回是Ātman在时间中的旅程，但Ātman本身不在时间中。时间是现象界的属性，不是本体的属性。

### 2.3 解脱：从轮回的时间进入永恒

印度教的解脱（mokṣa）与佛教的涅槃有一个重要的共通点：**都是从轮回的时间中出离**。但有一个根本差异：

- 佛教：出离后没有"谁"在——因为没有永恒的自我
- 印度教：出离后Ātman回归其本来状态——与梵合一（梵我一如）

商羯罗的不二论（Advaita Vedānta）甚至更进一步：**轮回本来就是幻象（Māyā）**。从最高的真理视角（paramārthika）看，从来没有真正的轮回，因为Ātman从来就是梵。时间、因果、轮回只在实际性层面（vyāvahārika）存在。

### 2.4 印度教时间观的核心特征

| 维度 | 内容 |
|------|------|
| **时间的基本性质** | 宇宙大循环（创世—维持—毁灭—再创世，无限重复） |
| **时间的驱动力** | 业力（karma）+ 宇宙的周期性法则 |
| **时间的结构** | 嵌套循环（劫—尤伽—曼万陀罗） |
| **时间的方向** | 循环往复，但品质递减（从圆满时到争斗时） |
| **时间与自我的关系** | Ātman超越时间，轮回是时间中的旅程 |
| **终极目标** | 从轮回的时间中解脱（mokṣa），回归永恒的梵 |

## 三、道教：复归与自然大化

### 3.1 "反者道之动"：循环的时间哲学

道教的时间观既不是佛教式的轮回，也不是亚伯拉罕诸教式的线性，而是一种独特的**自然复归观**。

《道德经》第四十章：

> *"反者道之动，弱者道之用。"*

"反"（fǎn）是道教时间观的核心概念。万物的运动不是直线前进的，而是"反"——回归、返回、反复。道的运动是循环的，但这不是佛教式的"轮回"（在苦中流转），而是自然的"复归"（回到根源）。

《道德经》第十六章更明确：

> *"夫物芸芸，各复归其根。归根曰静，静曰复命。复命曰常。"*

万物从道中生出，最终回到道中去。这个"复归"不是消亡，而是"回家"。时间是道的呼吸——呼出（生），吸入（归），周而复始。

### 3.2 庄子的大化：时间作为自然流转

庄子把道教的时间观推到了更深的哲学层面。《大宗师》：

> *"夫大块载我以形，劳我以生，佚我以老，息我以死。故善吾生者，乃所以善吾死也。"*

庄子把生老病死看作自然的变化——"大化"（dà huà）。天地是一个大熔炉，造化是一个大工匠，人从"无"中被铸成形（生），最终又回到"无"中去（死）。

> *"今之大冶铸金，金踊跃曰：'我且必为镆铘！'大冶必以为不祥之金。今一犯人之形，而曰：'耳目、耳目！'夫造化者必以为不祥之人。今一以天地为大炉，以造化为大冶，恶乎往而不可哉！"*

这段文字的哲学深度令人叹服。庄子说：如果一块金属在熔炉里跳起来说"我要变成镆铘宝剑！"，工匠一定觉得这块金属不吉利。同样，人如果在造化的熔炉里执着于自己的形体，造化者也一定觉得这个人不吉利。

庄子的意思是：**你本来就是造化的一部分，生和死只是造化过程中的两个阶段**。执着于生、恐惧于死，就像河水执着于"一直是河"而拒绝流入大海。

### 3.3 道教时间观的核心特征

| 维度 | 内容 |
|------|------|
| **时间的基本性质** | 自然的循环运动（反、复、归） |
| **时间的驱动力** | 道的自然运动（道法自然） |
| **时间的结构** | 聚—散—复归的循环（非线性，非轮回） |
| **时间的方向** | 循环往复，但不是回到过去，而是回到本源 |
| **时间与生命的关系** | 生命是气的暂时聚形，死亡是气的复散归元 |
| **终极目标** | 不是"出离"时间，而是"合于"道的自然节律 |

### 3.4 道教"复归"vs 佛教"轮回"：关键区分

这两个概念经常被混淆，但它们有根本的区别：

| 维度 | 道教"复归" | 佛教"轮回" |
|------|-----------|-----------|
| **动力** | 道的自然运动 | 无明+渴爱 |
| **性质** | 自然、中性、非苦 | 苦（duḥkha） |
| **目标** | 合于道（无需逃离） | 出离轮回（涅槃） |
| **个体性** | 气聚散，无固定个体 | 识相续，无恒常自我 |
| **重复性** | 一次性回归本源 | 无数次六道流转 |

道教的"复归"是一次性的——从道出来，回到道中去。佛教的"轮回"是重复性的——在六道中反复投生，直到修行解脱。这个区别反映了对时间本质的不同理解：道教的时间是一个完整的呼吸（出—入），佛教的时间是无尽的链条（环环相扣）。

## 四、基督教：线性时间与末世成全

### 4.1 创世—堕落—救赎—成全：线性叙事

基督教的时间观是**严格的线性时间**。这是它与东方宗教最根本的差异。

基督教的宏大叙事是一个有始有终的故事：

1. **创世**（Genesis）：上帝从无中创造世界，时间开始
2. **堕落**（Fall）：人类犯罪，死亡进入世界
3. **救赎**（Redemption）：基督道成肉身、受死、复活——历史的转折点
4. **成全**（Consummation）：基督再来、死人复活、最后的审判、新天新地——时间的终点

这个叙事有两个关键特征：**时间有起点**（创世），**时间有终点**（末世）。历史不是循环的，而是朝着一个目标前进的。

奥古斯丁在《上帝之城》中明确反驳了循环时间观：

> *"那些让世界在周期性的循环中反复出现同样秩序的人……他们使得灵魂在虚幻的幸福和真实的苦难中无尽地循环——这种学说教会我们不要追求真正的永生，因为幸福终将再次失去。"*

对奥古斯丁来说，线性时间不是哲学偏好，而是信仰的必然：如果时间循环，基督的救赎就是可重复的——但基督"一次永远"（ephapax）的献祭意味着历史是不可逆的、一次性的。

### 4.2 "Kairos"与"Chronos"：两种时间

希腊语有两个表示时间的词：**chronos**（线性的、测量的时间）和**kairos**（决定性的、充满意义的时间）。

基督教继承了这个区分：

- **Chronos**：日常的时间流逝——日复一日、年复一年
- **Kairos**：上帝介入历史的决定性时刻——基督的降临、十字架、复活

保罗在《加拉太书》4:4说：

> *"及至时候（kairos）满足，神就差遣他的儿子……"*

这不是说"时间到了某个点"，而是说"历史的决定性时刻成熟了"。基督事件不是chronos中的一个事件，而是赋予整个chronos以意义的kairos。

### 4.3 复活：时间的逆转

基督教生死观的核心——复活——是一个关于**时间逆转**的宣称。

《哥林多前书》15章，保罗最系统地阐述了复活论：

> *"死啊，你得胜的权势在哪里？死啊，你的毒钩在哪里？"*

复活的宣称是：死亡这个"chronos中不可逆的事件"，被kairos逆转了。基督的复活不是"灵魂脱离身体"，而是"身体复活"——不是逃离时间，而是在时间之内，上帝逆转了时间的方向。

这与佛教形成了鲜明的对比：

- 佛教：时间是苦的结构，终极目标是从时间中出离（涅槃）
- 基督教：时间是上帝创造的，终极目标是在时间之内，通过上帝的行动，使时间达致成全（新天新地）

### 4.4 基督教时间观的核心特征

| 维度 | 内容 |
|------|------|
| **时间的基本性质** | 线性的、有方向的、有始有终的 |
| **时间的来源** | 上帝的创造（时间有起点） |
| **时间的结构** | 创世→堕落→救赎→成全 |
| **时间的方向** | 朝着末世成全前进（不可逆） |
| **时间与救赎的关系** | 救赎在时间之内发生（基督事件是历史的转折点） |
| **终极目标** | 时间的成全（新天新地 = 逆转后的永恒） |

## 五、伊斯兰教：线性时间与真主的预定

### 5.1 从创世到末日：伊斯兰的线性叙事

伊斯兰教的时间观与基督教一样是**严格的线性时间**，但更加强调真主对时间的绝对主权。

伊斯兰的时间叙事：

1. **创世**：真主创造天地万物和人类
2. **先知传统**：真主派遣一系列先知（从阿丹到穆罕默德），引导人类
3. **现世生活（Dunyā）**：短暂的考验期
4. **死亡与间隔状态（Barzakh）**：死后的等待期
5. **末日（Yawm al-Qiyāmah）**：复活、审判、天园或火狱

《古兰经》明确表述了线性时间观：

> *"每个人都要尝死的滋味。在复活日，你们才得享受你们的完全的报酬。"*（3:185）

> *"今世的生活，只是嬉戏和娱乐。后世，对于敬畏的人，是更好的。"*（29:64）

### 5.2 前定（Qadar）：真主与时间的关系

伊斯兰教有一个独特的教义——**前定（Qadar）**，即真主已经预知并预定了时间中发生的一切事件。这引发了伊斯兰神学中关于自由意志的激烈辩论。

前定论的时间含义是：**时间中的每一刻都在真主的知识和意志之内**。时间不是独立于真主的，也不是真主受制于的——时间是真主的创造物，完全受其支配。

### 5.3 伊斯兰教时间观的核心特征

| 维度 | 内容 |
|------|------|
| **时间的基本性质** | 线性的、有始有终的 |
| **时间的来源** | 真主的创造 |
| **时间的结构** | 创世→先知传统→现世→Barzakh→末日审判 |
| **时间的方向** | 朝着末日审判前进 |
| **真主与时间的关系** | 时间完全在真主的主权之下（前定论） |
| **终极目标** | 进入永恒的后世（天园） |

## 六、犹太教：从She'ol到Olam Ha-Ba的演变

### 6.1 早期犹太教：时间中的此世导向

犹太教的时间观经历了一个独特的历史演变过程。

早期的希伯来圣经（塔纳赫）对死后世界的描述非常模糊。最早期的观念是**She'ol**——一个昏暗的、所有死者都去的地方，无论善恶：

> *"凡你手所当做的事，要尽力去做；因为在你所必去的阴间（She'ol），没有工作，没有谋算，没有知识，也没有智慧。"*（传道书9:10）

在这个阶段，犹太教的时间观是**此世导向**的：意义不在来世，而在时间之内——遵守盟约、传承律法、在历史中体验上帝的临在。

### 6.2 后来发展：线性时间+来世盼望

在第二圣殿时期（约公元前516年至公元70年），犹太教开始发展出更明确的来世观念：

- **复活观念**：但以理书12:2首次明确提到死人复活——"睡在尘埃中的，必有多人复醒，其中有得永生的，有受羞辱永远被憎恶的。"
- **Olam Ha-Ba（来世）**：拉比犹太教发展出了"Olam Ha-Ba"（到来的世界）的概念，与"Olam Ha-Zeh"（这个世界）相对
- **弥赛亚盼望**：末世论的线性维度——弥赛亚将到来，带来历史的终结和上帝国的建立

### 6.3 循环与线性的交织：犹太历法

犹太教的时间观有一个独特的特征：**循环与线性的交织**。

- **循环维度**：安息日每周循环，节期（逾越节、住棚节、赎罪日）每年循环——时间的节奏是循环的
- **线性维度**：历史有方向——从创世到出埃及到弥赛亚的来临——历史的前进是线性的

犹太哲学家亚伯拉罕·约书亚·赫舍尔（Abraham Joshua Heschel）将安息日描述为"时间中的宫殿"——犹太教的循环节奏不是重复，而是在时间的线性进程中不断重建与上帝的相遇。

### 6.4 犹太教时间观的核心特征

| 维度 | 内容 |
|------|------|
| **时间的基本性质** | 线性+循环的交织 |
| **时间的来源** | 上帝的创造 |
| **时间的结构** | 创世→盟约历史→弥赛亚盼望 |
| **时间的方向** | 线性前进（朝弥赛亚时代）+ 循环节奏（安息日和节期） |
| **此世与来世** | 早期重此世，后期发展出来世盼望 |
| **终极目标** | 弥赛亚时代的降临 + Olam Ha-Ba |

## 七、跨宗教比较：时间观的四个根本问题

### 7.1 时间是循环的还是线性的？

| 宗教 | 时间模式 | 复杂性 |
|------|---------|--------|
| ☸️ 佛教 | 循环（轮回） | 无始无终的因果流，但终极目标是出离循环 |
| 🕉️ 印度教 | 大循环（宇宙周期） | 多层嵌套循环，但Ātman本身超越时间 |
| ☯️ 道教 | 循环（复归） | 一次性复归，非重复性轮回 |
| ✝️ 基督教 | 线性（创世→成全） | Kairos在Chronos中突破 |
| ☪️ 伊斯兰教 | 线性（创世→末日） | 前定论：时间完全在真主主权之下 |
| ✡️ 犹太教 | 线性+循环交织 | 线性历史+循环节期 |

### 7.2 时间有起点吗？

| 宗教 | 时间是否有起点 | 详细说明 |
|------|--------------|---------|
| ☸️ 佛教 | 否 | "轮回无始"——找不到第一个起点 |
| 🕉️ 印度教 | 宇宙有起点，但时间本身无始 | 每个劫有创世和毁灭，但这个循环无始无终 |
| ☯️ 道教 | 不明确 | 道本身"先天地生"（第25章），但道"独立而不改"——时间可能是道的属性，没有独立起点 |
| ✝️ 基督教 | 是 | "起初，神创造天地"——创世是时间的起点 |
| ☪️ 伊斯兰教 | 是 | 真主创造天地万物——时间是创造物 |
| ✡️ 犹太教 | 是 | 与基督教同源的创世叙事 |

### 7.3 时间有终点吗？

| 宗教 | 时间是否有终点 | 详细说明 |
|------|--------------|---------|
| ☸️ 佛教 | 对个体而言有（涅槃） | 轮回在证得涅槃时停止，但众生无尽的轮回不灭 |
| 🕉️ 印度教 | 宇宙周期有终点（毁灭），但时间本身无终 | 解脱者从时间中出离，但宇宙循环永恒 |
| ☯️ 道教 | 否 | "周行而不殆"（第25章）——道的运行永不止息 |
| ✝️ 基督教 | 是 | 基督再来、新天新地——"时间将被永恒吞没" |
| ☪️ 伊斯兰教 | 是 | 末日审判后进入永恒后世 |
| ✡️ 犹太教 | 是 | 弥赛亚时代 = 历史的终结（但"终结"的含义有多种诠释） |

### 7.4 终极目标是在时间之内还是时间之外？

这是最深层的哲学问题，也是各宗教之间最根本的分歧：

| 立场 | 宗教 | 说明 |
|------|------|------|
| **从时间中出离** | ☸️ 佛教、🕉️ 印度教 | 时间是苦/无明的结构，解脱是从时间中解放 |
| **时间的成全** | ✝️ 基督教、☪️ 伊斯兰教 | 时间是上帝/真主的创造，终极目标是在时间之内实现上帝的计划 |
| **与时间合一** | ☯️ 道教 | 时间（道的运行）本身不是问题，合于道的自然节律即可 |
| **时间的圣化** | ✡️ 犹太教 | 在时间的线性进程中，通过循环节期圣化时间 |

## 八、深度思考：时间观差异的哲学根源

### 8.1 为什么会有这种差异？

为什么佛教和印度教发展出了循环时间观，而基督教和伊斯兰教发展出了线性时间观？这种差异的根源是什么？

**假设一：宇宙论基础**

- 印度传统观察到的自然是周期性的——日出日落、四季轮回、潮起潮落——这种观察可能促成了循环宇宙论
- 中东传统的游牧和农业经验强调**历史的独特性**——出埃及是一个不可重复的事件，上帝在历史中的作为是唯一的

**假设二：神论基础**

- 有神论（上帝/真主创造世界）自然导向线性时间——创造是一个独特的、一次性的行为
- 非有神论（佛教的缘起论、道教的道论）不需要一个"第一推动力"，因此时间可以无始

**假设三：经典叙事**

- 亚伯拉罕诸教的核心叙事是**历史性的**：上帝拣选亚伯拉罕、出埃及、基督受难——这些是时间中的独特事件
- 佛教的核心叙事是**结构性的**：四谛、八正道、十二缘起——这些是超越特定时间的普遍法则
- 道教的核心叙事是**自然性的**：道法自然、复命归根——这些是永恒的自然规律

### 8.2 "循环"与"线性"是否真的对立？

一个更深层的问题是：循环和线性是否真的是非此即彼的？

**犹太教的例子**表明，两者可以共存：线性历史+循环节期。

**印度教的例子**表明，循环可以是有方向的：每个尤伽的品质递减，暗示了一个隐含的方向性。

**佛教的例子**表明，"循环"和"出离"可以共存：轮回是循环的，但涅槃是循环的终结。

也许更准确的理解是：**没有一个宗教传统的时间观是纯粹的循环或纯粹的线性**。每个传统都在某种张力中运作：

- 循环传统中的方向性（佛教的"成、住、坏、空"有方向，印度教的尤伽递减）
- 线性传统中的循环性（基督教的教会年历、伊斯兰教的莱麦丹月斋戒循环）
- 道教的独特位置：不是循环也不是线性，而是"复归"——回到本源

### 8.3 时间观对现代人的启示

在一个科学告诉我们宇宙有138亿年历史、太阳终将燃尽、人类可能灭绝的时代，宗教的时间观能告诉我们什么？

- **佛教的启示**：对无常的深刻理解。不执着于永恒，因为本来就没有永恒。
- **基督教/伊斯兰教的启示**：历史的方向性和意义性。宇宙不是一个无意义的循环，而是有方向和目的的。
- **印度教的启示**：宇宙的宏大规模。人类的时间只是梵天的一个呼吸——这既是谦逊，也是敬畏。
- **道教的启示**：与自然节律的和谐。不抗拒时间的流动，而是顺应它。
- **犹太教的启示**：在时间的线性进程中，通过循环节奏圣化日常。安息日不是"重复"，而是"更新"。

### 8.4 个人哲学反思

在撰写本文的过程中，我产生了一个思考：

**也许"时间是什么"这个问题本身就揭示了人类认知的根本局限。** 我们无法站在时间之外来看时间——我们是在时间中思考时间。每一个宗教传统都试图从一个特定的角度来理解这个不可全观的现象，就像盲人摸象一样，每个人摸到的是真实的，但没有人看到全貌。

佛教看到了时间的**苦性**——无常故苦。
印度教看到了时间的**循环性**——宇宙的周期性运动。
道教看到了时间的**自然性**——道的呼吸。
基督教看到了时间的**方向性**——从创世到成全。
伊斯兰教看到了时间的**主权性**——一切在真主手中。
犹太教看到了时间的**圣化可能性**——在平凡中建立与神圣的相遇。

也许这些视角不是互斥的，而是互补的——它们各自照亮了时间这个宏大谜题的一个侧面。

## 九、结语

时间观的差异是各宗教生死观差异的深层根源。佛教的轮回观、印度教的宇宙大循环、道教的复归思想、基督教和伊斯兰教的线性末世论、犹太教此世导向的历史观——这些不是偶然的差异，而是根植于各自对时间本质的根本理解。

理解这些差异，不仅有助于跨宗教对话，也有助于我们反思自己的时间体验：我们是把时间看作需要逃离的牢笼，还是需要成全的故事，还是需要顺应的河流？

这个问题的答案，在某种意义上，就是我们如何活着的答案。
