**产出**：`🔍专题研究/生死观/共通主题分析.md`（约9,000字）

### 核心发现

从已有的三个深度比较研究（时间观、灵魂观、终极目标）中提炼出六大共通主题：

1. **死亡不是终结** — 所有宗教都拒绝"死亡是绝对的终点"，即使是否定灵魂的佛教也坚持识的相续
2. **此生有永恒分量** — 没有宗教说"死后还有机会，此生无所谓"，反而都加重了此生的道德分量
3. **痛苦需要回应** — 所有宗教都不接受"痛苦是无意义的"，发展出超越策略和战胜策略两种回应
4. **更高维度存在** — 日常感官经验不是全部实在，死亡是通往更高维度的门槛
5. **路径是必要的** — 从迷到悟需要具体的修行/信仰实践，不是仅凭理解就够
6. **个人与宇宙交织** — 死亡不是纯粹的私人事件，个人生死嵌入更大的宇宙秩序

### 关键洞察

- 共通主题不是"殊途同归"的证据，而是人类意识面对有限性时的"建筑必然性"
- 六大主题对应六个普遍人类心理特征：意识无法想象自身不存在、有限性赋予选择意义、意义需求是基本需求、超越性体验的普遍性、自我改善的驱力、归属需求
- 共通主题划出了人类精神生活的"必然领地"，即使不信教也需要面对这六个领域
