## 📋 归档规范

### 归档条件
- 任务清单中**所有条目**均已完成（全部打 ✅）
- 已在「任务总览.md」中将状态更新为 ✅ 已完成

### 归档流程
1. 确认任务清单所有条目完成
2. 将文件从 `📋任务/` 移入 `📋任务/归档/`
3. 在「任务总览.md」中更新状态为 📦 已归档，并记录归档日期

### 文件保持原名
- 归档时不修改文件名，保持 `TASK-XXX-标题.md` 格式
- 通过任务总览中的归档日期追踪时间线
