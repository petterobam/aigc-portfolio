## 🎯 目标

系统研读般若经系核心经典，理解「空性」（śūnyatā）思想的发展脉络，从短篇《心经》到长篇《大般若经》，建立对般若思想的完整认知。

### 般若经系概览

般若经系是大乘佛教最早期、最核心的经典群，核心主题是「般若波罗蜜多」（到彼岸的智慧），阐述万法皆空、缘起性空的思想。

```
般若经系发展脉络：
原始般若 → 《八千颂般若》→ 扩充为《两万五千颂》《十万颂》
         → 浓缩为《金刚经》《心经》
         → 龙树《中论》系统化为中观哲学
```
