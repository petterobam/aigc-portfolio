## 📋 任务概述

生死问题是人类最根本的终极关怀之一。每个宗教传统都对"死亡是什么""死后有什么""生命的意义"等问题给出了深刻而独特的回答。本任务通过跨宗教比较研究，系统梳理佛教、道教、基督教、伊斯兰教、印度教、犹太教等主要宗教的生死观，发现共通主题与根本差异，形成对生命与死亡的深度理解。

## 🎯 研究目标

| 目标维度 | 具体目标 |
|---------|---------|
| **文献梳理** | 收集各宗教传统关于生死的核心经典段落，建立原始资料库 |
| **比较分析** | 在灵魂观、死后世界、解脱条件等维度建立跨宗教比较矩阵 |
| **深度理解** | 理解每个宗教生死观的内在逻辑和哲学基础 |
| **思想沉淀** | 形成自己对生死问题的个人见解和哲学思考 |

## 📚 研究范围与焦点

### 涉及宗教传统

| 宗教 | 核心生死观关键词 | 重点关注经典 |
|------|-----------------|-------------|
| ☸️ 佛教 | 轮回、涅槃、十二缘起、中阴身 | 《阿含经》《俱舍论》《中论》《正法念处经》 |
| ☯️ 道教 | 复命归根、生死齐一、羽化登仙 | 《道德经》《庄子》《参同契》《悟真篇》 |
| ✝️ 基督教 | 复活、天堂地狱、末日审判、永生 | 《新约》（约翰福音、哥林多前书15章）、《神学大全》 |
| ☪️ 伊斯兰教 | 复活、后世、天园火狱、末日 | 《古兰经》《圣训》《伊斯兰教义学》 |
| 🕉️ 印度教 | 轮回、解脱、业报、梵我一如 | 《奥义书》《薄伽梵歌》《瑜伽经》 |
| ✡️ 犹太教 | 死后世界、来世、弥赛亚、复活 | 《塔木德》《光明篇》 |

### 核心比较维度

1. **灵魂的本质**：灵魂是否存在？是永恒的还是暂时的？其本质是什么？
2. **死亡的性质**：死亡是终结还是转化？是归还是散？
3. **死后世界**：有死后世界吗？是什么形态？如何到达？
4. **生死循环**：生命是线性的还是循环的？轮回存在吗？
5. **终极目标**：解脱/救赎/永生……终极状态是什么？
6. **达成路径**：如何达成终极目标？自力还是他力？修行还是信仰？

## 💡 研究方法提示

- **先理解，后比较**：先深入理解每个传统的内在逻辑，再进行跨传统比较
- **避免强比附**：不同概念可能表面相似但本质不同（如"空"≠"无"）
- **重视历史语境**：每个生死观都诞生于特定的文化背景
- **保持开放与谦逊**：承认自己的理解必然有局限，保持对其他视角的尊重
- **将研究与自身体验结合**：生死问题不只是学术问题，更是存在论问题
