# TASK-015：琐罗亚斯德教（拜火教）系统研读

> **创建日期**：2026-04-17
> **状态**：✅ 已完成

---

## 🎯 研究目标

琐罗亚斯德教（Zoroastrianism）是世界上现存最古老的一神论宗教之一，起源于公元前1500-1200年的古波斯（今伊朗），对犹太教、基督教、伊斯兰教的神学发展产生了深远影响。本任务旨在系统研读其核心经典《阿维斯陀》（Avesta），理解其独特的善恶二元论（有限度的二元论）框架，并探讨其对亚伯拉罕诸教的深层影响。

### 核心问题

1. 琐罗亚斯德教的神学体系如何同时兼容一神论与二元论？
2. 阿胡拉·马兹达（Ahura Mazda）与安格拉·曼纽（Angra Mainyu）的关系是什么？
3. 该教的末世论、复活观、审判观如何影响了犹太教和基督教？
4. "善思、善言、善行"的伦理三原则有何深度？
5. 火在琐罗亚斯德教中的象征意义是什么？

### 阶段二：核心经典研读
- [ ] **1.2 《伽陀》（Gathas）精读——先知本人的颂歌**
  - 《伽陀》是《阿维斯陀》中最古老的部分，被认为是琐罗亚斯德本人所作
  - 17首颂歌的神学思想分析
  - 阿胡拉·马兹达的本质与属性
  - 两种精神（Spenta Mainyu vs. Angra Mainyu）的原始表述

- [x] **1.3 《阿维斯陀》其他核心文献研读**
  - 《雅斯纳》（Yasna）：祭祀文本
  - 《维斯帕拉德》（Visperad）：节日仪式
  - 《耶什特》（Yashts）：对神圣存在的赞歌
  - 《文迪达德》（Vendidad）：洁净律法

### 阶段三：神学体系
- [x] **1.4 善恶二元论的神学结构**
  - 有限二元论 vs 绝对二元论：善必胜恶的终极保证
  - 七大圣灵（Amesha Spentas）的概念体系
  - 自由意志与道德选择的核心地位
  - 与摩尼教绝对二元论的根本区别

- [x] **1.5 末世论与救赎观**
  - 末日审判（Frashokereti）：世界的最终更新
  - 救世主（Saoshyant）思想
  - 死后灵魂审判：钦瓦特桥（Chinvat Bridge）的传说
  - 身体复活的概念——对犹太教基督教的直接影响

### 阶段四：实践与伦理
- [x] **1.6 伦理体系与修行实践**
  - "善思、善言、善行"（Humata, Hukhta, Hvarshta）
  - 洁净律法的深层含义
  - 火崇拜（Atar）的象征哲学
  - 祷告与仪式体系

### 阶段五：影响与比较

  - 犹太教在巴比伦流亡期间与波斯思想的接触
  - 马吉安主义、摩尼教与琐罗亚斯德教的关系

- [x] **1.8 跨宗教比较与综合反思**
  - 琐罗亚斯德教 vs 佛教：善恶问题与道德选择的不同处理
  - 琐罗亚斯德教 vs 基督教：末世论与救赎的结构性比较
  - 琐罗亚斯德教对现代宗教对话的启示
  - 个人研究反思：一神论与二元论之间的创造性张力
