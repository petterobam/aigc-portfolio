#### 五、4大使用场景

**场景1：我要批量发布已完成的故事**
- 调用发布调度器 → 自动选择最佳发布时间 → 自动执行登录、填充、发布、监控 → 生成发布报告
- 预期耗时：5-10分钟（准备）+ 自动发布（后台运行）

**场景2：我要监控审核状态和收集数据**
- 启动审核监控器 → 定时查询审核状态 → 审核通过自动通知 → 定时采集发布数据 → 生成数据分析报告
- 预期耗时：1分钟（启动）+ 自动监控（后台运行）

**场景3：我要优化0阅读作品的标题**
- 调用数据分析器，分析0阅读作品 → 基于爆款标题公式生成新标题 → 修改作品标题 → 监控标题修改后的数据变化 → 生成标题优化报告
- 预期耗时：5-10分钟（分析）+ 自动优化（后台运行）

**场景4：我要制定下周发布计划**
- 调用数据分析器，分析历史数据 → 基于最佳发布时间生成发布计划 → 基于发布日历安排发布节奏 → 自动调度下周发布任务
- 预期耗时：5分钟（分析）+ 自动调度（后台运行）
