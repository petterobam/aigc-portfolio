### 发现1: self-improvement-agent 技能是持续改进的解决方案 ⭐⭐⭐⭐⭐

**核心功能**:
1. **日志记录**：记录学习、错误和修正，实现持续改进
2. **自动提升**：编码代理可以处理这些学习日志为修复
3. **知识提升**：重要学习提升到项目记忆
4. **模式检测**：检测重复模式，简化和加固流程

**使用场景**:
- 命令或操作意外失败
- 用户纠正 Claude（"No, that's wrong..."）
- 用户请求不存在的功能
- 外部 API 或工具失败
- Claude 发现知识已过时或不正确
- 为重复任务发现更好的方法

### 发现2: 学习日志系统有完整的工作流 ⭐⭐⭐⭐⭐

**工作流**:
1. **记录学习**：当发现非显而易见的解决方案时，记录到 `.learnings/LEARNINGS.md`
2. **记录错误**：当命令失败或出现异常时，记录到 `.learnings/ERRORS.md`
3. **记录功能请求**：当用户请求功能时，记录到 `.learnings/FEATURE_REQUESTS.md`
4. **定期审查**：每周审查 `.learnings/` 目录
5. **解决已修复条目**：更新状态为 resolved
6. **提升适用学习**：将学习提升到项目记忆

**提升目标**:
- CLAUDE.md（项目事实和约定）
- AGENTS.md（工作流和自动化）
- `.github/copilot-instructions.md`（项目上下文和约定）
- SOUL.md / TOOLS.md（OpenClaw 工作区级指导）
