### 核心知识点

#### 1. LLM 训练流程（两大阶段）

**预训练（Pre-Training）- 基础构建**
- 数据：大规模无标注文本（网页、书籍、代码等），TB级数据量
- 范式：自监督学习
  - 掩码语言模型（BERT）：预测被遮盖的词
  - 因果语言模型（GPT）：预测下一个词
- 目标：语言基础能力
  - 文本向量表示
  - 语义语法理解
  - 基础事实知识

**后训练（Post-Training）- 能力迭代**
分为三个子阶段：

**第一阶段：基础能力强化（SFT/蒸馏/拒绝采样）**
- 数据：对话、问答等结构化任务数据
- 范式：监督微调、知识蒸馏、拒绝采样
- 目标：对话能力、指令跟随、工具调用

**第二阶段：偏好对齐（DPO/RLHF）**
- 数据：人类偏好数据集（对回答好坏的判断）
- 范式：DPO（直接偏好优化）、RLHF（基于人类反馈的强化学习）
- 目标：有用性、价值观对齐、安全性

**第三阶段：深度思考能力训练**
- 数据：Prompt提示词
- 范式：GRPO、CoT（思维链）、PRM（过程奖励模型）、Self-Play
- 目标：复杂问题推理、多步决策

#### 2. 前向与反向传播

**前向传播（Forward Propagation）**
- 数据从输入层逐层传递
- 每层进行：线性变换 → 激活函数 → 生成激活值
- 最终计算损失：Loss = L(预测值, 真实标签)
- 关键：缓存每层的激活值（反向传播需要）

**反向传播（Backward Propagation）**
- 从损失开始，利用链式法则反向计算梯度
- 梯度流动：Loss → Gradient 4 → Gradient 3 → Gradient 2 → Gradient 1
- 目的：计算各层权重和偏置的梯度
- 最终：通过优化算法更新参数
  - W_new = W_old - learning_rate · ∂Loss/∂W
  - b_new = b_old - learning_rate · ∂Loss/∂b

**为什么需要激活值缓存？**
- 反向传播用链式法则计算梯度
- 激活函数的导数需要正向传播时的激活值
- 前向传播必须缓存每层的激活值

#### 3. SFT 的交叉熵损失（Cross-Entropy Loss）

**基础公式**
```
L_CE = -Σ y_i · log(ŷ_i)
```
- y_i: 真实标签（one-hot，正确token为1）
- ŷ_i: 模型预测概率
- i: token索引

**SFT 序列损失**
```
L_SFT = -(1/N) · Σ_j Σ_i y_ji · log(ŷ_ji)
```
- N: 样本数
- j: 样本索引
- i: token索引

**直观含义**
- 模型预测正确token的概率越高，损失越小
- ŷ_correct = 1 → Loss = 0（最优）
- ŷ_correct = 0.5 → Loss ≈ 0.693
- ŷ_correct = 0.01 → Loss = 4.605（极差）

**在 LLM 训练中的作用**
1. 定义"正确性"标准：告诉模型生成与参考答案一致的token是正确的
2. 引导参数更新：通过梯度下降调整参数，提高正确token的概率
3. 优化生成能力：让模型理解prompt意图，生成连贯准确的response

**示例分析**
Prompt: "虎皮青椒用什么做？"
Response: "青椒，大蒜。"

Loss计算（6个token平均）：
- "青": P=0.0009 → log(0.0009) ≈ -7 → loss≈7（差）
- "椒": P=0.99 → log(0.99) ≈ -0.01 → loss≈0.01（好）
- "," : P=0.03 → log(0.03) ≈ -3.5 → loss≈3.5
- "大": P=0.0003 → log(0.0003) ≈ -8 → loss≈8（差）
- "蒜": P=0.014 → log(0.014) ≈ -4.2 → loss≈4.2
- "。" : P=0.327 → log(0.327) ≈ -1.1 → loss≈1.1

平均Loss ≈ 3.99

### 预训练 vs 后训练对比表

| 维度 | 预训练 | 后训练 |
|------|--------|--------|
| 数据类型 | 大规模无标注文本 | 结构化任务数据、偏好数据 |
| 训练范式 | 自监督学习 | 监督微调、强化学习 |
| 核心目标 | 学习语言基础 | 提升高级能力（对话、指令跟随、价值观对齐） |
| 模型能力 | 理解语言，生成通顺文本 | 对话流畅、遵循指令、安全可控、深度推理 |

### 关键洞察

1. **训练是分阶段的**：先学基础（预训练），再学高级能力（后训练）
2. **前向+反向是训练核心**：前向预测，反向优化
3. **交叉熵是监督信号**：量化模型生成与参考答案的差异
4. **多阶段后训练迭代**：从"能说话"到"说得好"到"说对事"
