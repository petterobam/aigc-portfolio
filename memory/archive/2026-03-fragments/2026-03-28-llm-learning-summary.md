### 2. 深入学习重复内容检测技术

#### 2.1 重复段落检测脚本（check-story-duplicates.js）

**核心功能**：
1. 读取完整故事文件
2. 只检查正文内容，过滤标题、元数据和检查清单
3. 分析是否存在重复段落
4. 统计重复段落的数量和位置
5. 生成详细的检查报告

**关键技术**：
- **文本过滤**：过滤标题（以"#"开头）、元数据段落、检查清单
- **段落提取**：将相邻的非空行合并为一个段落
- **重复检测**：
  - 完全重复：使用段落内容作为 key，统计出现次数
  - 高度相似：使用编辑距离计算相似度
- **报告生成**：Markdown 格式，包含摘要、详情、优化建议

**检测标准（v2.0优化版）**：
- **过滤规则**：
  - 排除短段落（< 30字符）
  - 排除对话（包含引号）
  - 排除心声和心理描写（含"心想"、"心里"等关键词）
  - 排除纯动作描写（如"他点了点头。"）
  - 只检查叙事性段落
- **相似度阈值**：0.95（更严格，减少误判）

#### 2.2 编辑距离算法（Levenshtein Distance）

**原理**：
- 计算两个字符串之间的最小编辑距离
- 编辑操作包括：插入、删除、替换
- 编辑距离越小，两个字符串越相似

**应用**：
- 计算段落之间的相似度
- 识别高度相似的重复段落
- 设置相似度阈值（0.95 或 0.80）

**相似度计算公式**：
```
相似度 = 1 - (编辑距离 / max(len1, len2))
```

### 3. 重复内容产生的原因分析

#### 3.1 创作流程不完整 ⭐⭐⭐⭐⭐

**证据**：
- 只创建了创作模板（v0.1）
- 没有执行创作任务（任务4-7）
- 文件状态仍为"待创作"

**原因**：
- 可能是 AI 只创建了模板，但没有执行创作
- 可能是创作过程中断，没有完成
- 可能是质量检查流程缺失

**教训**：
- 创作流程必须有验证机制
- 不能只创建模板，必须确保有实际内容
- 需要检查"待创作"状态

#### 3.2 质量检查机制缺失 ⭐⭐⭐⭐⭐

**证据**：
- 没有检查文件内容是否为模板
- 没有检查章节是否真正创作完成
- 没有检查文件末尾的"状态：待创作"

**原因**：
- 质量检查脚本没有检查"待创作"状态
- 质量检查脚本没有识别模板内容
- 缺少创作完成验证机制

**教训**：
- 质量检查必须包含"完成度"检查
- 必须检查文件状态（待创作 → 已完成）
- 必须识别模板内容

#### 3.3 重复段落检测机制不完善 ⭐⭐⭐⭐

**证据**：
- 重复段落检测脚本没有检测到所有章节完全相同
- 可能是脚本只检测了段落内容的相似度，但没有检测文件内容的重复

**原因**：
- 需要改进重复段落检测逻辑
- 需要添加文件级重复检测
- 需要添加模板内容识别

**教训**：
- 重复检测必须多层次（文件级、段落级）
- 必须识别模板内容
- 必须检测文件之间的相似度

### 4. 优化写作技巧

#### 4.1 从源头避免重复

**技巧1：多样化动作描写**

**问题**：
```
我握紧拳头,摇头。
我握紧拳头,点头。
我握紧拳头,感到一阵悲伤。
我握紧拳头,感到一阵痛苦。
```

**优化**：
```
我紧握双拳,眉头紧锁,轻轻摇头。
我点了点头,深吸一口气,努力平复心情。
一股悲伤涌上心头,我感到眼眶有些湿润。
痛苦如潮水般袭来,我几乎要支撑不住。
```

**原则**：
- 使用不同的动词（握紧 → 紧握、攥紧、抓紧）
- 添加细节描写（眉头紧锁、深吸一口气、眼眶湿润）
- 使用比喻（痛苦如潮水般袭来）

---

**技巧2：避免短句重复**

**问题**：
```
妹妹笑了。（出现多次）
但我不想抛下她,我不想一个人离开。（出现多次）
```

**优化**：
```
妹妹笑了,眼中闪烁着喜悦的光芒。
我无法抛下她,更无法独自逃离这个噩梦。
她的笑容温暖如春阳,让我心中涌起一股暖流。
我下定决心,绝不离开她,绝不独自求生。
```

**原则**：
- 扩展短句，添加细节
- 使用不同的表达方式
- 改变句式结构（简单句 → 复杂句）

---

**技巧3：避免情节重复**

**问题**：
```
编剧可以改变剧本,可以决定结局...
但如果你做编剧,你可以改变剧本,可以决定结局...
```

**优化**：
```
作为编剧,我拥有改变剧本、决定结局的权力。这是一份沉甸甸的责任。
此刻我才明白,编剧不仅是创作者,更是命运的掌控者。我的每一个决定,都影响着故事的走向。
```

**原则**：
- 合并相似段落
- 深化情节表达
- 增加情感层次
