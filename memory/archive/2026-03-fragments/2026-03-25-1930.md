- constitution → specify → clarify → plan → tasks → write → analyze
- 系统化的创作流程
- 每个步骤都有明确的输入和输出

- 随时可升级到更详细层级

#### 黄金开篇法则

- 核心冲突前置
- 避免信息轰炸
- 限制出场人数
- 快速展现金手指
