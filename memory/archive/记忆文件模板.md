## 💡 使用说明

### 1. 选择合适的模板

根据记录类型选择合适的模板:
- 日常记录 → 日常记录模板
- 心跳记录 → 心跳记录模板
- 总结报告 → 总结报告模板
- 特定主题 → 特定主题模板

### 2. 替换占位符

替换模板中的占位符:
- `YYYY-MM-DD` → 日期
- `HH:MM` → 时间
- `X` → 具体数值
- `主题名称` → 具体主题

### 3. 删除不需要的部分

根据实际情况删除不需要的部分:
- 没有待办事项 → 删除待办事项部分
- 没有关键发现 → 删除关键发现部分
- 没有经验总结 → 删除经验总结部分

### 4. 添加具体内容

添加具体的内容:
- 完成的工作 → 具体描述
- 关键发现 → 具体发现
- 核心成果 → 具体成果

### 5. 更新相关文档

更新相关文档的引用:
- 工作日志
- 心跳历史
- 记忆文件
