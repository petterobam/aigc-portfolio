**核心要点**：

1. **大模型算法总体架构**
   - 主要由三个核心部分组成：输入层、模型层、输出层
   - 输入层负责将文本转换为数值矩阵（embedding）
   - 模型层通过多层 Transformer 进行特征提取和语义理解
   - 输出层将隐藏状态转换为概率分布，用于生成下一个 token

2. **LLM 结构总图**
   - Tokenizer → Embedding → Transformer Blocks → LM Head
   - 每个层级都有明确的数据流动方向
   - 支持从输入到输出的端到端训练

3. **Decoder-Only vs MoE 架构**
   - **Decoder-Only**：传统 GPT 系列，所有层都共享参数，计算均匀
   - **MoE（Mixture of Experts）**：混合专家模型，每个 token 只激活部分专家
     - 优点：参数量大但计算量可控，性能更好
     - 挑战：路由机制设计、负载均衡
   - 代表模型：DeepSeek-V3（MoE）、Qwen 系列（Decoder-Only）
