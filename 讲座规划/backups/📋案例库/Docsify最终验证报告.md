## 🔍 路径处理机制

### Docsify路由

**Hash模式**:
```
浏览器URL: http://localhost:3000/#/01_编剧部门/剧情结构/案例
实际文件: 01_编剧部门/剧情结构/案例.md
```

**相对路径**:
- 所有路径相对于案例库根目录
- Markdown中的相对路径自动解析
- SVG流程图路径自动处理

### TOC目录

**自动生成逻辑**:
1. 扫描内容区域
2. 提取H2-H4标题
3. 生成层级列表
4. 创建锚点链接
5. 绑定滚动监听

## 🐛 故障排除

### 问题1：侧边栏不显示

**检查**:
```bash
# 1. 检查sidebar.md是否存在
ls -la sidebar.md

# 2. 检查文件内容
head -20 sidebar.md

# 3. 检查HTTP访问
curl -I http://localhost:3000/sidebar.md
```

### 问题2：TOC不显示

**原因**: 文章没有H2-H4标题

**解决**:
- 确认案例文件有标题
- 标题格式：`## 标题` 或 `### 标题`

### 问题3：案例404错误

**检查**:
```bash
# 1. 检查案例文件是否存在
ls -la "01_编剧部门/剧情结构/02_节奏点识别和优化.md"

# 2. 测试HTTP访问
curl -I "http://localhost:3000/01_编剧部门/剧情结构/02_节奏点识别和优化.md"
```

3. 刷新浏览器即可看到

### 修改主题

编辑`index.html`中的样式：
```html
<style>
  :root {
    --theme-color: #667eea;      /* 主题色 */
    --theme-color-dark: #5568d3; /* 深色 */
    --sidebar-width: 280px;      /* 侧边栏宽度 */
    --toc-width: 250px;          /* TOC宽度 */
  }
</style>
```
