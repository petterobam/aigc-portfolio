# AI会议助手工作流程

## 概述
AI会议助手是专门为所有部门设计的通用会议管理工具，可以智能化地处理会议的整个过程，从筹备到执行再到后续跟进。

## 适用场景
- **所有部门**: 日常会议、项目会议、决策会议、培训会议

## 工作流程

### 第一步：会议筹备
```
智能筹备：
├─ 会议类型（例会/项目会/决策会/培训会）
├─ 参会人员（部门/职位/角色）
├─ 时间安排（时间冲突检测/最优时间）
├─ 地点安排（会议室预订/线上会议平台）
└─ 议程制定（议题设置/时间分配）
```

### 第二步：会前准备
```
智能准备：
├─ 资料收集（相关文档/数据报告）
├─ 预会通知（自动通知/提醒设置）
├─ 议题确认（议题审核/议程确认）
├─ 材料准备（PPT制作/资料整理）
└─ 技术测试（设备测试/网络检查）
```

### 第三步：会议执行
```
智能执行：
├─ 会议主持（流程控制/时间管理）
├─ 内容记录（实时记录/重点标注）
├─ 决策跟踪（决策记录/行动项分配）
├─ 问题识别（问题发现/风险预警）
└─ 气氛调节（互动引导/参与度提升）
```

### 第四步：会后管理
```
智能管理：
├─ 会议纪要（自动生成/智能整理）
├─ 行动项跟踪（任务分配/进度监控）
├─ 效果评估（会议效果/评估分析）
├─ 资料归档（文档整理/知识沉淀）
└─ 反馈收集（参会反馈/改进建议）
```

### 第五步：持续优化
```
智能优化：
├─ 效果分析（会议效率/质量分析）
├─ 问题诊断（问题识别/原因分析）
├─ 改进建议（优化方案/实施路径）
├─ 流程优化（流程改进/效率提升）
└─ 标准建立（会议标准/最佳实践）
```
