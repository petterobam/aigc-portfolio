## 🎨 主题定制

### 修改主题色

编辑 `index.html`:

```html
<style>
  :root {
    --theme-color: #667eea;      /* 主题色 */
    --theme-color-dark: #5568d3; /* 深色 */
  }
</style>
```

### 修改侧边栏宽度

```css
:root {
  --sidebar-width: 280px;  /* 侧边栏宽度 */
  --toc-width: 250px;      /* TOC宽度 */
}
```

## 🔍 故障排除

### 问题1：侧边栏无法展开

**解决**：
- 刷新页面（F5）
- 清除浏览器缓存（Ctrl+Shift+Delete）
- 检查控制台错误（F12）

### 问题2：TOC不显示

**原因**：文章没有H2-H4标题

**解决**：
- 确认案例文件有标题
- 标题格式：`## 标题` 或 `### 标题`

### 问题3：案例无法加载

**检查**：
```bash
# 检查案例文件是否存在
ls -la "01_编剧部门/剧情结构/02_节奏点识别和优化.md"

# 检查文件权限
chmod 644 "01_编剧部门/剧情结构/02_节奏点识别和优化.md"
```

### 问题4：SVG无法显示

**原因**：路径错误或文件不存在

**解决**：
```bash
# 检查SVG文件
ls -la "01_编剧部门/images/topic-creative-generation-workflow.svg"

# 测试Web访问
curl -I "http://localhost:3000/01_编剧部门/images/topic-creative-generation-workflow.svg"
```
