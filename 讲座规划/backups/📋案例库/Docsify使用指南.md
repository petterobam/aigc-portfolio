## 🔍 故障排除

### 问题1: 页面空白

**原因**: Docsify未正确加载

**解决**:
1. 检查浏览器控制台错误（F12）
2. 确认网络可访问CDN资源
3. 刷新页面（Ctrl+Shift+R）

### 问题2: SVG无法显示

**原因**: 浏览器CORS限制

**解决**:
1. 使用Firefox浏览器
2. 或禁用浏览器安全策略
3. 或使用本地服务器

### 问题3: 侧边栏不显示

**原因**: sidebar.md加载失败

**解决**:
```bash
# 检查sidebar.md是否存在
ls -la sidebar.md

# 检查文件内容
head -20 sidebar.md
```

### 问题4: 搜索不工作

**原因**: 搜索索引未建立

**解决**:
- Docsify会自动建立搜索索引
- 等待几秒钟让索引完成
- 刷新页面

## 💡 高级功能

### 1. 自定义主题

编辑`index.html`中的样式：

```html
<style>
  :root {
    --theme-color: #667eea;  /* 主题色 */
    --theme-color-dark: #5568d3;  /* 深色 */
  }
</style>
```

### 2. 添加新案例

1. 在对应部门文件夹创建`.md`文件
2. 更新`sidebar.md`添加链接
3. 刷新浏览器

### 3. 导航到特定案例

直接访问Hash路由：
```
http://localhost:3000/#/01_编剧部门/创意激发/02_题材创意生成
```
