```javascript
markdown = markdown.replace(/!\[(.*?)\]\((.*?)\.svg\)/g, (match, alt, svgPath) => {
    const caseDir = casePath.substring(0, casePath.lastIndexOf('/'));
    const fullPath = `${caseDir}/${svgPath}.svg`;
    return `![${alt}](${fullPath})`;
});
```

### SVG路径处理方案

**问题**:
- 案例文档使用相对路径引用SVG
- Web访问需要正确的URL路径

**解决方案**:
1. 解析Markdown中的SVG引用
2. 获取案例文件所在目录
3. 拼接完整路径
4. 替换为正确的URL

**代码实现**:
```javascript
// 正则匹配SVG引用
/!\[(.*?)\]\((.*?)\.svg\)/g

// 路径转换
const caseDir = casePath.substring(0, casePath.lastIndexOf('/'));
const fullPath = `${caseDir}/${svgPath}.svg`;
```
