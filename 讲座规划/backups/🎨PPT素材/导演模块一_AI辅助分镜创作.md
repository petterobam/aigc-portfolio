## 第三部分：分镜提示词设计（20分钟）

---

### 分镜提示词的核心要素

**1. 景别**
- 远景（Wide shot）
- 中景（Medium shot）
- 近景（Close-up）
- 特写（Extreme close-up）

**2. 角度**
- 平视（Eye level）
- 仰视（Low angle）
- 俯视（High angle）
- 侧视（Side angle）

**3. 构图**
- 三分法（Rule of thirds）
- 对角线（Diagonal）
- 黄金螺旋（Golden spiral）
- 中心构图（Centered）

**4. 场景**
- 室内/室外
- 时间（白天/夜晚/黄昏）
- 环境（办公室/家里/街道）

**5. 人物动作**
- 姿态、表情、动作

**6. 光影**
- 自然光/室内光
- 侧光/逆光/顺光
- 温暖/冷峻

**7. 风格**
- 写实/电影感
- 古风/现代
- 艺术风格

---

### 分镜提示词公式

**通用公式**

```
[景别] + [角度] + [构图] + [场景] + [人物动作] + [光影] + [风格]
```

**示例1：室内对话戏（中景）**

```
中景 + 平视 + 三分法 + 办公室，白天 + 男女主角坐在桌子两端交谈 + 柔和室内光 + 电影感
```

**示例2：室外追逐戏（远景）**

```
远景 + 俯视 + 对角线 + 城市街道，黄昏 + 男主角在街道上奔跑，身后有人追赶 + 侧光，阴影长 + 动感
```

**示例3：情感特写（特写）**

```
特写 + 平视 + 中心构图 + 室内，夜晚 + 女主角流泪，眼神悲伤 + 逆光，轮廓光 + 写实
```

### 如何控制风格一致性

**1. 使用固定 seed**

- 在生成时指定 seed 值
- 保持人物外观、场景风格一致

**2. 使用参考图像**

- 上传参考图像作为风格基准
- 确保风格统一

**3. 风格关键词**

- 在每个提示词中加入风格关键词
- 如"电影感"、"写实"、"古风"

**4. OpenClaw 批量生成**

- 使用相同的提示词模板
- 只改变必要的参数（景别、角度、构图）

- 使用固定 seed

**2. 场景一致性**

**4. 提示词模板**

- 使用统一的提示词格式

**2. 风格迁移**

**3. 构图调整**

- 调整人物位置
- 优化构图平衡

**4. 批量优化**

- 使用 OpenClaw 批量处理
- 应用相同的优化方案
- 提高效率

### 批量生成技巧

**使用 OpenClaw 批量处理多个镜头**

```bash
# 创建镜头列表
shots=(
  "shot_1:中景+平视+三分法+咖啡馆，下午+男女主角坐在一起+温暖柔和光+电影感"
  "shot_2:近景+平视+三分法+咖啡馆，下午+女主角表情悲伤+温暖柔和光+电影感"
  "shot_3:特写+平视+中心构图+咖啡馆，下午+女主角流泪+温暖柔和光，侧光+电影感"
  # ... 更多镜头
)

# 批量生成
for shot_info in "${shots[@]}"; do
  shot_id=$(echo $shot_info | cut -d: -f1)
  prompt=$(echo $shot_info | cut -d: -f2)
  echo "生成镜头 $shot_id"
  openclaw message "请生成分镜：$prompt" \
    --model zai/glm-4.7 \
    > "shots/${shot_id}.md"
done
```

**优势**：一键生成多个镜头，提高效率

**要求**：
1. 选择一段完整的剧本（建议3-5分钟的内容）
2. 分析剧本，确定关键镜头
3. 设计8-10个分镜（包括不同景别、角度、构图）
4. 使用提示词模板生成分镜
5. 输出分镜脚本（包含提示词和生成结果）

**提交格式**：
- 分镜脚本文档（Markdown 或 PDF）
- 包含：剧本片段、分镜设计（景别、角度、构图）、提示词、生成图像

### 问答环节

**Q1**：AI 生成的分镜质量如何保证？

**A**：
- AI 是辅助工具，不是替代工具
- 通过多轮对话和优化，逐步提升质量
- 导演的创意判断不可替代，AI 提供选项，导演做最终选择
- 定期训练 AI，使其更符合华策的风格和需求

---

**Q2**：AI 工具学习成本高吗？

**A**：
- 本模块针对零基础学员，循序渐进
- 提供完整的提示词模板和工作流
- 课程后有 1 对 1 辅导和交流群支持
- 实战练习占 50%+，确保学员掌握核心技能

---

**Q3**：AI 会限制创作自由吗？

**A**：
- AI 的优势是提供多个选项，而不是限制创作
- 导演可以在 AI 生成的基础上，融入自己的创意
- AI 可以激发新的创意灵感，而不是替代创意
- 关键是如何使用 AI，而不是 AI 本身

---

**Q4**：如何处理 AI 生成的不符合预期的结果？

**A**：
- 调整提示词，重新生成
- 使用局部重绘修改细节
- 参考风格迁移，统一风格
- 多方案对比，选择最佳方案
- 与团队讨论，获取反馈
