# 导演模块二：AI 辅助场景可视化

## 课程信息

**课程名称**：AI 辅助场景可视化
**授课对象**：导演团队（导演、副导演、执行导演、分镜师）
**课程时长**：2 小时
**难度级别**：进阶级
**前置要求**：完成模块一（AI 辅助分镜创作），或具备 AI 图像生成基础

**课程目标**：
- 掌握 AI 工具生成场景概念图和氛围图的方法
- 学会使用 AI 优化场景设计和氛围营造
- 能够快速创建可视化的场景方案，降低与美术、摄影团队的沟通成本

---

## 第一部分：课程导入（10分钟）

### 1.1 传统场景可视化的挑战

**问题 1：依赖美术团队**

- 场景设计需要美术团队参与，等待时间长
- 导演的构思可能无法准确传达给美术团队
- 修改周期长，影响拍摄进度

**问题 2：沟通成本高**

- 用文字描述场景，容易产生理解偏差
- "温馨的氛围"、"紧张的感觉"等抽象描述难以统一
- 需要多轮沟通才能达成一致

**问题 3：修改周期长**

- 每次调整都需要美术团队重新绘制
- 拍摄现场临时调整需求，难以快速响应
- 时间成本高，影响拍摄效率

**问题 4：成本高**

- 美术团队的人力成本
- 反复修改的时间成本
- 因沟通问题导致的返工成本

### 1.2 AI 辅助场景可视化的优势

**优势 1：快速生成**

- 几分钟内生成多个场景方案
- 无需等待美术团队
- 现场调整可以快速响应

**优势 2：多样选项**

- 生成不同风格、不同氛围的多个方案
- 提供更多创意灵感
- 可以快速对比不同方案

**优势 3：低成本**

- 无需投入大量人力
- 快速迭代，节省时间成本
- 降低沟通成本

**优势 4：可视化清晰**

- 直接看到场景效果，避免文字描述的偏差
- 可以精确控制光影、色彩、氛围
- 与美术、摄影团队沟通更高效

### 1.3 AI 在场景可视化中的应用场景

**场景 1：概念设计**

- 在剧本创作阶段，快速生成场景概念图
- 探索不同的风格和氛围
- 为剧本创作提供视觉参考

**场景 2：氛围营造**

- 根据剧情需求，生成不同情绪氛围的场景
- 快速调整光影、色彩、构图
- 为演员表演提供情绪参考

**场景 3：场景优化**

- 在拍摄前，优化场景设计
- 调整光影、色彩、细节
- 确保拍摄效果符合预期

**场景 4：团队沟通**

- 用可视化方案与美术团队沟通
- 与摄影团队确认布光方案
- 与演员说明场景氛围

### 1.4 课程内容概览

**本课程将学习**：

## 第二部分：AI 场景可视化工具介绍（20分钟）

### 2.1 主流 AI 场景生成工具对比

#### 工具 1：Midjourney

**特点**：
- 艺术风格强，适合创意探索
- 支持多种艺术风格（写实、绘画、动漫等）
- 社区活跃，有大量参考案例

**优势**：
- 生成的画面质量高
- 风格多样，创意丰富
- 操作简单，易于上手

**劣势**：
- 精确控制能力较弱
- 需要多轮对话才能达到理想效果
- 对中文提示词理解有限

**适用场景**：
- 创意探索和风格尝试
- 需要艺术感强的场景设计
- 不需要精确控制的场景

#### 工具 2：Stable Diffusion + ControlNet

**特点**：
- 开源免费，可高度定制
- ControlNet 支持精确控制（草图、深度图、姿态）
- 可以本地部署，保护隐私

**优势**：
- 控制能力强，可以精确控制构图和细节
- 支持多种控制模式（草图、深度图、Canny、Pose）
- 可以批量处理，适合大量场景生成

**劣势**：
- 配置复杂，需要一定的技术基础
- 需要硬件支持（需要较好的 GPU）
- 学习曲线较陡

**适用场景**：
- 需要精确控制的场景设计
- 批量生成多个场景
- 有技术团队支持的团队

#### 工具 3：DALL-E 3

**特点**：
- 文字理解能力强，适合精确描述
- 与 ChatGPT 集成，便于对话式创作
- 操作简单，无需技术背景

**优势**：
- 对提示词理解准确，能够精确控制
- 对话式创作，便于快速迭代
- 界面友好，易于上手

**劣势**：
- 风格选择较少
- 艺术感不如 Midjourney
- 控制能力不如 Stable Diffusion

**适用场景**：
- 需要精确控制场景细节
- 快速生成场景方案
- 不需要复杂控制的场景

### 2.2 工具选择指南

**根据需求选择合适的工具**：

| 需求场景 | 推荐工具 | 原因 |
|---------|---------|------|
| 创意探索、风格尝试 | Midjourney | 艺术风格强，创意丰富 |
| 精确控制、批量生成 | Stable Diffusion + ControlNet | 控制能力强，可批量处理 |
| 快速生成、精确描述 | DALL-E 3 | 文字理解强，操作简单 |
| 初学者、零基础 | DALL-E 3 | 操作简单，无需技术背景 |
| 技术团队支持 | Stable Diffusion + ControlNet | 可定制，控制能力强 |
| 艺术感要求高 | Midjourney | 画面质量高，风格多样 |

**工具组合策略**：
- **创意阶段**：使用 Midjourney 探索多种风格和创意
- **精确控制阶段**：使用 Stable Diffusion + ControlNet 控制细节和构图
- **快速迭代阶段**：使用 DALL-E 3 快速生成多个方案
- **批量处理阶段**：使用 Stable Diffusion 批量生成多个场景

### 2.3 ControlNet 的应用

**ControlNet 是什么**：
- Stable Diffusion 的一个插件
- 允许通过输入图像精确控制生成结果
- 支持多种控制模式（草图、深度图、Canny、Pose 等）

**ControlNet 的核心功能**：

#### 2.3.1 草图控制（Sketch ControlNet）

**用途**：
- 通过手绘草图控制场景构图
- 快速探索不同的构图方案
- 降低美术门槛，人人都能画草图

**使用方法**：
1. 手绘场景草图（可以是简笔画）
2. 上传草图到 ControlNet
3. 生成基于草图的高质量场景图

**应用场景**：
- 探索不同构图方案
- 快速验证创意
- 降低美术团队沟通成本

#### 2.3.2 深度图控制（Depth ControlNet）

**用途**：
- 通过深度图控制场景的空间关系
- 确保人物和环境的透视关系正确
- 优化场景的空间布局

**使用方法**：
1. 生成或绘制深度图
2. 上传深度图到 ControlNet
3. 生成基于深度图的高质量场景图

**应用场景**：
- 确保场景透视关系正确
- 优化人物和环境的空间布局
- 制作复杂的场景设计

#### 2.3.3 姿态控制（Pose ControlNet）

**用途**：
- 通过姿态图控制人物的姿势和位置
- 快速探索不同的人物布局
- 确保人物动作符合剧情需求

**使用方法**：
1. 生成或绘制人物姿态图
2. 上传姿态图到 ControlNet
3. 生成基于姿态图的高质量场景图

**应用场景**：
- 设计人物的姿势和位置
- 快速探索不同的人物布局
- 确保人物动作符合剧情

#### 2.3.4 Canny 边缘控制（Canny ControlNet）

**用途**：
- 通过边缘图控制场景的轮廓和形状
- 保持场景的整体结构
- 快速调整场景细节

**使用方法**：
1. 生成场景的 Canny 边缘图
2. 上传边缘图到 ControlNet
3. 生成基于边缘图的高质量场景图

**应用场景**：
- 保持场景的整体结构
- 快速调整场景细节
- 批量生成相似但细节不同的场景

### 2.4 OpenClaw 在场景可视化中的应用

**OpenClaw 的核心优势**：

#### 2.4.1 统一入口管理

**功能**：
- 通过 OpenClaw 统一调用多个 AI 工具
- 无需切换不同的平台和账号
- 集中管理所有的 AI 生成任务

**应用场景**：
- 同时调用 Midjourney、Stable Diffusion、DALL-E 3
- 对比不同工具的生成效果
- 选择最优方案

#### 2.4.2 工作流自动化

**功能**：
- 使用 OpenClaw 的 cron 功能自动化场景生成
- 批量生成多个场景方案
- 定期更新场景库

**应用场景**：
- 自动批量生成多个场景方案
- 定期更新场景库
- 自动化场景优化流程

**示例工作流**：
```bash
# 批量生成不同氛围的场景
for atmosphere in 温馨 紧张 浪漫 悬疑; do
  echo "生成氛围: $atmosphere"
  openclaw message "请为以下场景生成$atmosphere氛围的方案：[场景描述]" \
    --model zai/glm-4.7 \
    > "scenes/scene_$atmosphere.md"
done
```

#### 2.4.3 提示词管理

**功能**：
- 使用 OpenClaw 管理提示词模板
- 快速调用常用的提示词
- 优化提示词效果

**应用场景**：
- 管理不同类型场景的提示词模板
- 快速调用提示词模板
- 优化提示词效果

**示例模板库**：
```
温馨氛围：[环境类型] + [时间] + 温馨 + [风格] + 柔和光影 + [细节描述]
紧张氛围：[环境类型] + [时间] + 紧张 + [风格] + 强烈光影 + [细节描述]
浪漫氛围：[环境类型] + [时间] + 浪漫 + [风格] + 温暖光影 + [细节描述]
悬疑氛围：[环境类型] + [时间] + 悬疑 + [风格] + 冷峻光影 + [细节描述]
```

### 3.4 如何营造氛围

#### 3.4.1 通过色彩营造氛围

**暖色调**：
- 红色、橙色、黄色
- 营造温馨、浪漫、欢快、热情的氛围
- 适用场景：家庭、聚会、爱情戏

**冷色调**：
- 蓝色、绿色、紫色
- 营造冷静、神秘、悲伤、紧张的氛围
- 适用场景：悬疑、悲伤、科幻

**中性色调**：
- 灰色、白色、黑色
- 营造简洁、现代、高雅、冷漠的氛围
- 适用场景：现代办公、科技、极简

**高饱和度**：
- 色彩鲜艳、明快
- 营造欢快、活力、童话般的氛围
- 适用场景：喜剧、儿童剧、广告

**低饱和度**：
- 色彩暗淡、柔和
- 营造忧郁、压抑、怀旧的氛围
- 适用场景：悲剧、历史剧、文艺片

#### 3.4.2 通过光影营造氛围

**柔和光线**：
- 柔和的光线、低对比度
- 营造温馨、舒适、梦幻的氛围
- 适用场景：温馨家庭、浪漫场景

**强烈光线**：
- 强烈的光线、高对比度
- 营造戏剧性、紧张、冲击力强的氛围
- 适用场景：冲突戏、悬疑、动作戏

**侧光**：
- 光线从侧面照射
- 营造立体感、戏剧性、神秘的氛围
- 适用场景：人物肖像、悬疑场景

**逆光**：
- 光线从背后照射
- 营造剪影效果、戏剧性、神圣的氛围
- 适用场景：日出日落、神圣场景

**多光源**：
- 多个光源同时照射
- 营造层次丰富、复杂的氛围
- 适用场景：复杂的室内场景、酒吧、餐厅

#### 3.4.3 通过构图营造氛围

**对称构图**：
- 左右对称、平衡
- 营造稳定、正式、庄严的氛围
- 适用场景：正式场合、历史剧

**三分法构图**：
- 将画面分为三等分
- 营造和谐、平衡、舒适的氛围
- 适用场景：日常场景、风景

**对角线构图**：
- 主元素沿对角线分布
- 营造动感、紧张、冲击力强的氛围
- 适用场景：动作戏、冲突戏

**黄金螺旋构图**：
- 基于黄金螺旋的布局
- 营造优美、和谐、艺术感的氛围
- 适用场景：艺术性强的场景

**低角度构图**：
- 相机位置较低
- 营造压迫感、崇高感、戏剧性
- 适用场景：英雄场景、冲突戏

**高角度构图**：
- 相机位置较高（鸟瞰）
- 营造宏大、全景、俯视的氛围
- 适用场景：大场面、风景

#### 3.4.4 通过细节营造氛围

**植物**：
- 花朵：浪漫、温馨、美好
- 枯树：萧条、悲伤、荒凉

## 第四部分：AI 场景生成实战（40 分钟）

### 4.1 完整的场景生成工作流

**步骤 1：剧本分析（5 分钟）**

**目标**：
- 理解剧本中的场景需求
- 确定场景的环境、时间、氛围

**方法**：
1. 仔细阅读剧本
2. 提取关键信息：
   - 场景发生在哪里？（环境类型）
   - 什么时候发生的？（时间）
   - 剧情是什么？需要什么氛围？（氛围）
   - 有哪些人物和道具？（细节描述）

**示例**：
- 剧本片段："深夜，男主角独自一人在办公室加班，窗外是城市的灯光。他感到疲惫不堪。"
- 提取信息：
  - 环境类型：办公室（室内）
  - 时间：深夜
  - 氛围：疲惫、孤独、压抑
  - 细节：电脑、文件、台灯、窗外的城市灯光

**步骤 2：场景规划（5 分钟）**

**目标**：
- 设计场景的整体布局
- 确定需要突出的元素

**方法**：
1. 画出场景草图（可以是简笔画）
2. 确定场景的主要元素：
   - 人物位置
   - 主要家具和道具
   - 窗户和光源位置
3. 确定场景的构图：
   - 对称/不对称
   - 平视/仰视/俯视

**示例**：
- 场景草图：办公室中央是办公桌，男主角坐在桌前，背后是窗户，窗外是城市夜景
- 主要元素：办公桌、电脑、文件、台灯、窗户、城市灯光
- 构图：平视，人物在画面中央

**步骤 3：提示词设计（10 分钟）**

**目标**：
- 编写精确的提示词，控制场景的生成效果

**方法**：
1. 使用提示词公式：`[环境类型] + [时间] + [氛围] + [风格] + [光影] + [细节描述]`
2. 逐步添加细节，从简单到复杂
3. 迭代优化提示词，测试效果

**示例**：
- 基础版：办公室，深夜，疲惫，压抑，冷色调，台灯、电脑、文件
- 优化版：现代办公室，深夜，冷色调，疲惫、孤独、压抑的氛围，台灯、电脑、文件堆、窗外是城市灯光、阴影、低饱和度
- 最终版：现代办公风格，深夜，疲惫、孤独、压抑的氛围，台灯、电脑、文件堆、窗外是城市夜景，冷色调、阴影、低饱和度、细节丰富

**步骤 4：AI 生成（10 分钟）**

**目标**：
- 使用 AI 工具生成场景图像

**方法**：
1. 选择合适的 AI 工具（Midjourney、Stable Diffusion、DALL-E 3）
2. 输入提示词
3. 生成多个版本（建议 4-8 个）
4. 选择最优版本

**示例**：
- 工具选择：DALL-E 3（对提示词理解准确）
- 提示词：现代办公风格，深夜，疲惫、孤独、压抑的氛围，台灯、电脑、文件堆、窗外是城市夜景，冷色调、阴影、低饱和度、细节丰富
- 生成数量：4 个
- 选择：选择最符合剧本氛围的版本

**步骤 5：优化调整（10 分钟）**

**目标**：
- 优化生成的场景，调整细节和氛围

**方法**：
1. 分析生成结果，找出需要调整的地方
2. 使用以下方法优化：
   - 局部重绘：调整特定区域
   - 风格迁移：改变整体风格
   - 色彩调整：调整色调和饱和度
   - 光影调整：调整光影效果
3. 迭代优化，直到满意

**示例**：
- 分析：生成的场景光影不够强，压抑感不够明显
- 优化：增加阴影、降低饱和度、增强光影对比
- 迭代：再次生成，对比效果
- 最终：选择最符合剧本氛围的版本

### 4.2 如何控制场景的一致性

#### 4.2.1 光影一致性

**为什么重要**：
- 光影是营造氛围的关键
- 不一致的光影会破坏场景的真实感

**如何控制**：
1. 固定光源位置和方向
2. 使用固定的光影关键词（柔光、强光、侧光、逆光）
3. 使用 ControlNet 控制光影分布
4. 使用固定的 seed 值

**示例**：
- 光源位置：台灯在桌面左侧，光线从左向右照射
- 光影关键词：侧光、冷光、强烈对比
- ControlNet：使用深度图控制光影分布
- Seed：使用固定的 seed 值（如 12345）

#### 4.2.2 色彩一致性

**为什么重要**：
- 色彩是营造氛围的重要手段
- 不一致的色彩会破坏场景的统一感

**如何控制**：
1. 固定主色调（暖色/冷色/中性色）
2. 使用固定的色调关键词（温馨色调、冷色调、灰色调）
3. 使用风格迁移统一色调
4. 使用参考图像统一风格

**示例**：
- 主色调：冷色调（蓝色、灰色）
- 色调关键词：冷色调、低饱和度、蓝色调
- 风格迁移：使用一张冷色调的参考图像
- 参考图像：选择一张冷色调的办公室图像作为参考

#### 4.2.3 风格一致性

**为什么重要**：
- 风格统一才能保持场景的整体感
- 不一致的风格会让人感觉混乱

**如何控制**：
1. 固定风格关键词（写实、艺术感、古风、现代感、科幻、奇幻）
2. 使用风格迁移统一风格
3. 使用参考图像统一风格
4. 使用固定的 seed 值

**示例**：
- 风格关键词：现代简约风格、冷色调、细节丰富
- 风格迁移：使用一张现代简约风格的参考图像
- 参考图像：选择一张现代简约风格的办公室图像作为参考
- Seed：使用固定的 seed 值（如 12345）

### 4.3 如何优化 AI 生成的场景

#### 4.3.1 局部重绘（Inpainting）

**用途**：
- 调整场景中的特定区域
- 修改或替换某些元素
- 优化细节

**方法**：
1. 选择需要修改的区域
2. 输入新的提示词
3. 生成新的图像

**示例**：
- 场景：办公室，窗外的城市灯光不够明亮
- 操作：选择窗户区域，输入提示词："窗外是明亮的城市夜景，灯光辉煌"
- 结果：窗外的城市灯光更加明亮

#### 4.3.2 风格迁移（Style Transfer）

**用途**：
- 改变场景的整体风格
- 统一多个场景的风格
- 营造特定的艺术效果

**方法**：
1. 选择一张参考图像（具有目标风格）
2. 使用风格迁移工具
3. 将参考图像的风格应用到当前场景

**示例**：
- 场景：办公室，需要更压抑的氛围
- 操作：选择一张冷色调、低饱和度的参考图像，使用风格迁移工具
- 结果：场景色调更冷，饱和度更低，压抑感更强

#### 4.3.3 构图调整

**用途**：
- 调整场景的构图
- 改变视角
- 优化视觉效果

**方法**：
1. 使用 ControlNet 控制构图
2. 使用草图控制改变构图
3. 使用 Canny 边缘控制调整布局

**示例**：
- 场景：办公室，需要从平视改为低角度
- 操作：使用 ControlNet，输入低角度的草图
- 结果：场景视角从平视改为低角度，压迫感更强

#### 4.3.4 细节优化

**用途**：
- 优化场景的细节
- 增加细节的真实感
- 提升整体质量

**方法**：

#### 4.4.2 使用变量控制生成

**优势**：
- 灵活控制生成参数
- 快速生成多个版本
- 便于对比效果

**方法**：
1. 定义变量（氛围、时间、光影等）
2. 使用循环生成多个版本
3. 对比不同版本的效果

**示例脚本**：
```bash
# 定义变量
atmospheres=("温馨" "紧张" "浪漫" "悬疑")

done
```

#### 4.4.3 使用 Prompt Template

### 6.4 常见问题解答

#### Q1：AI 生成的场景质量如何保证？

**A**：
- AI 是辅助工具，不是替代工具
- 通过多轮对话和优化，逐步提升质量
- 导演和美术团队的创意判断不可替代，AI 提供选项，团队做最终选择
- 定期训练 AI，使其更符合华策的风格和需求

**保证质量的方法**：
1. 精确的提示词：使用提示词公式，确保提示词准确描述场景
2. 多轮优化：根据反馈调整提示词，重新生成场景
3. 与美术团队紧密合作：美术团队的经验非常重要，需要尊重他们的意见
4. 选择合适的 AI 工具：根据需求选择合适的工具，提升生成质量

#### Q2：AI 场景生成的时间成本如何？

**A**：
- AI 生成场景的时间成本远低于传统方式
- 单个场景生成时间：2-3 小时（包含分析和优化）
- 传统方式：2-3 天/场景
- 节省时间：70%+

**时间成本对比**：
- AI 生成场景：2-3 小时
- 传统方式：2-3 天
- 时间节省：70%+

**影响因素**：
- 提示词的精确程度
- AI 工具的选择
- 场景的复杂程度
- 优化的轮次

#### Q3：AI 生成的场景如何与实际拍摄结合？

**A**：
- AI 生成的场景作为概念设计，不是最终场景
- 美术团队根据 AI 生成的场景进行实际设计和搭建
- 摄影团队根据 AI 生成的场景确认布光方案
- 演员根据 AI 生成的场景理解场景氛围
- 后期团队根据 AI 生成的场景确认特效需求

**结合方式**：
1. 概念设计：AI 生成的场景作为概念设计
2. 实际设计：美术团队根据 AI 生成的场景进行实际设计
3. 布光方案：摄影团队根据 AI 生成的场景确认布光方案
4. 表演参考：演员根据 AI 生成的场景理解场景氛围
5. 特效需求：后期团队根据 AI 生成的场景确认特效需求

#### Q4：AI 在场景可视化中的局限性是什么？

**A**：
- AI 生成的场景细节不够精确
- AI 生成的场景需要美术团队的调整和优化
- AI 无法替代真实的场景搭建
- AI 工具的稳定性和准确性有待提高

**局限性**：
1. **细节精确性**：AI 生成的场景细节不够精确，需要美术团队的调整和优化
2. **替代性**：AI 无法替代真实的场景搭建，AI 生成的场景作为概念设计
3. **稳定性**：AI 工具的稳定性和准确性有待提高，需要多轮优化
4. **中文理解**：AI 对中文提示词的理解有限，需要使用英文提示词或精确描述

**应对方法**：
1. 多轮优化：通过多轮对话和优化，逐步提升质量
2. 与团队紧密合作：AI 是辅助工具，不是替代工具，需要与团队紧密合作
3. 选择合适的工具：根据需求选择合适的工具，提升生成质量
4. 精确的提示词：使用提示词公式，确保提示词准确描述场景

### 附录 C：场景评估标准

#### 评估维度

**1. 氛围准确性（30%）**
- 场景氛围是否符合剧本需求
- 情绪表达是否准确
- 氛围营造是否到位

**2. 视觉质量（30%）**
- 画面质量是否高
- 细节是否丰富
- 光影效果是否自然

**3. 风格一致性（20%）**
- 场景风格是否统一
- 色彩搭配是否协调
- 光影是否一致

**4. 实用性（20%）**
- 场景是否可用于实际拍摄
- 是否与美术团队沟通顺畅
- 是否满足拍摄需求

#### 评分标准

**优秀（90-100分）**：
- 氛围营造非常准确，完全符合剧本需求
- 视觉质量非常高，细节丰富，光影自然
- 风格高度统一，色彩搭配协调
- 实用性非常强，可直接用于拍摄

**良好（80-89分）**：
- 氛围营造准确，符合剧本需求
- 视觉质量高，细节丰富，光影自然
- 风格统一，色彩搭配协调
- 实用性强，可用于拍摄

**中等（70-79分）**：
- 氛围营造基本准确，基本符合剧本需求
- 视觉质量中等，细节较丰富，光影较自然
- 风格基本统一，色彩搭配较协调
- 实用性中等，需要优化后用于拍摄

- 氛围营造有偏差，部分符合剧本需求
- 视觉质量较低，细节较少，光影不够自然
- 风格不够统一，色彩搭配不够协调
- 实用性较低，需要大量优化后用于拍摄

**不及格（<60分）**：
- 氛围营造偏差大，不符合剧本需求
- 视觉质量低，细节少，光影不自然
- 风格不统一，色彩搭配不协调
