## 第一部分：人物档案生成

### 🎯 为什么需要人物档案？

**常见问题**：
- ❌ 人物扁平，缺乏立体感
- ❌ 行为逻辑前后矛盾
- ❌ 对白风格千人一面
- ❌ 关系网络不清晰

**人物档案的价值**：
- ✅ 确保人物行为有逻辑
- ✅ 统一对白风格
- ✅ 明确人物关系
- ✅ 加速创作进程

---

### 📋 人物档案的核心要素

**基础信息**：
- 姓名、年龄、职业
- 外貌特征、标志性习惯
- 教育背景、成长经历

**性格维度**：
- 性格关键词（3-5 个）
- 优点与缺点
- 核心信念和价值观
- 情绪触发点

**动机与目标**：
- 表层目标（显性）
- 深层动机（隐性）
- 恐惧与渴望

**对白风格**：
- 语言习惯（口语化/书面/方言）
- 常用词和口头禅
- 说话节奏和语气

---

### 🎮 实战：生成人物档案（30 分钟）

**任务**：为一部都市职场剧生成 3 个主要人物档案

**操作步骤**：
1. **明确人物类型**（主角、反派、配角）
2. **设计提示词**（人物定位 + 故事背景）
3. **与 AI 对话生成**（迭代优化）
4. **验证人物逻辑**（检查一致性）

**提示词模板**：
```
我正在创作一部[类型]剧，需要设计一个[人物类型]人物。
人物定位：[人物角色和功能]
故事背景：[简要背景]
请帮我生成一份详细的人物档案，包括：
1. 基础信息（姓名、年龄、职业等）
2. 性格维度（关键词、优缺点、价值观）
3. 动机与目标（表层目标、深层动机）
4. 对白风格（语言习惯、常用词、语气）

要求：
- 人物要有独特性和记忆点
- 行为逻辑自洽
- 对白风格鲜明
```

---

### 📝 示例：主角人物档案

**基础信息**：
- 姓名：林小雅
- 年龄：26 岁
- 职业：广告公司策划
- 外貌：中等身材，常戴黑框眼镜

**性格维度**：
- 关键词：坚韧、敏感、完美主义
- 优点：创意丰富、执行力强
- 缺点：过度焦虑、缺乏自信
- 价值观：相信努力就会有回报

**对白风格**：
- 语速中等偏快
- 喜欢用反问句
- 紧张时会重复词语

**你的任务**：生成第 2、3 个人物档案！

## 第二部分：人物一致性检查

### ⚠️ 什么是一致性问题？

**OOC（Out of Character）问题**：

| 问题类型 | 典型表现 | 影响 |
|----------|----------|------|
| 行为不一致 | 勇敢的人突然胆怯 | 观众出戏 |
| 对白风格突变 | 粗犷的人突然文雅 | 角色崩塌 |
| 价值观冲突 | 正直的人突然黑化 | 逻辑混乱 |
| 关系错乱 | 陌生人突然变密友 | 情节不可信 |

**长剧的挑战**：
- 集数多，容易忘记设定
- 编剧团队协作，信息不同步
- 后期修改，影响前文

---

### 🔍 AI 如何检查一致性？

**检查维度**：
1. **行为逻辑**
   - 评估行为是否符合性格设定
   - 检查情绪反应是否合理
   - 验证行为动机是否一致

2. **对白风格**
   - 识别语言习惯是否统一
   - 检查口头禅是否过度使用
   - 确认语气是否符合角色身份

3. **人物关系**
   - 验证关系发展的合理性
   - 检查情感进展的逻辑
   - 确保关系冲突的成因明确

---

### 🎮 实战：检查人物一致性（30 分钟）

**任务**：检查一段剧本中的人物一致性问题

**操作步骤**：
1. **提供人物档案**（作为参考标准）
2. **提供待检查文本**（剧本片段）
3. **让 AI 识别问题**（标注不一致之处）
4. **分析问题原因**（为什么会出问题）
5. **生成修改建议**（如何修正）

**提示词模板**：
```
我正在检查剧本中的人物一致性问题。

**人物档案**：
[粘贴人物档案内容]

**待检查文本**：
[粘贴剧本片段]

请帮我检查：
1. 行为是否符合人物性格设定？
2. 对白风格是否统一？
3. 是否存在 OOC（Out of Character）问题？
4. 如果有问题，请指出具体位置并说明原因
5. 提供修改建议

输出格式：
- 问题位置：[集数/场数]
- 问题描述：[具体问题]
- 问题原因：[分析]
- 修改建议：[建议]
```

---

### 📚 案例分析：角色 OOC 问题解决

**问题场景**：
- 人物设定：谨慎、保守的财务经理
- 问题场景：第 8 集突然冒险投资

**AI 分析结果**：
1. 问题位置：第 8 集第 3 场
2. 问题描述：谨慎的角色突然冒险
3. 问题原因：缺少合理铺垫和动机
4. 修改建议：
   - 增加前期铺垫（压力、诱惑）
   - 赋予强烈的动机（家庭危机）
   - 设计内心挣扎戏份

**修改后效果**：
- 行为有逻辑，观众信服
- 人物成长弧光更完整
- 戏剧张力提升

## 第三部分：对白优化技巧

### 🗣️ 好对白的标准

**自然流畅**：
- 符合人物身份和语言习惯
- 避免书面语、长篇大论
- 口语化、生活化

**信息传递**：
- 推进剧情发展
- 展现人物性格
- 暗示潜台词

**情感张力**：
- 有冲突，有层次
- 有节奏，有留白
- 有记忆点

---

### 🔧 AI 优化对白的三个层次

**层次一：基础修正**
- 语法错误、用词不当
- 逻辑不通、表达不清
- 风格不统一

**层次二：风格优化**
- 调整语气和语速
- 加入个人特色
- 符合时代背景

**层次三：情感升华**
- 增强冲突张力
- 加入潜台词
- 制造戏剧效果

---

### 🎮 练习一：对白自然度检查（15 分钟）

**任务**：检查并优化一段生硬的对白

**原始对白**（示例）：
```
林小雅：张经理，我有一个建议，我觉得我们应该采用那个方案，因为它有很多优点。
张经理：好的，我同意你的意见，我们就这样做吧。
```

**问题诊断**：
- ❌ 过于书面化
- ❌ 缺乏个性
- ❌ 信息量低
- ❌ 情感平淡

**操作步骤**：
1. 分析对白问题
2. 明确人物身份和关系
3. 设计场景背景和情绪
4. 让 AI 生成优化版本
5. 对比选择最优方案

---

### 🎮 练习二：冲突场景对白优化（15 分钟）

**任务**：优化一场吵架戏的对白

**场景设定**：
- 人物：夫妻吵架

- 情绪：愤怒、失望、不解

**优化目标**：

- 制造情感张力

```

**场景设定**：
[描述场景背景]

**人物关系**：
[描述人物关系和当前情绪]

**原始对白**：
[粘贴原始对白]

优化要求：
1. 对白要符合人物性格
2. 推进矛盾冲突
3. 增强情感张力
4. 加入潜台词
5. 提供 3 个不同风格的版本（激烈型、压抑型、讽刺型）
```

---

### 🎮 练习三：不同角色对白风格（10 分钟）

**任务**：为不同角色生成风格统一的对白

**角色类型**：
1. 20 岁实习生（活泼、话痨）
2. 40 岁总监（沉稳、话少）
3. 50 岁老板（威严、果断）

**场景**：开会讨论方案

**操作步骤**：
1. 明确每个人物的对白风格
2. 提供统一的场景背景
3. 让 AI 生成每个人的台词
4. 检查风格是否鲜明

**关键点**：
- 词汇选择（专业/通俗）
- 句式长短（短促/冗长）
- 语气助词（嗯、嘛、吧）
- 说话方式（直截了当/含蓄委婉）

## 第四部分：OpenClaw 工具实践

### 🛠️ OpenClaw 工作流演示

**完整工作流**：
1. **人物档案创建** → 保存到文件
2. **一致性检查** → 批量处理剧本
3. **对白优化** → 生成多个版本
4. **质量审核** → 人工把关

**自动化脚本**（示例）：
```
# 人物档案管理
function checkCharacter(characterFile, scriptFile) {
    // 读取人物档案
    let character = readFile(characterFile);
    // 读取剧本
    let script = readFile(scriptFile);
    // AI 检查一致性
    let result = ai.checkConsistency(character, script);
    // 输出报告
    writeFile("report.md", result);
}
```
