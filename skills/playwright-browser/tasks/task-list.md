  ```bash
  # 步骤 2：以调试模式重新启动 Chrome

    --remote-debugging-port=9222 \
    --no-first-run \
    --no-default-browser-check \
    --profile-directory=Default
  ```
- **预期结果**：CDP 端口可达，支持自动刷新
- **依赖条件**：用户手动重新启动 Chrome
