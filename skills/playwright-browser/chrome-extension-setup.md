# Chrome 扩展安装指南

## 前置条件

- 已安装 Chrome 浏览器（版本 >= 144）
- 已登录番茄小说作者账号

## 安装步骤

### 1. 打开 Chrome 扩展商店

访问：https://chrome.google.com/webstore

### 2. 搜索并安装 Playwright MCP

1. 在搜索框中输入 "Playwright MCP"
2. 找到官方扩展（由 Microsoft 发布）
3. 点击"添加至 Chrome"按钮
4. 确认添加扩展

### 3. 配置扩展 Token

1. 点击浏览器工具栏中的扩展图标（拼图图标）
2. 找到 "Playwright MCP" 扩展
3. 点击扩展图标，打开配置面板
4. 输入 Token：`7U3d_3r_myx-TEchJ49XXKI8O5TjRrMwMP1ma5q4WZs`
5. 点击"保存"或"连接"

### 4. 验证扩展状态

- 扩展图标应该显示为激活状态（通常是蓝色或绿色）
- 点击图标，应该显示"已连接"或类似状态

### 5. 确保浏览器已登录

- 在浏览器中访问：https://fanqienovel.com/main/writer/login
- 确保已使用手机号和验证码登录成功
- 确认用户名为"帅帅它爸"

## 常见问题

### Q: 扩展无法连接？
A: 检查 Token 是否正确，确保与 `mcp-config.json` 中的 Token 一致。

### Q: 扩展图标显示未激活？
A: 尝试刷新扩展页面或重启 Chrome 浏览器。

### Q: 提示 Chrome 版本过低？
A: 更新 Chrome 浏览器到最新版本（需要 >= 144）。
