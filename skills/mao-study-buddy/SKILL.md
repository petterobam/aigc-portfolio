# 毛选学习助手

帮助欧阳洁系统学习《毛泽东选集》，按计划推进、记录进度、生成总结。

## 核心能力

### 1. 推荐今日学习内容

当用户问"今天学什么"或"继续学习"时：

1. 读取 `progress.json` 获取当前进度
2. 根据学习计划推荐1-2篇文章
3. 简要介绍文章背景和重要性

### 2. 生成文章摘要

当用户要求"总结这篇文章"或阅读指定文章时：

1. 读取 `~/.openclaw/workspace/books/MaoZeDongAnthology/src/XXX-文章名.md`
2. 生成结构化摘要：
   - **写作背景**：时间、地点、历史情境
   - **核心论点**：3-5个要点
   - **经典语句**：摘录2-3句名言
   - **现实意义**：简要点评
3. 更新 `progress.json` 中的 completedArticles

### 3. 记录学习进度

进度文件位于：`progress.json`

```json
{
  "currentStage": 1,
  "currentArticle": "000",
  "completedArticles": ["000", "001"],
  "lastStudyDate": "2026-03-10",
  "notes": {
    "016": "实践论核心：实践是检验真理的唯一标准"
  }
}
```

每次学习后自动更新进度。

### 4. 生成每日学习日志

当需要推送每日总结时：

1. 汇总当天学习的文章
2. 生成 Markdown 格式的学习日志
3. 保存到 `docs/study-logs/YYYY-MM-DD-{标题}.md`
4. **推送方式**：使用 `MEDIA:<文件绝对路径>` 推送文件本身，而不是发送文件内容
   - 例如：`MEDIA:~/.openclaw/workspace/skills/mao-study-buddy/docs/study-logs/2026-03-10-实践论.md`
   - 这样用户的消息列表更简洁，点击即可查看内容

日志格式：
```markdown
# 学习日志 - YYYY年MM月DD日

## 今日学习
- [文章名] - 一句话核心观点

## 核心收获
1. 要点一
2. 要点二

## 经典语句
> 引用原文

## 思考与感悟
（如有）
```
