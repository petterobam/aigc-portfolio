# 学习日志 - 2026年3月25日（周三）

## 📚 今日学习计划

**《井冈山的斗争》（003）** ⭐⭐⭐⭐⭐

这是毛泽东于1928年11月25日写给中共中央的报告，系统总结井冈山革命根据地一年多的斗争经验。

---

## 🎯 文章概览

### 写作背景
- **时间**：1928年11月25日
- **情境**：井冈山斗争进入关键时期，需要向中央汇报经验教训
- **意义**：这是对《中国的红色政权为什么能够存在？》的实践展开

### 核心内容预告

文章从八个方面系统总结了井冈山斗争的经验：

1. **湘赣边界的割据和八月失败**
   - 红色政权存在的五个条件
   - 统治阶级稳定与破裂时期的不同战略
   - 八月失败的深刻教训

2. **割据地区的现势**
   - 根据地的发展历程
   - 全盛时期的区域范围

3. **军事问题**
   - 红军的成分与改造
   - "支部建在连上"的重要意义
   - 军队内的民主主义制度
   - 游击战术的运用

4. **土地问题**
   - 边界土地状况分析
   - 中间阶级的复杂性
   - 土地分配的标准和方法

5. **政权问题**
   - 工农兵代表会的建设
   - 民主集中制的实行
   - 党与政权的关系

6. **党的组织问题**
   - 与机会主义的斗争
   - 克服地方主义和土客籍问题
   - 投机分子的清洗

7. **革命性质问题**
   - 中国革命处于资产阶级民权革命阶段
   - 对小资产阶级的政策

8. **割据地区问题**
   - 罗霄山脉中段的战略价值
   - 为什么选择井冈山

---

## 💡 学习重点提示

### 重点关注

1. **"支部建在连上"**
   > "红军所以艰难奋战而不溃散，'支部建在连上'是一个重要原因。"
   
   这是建军史上的重大创举，确立了党对军队的绝对领导。

2. **军队内的民主主义**
   > "中国不但人民需要民主主义，军队也需要民主主义。军队内的民主主义制度，将是破坏封建雇佣军队的一个重要的武器。"
   
   官兵平等、经济公开、士兵有开会说话的自由。

3. **八月失败的教训**
   - 不明了统治阶级暂时稳定时期
   - 分兵冒进的错误
   - 脱离群众基础

4. **中间阶级问题**
   - 农村阶级关系的复杂性
   - 白色恐怖下中间阶级的反水
   - 经济封锁的影响

5. **根据地建设的辩证法**
   - 集中与分散
   - 军事发展与群众工作
   - 波浪式推进政策
