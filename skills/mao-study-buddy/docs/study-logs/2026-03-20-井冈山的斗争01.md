# 学习日志 - 2026年03月20日

## 今日学习

**《井冈山的斗争》（第003篇，1928年11月25日）** - 毛泽东写给中共中央的报告，系统总结井冈山革命根据地创建一年来的经验教训，阐述工农武装割据的理论与实践

## 文章概要

### 写作背景
- **时间**：1928年11月25日
- **性质**：给中共中央的报告
- **历史情境**：井冈山根据地创建一年，经历了四月到七月的发展期和八月的失败，正处于恢复重建阶段

### 核心内容

毛泽东从八个方面系统总结了井冈山斗争的经验：

#### 1. 湘赣边界的割据和八月失败

**红色政权存在的条件：**
- 中国买办豪绅阶级间的不断分裂和战争
- 五个必备条件：
  1. 有很好的群众
  2. 有很好的党
  3. 有相当力量的红军
  4. 有便利于作战的地势
  5. 有足够给养的经济力

**战略策略的区分：**
- **统治阶级破裂时期**：可以比较冒进，军事发展可以比较广大
- **统治阶级稳定时期**：必须逐渐推进，最忌分兵冒进

**八月失败的原因：**
- 不明了当时正是统治阶级暂时稳定时期
- 采取在破裂时期的政策，分兵向湘南冒进
- 第二十九团官兵动摇思家，缺乏战斗力

#### 2. 割据地区的现势

- **全盛时期（六月）**：宁冈、永新、莲花三个全县，吉安、安福各一小部，遂川北部，酃县东南部
- **八月失败后**：县城及平原地区尽为敌据，山区始终无法被夺取
- **当前形势**：南自遂川井冈山南麓，北至莲花边界，成一南北狭长的整块

#### 3. 军事问题

**红军成分的六大来源：**
1. 潮汕叶贺旧部
2. 前武昌国民政府警卫团
3. 平浏的农民
4. 湘南的农民和水口山的工人
5. 敌军俘虏兵（最多）
6. 边界各县的农民

**红军的特点：**
- 废除雇佣制，士兵为自己为人民打仗
- 物质生活菲薄（每天每人只有五分大洋的油盐柴菜钱）
- 官兵待遇平等，经济公开
- 实行军队内的民主主义

**关键制度：**
- **"支部建在连上"**：红军艰难奋战而不溃散的重要原因
- **党代表制度**：特别是连一级，党代表担任党支部书记，负责政治训练和民运工作
- **士兵委员会**：代表士兵利益，做政治工作和民众工作

**对敌军的宣传：**
- 释放俘虏和医治伤兵最有效
- 打破敌人"共匪见人就杀"的欺骗宣传

#### 4. 土地问题

**土地状况：**
- 土地的60%以上在地主手里，40%以下在农民手里
- 遂川最集中：80%是地主的
- 永新次之：70%是地主的

**中间阶级问题：**
- 小地主、富农是中间阶级
- 革命初期：表面投降，实际恐吓贫农，延宕分田
- 革命高涨：才敢于中间阶级积极行动
- 白色恐怖一来：马上反水，带领反动军队烧屋捉人

**政策问题：**
- 对小资产阶级处理失当，导致贸易断绝
- 食盐、布匹、药材缺乏昂贵
- 中等阶级忍不住就投降豪绅

#### 5. 政权问题

**存在的问题：**
- 工农兵代表会名不副实
- 一哄而集的群众会，不能讨论问题
- 选举完毕，大权揽于委员会
- 封建独裁恶习惯深中于群众头脑

**纠正方向：**
- 制订详细的各级代表会组织法
- 党执行领导政府的任务，但不能直接向政府下命令
- 政权机关必须建立党团组织

#### 6. 党的组织问题

**与机会主义的斗争：**
- 一部分党员无斗争决心，敌来躲入深山（"打埋伏"）
- 一部分党员富有积极性，却流于盲目暴动
- 经过长期斗争锻炼和党内教育，逐渐减少

**地方主义问题：**
- 农业经济，社会组织以一姓为单位的家族组织
- 支部会议同时就是家族会议
- 要靠白色势力的非地方主义的压迫才能打破

**土客籍问题：**
- 土籍本地人和客籍人之间存在很大界限
- 历史仇怨非常深，有时发生激烈斗争
- 反映到党内，时常发生无谓斗争

**投机分子的反水：**
- 革命高涨时混入党内，白色恐怖到来就反水
- 九月以后厉行洗党，重新登记
- 党员数量减少，战斗力反而增加

#### 7. 革命性质问题

**明确革命阶段：**
- 中国现时处在资产阶级民权革命的阶段
- 包括：推翻帝国主义、完成土地革命、消灭封建关系、推翻军阀政府
- 必定要经过这样的民权主义革命，方能造成过渡到社会主义的基础

**对小资产阶级的政策：**
- 二月以前执行得比较好
- 三月湘南特委批评太右，改变政策
- 对城市中等商人没收和乡村小地主富农派款，做得十分厉害
- 把小资产阶级大部驱到豪绅一边
- 近来逐渐改变，情形渐渐好些

#### 8. 割据地区问题

**罗霄山脉中段的优势：**
1. 有经营了一年多的群众基础
2. 党的组织有相当的基础
3. 创造了富有斗争经验的地方武装
4. 有很好的军事根据地——井冈山
5. 影响两省，且能影响两省的下游

**中段的缺点：**
- 割据已久，"围剿"军多
- 经济问题，特别是现金问题，十分困难

**坚持井冈山的意义：**
- 边界的红旗子始终不倒，表示了共产党的力量
- 表示了统治阶级的破产
- 在全国政治上有重大的意义

## 核心收获

### 1. 红色政权存在的理论

毛泽东在《中国的红色政权为什么能够存在？》中提出问题，在这篇报告中用实践回答了问题：

**客观条件：**
- 中国买办豪绅阶级间的分裂和战争
- 白色政权的不统一

**主观条件：**
- 有很好的群众、党、红军
- 有有利的地势和给养

**策略条件：**
- 正确区分统治阶级稳定与破裂时期
- 采取不同的战略策略

### 2. "支部建在连上"的重大意义

这是毛泽东对军队建设的重大创新：

- **组织基础**：连有支部，班有小组，党在军队中有坚实的组织基础
- **思想保证**：保证党对军队的绝对领导
- **战斗力保证**：红军艰难奋战而不溃散的重要原因

对比：两年前在国民党军中，每团只有一个支部，经不起严重考验

### 3. 军队民主主义的威力

红军物质生活菲薄，战斗频繁，仍能维持不敝，靠的是：

- 官长不打士兵，官兵待遇平等
- 士兵有开会说话的自由
- 废除烦琐礼节，经济公开
- 士兵管理伙食，有"伙食尾子"

**效果**：
- 俘虏兵感觉红军和白军是两个世界
- 虽然物质不如白军，但精神得到解放
- 昨天在敌军不勇敢，今天在红军很勇敢

**启示**：中国不但人民需要民主主义，军队也需要民主主义

### 4. 中间阶级的策略问题

这是井冈山斗争中的重大教训：

**革命初期**：中间阶级表面投降，实际阻挠
**革命高涨**：才敢于中间阶级斗争
**白色恐怖**：中间阶级马上反水

**原因**：
- 受到革命过重打击
- 经济封锁下日常生活压迫

**教训**：
- 争取中间阶级极其重要
- 政策不能过左
- 要保护中小商人利益

### 5. 调查研究的典范

这篇报告是调查研究的经典范例：

- 用数据说话（土地分配比例、枪支数量、兵员来源）
- 总结实践经验（四月到七月的发展，八月的失败）
- 分析具体问题（土客籍问题、地方主义问题）
- 提出具体建议（需要中央派人来、送药品等）

### 6. 革命阶段的清醒认识

毛泽东对中国革命性质有清醒认识：

- 明确指出中国还处在资产阶级民权革命阶段
- 不能跳过这个阶段直接搞社会主义
- 要通过民权革命造成过渡到社会主义的基础

这为后来新民主主义理论奠定了基础。

## 思考与感悟

### 1. 理论与实践的结合

如果说《实践论》从哲学高度论述了认识来源于实践，那么《井冈山的斗争》就是"从实践到认识"的典范：

- 不是从书本出发，而是从一年多的斗争实践出发
- 总结成功经验（四月到七月）和失败教训（八月）
- 形成系统性认识（红色政权理论、军队建设理论、土地革命理论）

这种从实践中来、到实践中去的认识路线，正是马克思主义中国化的生动体现。

### 2. 对比《中国的红色政权为什么能够存在？》

这两篇文章是姊妹篇：

**《红色政权》**（1928年10月）：
- 提出问题：为什么能够存在？
- 分析客观原因：白色政权的分裂和战争
- 指出五个条件

**《井冈山的斗争》**（1928年11月）：
- 回答问题：如何存在和发展？
- 总结实践经验：成功和失败
- 形成系统策略：军事、政治、经济、党的建设

从"为什么"到"怎么做"，体现了认识的深化过程。

### 3. "支部建在连上"的现代启示

这个制度创新对今天依然有启发：

**组织建设**：
- 基层组织是基础，"基础不牢，地动山摇"
- 要把组织建设到最基层（连、班）
- 党员要发挥先锋模范作用

**领导方式**：
- 党的领导要深入基层，不能浮在上面
- 党代表（指导员）要既懂政治又懂业务
- 政治工作和业务工作要结合

### 4. 中间阶级问题的普遍性

中间阶级（小资产阶级）的动摇性是革命中的普遍现象：

- 受到革命冲击时会不满
- 面临白色恐怖时会退缩
- 经济困难时会动摇

**对今天工作的启示**：
- 要团结大多数，不能只依靠少数基本群众
- 政策不能过左，要照顾中间阶层利益
- 要善于争取和转化中间力量

### 5. 民主主义的双重意义

毛泽东强调军队需要民主主义，这个观点非常深刻：

**对内**：
- 调动士兵积极性
- 增强军队凝聚力
- 提高战斗力

**对外**：
- 区别于旧军队
- 吸引敌军士兵
- 赢得人民拥护

启示我们：民主不仅是政治原则，也是管理方法，是调动人的积极性的有效手段。

### 6. 实事求是的方法论

这篇报告处处体现实事求是的精神：

- 不夸大成绩（承认八月失败）
- 不回避问题（经济困难、投机分子反水）
- 具体分析（土客籍问题、地方主义问题）
- 提出切实建议（要人、要药、要指导）

这种实事求是的态度，是共产党人应该始终坚持的。
