## 📋 段落结构规范

### 基本要求

| 检查项 | 标准 | 验证方法 |
|--------|------|---------|
| 单句成段比例 | 30%-50% | 统计单句成段数量 / 总段落数 |
| 每段字数 | 50-100字 | 统计平均段落长度 |
| 重点信息独立成段 | ✅ | 人工检查 |
| 场景转换 | 空白行分隔 | 人工检查 |

### 段落规范详解

#### ✅ 单句成段（30%-50%）
- **目的**: 提高阅读速度，降低阅读疲劳
- **示例**:
  ```
  沈清婉冷笑一声。
  （单句成段，节奏紧凑）

  她知道王宫女的心思，却不说破。
  （单句成段，重点突出）
  ```

#### ✅ 每段50-100字
- **目的**: 控制视觉长度，避免长段落吓退读者
- **示例**:
  ```
  沈清婉冷笑一声，她知道王宫女的心思，却不说破。王宫女看着她的眼神，心中暗想："这个废后怎么还这么自信？难道她不知道太子已经不喜欢她了吗？"（80字，符合规范）
  ```

#### ✅ 重点信息独立成段
- **目的**: 突出关键信息，提高阅读效率
- **示例**:
  ```
  沈清婉听到了一个声音。

  "太子殿下心中想的是废后，不是我。"王宫女的心声。

  （重点信息独立成段）
  ```

#### ✅ 场景转换使用空白行分隔
- **目的**: 帮助读者理解场景转换
- **示例**:
  ```
  沈清婉离开御花园，回到了自己的寝宫。

  （空白行分隔场景转换）

  寝宫内，李公公正在等候。
  ```

#### ⛔ 禁止数字标记分段
- **目的**: 保持阅读沉浸感
- **错误示例**:
  ```
  一、沈清婉冷笑一声。
  二、王宫女看着她的眼神。
  三、李公公走了进来。
  ```
- **正确做法**: 直接用段落分隔，不要用数字标记

## 🎭 去 AI 腔规范（基于腾讯朱雀标准）

### 禁止的写作模式

#### ❌ 禁止堆砌感官描写
- **错误示例**:
  ```
  沈清婉闻到了花香，听到了鸟叫，看到了花开，心中暗想："春天真的来了。"
  ```
- **正确做法**: 不强行凑"3种感官"，自然描写
  ```
  御花园里春意正浓。沈清婉停下脚步，看着盛开的牡丹，嘴角微微上扬。
  ```

#### ❌ 禁止华丽套话
- **禁止词汇**:
  - `摇摇欲坠`（用"差点倒下"替代）
  - `空气凝固`（用"没人敢说话"替代）
  - `弥漫着`（用"充满了"或直接删除替代）
  - `心中一紧`（用"很紧张"替代）
  - `心中一颤`（用"很害怕"替代）

- **错误示例**:
  ```
  沈清婉心中一紧，摇摇欲坠的身体仿佛要倒下，空气弥漫着紧张的气氛。
  ```
- **正确做法**:
  ```
  沈清婉很紧张，差点倒下。没人敢说话，气氛很压抑。
  ```

#### ❌ 禁止 AI 高频词
- **禁止词汇**:
  - `唯一的`（用"只有"替代）
  - `直到`（用"一直等到"替代）
  - `不禁`（用"忍不住"替代或直接删除）
  - `顿时`（用"马上"替代）
  - `心中暗想`（用直接心理描写替代）
  - `心中`（大部分时候可以删除）
  - `一瞬间`（用"立刻"替代）

- **错误示例**:
  ```
  沈清婉心中暗想："这是唯一的办法。"不禁松了口气，顿时感觉轻松了很多。
  ```
- **正确做法**:
  ```
  沈清婉想："这是唯一的办法。"她松了口气，马上感觉轻松了很多。
  ```

### 推荐的写作模式

#### ✅ 用行为暗示替代直白心理描写
- **错误示例**:
  ```
  沈清婉很生气，她狠狠地瞪了王宫女一眼。
  ```
- **正确做法**:
  ```
  沈清婉的手指掐进了掌心，但她没有说话，只是狠狠地瞪了王宫女一眼。
  ```

#### ✅ 对话加入停顿、语病，更口语化
- **错误示例**:
  ```
  "你为什么要陷害我？"沈清婉问道，语气中充满了愤怒。
  ```
- **正确做法**:
  ```
  "你……为什么要陷害我？"沈清婉的声音在发抖。
  ```

#### ✅ 具象化替代抽象
- **错误示例**:
  ```
  很多人都在议论这件事。
  ```
- **正确做法**:
  ```
  至少有五个宫女在议论这件事。
  ```
- **错误示例**:
  ```
  最近太子很少来御花园。
  ```
- **正确做法**:
  ```
  从上周三开始，太子就没来过御花园。
  ```

## 📊 质量检查清单

### 段落结构检查

- [ ] 单句成段比例在30%-50%之间
- [ ] 平均段落长度在50-100字之间
- [ ] 重点信息独立成段
- [ ] 场景转换使用空白行分隔
- [ ] 没有使用"一"、"二"、"三"等数字标记分段

### 去 AI 腔检查

- [ ] 没有堆砌感官描写
- [ ] 没有使用"摇摇欲坠"、"空气凝固"、"弥漫着"等华丽套话
- [ ] 没有使用"唯一的"、"直到"、"不禁"、"顿时"、"心中暗想"等 AI 高频词
- [ ] 用行为暗示替代了部分直白心理描写
- [ ] 对话加入了停顿、语病，更口语化
- [ ] 用具象化替代了抽象表达

## 🎯 使用建议

### 写作前
1. 阅读本规范，熟悉段落结构和去AI腔要求
2. 重点关注单句成段比例（30%-50%）和每段字数（50-100字）
3. 重点关注禁止的 AI 高频词（"唯一的"、"直到"、"不禁"、"顿时"、"心中暗想"）

### 写作中
1. 每写完一段，检查是否符合段落结构规范
2. 每写完一段，检查是否使用了禁止的 AI 高频词
3. 每写完一段，检查是否堆砌了感官描写

### 写作后
1. 使用质量检查清单全面检查
2. 统计单句成段比例是否在30%-50%之间
3. 统计平均段落长度是否在50-100字之间
4. 检查是否使用了禁止的 AI 高频词
5. 如有不符合规范的地方，立即修改
