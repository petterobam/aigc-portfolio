# 用户指令手册

## 基础指令

### 继续学习

触发当天学习内容推进，完成后自动保存日志并推送。

```
继续学习
```

---

### 查看进度

显示当前学习天数、完成百分比、连续学习天数（streak）。

```
学习进度
```

**输出示例：**
```
📊 学习进度
当前：第 17 天 / 共 35 天（48.6%）
连续学习：16 天
上次学习：2026-03-16
最近完成：功能调用（Function Calling）
```

---

### 查看历史

查询某天的学习内容或已完成的知识点列表。

```
学习历史
```

```
学习历史 第5天
```

```
学习历史 第10天到第12天
```

---

### 跳转天数

跳过或回溯到指定天数，不影响已有日志，但会覆盖 `progress.json` 中的 `currentDay`。

```
跳转到第X天
```

**示例：**
```
跳转到第20天
```

---

### 重置进度

将进度归零（`currentDay` 重置为 1），streak 清零，已有日志文件保留不删除。

```
重置学习进度
```

---

## 日志指令

### 推送指定天日志

直接上传某天的日志文件到飞书，不重新生成内容。

```
推送第X天日志
```

**示例：**
```
推送第3天日志
```

---

### 批量推送日志

推送连续多天的日志文件。

```
推送第X天到第Y天日志
```

**示例：**
```
推送第1天到第5天日志
```

---

### 补推遗漏日志

查找 `daily-logs/` 目录中存在但未推送的文件并批量补推。

```
补推遗漏日志
```

---

## 内容指令

### 回顾知识点

针对某个主题或天数进行知识点复习，不更新进度。

```
回顾 [主题或天数]
```

**示例：**
```
回顾 DPO
回顾 第11天
回顾 LoRA
```

---

### 深入讲解

对某个知识点进行更详细的展开，不更新进度。

```
深入讲解 [主题]
```

**示例：**
```
深入讲解 PPO-Clip
深入讲解 KV Cache
```

---

### 代码示例

获取某个知识点的代码示例片段。

```
代码示例 [主题]
```

**示例：**
```
代码示例 LoRA
代码示例 DPO 训练
```

---

## 指令速查表

| 指令 | 说明 | 是否更新进度 |
|------|------|------------|
| `继续学习` | 推进当天学习 + 保存日志 + 推送 | ✅ 是 |
| `学习进度` | 查看当前进度概览 | ❌ 否 |
| `学习历史 [第N天]` | 查看历史学习记录 | ❌ 否 |
| `跳转到第X天` | 手动设置当前天数 | ✅ 覆盖 |
| `重置学习进度` | 归零进度，保留日志 | ✅ 清零 |
| `推送第X天日志` | 上传指定日志文件 | ❌ 否 |
| `推送第X天到第Y天日志` | 批量上传日志文件 | ❌ 否 |
| `补推遗漏日志` | 补推未推送的日志 | ❌ 否 |
| `回顾 [主题]` | 复习知识点 | ❌ 否 |
| `深入讲解 [主题]` | 展开详细讲解 | ❌ 否 |
| `代码示例 [主题]` | 获取代码片段 | ❌ 否 |
