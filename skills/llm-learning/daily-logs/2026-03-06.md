## 核心知识点

### 1. LLM 训练的两大阶段

**预训练（Pre-Training）**
- 目标：让模型学会语言基础（语法、语义、知识）
- 数据：海量无标注文本
- 特点：
  - 数据量最大
  - 最消耗算力
  - 以自监督学习为主
- 包括：
  - 初期训练（短上下文）
  - 中期训练（长上下文）
  - 退火训练（Annealing）

**后训练（Post-Training）**
- 目标：提升模型的高级能力和对齐性
- 数据：高质量指令数据、偏好数据
- 方法多样化：
  - SFT（监督微调）
  - 蒸馏
  - RSFT（拒绝采样微调）
  - RLHF（基于人类反馈的强化学习）
  - DPO（直接偏好优化）
  - GRPO（群体相对策略优化）
  - PPO（近端策略优化）
- 特点：
  - 可以多轮迭代
  - 数据量比预训练小
  - 更关注质量和对齐

### 2. 训练的核心机制

**前向传播（Forward Propagation）**
```
输入 → Embedding → Transformer层 → 输出
```
- 逐层计算，生成预测结果
- 每层都进行自注意力计算和前馈网络处理
- 输出：Logits（未归一化的得分）

**反向传播（Backward Propagation）**
```
Loss → 梯度计算 → 参数更新
```
- 计算损失与预测之间的误差
- 反向计算每个参数的梯度
- 更新模型参数以减少损失

**关键点：前向必须缓存激活值**
- 原因：反向传播需要用到前向传播的中间结果
- 作用：链式求导需要每一层的输出
- 优化：
  - 完整缓存：显存占用大，速度快
  - Gradient Checkpoint：部分缓存，重新计算，省显存
  - Full Recomputation：不缓存，全部重新计算，最省显存

### 3. 交叉熵损失函数

**公式**
```
L = -Σ y_i · log(ŷ_i)
```

**直观理解**
- y_i：正确标签（1表示正确token，0表示错误）
- ŷ_i：模型预测概率（0-1之间）
- log(ŷ_i)：对概率取对数
- 负号：因为log概率是负数，取负使损失为正

**示例**
```
假设词表有3个词：["我", "是", "人"]
正确答案："我" → y = [1, 0, 0]
模型预测：ŷ = [0.7, 0.2, 0.1]

L = -(1·log(0.7) + 0·log(0.2) + 0·log(0.1))
  = -(-0.357 + 0 + 0)
  = 0.357

如果预测更准：ŷ = [0.9, 0.05, 0.05]
L = -(1·log(0.9) + 0 + 0) = 0.105

损失越小 → 预测越准
```

**为什么用交叉熵？**
- 正确token概率越高，损失越小
- 错误token概率越低，损失越小
- 指导模型学会生成人类期望的文本
- 对概率分布敏感，能区分细微差异
