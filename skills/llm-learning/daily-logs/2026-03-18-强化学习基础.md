## 📚 今日核心知识点

### 1. 奖励、回报、价值的区别

**奖励（Reward）**：
- 即时奖励，某一状态下的局部收益或短期收益
- 记作 r_t，表示第 t 步获得的即时反馈

**回报（Return）**：
- 未来回报，从当前状态 s_t 开始计算的奖励累积和
- 记作 G_t = r_t + γ·r_{t+1} + γ²·r_{t+2} + ... + γ^{T-t}·r_T
- γ 是折扣因子，0 < γ ≤ 1，表示未来奖励的重要程度

**价值（Value）**：
- 回报的期望值，所有可能轨迹中回报乘以其发生概率的加权平均值
- 记作 V_π(s) 或 Q_π(s,a)
- 表示在平均情况下的未来总收益

**核心区别**：
- 奖励 = 即时的（单步）
- 回报 = 累积的（完整轨迹）
- 价值 = 期望的（统计平均）

---

### 2. 价值函数 Q_π 与 V_π 的关系

**状态价值函数 V_π(s)**：
- 在策略 π 下，智能体处于状态 s 时，未来能获得的回报期望
- 记作：V_π(s) = E[G_t | S_t = s, π]
- 关注"状态本身"的价值

**动作价值函数 Q_π(s,a)**：
- 在策略 π 下，智能体在状态 s 采取动作 a 后，未来能获得的回报期望
- 记作：Q_π(s,a) = E[G_t | S_t = s, A_t = a, π]
- 关注"状态-动作对"的价值

**关系式**：
```
V_π(s) = Σ_a π(a|s) · Q_π(s,a)
```
- 状态价值是对所有可能动作的加权平均
- 权重是策略 π 下选择动作 a 的概率

---

### 3. Bellman 方程

**Bellman 方程**是强化学习的基石，描述了价值函数的递归关系。

**Bellman 期望方程**（针对 V_π）：
```
V_π(s) = E[R_t + γ·V_π(S_{t+1}) | S_t = s]
```

展开形式：
```
V_π(s) = Σ_a π(a|s) Σ_{s', r} p(s', r | s, a) [r + γ·V_π(s')]
```

**核心思想**：
- 当前状态的价值 = 即时奖励的期望 + 折扣后的下一状态价值
- 递归定义：价值函数可以通过自身来定义

**Bellman 方程的作用**：
- 将无限时序问题转化为有限递归问题
- 提供价值函数的计算和更新方法
- 是动态规划、TD 方法、Q-learning 等算法的理论基础

---

### 4. TD 目标与 TD 误差

**TD Target（时序差分目标）**：
```
TD_target = r_t + γ·V(S_{t+1})
```
- 包含最新的环境反馈信息 r_{t+1}
- 被认为比 V(s_t) 更接近真实值
- 作为价值函数拟合的目标

**TD Error（时序差分误差）**：
```
δ_t = TD_target - V(s_t) = r_t + γ·V(S_{t+1}) - V(s_t)
```
- 衡量当前价值估计与 TD 目标之间的差异
- 反映当前估计与最新信息的不一致程度
- **指导价值函数的更新方向和幅度**

**价值函数更新**：
```
V(s_t) ← V(s_t) + α · δ_t
```
- α 是学习率
- 通过 TD 误差调整价值估计

---

### 5. TD(0)、多步 TD 与蒙特卡洛的关系

**TD(0)**（单步 TD）：
```
TD(0): δ_t = r_t + γ·V(S_{t+1}) - V(s_t)
```
- 只利用下一状态的价值估计
- 高偏差、低方差
- 在线学习，每步更新

**多步 TD**（n-step TD）：
```
n-step TD: δ_t^{(n)} = (r_t + γ·r_{t+1} + ... + γ^n·V(S_{t+n})) - V(s_t)
```
- 利用 n 步后的价值估计
- n=1 时退化为 TD(0)
- n→∞ 时退化为蒙特卡洛

**蒙特卡洛方法**：
```
MC: G_t = r_t + γ·r_{t+1} + ... + γ^{T-t}·r_T
```
- 使用完整轨迹的实际回报
- 低偏差、高方差
- 需要等到轨迹结束才能更新

**三者关系**：
```
n=1      → TD(0)
n=有限   → 多步 TD
n→∞      → 蒙特卡洛（MC）
```

**偏差-方差权衡**：
- TD：高偏差（用估计值估计）、低方差
- MC：低偏差（用真实回报）、高方差
- 多步 TD：通过 n 在偏差和方差之间取得平衡

## 💡 实践建议

1. **理解价值函数的递归本质**：Bellman 方程的关键在于"用未来的价值估计当前的价值"，这是 RL 算法的核心思想

2. **掌握 TD 方法**：TD 方法是 RL 实际应用中最常用的方法，因为它可以在线学习，不需要等待完整轨迹

3. **实践 TD 更新**：
   ```python
   # 简单的 TD(0) 更新示例
   def td_update(V, state, next_state, reward, gamma, alpha):
       td_target = reward + gamma * V[next_state]
       td_error = td_target - V[state]
       V[state] += alpha * td_error
       return V
   ```

4. **理解偏差-方差权衡**：
   - 数据有限时，用 TD（偏向偏差）
   - 数据充足时，用 MC（偏向方差）
   - 多步 TD 提供灵活的中间选择
