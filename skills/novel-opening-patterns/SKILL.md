# Novel Opening Patterns - 番茄小说开篇套路工具

## Overview

番茄小说开篇套路工具，基于41号故事《读心诡校，同学都在演戏》的爆款开篇分析，提供系统化的开篇创作和优化方法论。

核心能力：
- **5步开篇法**：开篇钩子 → 持续铺垫 → 部分揭露 → 规则恐怖 → 反转结尾
- **爆款元素库**：规则恐怖、诡异学校、读心术、真相探索等高流量元素
- **开篇检查清单**：自动化检查开篇必备元素
- **可复用的创作技巧**：悬念制造、对话设计、氛围营造、单句成段

## Core Capabilities - 核心能力

### 1. 开篇质量检查

检查开篇是否包含必备元素，识别问题并提供优化建议。

**检查清单**（详见 [checklist.md](references/checklist.md)）：

- [ ] **开篇钩子**：开篇第1-2行必须制造悬念
- [ ] **诡异氛围**：通过细节描写营造诡异氛围
- [ ] **持续铺垫**：多个细节暗示诡异，不直接揭露
- [ ] **部分揭露**：在开篇中段揭露部分真相
- [ ] **规则恐怖**：使用"规则"制造恐怖氛围（推荐）
- [ ] **反转结尾**：结尾揭示主角特殊身份
- [ ] **强烈悬念**：结尾留下强烈悬念，吸引读者看下一章

**可选元素**：
- [ ] 超能力觉醒（第2章或第3章）
- [ ] 读心术（快速获得信息，加速剧情）
- [ ] 身份反转（揭示主角特殊身份）
- [ ] 重置/轮回（增加恐怖感和剧情张力）
- [ ] 阴谋论（增加剧情深度）
- [ ] 单句成段（30-50%占比，增强节奏感）

使用经过验证的创作技巧增强开篇质量。

- **双关对话**：表面正常，实则诡异
- **关键词重复**：通过关键词暗示真相
- **信息碎片化**：通过多个碎片信息拼凑真相

#### 氛围营造技巧
- **细节描写**：通过细节暗示诡异
- **节奏控制**：快节奏+慢节奏交替
- **留白艺术**：不说破，留出想象空间

- 占比：30-50%
- 效果：增强节奏感，适合移动端阅读

## Avoid Common Pitfalls - 避坑指南

### ❌ 不要做的事

1. **开篇直接说"这是诡异学校"**
   - 原因：失去悬念，读者没有探索欲
   - 正确做法：通过细节暗示，让读者自己发现

2. **一次性揭露所有真相**
   - 原因：没有后续悬念，读者不想继续看
   - 正确做法：部分揭露，留出想象空间

3. **没有规则恐怖**
   - 原因：失去番茄小说的爆款元素
   - 正确做法：使用"规则"制造恐怖氛围

4. **单句成段占比过低**
   - 原因：节奏感差，移动端阅读体验不好
   - 正确做法：单句成段占比30-50%

5. **开篇没有钩子**
   - 原因：前200字没抓住读者，读者就走了
   - 正确做法：开篇第1行就制造悬念

## Best Practices - 最佳实践

1. **开篇黄金法则**：开篇第1-2行必须制造悬念
2. **持续悬念**：每200字都要有新信息或新悬念
3. **结尾强钩子**：结尾必须留下强烈悬念，吸引读者看下一章
4. **爆款元素组合**：融合至少3个高流量元素
5. **单句成段占比**：30-50%，增强节奏感
6. **移动端优化**：短段落、快节奏、高悬念
