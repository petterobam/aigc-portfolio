# 番茄短篇故事优化助手

基于29个已发布故事的数据分析，为低表现作品生成优化方案。

## 成功公式

```
明确金手指(40%) + 社会共鸣强(30%) + 标题悬念(20%) + 合适字数(10%) = 阅读量
```

## 核心优化原则

### 1. 金手指必须在标题中明确体现
- **优先类型**：读心术、重生+复仇、隐藏身份
- **避免**：金手指模糊、类型不明确
- **数据验证**：明确金手指作品平均阅读6.5次，不明确仅1.5次

### 2. 标题长度优化
- **最佳**：≤15字（平均阅读6.5次）
- **可接受**：16-25字（平均阅读4.8次）
- **避免**：>25字（平均阅读3.2次）

### 3. 开篇节奏优化
- **前500字必须出现金手指觉醒**
- **第一个爽点在1000字内出现**
- **冲突密度：3-5个冲突/章**

### 4. 题材选择
- **优先**：重生复仇（平均7.2阅读）、读心术（平均6.5阅读）、家庭矛盾（平均11.3阅读）
- **谨慎**：规则怪谈（竞争激烈）
- **避免**：科幻、历史穿越（平均<1阅读）

### 5. 字数控制
- **最佳范围**：10k-15k字
- **避免**：>20k字（负面影响明显）

## 使用流程

### 场景1：为单个0阅读作品生成优化方案

```bash
# 运行优化脚本
node ${skillDir}/../scripts/optimize-story.js [story-id]
```

脚本会自动：
1. 读取故事数据（从 data/all-stories-*.json）
2. 分析问题诊断
3. 生成3个备选新标题
4. 提供开篇优化建议
5. 输出完整优化报告

### 场景2：批量优化多个低表现作品

```bash
# 批量优化所有0阅读作品
node ${skillDir}/../scripts/batch-optimize.js --filter zero-reading

# 批量优化阅读量<3的作品
node ${skillDir}/../scripts/batch-optimize.js --filter low-reading
```

### 场景3：新作品发布前的质量检查

```bash
# 检查新故事是否符合成功公式
node ${skillDir}/../scripts/validate-new-story.js <story-file>
```

检查项：
- ✅ 标题长度是否≤15字
- ✅ 金手指是否明确
- ✅ 开篇前500字是否出现金手指觉醒
- ✅ 题材是否属于高表现类型
- ✅ 字数是否在10k-15k范围

## 问题诊断模板

当接到优化请求时，按此顺序诊断：

### 1. 检查金手指明确性
- 标题中是否明确提到金手指类型？（读心术、重生、隐藏身份等）
- 开篇前500字是否出现金手指觉醒？
- 金手指是否贯穿全文并持续发挥作用？

### 2. 检查社会共鸣强度
- 题材是否涉及家庭矛盾、社会热点、现实话题？
- 是否有"读者能代入"的情节？
- 是否避免了小众题材（科幻、历史穿越）？

### 3. 检查标题质量
- 标题长度是否≤15字？
- 是否同时体现金手指和爽点？
- 是否有悬念或冲突？

### 4. 检查开篇节奏
- 前1000字是否出现第一个爽点？
- 冲突密度是否足够（3-5个/章）？
- 是否快速建立主角优势？

### 5. 检查字数合理性
- 字数是否在10k-15k范围？
- 是否有冗长拖沓的章节？

## 优化方案生成

### 标题优化（生成3个备选）

**原则**：
1. 长度≤15字
2. 明确金手指类型
3. 包含爽点或悬念

**模板**：
- `模板1`：[金手指] + [爽点]
  - 例："能听到老板心声后，我带走了所有客户"
- `模板2`：[共鸣] + [金手指]
  - 例："婆婆逼我离婚后，我觉醒读心术了"
- `模板3`：[冲突] + [金手指]
  - 例："背锅后觉醒读心术，我带客户全走了"

### 开篇优化（重写前500字）

**优化前检查**：
- 金手指在第几百字出现？
- 第一个爽点在第几百字出现？

**优化目标**：
- 金手指在≤100字出现
- 第一个爽点在≤400字出现

**优化技巧**：
- 直接在开头插入金手指觉醒场景
- 使用"心声"提示符号：【心声内容】
- 立即建立"外表vs内心"的对比
- 快速让主角获得优势

### 题材调整建议

**如果当前题材表现差**：
- 规则怪谈 → 改为读心术+规则怪谈（双重金手指）
- 末世求生 → 改为重生+读心术（前置优势）
- 历史穿越 → 改为现代重生复仇

**数据依据**：
- 重生复仇类：平均7.2阅读
- 读心术类：平均6.5阅读
- 家庭矛盾类：平均11.3阅读
- 规则怪谈类：平均0.5阅读
- 历史穿越类：平均0.3阅读
