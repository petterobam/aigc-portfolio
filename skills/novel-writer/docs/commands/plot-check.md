# `/novel:plot-check` — 情节一致性检查

**触发条件**：用户输入 `/novel:plot-check`

**执行脚本**：`.specify/scripts/bash/check-plot.sh`

---

## 检查维度

1. **情节逻辑链**：事件因果是否完整，有无无解释的突然转变
2. **伏笔闭合状态**：已埋设未回收的伏笔，超期的 critical 级伏笔
3. **角色行为一致性**：行为是否符合性格设定和已知信息
4. **能力自洽**：特殊能力/金手指是否有无来由的变化

---

## 参数

```
/novel:plot-check             # 全部检查
/novel:plot-check 第1-5章     # 指定范围
/novel:plot-check 伏笔         # 只检查伏笔闭合状态
/novel:plot-check 角色         # 只检查角色行为一致性
```

---

## 输出格式

```markdown
## 情节一致性检查报告

### ✅ 一致的部分（N项）

### ⚠️ 需关注（N项）
  - [章节]：[问题描述]
    建议：[解决方向]

### ❌ 明确矛盾（N项）
  - [章节X] vs [章节Y]：[冲突描述]
    建议修改：[具体方案]

### 🌱 未回收伏笔清单
  | 伏笔 | 埋入章节 | 计划回收 | 距截止 | 重要度 |

### 下一步建议
```
