# `/novel:timeline` — 时间线管理

**触发条件**：用户输入 `/novel:timeline`

**数据文件**：`spec/tracking/timeline.json`

**执行脚本**：`.specify/scripts/bash/check-timeline.sh`

---

## 子命令

```
/novel:timeline show              # 展示完整时间线可视化表
/novel:timeline check             # 检查时间矛盾
/novel:timeline add <事件描述>    # 手动添加节点
/novel:timeline fix <矛盾ID>      # 修复指定矛盾
```

---

## 时间线检查项

- 旅行时间合理性（A到B的时间是否与描述一致）
- 年龄随时间线的变化一致性
- 季节/天气描写与时间线的一致性
- "X天后"表述的数学正确性
- 并行事件的时间重叠是否冲突

---
