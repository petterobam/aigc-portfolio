# 任务管理

这个目录包含记忆优化器的所有任务和计划。

---

## 📂 文件结构

```
tasks/
├── README.md - 任务管理说明
├── task-list.md - 任务列表（主要文件）
└── (未来可能添加)
    ├── completed/ - 已完成任务
    ├── in-progress/ - 进行中任务
    └── archived/ - 归档任务
```

---

## 🎯 任务管理规则

### 1. 优先级定义

| 优先级 | 说明 | 时间框架 |
|--------|------|----------|
| **P0** | 正在进行，必须完成 | 立即 |
| **P1** | 高优先级 | 本周 |
| **P2** | 中优先级 | 本月 |
| **P3** | 低优先级 | 下月 |

### 2. 任务状态

- [ ] 待开始
- [x] 已完成
- ⏳ 进行中
- ❌ 已取消
- 🔄 需要重做

### 3. 更新频率

- **每周更新**：回顾任务进度，调整优先级
- **每月更新**：添加新任务，归档旧任务
- **每季度更新**：重新评估长期任务

---

## 📝 使用方法

### 添加新任务

1. 打开 `task-list.md`
2. 找到合适的分类（P0/P1/P2/P3）
3. 添加任务和子任务
4. 评估预估时间

### 更新任务状态

1. 勾选已完成的子任务
2. 更新任务统计
3. 记录到"任务更新记录"

### 归档任务

1. 将已完成/取消的任务移到 `completed/` 或 `archived/`
2. 更新 `task-list.md`
3. 更新任务统计
