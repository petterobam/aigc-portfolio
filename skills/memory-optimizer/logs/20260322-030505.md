- 如果同一天执行多次备份，会生成多个同名文件（但时间戳不同）

**潜在改进方向**:
1. 在文件名中加入时间戳: `memory-db-20260322-030505.tar.gz`
2. 使用小时-分钟-秒精度: `memory-db-2026-03-22-0305.tar.gz`
3. 使用序号: `memory-db-2026-03-22-01.tar.gz`, `memory-db-2026-03-22-02.tar.gz`

**建议**:
使用更精确的时间戳格式，例如：
```javascript
const timestamp = new Date().toISOString().replace(/[:.]/g, '-').split('T')[0] + '-' +
                 new Date().toTimeString().split(' ')[0].replace(/:/g, '');
// 结果示例: 20260322-030505
```
