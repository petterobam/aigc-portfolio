### 3. 识别向量生成失败问题

**问题**: 记忆 [1] 向量生成失败

**错误信息**: `Ollama API 错误: the input length exceeds the context length`

**原因分析**:
- gemma:2b 模型的上下文窗口限制较小（可能是 2048 或 4096 tokens）
- 记忆 [1] 的内容可能过长，超过了模型的上下文限制

**解决方案**:
1. **截断长内容**：只使用内容的前 N 个字符生成向量（例如 2000 字符）
2. **使用更大的模型**：切换到上下文窗口更大的模型（例如 gemma:7b, mistral, llama2）
3. **使用摘要代替内容**：优先使用 summary 字段生成向量，如果为空再使用 content
4. **分段向量化**：将长内容分成多个段落，分别生成向量，然后取平均

**推荐方案**: 优先使用 summary 字段，如果为空再截断 content 的前 2000 字符
