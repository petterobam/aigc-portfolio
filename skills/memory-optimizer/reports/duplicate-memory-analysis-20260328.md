## 附录

### A. 相似度计算方法

**向量相似度计算**：
- 使用 Ollama 生成文本向量（模型：gemma:2b）
- 计算余弦相似度（Cosine Similarity）
- 相似度 > 90% 认为是语义重复

**哈希相似度计算**：
- 使用 SHA-256 计算内容哈希
- 哈希相同认为是完全重复（本次未发现）

### B. 评分系统说明

**评分计算公式**：
```
importance_score =
  access_frequency * 0.30 +
  recency * 0.15 +
  content_quality * 0.25 +
  category_relevance * 0.20 +
  social_resonance * 0.10
```

**评分范围**：0.0 - 5.0
**归档阈值**：minImportance = 1.2

### C. 归档策略

**归档条件**：
- combined_days > 14（超过 14 天未访问）
- importance_score < 1.2（重要性低于阈值）
- 不包含 'protected' 标签

**归档后行为**：
- category 更新为 'archived'
- 不参与评分计算
- 不参与访问追踪
- 可手动恢复
