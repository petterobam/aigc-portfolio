## 配置参数（当前生效）

```javascript
{
  archiveAfterDays: 14,        // 超过 14 天未访问则归档
  minImportance: 1.2,          // 重要性低于 1.2 才会被归档（评分范围 0-5）
  minContentLength: 50,        // 内容少于 50 字符才会被删除
  protectedTag: 'protected',   // 此标签的记忆永不处理
  deleteArchived: false,       // 归档记忆不自动删除
  scoreUpdateThreshold: 0.2    // 评分变化超过此值才更新数据库
}
```

> 修改配置请编辑 `scripts/config.js` 中的 OPTIMIZE_CONFIG 对象
