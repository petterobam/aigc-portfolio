# 测试文件目录

本目录包含记忆优化系统的测试文件和调试脚本。

## 测试文件列表

### debug-score.js
- **用途**: 评分系统调试脚本
- **功能**: 测试评分计算逻辑，验证评分公式的准确性
- **使用**: `node debug-score.js`

### test-mem-1.js
- **用途**: 记忆系统基础测试
- **功能**: 测试记忆的创建、读取、更新、删除等基础操作
- **使用**: `node test-mem-1.js`

### test-score-calculation.js
- **用途**: 评分计算详细测试
- **功能**: 详细测试评分计算的各个维度（访问频率、时效性、内容长度、标题质量、类别权重、关键词密度）
- **使用**: `node test-score-calculation.js`

### test-vector.js
- **用途**: 向量生成和相似度计算测试
- **功能**: 测试 Ollama 向量生成、余弦相似度计算、语义重复检测
- **使用**: `node test-vector.js`

## 使用建议

1. **运行所有测试**: 建议在修改评分系统或向量系统后运行相关测试
2. **调试问题**: 使用 debug-score.js 快速定位评分计算问题
3. **验证功能**: 使用 test-score-calculation.js 详细验证评分计算的每个维度
4. **测试向量**: 使用 test-vector.js 测试向量生成和相似度计算

## 注意事项

- 所有测试文件都是独立的，可以直接运行
- 测试不会修改主数据库，使用测试数据进行验证
- 运行测试前请确保 Ollama 服务已启动（用于 test-vector.js）
