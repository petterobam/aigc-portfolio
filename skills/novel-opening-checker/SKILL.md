# Novel Opening Checker

## 概述

番茄小说开篇质量检查和优化工具，基于41号爆款故事的5步开篇法，提供自动化检查和详细参考文档。

**核心目标**：提高开篇吸引力和前3章留存率，实现>70%留存率目标。

**适用类型**：悬疑恐怖、校园灵异、读心术、规则恐怖、真相探索等高流量类型。

---

## 快速开始

### 1. 自动化检查开篇

```bash
node scripts/check-opening.js <开篇章节文件路径>
```

示例：
```bash
node scripts/check-opening.js content/chapter-1.md
node scripts/check-opening.js story-41/content/chapter1.md
```

**检查内容**：
- ✅ 开篇必备元素（7个）
- ✅ 单句成段占比（推荐30-50%）
- ✅ 5步开篇法执行情况
- ✅ 开篇优化建议

### 2. 参考5步开篇法

详见 [references/five-step-opening.md](references/five-step-opening.md)

**5步结构**：

## 开篇质量标准

### 必备元素（7个）

| 元素 | 说明 | 检查方法 |
|------|------|---------|
| 开篇钩子 | 开篇第1-2行必须制造悬念 | 是否包含"口误"、"诡异"等关键词 |
| 诡异氛围 | 通过细节描写营造诡异氛围 | 是否包含"诡异"、"阴森"等氛围词 |
| 持续铺垫 | 多个细节暗示诡异，不直接揭露 | 是否有≥2处细节暗示 |
| 部分揭露 | 在开篇中段揭露部分真相 | 是否包含"演戏"、"剧本"等揭露词 |
| 规则恐怖 | 使用"规则"制造恐怖氛围 | 是否包含"规则"、"禁止"等规则词 |
| 反转结尾 | 结尾揭示主角特殊身份 | 是否揭示主角身份 |
| 悬念密度 | 每章3-4个爽点/悬念 | 每章是否有≥3个悬念点 |

### 单句成段占比

**推荐范围**：30-50%
**优秀开篇**：41号故事，单句成段占比30-50%

**计算方法**：
```
单句成段占比 = 单句成段数量 / 总段落数量 × 100%
```

**效果**：
- 增强节奏感
- 突出重点
- 提升阅读体验

### 5步开篇法执行情况

**推荐篇幅分配**：
- 第1步（开篇钩子）：10%
- 第2步（持续铺垫）：20%
- 第3步（部分揭露）：30%
- 第4步（规则恐怖）：20%
- 第5步（反转结尾）：20%

**检查重点**：

### 可复用的创作技巧

#### 悬念制造技巧
1. **口误法**：通过"口误"暗示有秘密
   - 示例：班主任说"剧本中"后快速纠正

2. **窃窃私语法**：通过"窃窃私语"透露关键信息
   - 示例：同学们窃窃私语说"导演还没喊卡"

3. **规则恐怖法**：通过"规则"制造恐怖氛围
   - 示例：宿舍墙上贴着8条诡异规则

4. **身份反转法**：揭示主角的特殊身份
   - 示例：告诉主角"你是主角"

#### 对话设计技巧
1. **双关对话**：表面正常，实则诡异
   - 示例："路上小心，晚上...不安全"

2. **关键词重复**：通过关键词暗示真相
   - 示例：重复"剧本"、"演员"、"导演组"、"重置"

3. **信息碎片化**：通过多个碎片信息拼凑真相
   - 示例：不同同学的心声透露不同信息

#### 氛围营造技巧
1. **细节描写**：通过细节暗示诡异
   - 示例："嘴角上扬的角度都像用尺子量过"

2. **节奏控制**：快节奏+慢节奏交替
   - 示例：快节奏揭露真相，慢节奏营造氛围

3. **留白艺术**：不说破，留出想象空间
   - 示例：规则8被涂掉

## 开篇优化建议

### 优化1：增强开篇钩子

**问题**：开篇前200字废话多
**建议**：
- 开篇第1行就制造悬念
- 删掉不必要的背景介绍
- 直接进入诡异场景

### 优化2：增加细节暗示

**问题**：细节太少，读者无法拼凑真相
**建议**：
- 添加≥2处细节暗示
- 每个细节都指向"演戏"这一真相
- 不直接揭露，让读者自己发现

### 优化3：增加规则恐怖

**问题**：没有规则恐怖元素
**建议**：
- 添加3-8条诡异规则
- 每条规则都是伏笔
- 规则本身就是故事

### 优化4：优化单句成段

**问题**：单句成段占比低于30%
**建议**：
- 将重要句子单独成段
- 提高单句成段占比到30-50%
- 增强节奏感

## 典型开篇问题与解决方案

### 问题1：开篇前200字废话多

**症状**：开篇迟迟不进入正题，读者失去耐心

**解决方案**：
- 删掉不必要的背景介绍
- 开篇第1行就制造悬念
- 直接进入诡异场景

**示例**：
```
修改前：
我叫林晚，今年18岁，是一名高中生。今天是转学的第一天，我很紧张...

修改后：
"欢迎林晚同学,你就是剧本中的...我是说,班级的新成员。"
班主任站在讲台上,脸上挂着标准的微笑...
```

### 问题2：没有规则恐怖

**症状**：没有规则元素，缺少恐怖氛围

**解决方案**：
- 添加3-8条诡异规则
- 每条规则都是伏笔
- 规则本身就是故事

**示例**：
```
规则1:晚上22:00后,禁止离开校园。
规则2:禁止询问剧本内容。
规则3:晚上22:00后,禁止单独在走廊走动。
...
```

### 问题3：结尾没有反转

**症状**：结尾平淡，没有悬念

**解决方案**：
- 揭示主角的特殊身份
- 制造"你是主角"的悬念
- 留下强烈悬念

**示例**：
```
"林晚同学,欢迎来到诡校。"
"这里是演员学校,所有人都是演员。"
"除了你。"
"你是观众。"
"而且...你是主角。"
```
