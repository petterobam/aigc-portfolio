# 番茄小说标题优化技能

## 技能简介

番茄小说标题优化专家，基于数据分析的爆款标题公式库，提供系统化的标题优化方案。

## 技能定位

- **名称**：fanqie-title-optimizer
- **描述**：番茄小说标题优化技能。用于优化番茄小说故事标题，提高标题吸引力、点击率和阅读量。基于爆款标题公式库，提供可复用的标题优化流程。
- **类型**：内容创作技能
- **领域**：番茄小说运营
- **目标**：提高标题吸引力、点击率和阅读量

## 核心能力

1. **标题分析**：分析原标题的问题（长度、金手指、爽点、冲突）
2. **标题优化**：基于爆款公式库生成优化方案
3. **效果预测**：预估优化后的阅读量提升
4. **验证支持**：提供验证标准和效果对比方法

### 步骤3：开始优化标题

```bash
# 分析并优化单个标题
# 原标题："皇宫里全是弹幕"

# 分析原标题
# - 标题长度：7字（过短）
# - 有金手指：弹幕
# - 无爽点
# - 题材表现：历史穿越（优秀）

# 应用爆款公式（公式1+公式2组合）
# - 冲突前置："穿成废柴贵妃受尽凌辱"
# - 身份标签："废柴贵妃"
# - 设定吸引："发现能看见皇上心里的弹幕"
# - 悬念："原来他早已爱上我"

# 优化后标题："穿成废柴贵妃受尽凌辱，我发现能看见皇上心里的弹幕，原来他早已爱上我"

# 验证优化方案
# - 标题长度：32字（✅ 符合20-50字）
# - 包含金手指：✅ 读心（弹幕）
# - 包含爽点：✅ 受辱+反转
# - 包含冲突：✅ 前置冲突
# - 题材选择：✅ 历史穿越+读心

# 预期效果：阅读量从0提升到10-20阅读（10倍+）
```

## 使用场景

### 场景1：优化零阅读作品

**方法**：

2. 应用爆款标题公式
3. 验证优化方案
4. 实施标题修改

**预期效果**：阅读量从0提升到10-20阅读（10倍+）

### 场景2：优化中等阅读作品

**目标**：将中等阅读作品优化为高阅读作品

**方法**：
1. 分析原标题问题
2. 应用爆款标题公式
3. 验证优化方案
4. 实施标题修改

**预期效果**：阅读量从1-5提升到15-30阅读（5倍+）

### 场景3：批量标题优化

**目标**：批量优化多个作品的标题

**方法**：
1. 批量分析原标题
2. 批量应用爆款标题公式
3. 批量验证优化方案
4. 批量实施标题修改

**预期效果**：平均阅读量提升5-10倍

### 场景4：标题质量评估

**目标**：评估标题质量，识别需要优化的标题

**方法**：
1. 按照评分标准评估标题
2. 识别低质量标题（<60分）
3. 应用爆款标题公式优化
4. 提升标题质量

**预期效果**：低质量标题提升为高质量标题（≥80分）

### 场景5：优化验证

**目标**：验证标题优化效果

**方法**：
1. 发布优化后的作品
2. 收集阅读数据（3-7天）
3. 对比优化前后的阅读量
4. 计算提升倍数

**预期效果**：验证标题优化是否达到预期效果
