# 📖 知识库 (Knowledge Base)

> **用途**: 外部输入 - 收集、整理、阅读学习资料
> **核心环节**: 知识输入 (Knowledge Input)
> **目标**: 为方法论提炼提供高质量的理论基础

---

## 📚 核心内容

### 1. 理论实践学习
- **文件目录**: `理论实践学习/`
- **内容**: 书籍、课程、文章的阅读笔记和心得
- **原则**: 每次学习至少提炼一句核心金句写进知识库

### 2. 趋势研究
- **文件目录**: `趋势研究/`
- **内容**: 行业趋势、市场动态、政策变化
- **原则**: 关注未来3-5年的趋势，不仅仅是当下

### 3. 竞品分析
- **文件目录**: `竞品分析/`
- **内容**: 竞品产品分析、功能对比、运营策略
- **原则**: 竞品是最好的老师，学习他们的成功经验

---

## 🎯 使用规范

1. **文件命名**: 使用日期 + 主题格式（如 `20260321-复利效应研究.md`）
2. **内容结构**:
   - 核心观点
   - 论证逻辑
   - 可行性分析
   - 下一步行动
3. **标签体系**: 使用统一的标签系统便于检索（如 `#财富理论 #投资策略 #产品方法论`）
