### 1. GitHub 一键部署脚本
**目标**: 将复杂GitHub操作简化至5分钟完成

```bash
#!/bin/bash
# GitHub一键部署脚本 - 个人网站
# 使用方法: ./github-deploy.sh

PROJECT_DIR="~/.openclaw/workspace/财富自由之路/personal-website"
REPO_NAME="personal-website"
USERNAME="petterobam"

echo "🚀 开始GitHub一键部署..."
echo "📁 项目目录: $PROJECT_DIR"
echo "🏷️  仓库名称: $REPO_NAME"
echo "👤 用户名: $USERNAME"
echo ""

# 检查项目目录
if [ ! -d "$PROJECT_DIR" ]; then
    echo "❌ 错误: 项目目录不存在"
    exit 1
fi

# 进入项目目录
cd "$PROJECT_DIR"

# 检查Git状态
if [ ! -d ".git" ]; then
    echo "🔧 初始化Git仓库..."
    git init
    echo "✅ Git仓库初始化完成"
fi

# 添加远程仓库
echo "🔗 添加远程仓库..."
git remote remove origin 2>/dev/null
git remote add origin "https://github.com/$USERNAME/$REPO_NAME.git"
echo "✅ 远程仓库已添加"

# 创建提交
echo "📝 添加文件到Git..."
git add .
git commit -m "Initial commit: Personal website deployment

- VuePress-based personal website
- GitHub Actions auto-deployment
- Ready for Pages publishing

🤖 Generated by wealth-freedom automation system"
echo "✅ 提交完成"

# 推送到GitHub
echo "🚀 推送到GitHub..."
git branch -M main
git push -u origin main
echo "✅ 代码推送完成"

echo ""
echo "🎉 GitHub部署准备完成！"
echo "📋 下一步操作:"
echo "   1. 访问: https://github.com/$USERNAME/$REPO_NAME"
echo "   2. 进入 Settings → Pages"
echo "   3. Source 选择: GitHub Actions"
echo "   4. 等待自动部署完成"
echo "   5. 访问: https://$USERNAME.github.io/$REPO_NAME/"
```

**功能特点**:
- ✅ 自动初始化Git仓库
- ✅ 自动配置远程仓库
- ✅ 自动提交和推送
- ✅ 错误处理和状态提示
- ✅ 操作时间: 3-5分钟
