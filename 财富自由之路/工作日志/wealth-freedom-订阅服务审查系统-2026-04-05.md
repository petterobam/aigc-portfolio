## 📊 价值评估矩阵

### 评估标准
```markdown
**价值评分 (1-5分)**:
- 5分: 必需品，每天都用，价值极高
- 4分: 常用工具，提供明显便利
- 3分: 有用但非必需，使用频率一般
- 2分: 偶尔使用，价值存疑
- 1分: 基本不用，纯粹浪费

**ROI计算**:
ROI = (月价值 × 使用频率) / 月费
- ROI > 10: 极高价值，必须保留
- ROI 5-10: 高价值，建议保留
- ROI 2-5: 中等价值，可考虑优化
- ROI < 2: 低价值，建议取消

**使用频率**:
- 每日: 使用价值 × 1.0
- 每周2-3次: 使用价值 × 0.7
- 每周1次: 使用价值 × 0.4
- 每月1-2次: 使用价值 × 0.2
- 极少使用: 使用价值 × 0.1
```

### ROI分析结果
```markdown
**高价值服务 (ROI > 5)**:
- 网易云课堂: ROI = 7.2 (必需学习工具)
- 极客时间: ROI = 6.7 (核心技能提升)
- B站大会员: ROI = 5.4 (每日使用，内容丰富)
- 知识付费内容: ROI = 8.1 (投资回报最高)

**中等价值服务 (ROI 2-5)**:
- 爱奇艺: ROI = 3.2 (娱乐需求，但可优化)
- 腾讯视频: ROI = 1.8 (使用频率低，建议取消)
- PS Plus: ROI = 2.1 (偶尔娱乐，可降级)

**低价值服务 (ROI < 2)**:

- 某些小众APP: ROI = 0.1 (直接取消)
```
