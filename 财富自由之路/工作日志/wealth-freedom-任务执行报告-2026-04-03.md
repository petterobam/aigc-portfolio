### 🚨 认证问题解决方案

**方法一：Personal Access Token (推荐)**

```bash
# 1. 生成Personal Access Token
# 访问：https://github.com/settings/tokens
# 点击 "Generate new token" → "Generate new token (classic)"
# 勾选 "repo" 权限
# 点击 "Generate token"

# 2. 使用Token推送代码
cd "~/.openclaw/workspace/财富自由之路/personal-website"
git push origin main
# 用户名：输入GitHub用户名
# 密码：输入刚生成的Token（不是GitHub密码）
```

**方法二：GitHub CLI**

```bash
# 1. 安装GitHub CLI（如果未安装）
brew install gh

# 2. 登录GitHub
gh auth login

# 3. 推送代码
git push origin main
# 会自动处理认证
```

**方法三：配置Git凭据存储**

```bash
# 1. 配置Git凭据存储
git config --global credential.helper store

# 2. 推送代码（首次会提示输入用户名和密码）
git push origin main
# 输入用户名和密码后会自动保存

# 3. 后续推送无需重复输入
```
