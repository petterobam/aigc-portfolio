# 竞品研究

> 研究个人财务软件领域的竞争对手，学习优秀实践，找到差异化定位。

---

## 研究目的

1. 学习竞品的优秀功能设计
2. 了解用户引导和激活流程
3. 找到产品差异化定位
4. 避免重复造轮子

## 差异化定位

### 我们的独特价值

1. **三阶段财务自由框架**（保障→安全→自由）
2. **内置财富自由目标计算**
3. **方法论体系支持**（不只是记账工具）
4. **开源+隐私保护**（数据在本地）

### 竞品普遍缺少的

- 明确的财务自由路径规划
- 三阶段目标的自动计算
- 财富方法论的内置指导
- 开源透明的数据处理
