## 二、推荐定价策略

### 三层定价模型

```
┌─────────────────────────────────────────────────────┐
│  免费版 (Free)          │  基础记账 + 月度报表       │
│  ¥0                     │  2 个账户 + 手动录入       │
├─────────────────────────────────────────────────────┤
│  专业版 (Pro)           │  全功能 + AI 洞察          │
│  ¥99/年 或 ¥12.9/月    │  无限账户 + 自动导入       │
│                         │  投资追踪 + 目标规划       │
├─────────────────────────────────────────────────────┤
│  终身版 (Lifetime)      │  专业版全部功能            │
│  ¥299（一次性）         │  含 1 年 AI 高级功能      │
│                         │  早期支持者专属            │
└─────────────────────────────────────────────────────┘
```

### 定价理由

| 决策 | 理由 |
|------|------|
| 免费版存在 | 降低门槛，积累用户基础（国内必需） |
| ¥99/年 | 远低于 YNAB（~¥720/年），符合国内消费水平 |
| 终身版 ¥299 | 锚定效应：让年付看起来更划算；回收早期开发成本 |
| 月付 ¥12.9 | 心理门槛低，转化路径短 |

## 四、变现路径优先级

```
Phase 1 (0-6月): 免费开源 + 社区建设
  → 积累种子用户，建立信任
  → GitHub Stars + 技术文章引流

Phase 2 (6-12月): Pro 版上线 + 早期定价
  → ¥99/年 早鸟价 ¥49/年
  → 通过技术社区和内容营销获客

Phase 3 (12-24月): 移动端 + 订阅制成熟
  → iOS/Android 客户端
  → AI 功能作为核心差异化

Phase 4 (24月+): 平台化
  → 开放 API
  → 家庭版、企业版
  → 理财师接入分成
```

## 五、核心差异化策略

### vs 国内竞品

1. **开源透明**：代码开源，数据自主，信任度高
2. **AI 原生**：智能分类、预测支出、优化建议
3. **财务自由导向**：不只是记账，而是规划通往自由的路径
4. **开发者友好**：API 开放，可自托管

### vs 国际竞品

1. **本地化**：支持国内银行、支付宝、微信
2. **价格优势**：¥99 vs $99（10倍价格差）
3. **中文体验**：原生中文界面和客服
