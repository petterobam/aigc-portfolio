## 核心洞见速查

### 财富理论

- 大多数人高估了一年能做到的，低估了十年能做到的
- 整天工作的人没有时间赚钱——要构建系统，而非出卖时间
- 惊人的好运通常只是多年准备的结果
- 战胜债务的最佳方式是积累财富，而非只是节省开支
- 投资者在买入时获得利润，而非卖出时——买入即要有安全边际

### 投资理财

- 财务安全所需金额 = 月支出 × 150（基于 8% 年化收益）
- 财务自由所需金额 = 梦想月成本 × 150
- 40:40:20 原则：40% 低风险 + 40% 中风险 + 20% 高风险
- 私人住宅是奢侈品，不是投资——它不产生现金流
- 一位投资者会在买入时获利，想在卖出时获益的是投机者

### 心理认知

- 你把责任推给谁，就把权力给了谁
- 我们的行为基于信念，而非逻辑——改变信念才能改变财务状况
- 消费债的本质：用未来的钱换取当下短暂的快乐，代价是长期痛苦
- 自信不是偶然的，需要系统建立（成功日记法）
- 避免同情陷阱：讲述困难会收获同情，但维持同情需要持续处于糟糕状态

### 创业商业

- 74% 的百万富翁是企业家——真正的财富积累几乎都要建立自己的企业
- 业余策略注重短期收益；专业策略着眼长期，先支付自己
- 没有完成目标制定，不要继续——定好目标，成功了一半
