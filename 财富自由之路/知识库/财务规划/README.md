## 核心概念

### 三阶段路径

1. **财务保障** (6-12个月储备金)
   - 公式:每月支出 × 保障月数
   - 目的:应对突发情况
   - 原则:安全且容易支取,不用于投机

2. **财务安全** (靠利息覆盖日常支出)
   - 公式:月所需金额 × 150
   - 目的:不工作也能保证基本生活
   - 策略:40:40:20原则

3. **财务自由** (靠利息实现梦想生活)
   - 公式:(日常支出 + 梦想月成本) × 150
   - 目的:过梦想中的生活
   - 策略:50:50原则

### 投资策略

- **财务保障**:安全第一,不冒险
- **财务安全**:40%低风险 + 40%中风险 + 20%高风险
- **财务自由**:50%中风险 + 50%高风险

### 核心原则

- 绝不动用本金(本金是下金蛋的鹅)
- 目标拓宽机会意识
- 为了"赢"而比赛
- 一切都很重要(不存在中间状态)
- 建立自己的企业是快速致富的路径
