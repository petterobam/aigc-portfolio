## 为什么大多数人没有储蓄

### 四大常见理由

1. **将来会挣很多钱，现在不用储蓄**
   - 这是最大的谎言
   - 收入增加不等于财富增加

2. **现在想享受生活，储蓄意味着限制**
   - 误解了储蓄的本质
   - 储蓄不是限制，是选择

3. **不认为储蓄很重要，无法改变观念**
   - 信念决定行为
   - 可以随时改变信念

4. **储蓄一无是处（低利率、通货膨胀）**
   - 忽略了复利的力量
   - 混淆了储蓄和投资

### 真相

**使你变得富有的是储蓄，而非收入**

```
财富 = 对金钱的留存
```

- 现在就应该开始储蓄
- 即使挣到很多钱，也不改变财务习惯的人，依然不会变富
- **想让事物向有利方向发展，首先自己必须变得更好**

## 节俭是富人的美德

### 前提条件

**节俭不是富裕的唯一原因，但明显是一个前提条件**

- 没有节俭这个基础，富裕没有指望
- 找不出不节俭的公司创始人
- 成功者准备好了去做大多数失败者拒绝去做的事情

---

## 储蓄使人快乐，储蓄易如反掌

### 悖论

**你选择享受生活的时间，就是你开始储蓄的时间。**

### 真相

- 储蓄不是让处境变艰难
- 不是紧缩整个月的开支
- 即使不储蓄，月末也常常不剩什么
- 意外支出或分期账单会消耗 100% 的收入

### 视角转换

**你一直在付钱给别人，却从来没有为自己付过钱。**

## 支付自己原则（核心方法）

### 10% 原则

```
将每月收入的 10% 存入独立账户
使用收入的 90% 生活
与使用 100% 相比，其实没有什么区别
```

### 鹅账户设置方法

#### 步骤

1. **设立"鹅"账户**
   - 在常用账户下开通长期划付委托业务
   - 每月初将工资收入的 10% 转入"鹅"账户

2. **千万不动"鹅"账户**
   - 这是下金蛋的鹅
   - 本金永远不动用

3. **按投资原则管理"鹅"账户资金**
   - 学习投资知识
   - 让鹅长大

4. **感受成长带来的乐趣和自信**
   - 随着"鹅"账户资金增长
   - 生活会更有乐趣、更加自信

5. **增强自律**
   - 设立"鹅"账户是简单有效的自律方法

### 鹅账户的意义

**鹅 = 本金（不动用）**
**金蛋 = 利息（被动收入）**

- 鹅账户是财务自由的基础
- 只有鹅足够大，金蛋才够生活

## 加薪处理策略

### 50% 原则

```
将涨薪的 50% 存入鹅账户
```

### 六大好处

1. **生活水平不会提升太快**
   - 从加薪中提取 50%
   - 避免生活方式膨胀

2. **促使鹅加快成长**
   - 每次加薪都让鹅变大
   - 复利效应加速

3. **明确工作目的**
   - 向潜意识表明你有良好的理财能力
   - 知道自己为什么而工作

4. **更接近长远目标**
   - 每一次加薪都推进目标
   - 看得见的进步

5. **个体经营者也能执行**
   - 集老板和雇员于一身
   - 定期给自己加薪

6. **不会对你造成影响**
   - 还没有习惯加薪后的生活
   - 50% 不会降低生活质量

## 为什么储蓄计划会失败

### 两大原因

1. **花销超标**
   - 收入增加，支出也增加
   - 没有控制消费欲望

2. **只存月末剩下的钱**
   - 这是最致命的错误
   - 月末常常没有剩下的钱

### 正确做法

**先支付自己，再支付别人**

```
收入到账 → 10% 存入鹅账户 → 90% 用于生活
（而不是）
收入到账 → 支付所有账单 → 剩下的存起来（通常 = 0）
```

## 储蓄与投资的关系

### 关键认知

**储蓄使你成为百万富翁，投资让百万富翁保持富有**

- 由于通胀，利息增加的速度跟不上购买力下降的速度
- 储蓄之后必须学习投资的基本知识
- 鹅账户的钱需要投资增值

### 储蓄 → 投资 → 财务自由

```
1. 储蓄：建立鹅账户（10% + 加薪 50%）
2. 投资：让鹅长大（学习投资原则）
3. 财务自由：靠金蛋生活（被动收入 > 支出）
```
