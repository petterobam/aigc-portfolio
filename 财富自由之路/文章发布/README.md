## 🎯 发布执行清单

### 步骤 1：准备账号

- [ ] 知乎账号（已绑定手机号）
- [ ] 掘金账号（已绑定手机号）
- [ ] V2EX 账号（已绑定手机号）

### 步骤 2：获取文章原文

文章原文在 `../技术文章/` 目录，格式为 Markdown。

### 步骤 3：发布到知乎

1. 登录 [知乎](https://www.zhihu.com/)
2. 点击「写文章」
3. 复制文章内容（保留 Markdown 格式）
4. 添加封面图
5. 选择分类标签
6. 点击「发布」

**技巧**：
- 知乎支持 Markdown，粘贴后会自动转换
- 封面图尺寸建议 1200x675
- 添加 3-5 个相关话题标签

### 步骤 4：发布到掘金

1. 登录 [掘金](https://juejin.cn/)
2. 点击「写文章」
3. 复制文章内容
4. 添加封面图和分类
5. 选择技术标签（前端/后端/移动端/工具资源等）
6. 点击「发布」

### 步骤 5：发布到 V2EX

1. 登录 [V2EX](https://www.v2ex.com/)
2. 点击「发布」
3. 选择「分享」分类
4. 填写标题和链接（指向知乎/掘金文章）
5. 点击「发布主题」
