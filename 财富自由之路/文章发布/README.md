## 发布准备清单

每篇文章发布前需要完成以下检查：

### 内容准备
- [ ] 文章撰写完成
- [ ] 文章校对（错别字、语法、逻辑）
- [ ] 文章排版（Markdown 格式）
- [ ] 插图准备（文章内插图、封面图）

### 封面图准备
- [ ] 封面图生成（1200x675 像素，PNG 格式，<5MB）
- [ ] 封面图风格统一（渐变色、字体、Logo）
- [ ] 封面图文件命名规范（文章名-封面图.png）

### 发布平台准备
- [ ] 知乎账号登录
- [ ] 个人网站发布功能测试
- [ ] 发布时间规划（避免高峰期，选择用户活跃时间）

### 发布后检查
- [ ] 文章链接验证
- [ ] 评论互动
- [ ] 数据追踪（阅读量、点赞数、收藏数、分享数）
- [ ] 用户反馈收集
