## 🎯 发布流程（每篇 5-8 分钟）

### 知乎发布步骤

1. ✅ 登录知乎 → 点击「写文章」
2. ✅ 复制文章标题和正文（从 `../技术文章/` 获取）
3. ✅ 粘贴到知乎编辑器
4. ✅ 上传封面图（可选）
5. ✅ 添加话题标签（3-5 个相关话题）
6. ✅ 点击「发布」

### 掘金发布步骤

1. ✅ 登录掘金 → 点击「写文章」
2. ✅ 复制文章标题和正文
3. ✅ 粘贴到掘金编辑器
4. ✅ 选择技术分类标签
5. ✅ 上传封面图（可选）
6. ✅ 点击「发布」

### V2EX 发布步骤

1. ✅ 登录 V2EX → 点击「发布」
2. ✅ 选择「分享」分类
3. ✅ 填写标题（文章标题 + 平台链接）
4. ✅ 粘贴文章摘要或链接
5. ✅ 点击「发布主题」

## ⚠️ 常见问题

### Q：文章格式错乱怎么办？
A：知乎和掘金都支持 Markdown，直接粘贴即可。如有格式问题，用平台自带的富文本编辑器微调。

### Q：封面图尺寸不对？
A：可使用在线工具裁剪，如 https://www.iloveimg.com/crop-image

### Q：话题标签怎么选？
A：选择 3-5 个相关话题，如 #程序员 #独立开发 #AIGC #财务自由

### Q：文章可以同时发多个平台吗？
A：可以！同一篇文章可以同时发布到多个平台，不会有影响。
