## 一、任务概述

### 目标
优化所有 5 个财务规划工具的图表交互体验，提升用户体验。

### 范围
- 复利计算器（Calculator.vue）
- 提前还款计算器（PrepaymentCalculator.vue）

- 大额支出规划（LargeExpensePlanner.vue）
- 情景模拟（ScenarioSimulator.vue）

---

## 二、问题分析

### 原有问题
1. **缺少图表**：部分工具（提前还款计算器、退休规划工具、大额支出规划、情景模拟）没有可视化图表
2. **缺少缩放功能**：无法放大/缩小图表查看细节
3. **缺少平移功能**：无法拖动图表查看不同时间段
4. **tooltip 信息简单**：只显示基本数值，没有详细分析
5. **缺少交互反馈**：鼠标悬停、点击等交互效果不够明显

### 用户体验影响
- 数据可视化不够直观
- 长期数据无法查看细节
- 无法快速定位关键数据点
- 图表美观度和专业感不足

## 五、技术实现细节

### 5.1 ECharts 配置优化

#### Tooltip 格式化
```typescript
formatter: (params: any) => {
  // 显示年份
  let result = `<div style="font-weight: bold;">${params[0].axisValue}</div>`

  // 显示各项数据
  params.forEach((param: any) => {
    result += `<div>
      <span>${param.marker}${param.seriesName}</span>
      <span style="font-weight: 600;">${formatCurrency(value)}</span>
    </div>`

    // 显示详细信息
    if (param.data && param.data.details) {
      Object.entries(param.data.details).forEach(([key, val]) => {
        result += `<div>${key}: ${formatCurrency(val)}</div>`
      })
    }
  })

  return result
}
```

#### DataZoom 配置
```typescript
dataZoom: [
  {
    type: 'slider',      // 滑块缩放
    show: true,
    xAxisIndex: [0],

    type: 'inside',      // 鼠标滚轮和拖动
```

### 5.2 图表实例管理

每个工具都维护一个图表实例引用：
```typescript
const chartRef = ref<HTMLElement>()
const chartInstance = ref<any>(null)

// 初始化
const initChart = () => {
  const chart = echarts.init(chartRef.value)
  chart.setOption(getXXXChartConfig(...))
  chartInstance.value = chart
}

// 导出
const handleExportChart = () => {
  exportChartToImage(chartInstance.value, '图表名称')
}
```

### 5.3 数据处理

每个工具都有独特的数据计算逻辑：
- **复利计算器**：逐年计算总资产、总投入、总收益
- **提前还款计算器**：逐年计算剩余本金、累计还款
- **退休规划工具**：逐年计算储蓄、支出（考虑通胀）
- **大额支出规划**：逐年计算目标金额、未来资产
- **情景模拟**：逐年对比两个场景的资产变化
