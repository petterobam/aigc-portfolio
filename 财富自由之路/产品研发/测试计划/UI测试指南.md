## ✅ UI/UX 问题记录清单

### 输入验证问题

- [ ] 输入负数时是否报错
- [ ] 输入极大数值时是否报错
- [ ] 年龄超出范围（<18 或 >100）时是否报错

### 边界情况问题

- [ ] 收益率为 0% 时，图表是否正常显示
- [ ] 月储蓄为 0 时，计算结果是否正常
- [ ] 目标金额为 0 时，是否报错

### 交互体验问题

- [ ] 按钮点击是否有反馈（加载动画、成功提示）
- [ ] 输入框是否有聚焦样式
- [ ] 表格排序是否正常
- [ ] 响应速度是否流畅（无明显卡顿）

### 视觉设计问题

- [ ] 颜色搭配是否合理（对比度、可读性）
- [ ] 字体大小是否合适
- [ ] 间距是否合理（不拥挤、不空旷）
- [ ] 图标是否清晰易懂

### 数据持久化问题

- [ ] 刷新页面后，输入参数是否保留
- [ ] 关闭应用后，输入参数是否保留
- [ ] 是否有"保存场景"功能
