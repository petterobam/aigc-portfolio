## 测试原则

1. **每个测试独立运行**：不依赖外部状态
2. **边界条件全覆盖**：包括 0、负数、极大值
3. **结果可验证**：每个测试都有明确的预期结果
4. **失败立即报告**：发现错误立即停止并报告

## 测试报告

测试结果会输出到控制台，包括：
- ✅ 通过的测试数量
- ❌ 失败的测试数量
- 📊 测试覆盖率（核心算法）
- ⏱️ 测试运行时间
