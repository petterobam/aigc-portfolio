## 一、产品定位

### 1.1 产品愿景

帮助用户系统化实现财务自由，从财务保障到财务安全，最终达到财务自由。

### 1.2 目标用户

- 有强烈财务自由意愿的个人
- 愿意正视财务现状、设定明确目标的人
- 需要工具辅助管理财务、追踪进度的用户

### 1.3 核心价值

**不是记账软件，是财富自由教练**

- 记账软件告诉你钱花哪了
- 我们告诉你离财务自由还有多远

## 三、MVP 核心功能

### 3.1 财务看板（核心）

**目标**：一眼看清财务全貌

| 模块 | 数据项 | 说明 |
|------|-------|------|
| 净资产 | 资产合计 - 负债合计 | 核心指标 |
| 月现金流 | 收入 - 支出 | 每月结余 |
| 三阶段进度 | 保障/安全/自由完成度 | 百分比+进度条 |
| 关键比率 | 储蓄率、负债率、被动收入占比 | 健康度指标 |

**交互**：
- 大数字展示净资产（最醒目）
- 进度条展示三阶段完成度
- 红黄绿三色健康度指示

### 3.2 目标追踪

**目标**：让目标成为绝对必需品

| 功能 | 说明 |
|------|------|
| 三阶段目标设定 | 输入月支出 → 自动计算保障/安全/自由目标金额 |
| 目标日期设定 | 预计何时达成每个阶段 |
| 进度可视化 | 实时显示当前/目标差距 |
| 达成预测 | 基于当前储蓄率预测达成时间 |

**公式**：
- 财务保障 = 月支出 × 6~12
- 财务安全 = 月支出 × 150（基于8%年化）
- 财务自由 = 梦想月成本 × 150

### 3.3 每日记账（轻量）

**目标**：追踪现金流，不追求繁琐

| 功能 | 说明 |
|------|------|
| 快速记账 | 收入/支出/转账 |
| 分类标签 | 固定支出/变动支出/被动收入等 |
| 月度汇总 | 自动生成月收支报告 |

**原则**：足够简单，3秒完成一笔记录

- 信念转变日志

## 五、非功能需求

### 5.1 平台支持

| 平台 | 优先级 | 说明 |
|------|-------|------|
| PC 桌面 | P0 | 主力使用场景 |
| 移动端 | P1 | 随时记账查看 |

### 5.2 数据安全

- 本地存储优先（敏感财务数据）
- 可选云同步（加密）
- 导出备份功能

### 5.3 技术栈建议

| 端 | 技术选型 | 理由 |
|------|---------|------|
| PC | Electron + Vue/React | 跨平台，本地存储 |
| 移动端 | Flutter / React Native | 跨平台复用 |
