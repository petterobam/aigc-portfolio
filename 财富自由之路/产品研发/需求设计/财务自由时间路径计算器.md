# 财务自由时间路径计算器

> 基于复利和时间价值理论，计算达到财务自由三个阶段所需的时间
> 
> 来源：《财富自由之路》第8章（储蓄）、第9章（复利）

---

## 核心公式

### 1. 三阶段目标金额

```
财务保障 = 月总支出 × 保障月数（6-12个月）
财务安全 = 月基本生活费用 × 150（基于8%年化收益）
财务自由 = 月梦想生活成本 × 150（基于8%年化收益）
```

> 150的由来：月所需金额 × 12个月 ÷ 8% = 月所需金额 × 150

### 2. 复利增长公式

```
终值 = 本金 × (1 + 年化收益率)^年数
```

或反推时间：

```
年数 = log(目标金额 / 当前本金) / log(1 + 年化收益率)
```

### 3. 定期投资复利公式

```
终值 = 月投资额 × [((1 + 月收益率)^月数 - 1) / 月收益率]
```

其中：月收益率 = 年化收益率 / 12

### 4. 72法则

```
翻倍时间 = 72 / 年化收益率
```

例子：
- 8%收益率 → 9年翻倍
- 12%收益率 → 6年翻倍
- 15%收益率 → 4.8年翻倍

## 计算模型

### 输入参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 当前净资产 | 现有可投资资产 | 用户填写 |
| 月收入 | 税后月收入 | 用户填写 |
| 月支出 | 当前月支出 | 用户填写 |
| 储蓄率 | 月储蓄 / 月收入 | 10% |
| 加薪比例 | 预期年化收入增长 | 5% |
| 年化收益率 | 投资组合预期收益 | 8% |
| 保障月数 | 财务保障目标月数 | 6个月 |
| 月基本生活 | 财务安全阶段月支出 | 用户填写 |
| 月梦想生活 | 财务自由阶段月支出 | 用户填写 |

### 计算逻辑

```javascript
// 1. 计算月储蓄额
月储蓄额 = 月收入 × 储蓄率

// 2. 计算三阶段目标
财务保障目标 = 月支出 × 保障月数
财务安全目标 = 月基本生活 × 150
财务自由目标 = 月梦想生活 × 150

// 3. 计算达到各阶段的时间（考虑复利+定期投资）
// 使用迭代法，因为每月投资额会随收入增长而增长

function 计算达成时间(目标金额, 当前资产, 月储蓄, 年化收益率, 年收入增长率) {
  let 资产 = 当前资产
  let 月数 = 0
  let 当前月储蓄 = 月储蓄
  const 月收益率 = 年化收益率 / 12
  
  while (资产 < 目标金额 && 月数 < 600) { // 最多50年
    // 资产增值（复利）
    资产 *= (1 + 月收益率)
    
    // 每月储蓄
    资产 += 当前月储蓄
    
    月数++
    
    // 每年调整储蓄额（收入增长）
    if (月数 % 12 === 0) {
      当前月储蓄 *= (1 + 年收入增长率)
    }
  }
  
  return {
    年数: Math.floor(月数 / 12),
    月数: 月数 % 12,
    最终资产: 资产
  }
}
```

## 实战建议

### 缩短达成时间的策略

1. **提高储蓄率**
   - 从10%提升到20%，时间缩短约40%
   - 加薪的50%直接存入鹅账户

2. **提高收益率**
   - 学习投资知识
   - 分散投资组合
   - 长期持有，避免频繁交易

3. **增加收入**
   - 发展副业
   - 成为专家
   - 创造被动收入来源

4. **降低目标**
   - 财务自由的月支出可以调整
   - 简化生活，降低欲望

### 风险提示

- 8%年化收益是长期平均值，实际会有波动
- 通胀会降低购买力，需要考虑实际收益率
- 意外支出可能打乱计划（因此需要财务保障）
- 收入增长不保证，需做好多种准备

## 示例计算

### 案例：小明的财务自由之路

**输入：**
- 当前净资产：¥50,000
- 月收入：¥15,000
- 月支出：¥10,000
- 储蓄率：20%（¥3,000/月）
- 年化收益率：8%
- 保障月数：6个月
- 月基本生活：¥8,000
- 月梦想生活：¥20,000

**计算：**
```
财务保障目标 = ¥10,000 × 6 = ¥60,000
财务安全目标 = ¥8,000 × 150 = ¥1,200,000
财务自由目标 = ¥20,000 × 150 = ¥3,000,000
```

**结果：**
- 财务保障：当前进度83%，预计4个月达成
- 财务安全：当前进度4%，预计12年达成
- 财务自由：当前进度2%，预计18年达成

**如果储蓄率提高到30%（¥4,500/月）：**
- 财务安全：预计10年达成（快2年）
- 财务自由：预计14年达成（快4年）

**如果收益率提高到10%：**
- 财务安全：预计10年达成（快2年）
- 财务自由：预计14年达成（快4年）
