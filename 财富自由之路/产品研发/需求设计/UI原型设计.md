# UI 原型设计

> 财富自由之路个人财务软件
> 版本：v0.1 MVP
> 更新日期：2026-03-22

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| 信息优先 | 核心数据一目了然 |
| 简洁克制 | 减少视觉噪音 |
| 行动导向 | 每个页面有明确行动点 |
| 渐进披露 | 复杂功能按需展开 |
