## 二、核心页面测试

### 1. 财务看板（Dashboard）
- [ ] 净资产显示正确（资产 - 负债）
- [ ] 本月收入/支出/结余计算正确
- [ ] 关键比率计算正确：
  - [ ] 储蓄率 = 月结余 / 月收入 × 100%
  - [ ] 被动收入覆盖率 = 被动收入 / 月支出 × 100%
  - [ ] 负债率 = 负债 / 资产 × 100%
  - [ ] 财务自由进度 = 当前净资产 / 财务安全目标 × 100%
- [ ] 资产结构图表正常显示
- [ ] 收支趋势图表正常显示
- [ ] 快捷记账按钮可用

### 2. 目标页面（Goals）
- [ ] 三阶段目标显示正确：
  - [ ] 财务保障：月支出 × 保障月数
  - [ ] 财务安全：月所需 × 150
  - [ ] 财务自由：梦想月成本 × 150
- [ ] 目标金额可编辑并保存
- [ ] 进度条计算正确
- [ ] 预计达成时间显示合理

### 3. 账户页面（Accounts）
- [ ] 能添加新账户
- [ ] 能编辑账户信息
- [ ] 能删除账户
- [ ] 账户余额正确汇总到总资产
- [ ] 多币种账户正确计算

### 4. 负债页面（Debts）
- [ ] 能添加新负债
- [ ] 能编辑负债信息
- [ ] 能记录还款
- [ ] 剩余金额计算正确
- [ ] 负债类型分类正确（消费债/贷款/其他）

### 5. 交易页面（Transactions）
- [ ] 能添加收入记录
- [ ] 能添加支出记录
- [ ] 交易分类可选
- [ ] 交易记录能编辑
- [ ] 交易记录能删除
- [ ] 月度统计正确

### 6. 梦想图册（Dreams）
