# 迭代记录

> 产品版本迭代历史和变更日志。

## 记录内容

- 版本号和发布日期
- 新增功能
- 问题修复
- 已知问题
- 下一步计划

## 记录模板

```markdown
# vX.Y.Z - YYYY-MM-DD

## 新增
- 功能一
- 功能二

## 修复
- 问题一
- 问题二

## 改进
- 优化一
- 优化二

## 已知问题
- 问题一（计划下版本修复）

## 下一版本计划
- 功能一
- 功能二
```

## 版本命名规范

- 主版本（X）：重大功能变更或架构调整
- 次版本（Y）：新功能添加
- 修订版（Z）：问题修复和小改进
