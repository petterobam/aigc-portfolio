## 🎯 产品介绍

财富自由之路是一款个人财务管理软件，帮助你：

- 📊 **可视化资产配置**：直观展示资产配置比例，支持交互式调整
- 📈 **追踪财务进度**：实时追踪财务三阶段进度（财务保障、财务安全、财务自由）
- 💡 **优化投资策略**：基于科学模型，提供最优资产配置建议
- 🎯 **设定财务目标**：个性化财务目标设定和追踪
- 📊 **数据导出**：支持导出配置报告（PDF、图片）

## 📥 安装说明

### macOS
1. 下载对应架构的 DMG 文件（x64 或 arm64）
2. 双击打开 DMG 文件
3. 将"财富自由之路.app"拖拽到 Applications 文件夹
4. 在 Launchpad 或应用程序文件夹中打开应用

**注意**：首次打开可能需要在系统偏好设置中允许运行（来自未知开发者）

### Windows
1. 下载 EXE 安装程序
2. 双击运行安装程序
3. 按照提示完成安装（可以更改安装路径）
4. 在桌面或开始菜单中打开应用

## 🚀 使用指南

### 快速开始
1. 打开"财富自由之路"应用
2. 选择财务阶段（财务保障、财务安全、财务自由）
3. 调整资产配置滑块（低风险、中风险、高风险）
4. 查看实时更新的饼图、雷达图和收益预测曲线
5. 导出配置报告（PDF 或图片）

### 核心功能
- **财务三阶段**：财务保障（6-12 个月储备）、财务安全（投资覆盖支出）、财务自由（梦想生活）
- **资产配置**：低风险（定期存款、国债）、中风险（债券基金、理财）、高风险（股票基金）
- **收益预测**：基于资产配置，预测 10 年、20 年、30 年的资产增长
- **数据导出**：支持导出 PDF（包含图表和配置详情）和图片（单个图表）
