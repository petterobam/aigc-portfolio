## 技术亮点

### 为什么网络请求统计显示没有 404，但控制台有错误？

因为 404 错误来自 JavaScript 代码内部（Vite 开发服务器的 HMR 机制），而不是网络请求。Playwright 的 `response` 事件只捕获 HTTP 响应，不捕获 JavaScript 内部的错误。

### 为什么 Playwright 无法捕获下载事件？

Playwright 的 `download` 事件依赖于浏览器的下载机制。在 Electron 环境中，下载机制可能与标准浏览器不同，导致 `download` 事件无法正确触发。但是，通过检查控制台日志和数据 URL，可以验证导出功能是否正常工作。
