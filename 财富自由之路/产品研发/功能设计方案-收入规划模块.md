# 收入规划模块 - 功能设计方案

> 设计于：2026-03-28（心跳 #157）
> 优先级：P1（重要功能，v1.2.0）
> 预计开发时间：5-7 天

---

## 一、功能定位

**核心目标**：帮助用户系统化规划收入增长，从主动收入扩展到被动收入，最终实现财务自由。

**价值主张**：
- 大多数人只关注省钱，但财务自由的关键是增加收入
- 提供从"打工思维"到"财富思维"的完整路径
- 系统化追踪收入结构，量化主动收入与被动收入的变化
- 明确收入提升策略，让每一份努力都有方向

**方法论支撑**：
- ✅ 投资与投机的本质区别（主动收入 vs 被动收入）
- ✅ 专家定位与收入增长（成为领域专家）
- ✅ 增加收入的策略（杠杆作用）
- ✅ 收入提升方法 - 成为专家展示强项
- ✅ 软件开发者的百万富翁路径（产品化收入）

**SMART 目标引导**：

- Relevant（相关）：关联财务自由三阶段进度
- Time-bound（有时限）：设置明确的截止日期

## 三、技术实现方案

### 3.1 数据模型

**新增数据表**：

```typescript
// 收入来源类型
enum IncomeSourceType {
  SALARY = 'salary',           // 工资收入
  FREELANCE = 'freelance',     // 兼职收入
  INVESTMENT = 'investment',   // 投资收益
  PRODUCT = 'product',         // 产品收入
  RENTAL = 'rental',           // 租金收入
  INTEREST = 'interest',       // 利息收入
  OTHER = 'other'              // 其他收入
}

// 收入类型（主动 vs 被动）
enum IncomeType {
  ACTIVE = 'active',           // 主动收入
  PASSIVE = 'passive'          // 被动收入
}

interface IncomeSource {
  id: string
  name: string                  // 收入来源名称
  type: IncomeType              // 主动收入/被动收入
  sourceType: IncomeSourceType  // 收入来源类型
  amount: number               // 金额（月）
  frequency: 'monthly' | 'yearly' | 'one-time'  // 频率
  stability: 1-5               // 稳定性评分（1-5 星）
  growthRate?: number          // 预期年增长率（%）
  description?: string         // 描述
}

interface IncomeRecord {
  id: string
  sourceId: string             // 收入来源 ID
  amount: number               // 实际收入金额
  date: Date                   // 收入日期
  note?: string                // 备注
}

interface IncomeGoal {
  id: string
  type: 'total' | 'passive_ratio' | 'source'  // 目标类型
  targetAmount?: number        // 目标金额
  targetRatio?: number         // 目标比例
  sourceId?: string            // 收入来源 ID（仅 source 类型）
  deadline: Date               // 截止日期
  status: 'in_progress' | 'completed' | 'overdue'
  currentProgress: number      // 当前进度（%）
}

interface IncomeAction {
  id: string
  strategyId: string           // 策略 ID
  title: string                // 行动标题
  description: string          // 行动描述
  priority: 'high' | 'medium' | 'low'
  deadline?: Date              // 截止日期
  status: 'todo' | 'in_progress' | 'completed'
  expectedIncome?: number      // 预期收入提升
  completedAt?: Date           // 完成时间
}

interface IncomeStrategy {
  id: string
  name: string                 // 策略名称
  type: 'expert' | 'product' | 'leverage' | 'investment'  // 策略类型
  description: string          // 策略描述
  steps: string[]              // 执行步骤
  estimatedTime: string         // 预计时间
  expectedIncome?: number      // 预期收入提升
  isActive: boolean            // 是否激活
  appliedAt?: Date             // 应用时间
}
```

### 3.4 关键算法

**1. 收入增长预测算法**：

```typescript
function predictIncome(
  currentIncome: number,
  activeGrowthRate: number,
  passiveGrowthRate: number,
  years: number
): PredictionResult {
  let predictions = [];
  let totalIncome = currentIncome;
  let activeIncome = currentIncome * 0.7;  // 假设当前 70% 主动
  let passiveIncome = currentIncome * 0.3; // 假设当前 30% 被动

  for (let year = 1; year <= years; year++) {
    // 主动收入按年增长率增长
    activeIncome *= (1 + activeGrowthRate / 100);

    // 被动收入按年增长率增长
    passiveIncome *= (1 + passiveGrowthRate / 100);

    // 总收入
    totalIncome = activeIncome + passiveIncome;

    predictions.push({
      year,
      totalIncome,
      activeIncome,
      passiveIncome,
      passiveRatio: (passiveIncome / totalIncome * 100)
    });
  }

  return predictions;
}
```

**2. 财务自由达成预测**：

```typescript
function predictFinancialFreedom(
  currentNetWorth: number,
  monthlyIncome: number,
  passiveIncomeGrowthRate: number,
  savingsRate: number,
  targetAmount: number
): FreedomPrediction {
  let netWorth = currentNetWorth;
  let month = 0;
  let monthlyPassiveIncome = monthlyIncome * 0.3; // 当前被动收入

  while (netWorth < targetAmount && month < 600) { // 最多 50 年
    month++;

    // 每月储蓄
    let monthlySavings = monthlyIncome * savingsRate;

    // 被动收入增长
    monthlyPassiveIncome *= (1 + passiveIncomeGrowthRate / 12 / 100);

    // 净资产增长 = 储蓄 + 投资收益（假设 8% 年化）
    netWorth += monthlySavings + (netWorth * 0.08 / 12);
  }

  return {
    months: month,
    years: Math.floor(month / 12),
    monthsRemaining: month % 12,
    targetDate: addMonths(new Date(), month)
  };
}
```
