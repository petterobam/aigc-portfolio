# 技术栈选型方案

> 财富自由之路个人财务软件
> 版本：v0.1 MVP
> 更新日期：2026-03-22

---

## 一、选型原则

| 原则 | 说明 |
|------|------|
| 本地优先 | 财务数据敏感，本地存储优先 |
| 跨平台 | 一套代码，PC + 移动端复用 |
| 轻量级 | MVP阶段，快速迭代 |
| 易维护 | 个人项目，技术栈要熟悉 |

---

## 二、技术栈决策

### 2.1 PC 端

| 组件 | 选型 | 备选 | 理由 |
|------|------|------|------|
| 框架 | **Electron** | Tauri | 生态成熟，调试方便 |
| 前端 | **Vue 3 + TypeScript** | React | 学习曲线平缓，组合式API好用 |
| UI库 | **Element Plus** | Ant Design Vue | 组件丰富，文档完善 |
| 状态管理 | **Pinia** | Vuex | Vue3官方推荐，更简洁 |
| 数据库 | **SQLite (better-sqlite3)** | LowDB | 关系型，查询灵活，性能好 |

### 2.2 移动端（后续）

| 组件 | 选型 | 备选 | 理由 |
|------|------|------|------|
| 框架 | **React Native** | Flutter | 可复用部分逻辑，生态成熟 |
| UI库 | **React Native Paper** | NativeBase | Material Design，美观 |

### 2.3 共享逻辑

| 组件 | 方案 |
|------|------|
| 数据模型 | TypeScript 接口定义，两端共用 |
| 业务逻辑 | 核心计算逻辑抽离为纯函数 |
| API | 暂无后端，本地存储 |

## 五、数据存储方案

### 5.1 SQLite 表结构

```sql
-- 用户表
CREATE TABLE users (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  currency TEXT DEFAULT 'CNY',
  guarantee_months INTEGER DEFAULT 6,
  expected_return_rate REAL DEFAULT 8.0,
  settings TEXT, -- JSON
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 账户表
CREATE TABLE accounts (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL,
  name TEXT NOT NULL,
  type TEXT NOT NULL,
  icon TEXT,
  color TEXT,
  balance REAL DEFAULT 0,
  currency TEXT DEFAULT 'CNY',
  is_reserved INTEGER DEFAULT 0,
  include_in_net_worth INTEGER DEFAULT 1,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 负债表
CREATE TABLE debts (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL,
  name TEXT NOT NULL,
  type TEXT NOT NULL,
  total_amount REAL NOT NULL,
  remaining_amount REAL NOT NULL,
  monthly_payment REAL,
  interest_rate REAL,
  due_date DATE,
  is_paid_off INTEGER DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 交易记录表
CREATE TABLE transactions (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL,
  account_id TEXT NOT NULL,
  type TEXT NOT NULL,
  category TEXT NOT NULL,
  amount REAL NOT NULL,
  date DATE NOT NULL,
  note TEXT,
  tags TEXT, -- JSON array
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 目标表
CREATE TABLE goals (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL,
  type TEXT NOT NULL,
  target_amount REAL NOT NULL,
  current_amount REAL DEFAULT 0,
  monthly_expense REAL,
  target_date DATE,
  status TEXT DEFAULT 'not_started',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 梦想表
CREATE TABLE dreams (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL,
  title TEXT NOT NULL,
  description TEXT,
  image_url TEXT,
  estimated_cost REAL,
  monthly_cost REAL DEFAULT 0,
  priority INTEGER DEFAULT 3,
  is_achieved INTEGER DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 创建索引
CREATE INDEX idx_transactions_user_date ON transactions(user_id, date);
CREATE INDEX idx_transactions_account ON transactions(account_id);
CREATE INDEX idx_accounts_user ON accounts(user_id);
CREATE INDEX idx_debts_user ON debts(user_id);
CREATE INDEX idx_goals_user ON goals(user_id);
CREATE INDEX idx_dreams_user ON dreams(user_id);
```
