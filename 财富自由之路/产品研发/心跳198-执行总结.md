## 技术要点

### Element Plus 滑块的正确交互方式

**关键理解**:
1. Element Plus 的滑块不使用原生 `<input type="range">`
2. 使用自定义 DOM 结构（`.el-slider`）
3. 交互方式是点击滑块轨道，而非设置 input 值

**DOM 结构**:
```html
<div class="el-slider">
  <div class="el-slider__runway">
    <div class="el-slider__bar" style="width: 40%; left: 0%;"></div>
    <div class="el-slider__button-wrapper" tabindex="0" role="slider" ...></div>
  </div>
</div>
```

**正确的测试代码**:
```javascript
const slider = page.locator('.allocation-sliders .el-slider').nth(test.index);
const box = await slider.boundingBox();

// 计算点击位置（基于百分比）
const clickX = box.x + (box.width * (value / 100));
const clickY = box.y + box.height / 2;

// 点击滑块轨道
await page.mouse.click(clickX, clickY);
```
