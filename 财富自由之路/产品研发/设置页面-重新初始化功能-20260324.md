
**步骤**：
1. 点击"重新初始化"按钮
2. 阅读警告信息

## 🔧 技术实现

### resetDatabase 流程

```typescript
// 1. 用户点击"重新初始化"
showResetDialog()

// 2. 输入"确认"后执行
handleReset()

// 3. 调用 API
window.electronAPI.resetDatabase()

// 4. IPC 通信
ipcRenderer.invoke('database:reset')

// 5. 主进程处理
handleDatabaseReset() {
  db.exec('DELETE FROM dreams')
  db.exec('DELETE FROM goals')
  db.exec('DELETE FROM transactions')
  db.exec('DELETE FROM debts')
  db.exec('DELETE FROM accounts')
  db.exec('DELETE FROM users')
  return { success: true }
}

// 6. 清空 store
userStore.$reset()
accountStore.$reset()
...

// 7. 刷新页面
window.location.reload()
```

## 📝 代码示例

### Settings.vue 关键代码

```vue
<template>
  <el-button 
    type="danger" 
    @click="showResetDialog"
    :loading="resetting"
  >
    <el-icon><RefreshRight /></el-icon>
    重新初始化
  </el-button>
</template>

<script setup lang="ts">
const handleReset = async () => {
  if (confirmForm.value.text !== '确认') {
    ElMessage.warning('请输入"确认"以继续')
    return
  }

  resetting.value = true

  try {
    const result = await window.electronAPI.resetDatabase()
    
    if (result.success) {
      ElMessage.success('数据已清除，即将重新初始化...')
      
      // 清空所有 store
      userStore.$reset()
      accountStore.$reset()
      // ...
      
      // 刷新页面
      setTimeout(() => {
        window.location.reload()
      }, 1000)
    }
  } catch (error) {
    ElMessage.error('重置失败，请查看控制台日志')
  } finally {
    resetting.value = false
  }
}
</script>
```
