# 资产配置可视化工具 - 功能需求文档

> **目标**：帮助用户直观理解资产配置策略，动态调整投资组合，实现风险-收益平衡
> **优先级**：P1（中等优先级，高价值功能）
> **预计开发时间**：6-8 小时
> **创建日期**：2026-03-28 18:48

---

## 1. 功能概述

### 1.1 核心价值

资产配置是投资成功的关键，但许多用户无法直观理解：
- 我现在的配置是否合理？
- 不同配置的风险和收益如何？
- 如何根据自己的财务阶段调整策略？

本工具通过**可视化图表 + 交互式调整**，让资产配置变得简单易懂。

### 1.2 目标用户

- 已有投资资产，希望优化配置的用户
- 处于不同财务阶段（保障/安全/自由）的用户
- 想了解风险-收益平衡的投资者

### 1.3 核心功能

1. **资产配置向导**：根据财务阶段自动推荐配置
2. **可视化展示**：饼图 + 雷达图 + 收益曲线
3. **交互式调整**：拖拽滑块调整比例，实时看到影响
4. **对比分析**：对比当前配置 vs 推荐配置
5. **模拟预测**：预测未来 5/10/20 年的资产增长

```typescript
// 计算配置的多维度指标
const calculateMetrics = (alloc: { low: number, medium: number, high: number }) => {
  // 安全性：低风险占比越高，安全性越高
  const safety = alloc.low * 1.0 + alloc.medium * 0.6 + alloc.high * 0.2

  // 收益性：高风险占比越高，收益性越高
  const returnRate = alloc.low * 0.3 + alloc.medium * 0.8 + alloc.high * 1.0

  // 流动性：低风险资产流动性最好
  const liquidity = alloc.low * 1.0 + alloc.medium * 0.7 + alloc.high * 0.4

  // 波动性：高风险占比越高，波动性越大
  const volatility = alloc.low * 0.2 + alloc.medium * 0.6 + alloc.high * 1.0

  // 分散度：各部分比例越均衡，分散度越高
  const dispersion = 100 - Math.abs(alloc.low - 33.3) - Math.abs(alloc.medium - 33.3) - Math.abs(alloc.high - 33.3)

  return [safety, returnRate, liquidity, volatility, dispersion]
}
```

#### 模块 5：收益预测曲线

**功能**：
- 模拟当前配置下未来 5/10/20 年的资产增长
- 支持切换年份：5 年 / 10 年 / 20 年
- 显示：最乐观、预期、最悲观三种情况

**技术实现**：
```typescript
const calculateFutureGrowth = (years: number) => {
  const scenarios = {
    optimistic: { low: 0.04, medium: 0.12, high: 0.25 },  // 最乐观
    expected: { low: 0.03, medium: 0.08, high: 0.15 },  // 预期
    pessimistic: { low: 0.02, medium: 0.04, high: 0.05 } // 最悲观
  }

  const result: { year: number, optimistic: number, expected: number, pessimistic: number }[] = []
  let currentAssets = totalAssets.value

  for (let y = 1; y <= years; y++) {
    const yearlyReturn =
      (currentAssets * allocation.value.low / 100) * scenarios.expected.low +
      (currentAssets * allocation.value.medium / 100) * scenarios.expected.medium +
      (currentAssets * allocation.value.high / 100) * scenarios.expected.high

    currentAssets += yearlyReturn

    result.push({
      year: y,
      optimistic: calculateWithScenario(currentAssets, y, scenarios.optimistic),
      expected: currentAssets,
      pessimistic: calculateWithScenario(currentAssets, y, scenarios.pessimistic)
    })
  }

  return result
}

const initGrowthChart = () => {
```

## 3. 数据模型

### 3.1 核心数据结构

```typescript
// 财务阶段配置
interface StageConfig {
  name: string
  description: string
  allocation: {
    low: number      // 低风险比例 (%)
    medium: number   // 中风险比例 (%)
    high: number     // 高风险比例 (%)
  }
  targetAmount: string
  riskLevel: string
}

// 用户资产配置
interface UserAllocation {
  low: number
  medium: number
  high: number
}

// 资产类型详情
interface AssetType {
  name: string
  currentPercent: number
  recommendedPercent: number
  stars: number          // 1-5 星
  riskLevel: 'low' | 'medium' | 'high'
  expectedReturn: string // "2%~4%"
  examples: string       // "货币基金、债券基金、定期存款"
}

// 未来增长预测
interface FutureGrowth {
  year: number
  optimistic: number
  expected: number
  pessimistic: number
}
```

```typescript
  if (totalAssets.value < 60000) return 'guarantee'
  if (totalAssets.value < 1500000) return 'safety'
  return 'freedom'
})
```

## 4. 交互流程

### 4.1 首次使用流程

1. 用户打开页面
2. 系统自动检测当前财务阶段（根据净资产）
3. 显示该阶段的推荐配置
4. 用户可以通过滑块调整配置
5. 实时看到配置变化对风险-收益的影响
6. 用户点击"应用推荐配置"或保存自定义配置

### 4.2 调整配置流程

1. 用户选择不同的财务阶段（如从"财务安全"切换到"财务自由"）
2. 系统更新推荐配置
3. 用户调整滑块，实时看到：
   - 饼图更新
   - 雷达图变化
   - 表格数据变化
   - 收益预测曲线变化
4. 用户满意后，保存配置或导出报告

### 4.3 导出报告流程

1. 用户点击"导出配置报告"
2. 系统生成包含所有图表和数据的 PDF
3. 用户保存到本地
