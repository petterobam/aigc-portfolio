# 故障排除 - 初始化失败

## 问题描述

点击"开始使用"按钮后提示：**初始化失败，请检查输入信息后重试**

## 解决方案

### 方案 1：重新启动应用（推荐）

1. **关闭当前应用**
   ```bash
   pkill -9 -f "electron"
   ```

2. **删除旧数据库**
   ```bash
   rm -rf ~/Library/Application\ Support/@wealth-freedom/
   ```

3. **重新启动**
   ```bash
   cd ~/.openclaw/workspace/财富自由之路/产品研发/code/wealth-freedom
   ./start.sh
   ```

### 方案 2：检查控制台日志

1. **打开开发者工具**
   - 在应用窗口中按 `Cmd+Option+I`（Mac）或 `Ctrl+Shift+I`（Windows）
   - 切换到 **Console** 标签

2. **查看错误信息**
   - 重新点击"开始使用"按钮
   - 查看控制台中的红色错误信息
   - 将错误信息复制下来

3. **常见错误及解决方法**

#### 错误 1：`SqliteError: table users has no column named email`

**原因**：数据库表结构不匹配

**解决方法**：
```bash
# 删除旧数据库
rm -rf ~/Library/Application\ Support/@wealth-freedom/

# 重新启动应用
cd ~/.openclaw/workspace/财富自由之路/产品研发/code/wealth-freedom
./start.sh
```

#### 错误 2：`Error: Attempted to register a second handler for 'user:get'`

**原因**：应用重复启动

**解决方法**：
```bash
# 杀掉所有 electron 进程
pkill -9 -f "electron"

# 等待 3 秒
sleep 3

# 重新启动
cd ~/.openclaw/workspace/财富自由之路/产品研发/code/wealth-freedom
./start.sh
```

#### 错误 3：`TypeError: Cannot read property 'id' of undefined`

**原因**：用户创建失败

**解决方法**：
1. 检查输入信息是否完整
2. 确保所有必填字段都已填写
3. 查看主进程日志（终端输出）

### 方案 3：清除测试数据

如果之前已经初始化过测试数据，可能会导致冲突：

```bash
# 1. 停止应用
pkill -9 -f "electron"

# 2. 删除数据库
rm -rf ~/Library/Application\ Support/@wealth-freedom/

# 3. 删除编译缓存
cd ~/.openclaw/workspace/财富自由之路/产品研发/code/wealth-freedom
rm -rf apps/desktop/out

# 4. 重新启动
./start.sh
```

## 调试步骤

如果以上方案都无法解决，请按以下步骤收集信息：

### 1. 查看主进程日志

在启动应用的终端窗口中，查看是否有错误输出。

### 2. 查看渲染进程日志

- 按 `Cmd+Option+I` 打开开发者工具
- 切换到 **Console** 标签
- 查看错误信息

### 3. 检查数据库

```bash
# 查看数据库文件是否存在
ls -la ~/Library/Application\ Support/@wealth-freedom/desktop/

# 如果存在，查看数据库内容
sqlite3 ~/Library/Application\ Support/@wealth-freedom/desktop/wealth-freedom/data.db
.tables
.schema users
SELECT * FROM users;
.quit
```

### 4. 检查网络请求

在开发者工具中：
- 切换到 **Network** 标签
- 重新点击"开始使用"按钮
- 查看是否有失败的请求

## 临时解决方案

如果初始化始终失败，可以：

1. **手动创建用户数据**

```sql
-- 打开数据库
sqlite3 ~/Library/Application\ Support/@wealth-freedom/desktop/wealth-freedom/data.db

-- 插入用户数据
INSERT INTO users (id, name, currency, guarantee_months, expected_return_rate, created_at, updated_at)
VALUES ('user-1', '测试用户', 'CNY', 6, 8.0, datetime('now'), datetime('now'));

-- 退出
.quit
```

2. **跳过欢迎页面**

修改 `src/renderer/src/App.vue`，将 `showWelcome` 默认设置为 `false`。
