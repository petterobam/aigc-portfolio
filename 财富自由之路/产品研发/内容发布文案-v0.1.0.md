# 90万资产如何配置？我用工具算出了科学方案

## 一、为什么你需要资产配置？

很多人有投资，但没有**资产配置**。

- 只有一支股票 = 赌博
- 只有一种基金 = 孤注一掷
- 什么都买一点 = 撒胡椒面，没有策略

**科学投资 = 多元化资产配置**

资产配置的意义：
- ✅ 降低风险：不要把鸡蛋放在一个篮子里
- ✅ 稳定收益：不同资产互补，平滑波动
- ✅ 长期复利：让时间成为你的朋友

## 二、90万资产如何配置？

我手头有90万可投资资产，想知道怎么配置才科学。

于是我开发了一款工具，输入数据后得到了这样的配置方案：

### 财务安全阶段配置

**我的财务状况**：
- 可投资资产：90万元
- 月支出：10,000元
- 目标：靠投资利息覆盖日常支出（财务安全）

**推荐配置方案**：
```
低风险资产：30% = 27万元
  - 银行存款/货币基金
  - 国债/债券
  - 稳健型理财

中风险资产：50% = 45万元
  - 指数基金/ETF
  - 混合基金
  - 蓝筹股

高风险资产：20% = 18万元
  - 个股
  - 行业主题基金
```

**预期收益**：
- 年化收益率：7.5%
- 年收入：67,500元
- 月被动收入：5,625元

**覆盖我的月支出（10,000元）**：56.3%

距离财务安全（被动收入 > 100%）还需时间，但至少路径清晰了。

## 三、不同财务阶段，配置策略不同

很多人用同一套配置策略，这是不对的。

**财务保障阶段**（目标：6个月储备金）
- 配置重点：100% 低风险
- 目标：快速积累储备金，不被意外击倒
- 投资产品：银行存款、货币基金、国债

**财务安全阶段**（目标：靠利息生活）
- 配置重点：30%低 + 50%中 + 20%高
- 目标：稳定增长 + 适度风险，年化 7-8%
- 投资产品：指数基金、蓝筹股、债券基金

**财务自由阶段**（目标：靠利息实现梦想）
- 配置重点：20%低 + 40%中 + 40%高
- 目标：追求更高收益，年化 10%+
- 投资产品：成长型基金、股票、REITs
