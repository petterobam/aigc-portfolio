## ⚠️ 注意事项

### 1. 打包失败处理
如果打包过程中遇到问题，可以尝试：
- 清理缓存：`rm -rf node_modules/.vite`
- 重新安装依赖：`pnpm install`
- 检查 electron-builder 配置：`apps/desktop/electron-builder.yml`
- 查看详细日志：`pnpm build -- --debug`

### 2. 网络问题
如果网络下载缓慢，可以尝试：
- 使用国内镜像（如淘宝镜像）
- 使用代理加速（如 Clash、V2Ray）
- 在非高峰时段打包（如凌晨 2-6 点）
- 提前下载 Electron 二进制文件到本地缓存

### 3. GitHub 认证
推送代码到 GitHub 时，可能需要：
- 配置 SSH 密钥：`ssh-keygen -t rsa -b 4096 -C "your.email@example.com"`
- 创建 Personal Access Token（Settings → Developer settings → Personal access tokens）
- 在推送时输入密码或 Token

### 4. 安装包签名
如果要进行公网分发，建议：
- macOS：申请 Apple Developer 证书（$99/年）
- Windows：购买代码签名证书（$100-500/年）
- 证书可以避免用户安装时的安全警告
