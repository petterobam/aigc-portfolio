# 财富自由之路 - 快速上手指南

> 5 分钟开始你的财务自由之旅

---

## 第一步：启动应用

打开终端，执行：

```bash
cd ~/.openclaw/workspace/财富自由之路/产品研发/code/wealth-freedom
./start.sh
```

首次运行会自动安装依赖，稍等片刻即可。

---

## 第二步：填写财务现状

应用启动后，进入「财务现状」页面，填写：

### 必填项目

| 类别 | 项目 | 说明 |
|------|------|------|
| **资产** | 现金及活期存款 | 银行卡、支付宝、微信余额 |
| | 定期存款 / 余额宝 | 货币基金、定存 |
| | 投资资产 | 股票、基金、债券（市值） |
| **负债** | 消费债 | 信用卡欠款、花呗、借呗 |
| | 贷款 | 房贷、车贷剩余本金 |
| **收支** | 月收入 | 税后到手工资 + 其他收入 |
| | 月支出 | 房租、餐饮、交通、购物等 |

### 填写原则

- **真实数字才有意义** - 不必精确到元，但数量级要对
- **不要逃避负债** - 正视现实是改变的第一步
- **定期更新** - 建议每周更新一次

---

## 第三步：设定财务自由目标

进入「目标设定」页面，系统会自动计算：

### 三阶段目标

| 阶段 | 公式 | 你的目标 |
|------|------|----------|
| 财务保障 | 月支出 × 6~12 | ______ 元 |
| 财务安全 | 月所需 × 150 | ______ 元 |
| 财务自由 | 梦想月成本 × 150 | ______ 元 |

> 基于 8% 年化收益率计算（12 ÷ 8% ≈ 150）

---

## 第四步：开始记账

进入「记账」页面：

1. **记录每笔收支** - 3 秒内完成一笔
2. **分类标注** - 必要支出 vs 消费欲望
3. **每周回顾** - 查看周报、月报

### 记账技巧

- 设定固定时间（如睡前 5 分钟）
- 大额支出单独记录
- 不要追求完美，持续比精确更重要

## 核心理念

### 财务自由三阶段

```
财务保障 → 财务安全 → 财务自由
(6-12月储备) (利息覆盖日常) (利息支撑梦想)
```

**不要跳步！** 先完成财务保障，再追求财务安全。

### 50/50 还债原则

如果你有消费债（信用卡、花呗等）：

- 每月余钱的 **50%** 用于还债
- 每月余钱的 **50%** 用于储蓄

> 不要把所有钱都还债，否则会失去积累的动力

### 支付自己原则

每月收入到账后：

1. **先** 把 10% 存入「鹅账户」（投资账户）
2. **再** 支付其他开支
3. 鹅账户的钱 **永远不花**，只用于投资
