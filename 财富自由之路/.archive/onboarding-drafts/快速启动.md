# 快速启动指南

## 一键启动

```bash
cd ~/.openclaw/workspace/财富自由之路/产品研发/code/wealth-freedom
pnpm dev
```

## 首次启动

首次启动会自动：
1. 安装所有依赖
2. 构建共享包
3. 启动 Electron 应用

应用会自动打开，展示测试数据。

## 下一步

1. 浏览各个页面，熟悉功能
2. 清空测试数据
3. 填写真实财务数据：
   - 账户余额（银行卡、支付宝、微信等）
   - 负债情况（房贷、信用卡等）
   - 月度收支（收入、支出分类）
   - 财务目标（保障、安全、自由）
   - 梦想清单（可视化目标）
