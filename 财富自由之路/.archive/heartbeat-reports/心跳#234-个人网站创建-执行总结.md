## 📅 下一步行动

### 短期（待用户执行）

1. **安装 Hugo** ⏳
   - 命令：`brew install hugo`
   - 验证：`hugo version`
   - 时间：5-10 分钟

2. **添加主题** ⏳
   - 命令：`git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke`
   - 时间：1-2 分钟

3. **本地预览** ⏳
   - 命令：`hugo server -D`
   - 访问：http://localhost:1313
   - 时间：1-2 分钟

4. **创建 GitHub 仓库** ⏳
   - 访问：https://github.com/new
   - 仓库名称：`yourname.github.io`
   - 设置为 Public
   - 点击 "Create repository"
   - 时间：1-2 分钟

5. **推送代码** ⏳
   - 命令：`git remote add origin https://github.com/YOUR_USERNAME/yourname.github.io.git`
   - 命令：`git push -u origin main`
   - 时间：2-5 分钟

6. **配置 GitHub Pages** ⏳
   - 进入仓库的 Settings → Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main" 和 "/ (root)"
   - 点击 Save
   - 时间：1-2 分钟

7. **访问网站** ⏳
   - 访问：`https://yourname.github.io`
   - 验证网站正常运行
   - 时间：1-2 分钟（GitHub Pages 构建时间）
