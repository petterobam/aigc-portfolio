## 下一步行动

### 立即行动（用户执行）

1. **安装 Hugo**
   ```bash
   brew install hugo
   ```

2. **创建站点**
   ```bash
   cd ~/.openclaw/workspace/企业家路径/personal-website
   hugo new-site .
   cd .
   ```

3. **添加主题**
   ```bash
   git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
   echo "theme = 'ananke'" >> config.toml
   ```

4. **创建页面**
   ```bash
   hugo new about.md
   hugo new services.md
   hugo new case-studies.md
   hugo new contact.md
   ```

5. **编写内容**
   - 将 `企业家路径/个人网站内容准备.md` 中的内容复制到对应页面
   - 编辑 `content/about.md`
   - 编辑 `content/services.md`
   - 编辑 `content/case-studies.md`
   - 编辑 `content/contact.md`

6. **预览网站**
   ```bash
   hugo server -D
   # 访问：http://localhost:1313
   ```
