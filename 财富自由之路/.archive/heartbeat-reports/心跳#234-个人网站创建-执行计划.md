### 2. 技术方案确认

**技术栈**：
- **静态网站生成器**：Hugo（Go 语言开发，速度快）
- **主题**：Ananke（简洁、响应式、现代设计）
- **托管平台**：GitHub Pages（免费、稳定、易于部署）
- **域名**：暂时使用 GitHub Pages 默认域名（后续可购买自定义域名）

**网站结构**：
- 首页（Home）：核心价值主张 + 服务简介 + 成功案例 + CTA
- 关于我（About）：个人简介 + 核心能力 + 工作经历 + 技术文章 + 开源项目
- 服务（Services）：AIGC 技术咨询、定制化开发、培训课程
- 案例（Case Studies）：3 个精选案例（电商、教育、媒体）
- 联系我（Contact）：联系方式 + 咨询表单 + 常见问题
- 技术博客（Blog）：技术文章列表
