# 图标导入错误修复 - 2026-03-25

## 问题描述

**错误信息**：
```
Uncaught (in promise) SyntaxError: The requested module '.../@element-plus/icons-vue.js' 
does not provide an export named 'Bulb'
```

**原因**：Element Plus Icons v2.3.2 中没有 `Bulb` 图标

## 修复方案

### 1. Goals.vue

**修改位置**：`apps/desktop/src/renderer/src/views/Goals.vue`

**修改内容**：
- 导入语句：`Bulb` → `Sunny`
- 模板使用：`<Bulb />` → `<Sunny />` (3处)

**修改前**：
```typescript
import { InfoFilled, PictureFilled, Clock, Bulb, SuccessFilled } from '@element-plus/icons-vue'
```

**修改后**：
```typescript
import { InfoFilled, PictureFilled, Clock, Sunny, SuccessFilled } from '@element-plus/icons-vue'
```

### 2. Dashboard.vue

**修改位置**：`apps/desktop/src/renderer/src/views/Dashboard.vue`

**修改内容**：
- 导入语句：`Bulb` → `Sunny`
- 模板使用：`<Bulb />` → `<Sunny />` (1处)

**修改前**：
```typescript
import { Plus, Flag, PictureFilled, Bulb } from '@element-plus/icons-vue'
```

**修改后**：
```typescript
import { Plus, Flag, PictureFilled, Sunny } from '@element-plus/icons-vue'
```

## 修复结果

- ✅ Goals.vue 修复完成（导入 + 3处使用）
