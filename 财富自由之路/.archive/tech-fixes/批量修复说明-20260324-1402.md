### 3. ❌ CSP 策略阻止加载外部图片

**问题**：
```
Refused to load the image 'https://images.unsplash.com/...' because it violates the following Content Security Policy directive: "default-src 'self'"
```

**影响的页面**：
- 梦想图册（Dreams.vue）

**解决方案**：
- ✅ 修改 CSP 策略，添加 `img-src 'self' https: data:`

**修改的文件**：
- `src/renderer/index.html`

**修改后的 CSP**：
```html
<meta http-equiv="Content-Security-Policy" content="default-src 'self'; script-src 'self'; style-src 'self' 'unsafe-inline'; img-src 'self' https: data:;">
```
