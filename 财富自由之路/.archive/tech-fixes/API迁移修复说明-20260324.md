# API 迁移修复说明

## 问题描述

启动应用时出现错误：
```
TypeError: Cannot read properties of undefined (reading 'user')
```

## 根本原因

编译后的 preload 脚本与源代码不一致：

**源代码（preload.ts）**：
- API 名称：`window.api`
- 结构：嵌套对象（`window.api.user.get()`）

**编译后（out/preload/index.js）**：
- API 名称：`window.electronAPI`
- 结构：扁平化（`window.electronAPI.getUser()`）

## 解决方案

### 1. 创建类型定义文件

创建了 `src/renderer/src/types/global.d.ts`，定义了正确的 TypeScript 类型。

### 2. 批量替换 API 调用

使用脚本批量替换所有 store 文件中的 API 调用：

| 原调用 | 新调用 |
|--------|--------|
| `window.api.user.get()` | `window.electronAPI.getUser()` |
| `window.api.user.create(data)` | `window.electronAPI.createUser(data)` |
| `window.api.account.getAll()` | `window.electronAPI.getAccounts()` |
| `window.api.account.create(data)` | `window.electronAPI.createAccount(data)` |
| ... | ... |

### 3. 修改的文件

- ✅ `src/renderer/src/types/global.d.ts` - 新建
- ✅ `src/renderer/src/App.vue`
- ✅ `src/renderer/src/stores/user.ts`
- ✅ `src/renderer/src/stores/accounts.ts`
- ✅ `src/renderer/src/stores/debts.ts`
- ✅ `src/renderer/src/stores/goals.ts`
- ✅ `src/renderer/src/stores/transactions.ts`
