# 重新初始化功能修复 - 紧急说明

## 🐛 问题

刷新页面后仍然报错：
```
TypeError: window.electronAPI.resetDatabase is not a function
```

## 🔍 根本原因

preload 脚本在 Electron **启动时**加载，而不是在页面刷新时加载。
修改编译后的文件（`out/preload/index.js`）后，需要**完全重启应用**才能生效。

## ✅ 解决方案

### 必须完全重启应用！

**步骤**：
1. **完全关闭应用**
   - 按 `Ctrl+C` 或 `Cmd+Q`
   - 或者在活动监视器中强制退出 Electron 进程

2. **重新启动应用**
```bash
cd ~/.openclaw/workspace/财富自由之路/产品研发/code/wealth-freedom
pnpm dev
```

或：
```bash
./start.sh
```

3. **测试功能**
   - 打开应用
   - 点击侧边栏的"设置"
   - 点击"重新初始化"
   - 输入"确认"
   - 点击确认按钮

## ⚠️ 重要提示

**刷新页面（Cmd+R）不会重新加载 preload 脚本！**

必须完全关闭并重新启动 Electron 应用。

## 🔍 验证 preload 脚本

如果重启后仍然报错，请在浏览器控制台（Cmd+Option+I）中检查：

```javascript

```

如果输出中没有 `resetDatabase` 方法，说明 preload 脚本没有正确加载。
