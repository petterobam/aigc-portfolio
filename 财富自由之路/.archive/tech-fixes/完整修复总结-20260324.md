## 🔧 修复详情

### 1. 数据库表结构修复

#### users 表
```sql
CREATE TABLE IF NOT EXISTS users (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  email TEXT,
  avatar TEXT,
  currency TEXT DEFAULT 'CNY',
  guarantee_months INTEGER DEFAULT 6,
  expected_return_rate REAL DEFAULT 8.0,
  settings TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
)
```

#### debts 表
```sql
CREATE TABLE IF NOT EXISTS debts (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL,
  name TEXT NOT NULL,
  type TEXT NOT NULL,
  total_amount REAL NOT NULL,
  paid_amount REAL DEFAULT 0,           -- ✅ 新增
  remaining_amount REAL NOT NULL,
  monthly_payment REAL,
  interest_rate REAL,
  priority INTEGER DEFAULT 3,           -- ✅ 新增
  due_date DATE,
  notes TEXT,                           -- ✅ 新增
  is_paid_off INTEGER DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id)
)
```

#### goals 表
```sql
CREATE TABLE IF NOT EXISTS goals (
  id TEXT PRIMARY KEY,
  user_id TEXT NOT NULL,
  stage TEXT NOT NULL,                  -- ✅ 修改（原为 type）
  target_amount REAL NOT NULL,
  current_amount REAL DEFAULT 0,
  target_date DATE,
  notes TEXT,                           -- ✅ 修改（原为 monthly_expense）
  status TEXT DEFAULT 'in_progress',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id)
)
```
