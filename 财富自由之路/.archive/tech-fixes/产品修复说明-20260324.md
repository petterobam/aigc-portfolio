### 3. 初始化失败问题

**问题**：填写所有信息后点击"开始使用"提示"初始化失败"

**根本原因**：
用户创建时缺少必需的数据库字段

**解决方案**：
- ✅ 修复了用户创建的 SQL 语句
- ✅ 添加了所有必需字段：id, name, email, avatar, currency, guarantee_months, expected_return_rate
- ✅ 为所有字段提供了合理的默认值

**修改文件**：
- `apps/desktop/src/main/ipcHandlers.ts`

**修复详情**：
```typescript
// 修复前：只插入 name, email, avatar（缺少必需字段）
INSERT INTO users (name, email, avatar, created_at, updated_at)
VALUES (?, ?, ?, datetime('now'), datetime('now'))

// 修复后：插入所有必需字段
INSERT INTO users (id, name, email, avatar, currency, guarantee_months, expected_return_rate, created_at, updated_at)
VALUES (?, ?, ?, ?, ?, ?, ?, datetime('now'), datetime('now'))
```
