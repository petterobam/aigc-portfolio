# 目标显示修复 - 2026-03-25

## 问题描述

**用户反馈**：录入了月支出数据后，软件里面的目标还是没算出来

**根本原因**：
1. 进度条计算使用的是 `netWorth`（净资产），显示正确
2. 但"当前储备/可投资资产"显示使用的是数据库中的 `goal.currentAmount`（静态值）
3. 两者不一致，导致用户看到进度条和数字不匹配

## 数据库实际数据

**用户数据**：
- 现金及存款：700,000 元
- 投资资产：400,000 元
- 负债：80,000 元
- **净资产**：1,020,000 元
- 月收入：30,000 元
- 月支出：10,000 元
- **月结余**：20,000 元
- **储蓄率**：66.7%

**三阶段目标**：
1. **财务保障**：目标 60,000 元
   - 当前进度：1,020,000 / 60,000 = **1700%** ✅ 已达成
   - 差距：0 元

2. **财务安全**：目标 1,500,000 元
   - 当前进度：1,020,000 / 1,500,000 = **68%**
   - 差距：480,000 元
   - 预计达成：24 个月（按每月储蓄 20,000 元）

3. **财务自由**：目标 3,000,000 元
   - 当前进度：1,020,000 / 3,000,000 = **34%**
   - 差距：1,980,000 元
   - 预计达成：99 个月（约 8 年）

## 修复方案

### 修改文件：`apps/desktop/src/renderer/src/views/Goals.vue`

#### 1. 财务保障 - 当前储备显示

**修改前**：
```vue
<span class="value highlight">{{ formatCurrency(securityGoal?.currentAmount || 0) }}</span>
```

**修改后**：
```vue
<span class="value highlight">{{ formatCurrency(netWorth) }}</span>
```

#### 2. 财务安全 - 当前可投资资产显示

**修改前**：
```vue
<span class="value highlight">{{ formatCurrency(safetyGoal?.currentAmount || 0) }}</span>
```

**修改后**：
```vue
<span class="value highlight">{{ formatCurrency(netWorth) }}</span>
```

#### 3. 财务自由 - 当前可投资资产显示

**修改前**：
```vue
<span class="value highlight">{{ formatCurrency(freedomGoal?.currentAmount || 0) }}</span>
```

**修改后**：
```vue
<span class="value highlight">{{ formatCurrency(netWorth) }}</span>
```

#### 4. 预计达成时间计算

**修改前**：
```typescript
const gap = goal.targetAmount - goal.currentAmount
const months = Math.ceil(gap / 5000)
```

**修改后**：
```typescript
const gap = goal.targetAmount - netWorth.value
const months = Math.ceil(gap / 20000)  // 基于用户的实际月结余
```
