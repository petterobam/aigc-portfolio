# 自我成长方法论

> 财富自由的心理基础：目标清晰、责任在己、持续成长。

## 包含的方法论

- **自我成长方法论** - 目标设定、责任承担、竹子理论、好运公式
- **金钱观改造方法论** - 信念识别、四步改变法、杠杆作用

## 核心原则

1. **责任 = 权力** - 把责任推给谁，就给了谁权力
2. **目标拓宽机会** - 目标越清晰，机会感知越强
3. **信念决定现实** - 改变信仰，才能改变财务状况

## 发生变化的五个层次

1. 意识到对现状不满意
2. 意识到仅仅行动还不够
3. 技巧起到了帮助
4. 思考世界观、金钱观
5. 改变对自身的认知，主动负责

## 好运的组成

1. 储存资本（准备）
2. 识别机会（眼光）
3. 果断行动（勇气）

---

**目标：成为最好的自己。**
