## 🔍 检查维度

### 维度1：理论理解度

- [ ] **核心概念**：能否用1-2句话说明白方法论的核心？
- [ ] **适用场景**：明确知道什么情况下使用这个方法论？
- [ ] **价值定位**：清楚这个方法论解决了什么问题？

### 维度2：实操可行性

- [ ] **步骤清晰**：能否按照步骤一步步执行？
- [ ] **工具准备**：所需工具和资源是否已准备？
- [ ] **时间预算**：知道需要投入多少时间？

### 维度3：落地执行度

- [ ] **已执行次数**：实际使用了多少次？
- [ ] **执行效果**：达到预期效果了吗？
- [ ] **遇到障碍**：遇到了哪些问题？

### 维度4：数据反馈

- [ ] **量化指标**：能否用数据衡量效果？
- [ ] **数据收集**：是否在收集相关数据？
- [ ] **定期回顾**：多久回顾一次效果？

### 维度5：持续优化

- [ ] **迭代计划**：是否有持续优化的计划？
- [ ] **经验沉淀**：是否记录了成功经验和失败教训？
- [ ] **分享输出**：是否可以分享给他人？
