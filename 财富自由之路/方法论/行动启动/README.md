## 核心概念

### 最小行动单元

每周只需填写3个数字：
1. 净资产（资产 - 负债）
2. 月收入（税后到手）
3. 月支出（所有支出）

**预计耗时**：1 分钟
**收益**：从这1分钟开始，构建10年的复利效应

### 启动的三个层次

1. **1分钟快速启动** — 最低门槛，只填3个数字
2. **10分钟完整启动** — 填写完整财务现状
3. **30分钟深度启动** — 系统化管理财务

### 核心信念

- 不完美的开始，胜过完美的等待
- 行动会带来新的行动
- 错误是成功的必要部分
- 从"知道"变成"做到"

---

## 使用场景

- 还没开始但想开始 → 阅读行动启动方法论
- 不知道从哪开始 → 1分钟快速启动
- 想一次性建立框架 → 10分钟完整启动
- 想系统化管理 → 30分钟深度启动
