# 收入增长方法论

> 整天工作的人没有时间赚钱。目标是构建系统，不是出卖时间。

## 核心洞察

**收入层次模型**：
```
第一层：主动收入（用时间换钱）
    → 工资、接单、咨询

第二层：半被动收入（一次投入，持续回报）
    → 软件产品、课程、内容版权

第三层：被动收入（资本工作）
    → 投资分红、利息、租金
```

## 核心原则

1. **开源 > 节流** - 节流有上限，开源无上限
2. **产品化服务** - 将专业技能变成可复用的产品
3. **专家定位** - 成为某个细分领域的专家

## 7年收入翻倍路径

- 专注主业（提升核心技能）
- 副业产品化（构建资产）
- 强制储蓄投资（让钱工作）

## 百万富翁的构成

- 74% 企业家
- 10% 高级管理人员
- 10% 从业者（医生、律师等）
- 5% 销售人员
- 1% 其他

---

**目标：从卖时间到卖产品，再到让钱工作。**
