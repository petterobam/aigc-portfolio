# 财务规划方法论

> 先保障，再安全，再自由。跳步是最大的风险。

## 包含的方法论

- **财务自由三阶段方法论** - 保障→安全→自由的完整路径
- **储蓄与复利方法论** - 支付自己、鹅账户、72法则
- **债务管理方法论** - 50/50还债原则、13条建议

## 核心原则

1. **财务保障优先** - 6-12个月储备金，不可用于投资
2. **绝不动用本金** - 本金是下金蛋的鹅
3. **开源 > 节流** - 节流有上限，开源无上限

## 三阶段目标公式

| 阶段 | 公式 | 说明 |
|------|------|------|
| 财务保障 | 月支出 × 6~12 | 储备金 |
| 财务安全 | 月所需 × 150 | 8%年化，靠利息生活 |
| 财务自由 | 梦想月成本 × 150 | 8%年化，靠利息圆梦 |
