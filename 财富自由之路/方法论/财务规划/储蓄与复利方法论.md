# 储蓄与复利方法论

> 来源：《财富自由之路》第8-9章
> 提炼日期：2026-03-22

---

## 核心原则

**使你变得富有的是储蓄，而非收入。**

财富产生于你对金钱的留存。
所谓"必要支出"总会增长到收入水平。

---

## 支付自己原则

**在月初将收入的 10% 存入独立账户**

| 原则 | 说明 |
|------|------|
| 先支付自己 | 月初存钱，而非月末剩多少存多少 |
| 鹅账户 | 独立账户，万万不可动用 |
| 10%起步 | 用90%生活与用100%没区别 |
| 加薪50% | 每次加薪将50%存入鹅账户 |

### 为什么储蓄计划失败？

1. 花销超标
2. 只存月末剩下的钱（往往没剩）
3. 认为储蓄=限制享受

### 正确做法

- 月初自动转账（设立长期划付委托）
- 从未习惯这笔钱的存在
- 加薪后立即存一半，趁还没习惯新收入

## 复利三要素

```
复利收益 = 时间 × 利润率 × 投入
```

| 要素 | 影响 | 行动 |
|------|------|------|
| **时间** | 越早开始，每月所需越少 | 现在就开始 |
| **利润率** | 决定翻倍速度 | 72÷利率=翻倍年数 |
| **投入** | 本金多少 | 持续存入 |

### 72法则

- **72 ÷ 利率 = 投资翻倍所需年数**
- **72 ÷ 通胀率 = 货币贬值一半所需年数**

例子：
- 8%利率 → 9年翻倍
- 4%通胀 → 18年购买力减半

---

## 鹅与金蛋

**不要杀鹅取卵**

- 鹅 = 本金（绝不动用）
- 金蛋 = 利息/收益（用来生活）

> 本金是下金蛋的鹅。动用本金 = 杀鹅。

---

## 创造新收入来源练习

**18个月储蓄挑战：**
- 每月存入金额是上月的2倍
- 从1元或10元开始
- 目的：锻炼寻找新收入来源的能力

---

## 储蓄信念重塑

| 旧信念 | 新信念 |
|--------|--------|
| 储蓄=限制享受 | 储蓄=支付自己 |
| 将来挣多了再存 | 现在就开始，10%起步 |
| 利率太低存钱没意义 | 复利+时间是关键 |
| 通胀会吃掉存款 | 有形资产对抗通胀 |

## 产品功能映射

| 方法论 | 产品功能 |
|--------|---------|
| 支付自己 | 自动储蓄提醒/设定 |
| 鹅账户 | 专项账户追踪（不可动用标记） |
| 10%原则 | 收入自动分配 |
| 复利计算 | 达成目标时间预测 |
| 72法则 | 翻倍/贬值计算器 |
