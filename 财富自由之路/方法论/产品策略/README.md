# 产品策略方法论

> 软件产品是最接近"让钱工作"的路径之一。

## 包含的方法论

- **产品变现方法论** - 五步变现法、定价策略、发布渠道

## 核心原则

1. **MVP优先** - 先验证核心需求，再扩展功能
2. **解决自己的问题** - 能解决自己的问题，才推给别人
3. **订阅制优先** - 稳定现金流，适合工具类产品

## 产品定位

- **目标用户**：有财务自由意识但缺乏系统工具的普通人
- **核心价值**：把复杂的财务规划变成可追踪的进度条
- **差异化**：三阶段财务自由框架内置，不只是记账工具

## 变现路径优先级

1. 订阅制（月付/年付）- 稳定现金流
2. 一次性买断 - 高单价、低更新频率
3. 增值服务 - AI财务顾问、定制报告

## 发布渠道

- 产品官网 + 支付系统
- 应用商店（Mac App Store、Microsoft Store）
- 社交媒体（小红书、知乎、B站）
- 口碑传播（用户推荐）

---

**目标：让产品产生被动收入，实现财务自由的一部分。**
