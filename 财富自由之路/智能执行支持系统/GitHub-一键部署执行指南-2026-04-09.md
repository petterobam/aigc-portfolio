## 🚨 注意事项

### GitHub Actions 权限
确保仓库已启用 GitHub Actions：
1. 仓库设置 → Actions → General
2. 确保 "Allow all actions and reusable workflows" 已启用

### 部署问题排查
如果部署失败：
1. 检查 GitHub Actions 日志
2. 确认 Node.js 版本配置正确
3. 检查 package.json 中的构建脚本

### 推送权限
确保有权限推送代码：
```bash
# 测试连接
git remote -v
```
