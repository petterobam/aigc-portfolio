# 产品数据

> 产品上线后的运营数据追踪。

## 追踪指标

### 用户指标
- 下载量
- 活跃用户（日活/周活/月活）
- 用户留存率
- 用户反馈数量

### 收入指标
- 付费用户数
- 月度收入（MRR）
- 年度收入（ARR）
- 付费转化率

### 产品指标
- 功能使用率
- 崩溃率
- 用户满意度

## 数据记录

- `downloads.csv` - 下载量记录
- `revenue.csv` - 收入记录
- `feedback.csv` - 用户反馈记录

## 更新频率

- 下载量：每日
- 收入：每日
- 活跃用户：每周
- 完整报告：每月
