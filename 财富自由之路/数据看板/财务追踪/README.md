# 财务追踪

> 真实的财务数据，是通向财务自由的第一步。

## 这个目录的作用

存放你的个人财务数据，包括：
- 净资产（资产 - 负债）
- 月度现金流（收入 - 支出）
- 财务自由三阶段进度
- 负债清单
- 被动收入追踪

## 核心文件

| 文件 | 用途 |
|------|------|
| `current-state.md` | 财务状态快照，每月更新一次 |
| `财务数据收集指南.md` | 如何收集和整理财务数据 |

## 为什么重要

**没有数据，一切都是空谈。**

只有填写真实数字，才能：
- 清楚知道自己有多少钱
- 计算离财务自由还有多远
- 追踪每月的进步
- 发现财务问题并及时调整

## 如何开始

1. 📄 打开 `财务数据收集指南.md`
2. ✏️ 按照指南收集数据（15-20分钟）
3. 💻 填写到 `current-state.md` 或产品中
4. 📊 查看财务自由进度

## 更新频率

- **每周/每月**：更新账户余额
- **每天/每周**：记录收支
- **每季度**：复盘财务自由进度

---

**记住：数字不会说谎。只有正视现实，才能改变现实。**
