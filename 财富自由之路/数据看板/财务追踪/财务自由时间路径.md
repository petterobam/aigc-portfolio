## 附录：复利计算公式

### 月度复利公式

```
FV = PV × (1 + r)^n + PMT × [(1 + r)^n - 1] / r
```

其中：
- FV = 未来价值（Future Value）
- PV = 现值（Present Value）
- r = 月收益率（年化8% → 月0.67%）
- n = 月数
- PMT = 每月投入金额

### 示例计算

**5年后（60个月）**：
```
FV = 0 × (1.0067)^60 + 20,000 × [(1.0067)^60 - 1] / 0.0067
   = 0 + 20,000 × [1.4922 - 1] / 0.0067
   = 20,000 × 73.46
   = ¥1,469,200
```

考虑更精确的月复利计算，实际值约为 ¥1,568,448
