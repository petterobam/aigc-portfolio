## 第五步：计算关键数据（2分钟）

### 5.1 净资产

```
资产合计 = 现金及活期 + 定期存款 + 投资资产 + 固定资产
负债合计 = 消费债 + 贷款 + 其他负债
净资产 = 资产合计 - 负债合计
```

### 5.2 月现金流

```
月收入 = 主动收入 + 被动收入
月支出 = 固定支出 + 变动支出
月结余 = 月收入 - 月支出
```

### 5.3 储蓄率

```
储蓄率 = 月结余 / 月收入 × 100%
```

## 第六步：填写产品数据

现在你有了所有数据，可以：

1. **启动产品**：运行 `./start.sh`
2. **创建账户**：在"账户管理"中添加所有资产账户
3. **记录负债**：在"债务管理"中添加所有负债
4. **记录交易**：在"记账"中录入本月收支
5. **查看仪表盘**：自动生成财务自由进度图

## 💡 小贴士

### 快速估算技巧

- **不记得精确数字？** 用近似值，比如"大约2万"比空着强
- **多张银行卡？** 先记主要的2-3张，其他的慢慢补充
- **不确定投资市值？** 打开股票/基金APP，记录当前市值
- **支出记不清？** 查看支付宝/微信的"账单"功能，有分类统计

### 数据安全

- 这些数据只存储在你的本地电脑
- 不会上传到任何服务器
- 定期备份数据库文件即可

### 保持更新

- **每周/每月**更新一次账户余额
- **每天/每周**记录一次收支
- **每季度**复盘一次财务自由进度
