## 为什么现在录入？

**月度现金流已经完成**（收入 30k / 支出 10k / 结余 20k），但净资产还是空白。

净资产 = 你真正的财富水平

- 月度现金流告诉你"每月能存多少"
- 净资产告诉你"现在真正有多少"
- 两者结合，才能计算财务自由时间

**只差这 5 个数字，就能看到完整的财务全景图。**

## 5 个数字清单

### 1. 现金及活期存款（___ 元）

**包括**：
- 银行卡余额（所有银行加起来）
- 支付宝余额
- 微信余额
- 手头现金

**快速估算方法**：
- 打开支付宝 + 微信，看余额
- 打开银行 APP，看所有银行卡余额
- 加起来，就是现金及活期存款

**如果不确定**：填一个大概的数字，以后可以调整

---

### 2. 定期存款 / 余额宝（___ 元）

**包括**：
- 余额宝、零钱通
- 银行定期存款
- 货币基金
- 其他低风险理财

**快速估算方法**：
- 打开支付宝，看余额宝
- 打开银行 APP，看定期存款
- 加起来，就是定期存款

**如果没有**：填 0

---

### 3. 投资资产（___ 元）

**包括**：
- 股票账户市值
- 基金账户市值
- 债券
- 其他投资（黄金、P2P 等）

**快速估算方法**：
- 打开证券 APP，看总资产
- 加起来，就是投资资产

**如果没有**：填 0

---

### 4. 固定资产（___ 元）

**包括**：
- 房产（市值估算，不是买入价）
- 车辆（市值估算）
- 其他大额资产

**快速估算方法**：
- 如果有房产，估算当前市值（可以参考同小区成交价）
- 如果有车，估算当前市值（可以参考二手车平台）
- 加起来，就是固定资产

**如果没有**：填 0

**注意**：这是可选的，如果你不想把房产算进去，可以填 0

---

### 5. 负债合计（___ 元）

**包括**：
- 信用卡欠款
- 花呗、借呗
- 房贷剩余本金
- 车贷剩余本金
- 其他借款

**快速估算方法**：
- 打开信用卡 APP，看当前欠款
- 打开花呗、借呗，看欠款
- 如果有房贷，看剩余本金（不是月还款额）
- 加起来，就是负债合计

**如果没有**：填 0

## 常见问题

### Q1：数字不精确怎么办？

**A**：填大概的数字，以后可以调整。重要的是开始追踪，而不是追求完美。

### Q2：固定资产要不要算进去？

**A**：看你的目标。如果你想靠房产增值实现财务自由，就算进去。如果你想靠投资收益实现财务自由，就不算。建议先不算，保持简单。

### Q3：负债是填剩余本金还是月还款额？

**A**：填剩余本金。月还款额已经在月度现金流里记录了。

### Q4：没有投资、没有房产怎么办？

**A**：填 0。这不丢人，这是起点。每个人都是从 0 开始的。

### Q5：录入后多久更新一次？

**A**：每月一次，建议在月初或发薪日更新。只需要 5 分钟。
