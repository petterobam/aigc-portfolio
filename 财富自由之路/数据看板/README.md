## 关键指标定义

- **净资产** = 所有资产（现金 + 投资 + 固定资产）- 所有负债（消费债 + 贷款）
- **财务保障目标** = 每月总支出 × 保障月数（建议 6~12 个月）
- **财务安全目标** = 每月所需金额 × 150（基于 8% 年化收益，12 ÷ 8% = 150）
- **财务自由目标** = 梦想月成本 × 150（同上，本金永不动用）
- **被动收入占比** = 被动收入 ÷ 总收入 × 100%（目标：>100% 即达财务安全）

## 使用规范

1. 每月至少更新一次「财务追踪」数据
2. 更新净资产时同步更新上方进度表
3. 产品上线后每周更新「产品数据」
4. 数据只增不删，历史记录保留在各子文件中
