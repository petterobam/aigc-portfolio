## 🔄 数据恢复方法

### 从备份恢复

1. **关闭产品**

2. **删除当前数据目录**（可选，建议先备份）
   ```bash
   # macOS
   rm -rf ~/Library/Application\ Support/@wealth-freedom/desktop/wealth-freedom/
   ```

3. **复制备份目录**
   - 将备份的 `wealth-freedom` 文件夹
   - 粘贴到原位置

4. **重新启动产品**
   - 数据应该已恢复

## ☁️ 云同步方案（高级）

### 方案1：使用云盘同步（不推荐）

**警告**：不推荐直接将数据目录放在云盘同步文件夹中，可能导致数据冲突。

**原因**：
- 数据库文件在产品运行时会频繁写入
- 云盘同步可能导致数据损坏
- 多设备同步可能产生冲突

### 方案2：定期备份到云盘（推荐）

**步骤**：
1. 每周手动备份一次数据目录
2. 将备份文件上传到云盘（iCloud、Google Drive、Dropbox等）
3. 保留多个版本的备份（如：最近4周）

**备份命名建议**：
```
wealth-freedom-backup-2026-03-24.zip
wealth-freedom-backup-2026-03-17.zip
wealth-freedom-backup-2026-03-10.zip
wealth-freedom-backup-2026-03-03.zip
```

## 🔒 数据安全建议

### 1. 定期备份

- **每周备份**：推荐
- **每月备份**：最低要求
- **重大变更后备份**：如录入大量数据后

### 2. 多重备份

- **本地备份**：外部硬盘、U盘
- **云端备份**：云盘、网盘
- **遵循3-2-1原则**：3份备份、2种介质、1份异地

### 3. 加密保护

如果你的财务数据非常敏感，可以：
- 使用加密的磁盘映像（macOS：磁盘工具 → 新建映像）
- 使用加密的压缩文件（zip -e）
- 使用专业的加密工具（VeraCrypt）

### 4. 测试恢复

- 每季度测试一次数据恢复
- 确保备份文件完整可用
- 记录恢复步骤

## 🚨 常见问题

### Q: 产品会自动备份吗？
A: 目前不会。自动备份是计划中的高级功能。

### Q: 数据会丢失吗？
A: 如果不备份，以下情况可能导致数据丢失：
- 电脑硬盘故障
- 系统崩溃
- 误删除数据目录
- 软件冲突

### Q: 备份文件很大吗？
A: 不大。通常只有几MB到几十MB（取决于你的交易记录数量）。

### Q: 可以在多台电脑上使用吗？
A: 可以，但需要手动备份和恢复数据。云同步功能正在计划中。

### Q: 数据会上传到服务器吗？
A: **不会**。所有数据都存储在本地，不会上传到任何服务器。
