# 季度计划

> 每季度财务自由行动计划。

## 计划模板

```markdown
# YYYY年QX季度计划

## 季度主题
一句话描述本季度核心目标

## 财务目标
- 净资产目标：¥XXX
- 储蓄率目标：XX%
- 被动收入目标：¥XXX

## 产品目标
- 产品里程碑

## 学习目标
- 学习重点

## 月度分解
- 第1月：XXX
- 第2月：XXX
- 第3月：XXX

## 季度复盘
- 目标达成情况：
- 关键成果：
- 经验教训：
- 下季度调整：
```

## 文件命名

- `YYYY-QX.md`
- 例：`2026-Q1.md`
