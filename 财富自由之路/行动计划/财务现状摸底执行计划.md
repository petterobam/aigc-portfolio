# 财务现状摸底执行计划

> **用途**: 分步骤指南，帮助用户完成财务现状摸底
> **目标**: 确保用户能够轻松、准确、完整地填写财务现状摸底表
> **创建时间**: 2026-03-21
> **项目**: 财富自由之路

---

## 📋 执行计划总览

**目标**：完成财务现状摸底（净资产、月收入、月支出、负债清单）

**核心步骤**:
1. **准备阶段**（10分钟）：准备账单、银行流水、理财收益证明等
2. **填写阶段**（30分钟）：按照填写指南，逐一填写资产清单、负债清单、收入清单、支出清单
3. **计算阶段**（10分钟）：计算净资产、月收入、月支出、总负债、储蓄率
4. **评估阶段**（10分钟）：评估财务健康度（财务保障度、财务安全度、财富自由度）
5. **复盘阶段**（10分钟）：分析财务现状，识别财务问题，制定优化方案

**总耗时**：约 70 分钟

---

## 📝 第一步：准备阶段（10分钟）

**目标**：收集所有必要的财务数据和证明材料

**行动清单**:
- [ ] 查询银行账户余额（储蓄卡、信用卡、理财账户）
- [ ] 查询支付平台余额（支付宝、微信、云闪付）
- [ ] 查询股票/基金市值（实时市值）
- [ ] 查询房贷剩余本金
- [ ] 查询信用卡账单
- [ ] 查询工资条/银行流水
- [ ] 准备一支笔和一张纸（或使用 Excel/Google Sheets）

**注意事项**:
- 使用实时数据（最新账单、最新市值）
- 如果是家庭，需要统计所有成员的资产和负债
- 建议使用电脑填写（比手机更方便）

---

## 📝 第二步：填写阶段（30分钟）

**目标**：按照填写指南，逐一填写财务现状摸底表

**行动清单**:
- [ ] 打开 `财务现状摸底表.md`
- [ ] 打开 `财务现状摸底填写指南.md`
- [ ] 填写资产清单（现金、银行理财、股票/基金、房产、其他资产）
- [ ] 填写负债清单（房贷、消费贷、信用卡、其他债务）
- [ ] 填写收入清单（工资收入、稿费收入、副业收入、其他收入）
- [ ] 填写支出清单（房租/房贷、水电燃气、餐饮、交通、通讯、娱乐、购物、人情、保险、其他）

**注意事项**:
- 参考填写指南中的详细示例和注意事项
- 确保所有数据真实、准确、完整
- 如果有疑问，查看填写指南中的相关章节

---

## 📝 第三步：计算阶段（10分钟）

**目标**：计算核心指标（净资产、月收入、月支出、总负债、储蓄率）

**行动清单**:
- [ ] 计算总资产（现金 + 银行理财 + 股票/基金 + 房产 + 其他资产）
- [ ] 计算总负债（房贷 + 消费贷 + 信用卡 + 其他债务）
- [ ] 计算净资产（总资产 - 总负债）
- [ ] 计算月收入（工资收入 + 稿费收入 + 副业收入 + 其他收入）
- [ ] 计算月支出（固定支出 + 可变支出）
- [ ] 计算储蓄率（(月收入 - 月支出) / 月收入 × 100%）

**计算公式**:
- 净资产 = 总资产 - 总负债
- 储蓄率 = (月收入 - 月支出) / 月收入 × 100%

**注意事项**:
- 使用计算器（电脑、手机、Excel）
- 如果计算出错，检查填写的数据是否正确
- 储蓄率为正：收入 > 支出，有储蓄
- 储蓄率为负：收入 < 支出，入不敷出

---

## 📝 第四步：评估阶段（10分钟）

**目标**：评估财务健康度（财务保障度、财务安全度、财富自由度）

**行动清单**:
- [ ] 评估财务保障度（拥有 6~12 个月的生活储备金）
- [ ] 评估财务安全度（靠投资利息覆盖日常支出）
- [ ] 评估财富自由度（靠利息实现梦想生活，本金永不动用）

**评估标准**:
- **财务保障度** (Financial Security):
  - 🟢 财务全（6~12个月生活储备金）
  - 🟡 需要改善（<6个月）
  - 🔴 危险（<3个月）
  - **计算方法**: 净资产 ÷ 月支出 = 储备月数

- **财务安全度** (Financial Safety):
  - 🟢 财务全（利息 > 月支出）
  - 🟡 需要改善（利息 < 月支出）
  - 🔴 危险（利息 = 0）
  - **计算方法**: 投资利息 ÷ 月支出 × 100%

- **财富自由度** (Wealth Freedom):
  - 🟢 已实现（利息 > 梦想生活支出）
  - 🟡 接近实现（利息 > 70% 梦想生活支出）
  - 🟡 还需要努力（利息 > 50% 梦想生活支出）
  - 🔴 还很遥远（利息 < 10% 梦想生活支出）
  - **计算方法**: 投资利息 ÷ 梦想生活支出 × 100%

**注意事项**:
- 如果财务保障度为危险（<3个月），需要立即增加储蓄
- 如果财务安全度为危险（利息 = 0），需要开始投资理财
- 如果财富自由度为很遥远（利息 < 10% 梦想生活支出），需要增加本金和投资收益

---

## 📝 第五步：复盘阶段（10分钟）

**目标**：分析财务现状，识别财务问题，制定优化方案

**行动清单**:
- [ ] 识别财务问题（入不敷出、负债过高、储蓄率过低）
- [ ] 分析财务问题原因（收入低、支出高、负债结构不合理、投资收益低）
- [ ] 制定优化方案（增加收入、减少支出、优化资产配置、优化负债结构）
- [ ] 设定财务目标（短期：6个月内）、中期：12个月内）、长期：36个月内）
- [ ] 制定行动计划（每月储蓄目标、投资计划、还款计划）

**优化方案建议**:
- **增加收入**:
  - 申请加薪或升职
  - 副业收入（咨询、培训、讲课、兼职）
  - 投资收益（股票、基金、理财产品）
- **减少支出**:
  - 控制可变支出（餐饮、交通、娱乐、购物）
  - 优化固定支出（房租、水电、保险）
  - 消除不必要支出（会员费、订阅费、人情往来）
- **优化资产配置**:
  - 增加高收益资产（股票、基金、理财产品）
  - 减少低收益资产（活期存款、现金）
  - 分散投资风险（不要把鸡蛋放在一个篮子里）
- **优化负债结构**:
  - 优先还清高利率负债（消费贷、信用卡）
  - 使用 50/50 还债原则（50% 用于还债，50% 用于储蓄）
  - 避免新增负债（不要使用信用卡透支、不要借新债还旧债）

**注意事项**:
- 设定可实现的目标（不要设定过高或过低的目标）
- 定期更新财务数据（每月或每季度）
- 持续监控财务状况（使用数据看板）

## 📝 附录

### A. 常见问题

**Q1: 我不知道某个资产或负债的具体金额，怎么办？**
A1: 查询账单、银行流水、理财收益证明。如果还是不知道，可以估算一个大概的金额（但最好使用真实数据）。

**Q2: 我的收入不稳定（如稿费收入），怎么办？**
A2: 记录近 6 个月的平均收入。如果收入为 0，记录 0。

**Q3: 我的支出不稳定（如娱乐、购物），怎么办？**
A3: 记录近 6 个月的平均支出。如果支出为 0，记录 0。

**Q4: 我没有投资（利息 = 0），怎么办？**
A4: 记录 0。财务安全度为危险（利息 = 0），需要开始投资理财。

### B. 技巧和建议

1. **使用 Excel/Google Sheets**: 使用表格软件自动计算，减少手动计算错误。
2. **使用记账 APP**: 如果已经使用记账 APP（如随手记、挖财、MoneyWiz），可以直接导出数据。
3. **定期更新**: 建议每月或每季度更新一次财务数据。
4. **保密性**: 财务数据属于个人隐私，注意保密。
