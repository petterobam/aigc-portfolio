# 月度计划

> 每月财务自由行动计划。

## 计划模板

```markdown
# YYYY年MM月计划

## 本月目标
1. 目标一
2. 目标二
3. 目标三

## 财务目标
- 储蓄目标：¥XXX
- 还债目标：¥XXX
- 投资目标：¥XXX

## 产品目标
- 产品相关任务

## 学习目标
- 学习相关任务

## 行动清单
- [ ] 任务一
- [ ] 任务二
- [ ] 任务三

## 月末复盘
- 完成情况：
- 经验教训：
- 下月调整：
```

## 文件命名

- `YYYY-MM.md`
- 例：`2026-03.md`

---

**更新频率**：每月初制定，月末复盘
