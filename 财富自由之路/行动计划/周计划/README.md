# 周计划

> 每周财务自由行动计划。

## 计划模板

```markdown
# 第X周计划（YYYY-MM-DD ~ YYYY-MM-DD）

## 本周重点
1. 最重要的任务
2. 次重要的任务

## 财务行动
- [ ] 记录每日收支
- [ ] 检查储蓄进度
- [ ] 其他财务任务

## 产品行动
- [ ] 产品相关任务

## 学习行动
- [ ] 学习相关任务

## 周末复盘
- 完成率：
- 下周调整：
```

## 文件命名

- `YYYY-WXX.md`（XX 为周数）
- 例：`2026-W12.md`

---

**更新频率**：每周一制定，周日复盘
