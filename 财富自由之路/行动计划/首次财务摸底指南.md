# 首次财务摸底指南

> **目标**: 用 10 分钟完成财务现状的第一次记录
> **原则**: 宁可用估算值开始，也不要因为追求完美而迟迟不动手

---

## 为什么必须填写真实数据？

1. **数字不会说谎** - 只有正视现实才能改变现实
2. **产品需要数据** - 没有真实数据，产品无法发挥作用
3. **进度需要基准线** - 没有起点，就看不到进步
4. **隐私绝对安全** - 数据只保存在本地，不上传云端

## 最简启动版（仅 4 个数字）

如果觉得完整版太复杂，先填这 4 个核心数字：

### 1️⃣ 净资产（资产 - 负债）

```
资产合计：_________ 元
负债合计：_________ 元
净资产：_________ 元（自动计算）
```

**如何估算**：
- 资产：银行存款 + 支付宝/微信余额 + 股票/基金市值 + 房产/车辆市值
- 负债：信用卡欠款 + 花呗/借呗 + 房贷/车贷剩余本金
- **可以用粗略估算**，比如"大概 30 万"这种，后期随时修正

### 2️⃣ 月收入

```
月收入：_________ 元
```

**包含**：
- 税后工资（到手金额，不是税前）
- 兼职/副业收入（如有）
- 被动收入（投资分红、利息等，如有）

### 3️⃣ 月支出

```
月支出：_________ 元
```

**快速估算方法**：
- 方法 A：查看支付宝/微信的「账单」-「月度账单」
- 方法 B：月收入 - 月储蓄 = 月支出
- **大致范围即可**：3000-5000 / 5000-8000 / 8000-12000 这样的区间

### 4️⃣ 月结余

```
月结余：_________ 元（自动计算）
```

## 三阶段目标计算

填写完上面数据后，立即计算财务自由三阶段目标：

### 第一阶段：财务保障

```
月总支出：_________ 元
目标月数：6 个月（建议值）
财务保障目标：月支出 × 6 = _________ 元
```

**意义**：拥有 6 个月的生活储备金，不被意外击倒

### 第二阶段：财务安全

```
月基本生活成本：_________ 元（可低于月总支出）
财务安全目标：月基本生活 × 150 = _________ 元
```

**意义**：靠 8% 年化收益覆盖日常支出，不再被迫工作

**公式解释**：
- 年支出 = 月基本生活 × 12
- 所需本金 = 年支出 ÷ 8% = 月基本生活 × 12 ÷ 0.08 = 月基本生活 × 150

### 第三阶段：财务自由

```
梦想生活月成本：_________ 元（加上旅行、娱乐、梦想支出）
财务自由目标：梦想月成本 × 150 = _________ 元
```

**意义**：靠利息实现梦想生活，本金永不动用

## 填写技巧

### 💡 不要追求 100% 精确

- 用估算值即可，后期随时修正
- 比如房产市值，可以用链家/贝壳的在线估价
- 比如月支出，可以看最近 3 个月支付宝/微信账单的平均值

### 💡 哪怕是负数也要填

- 净资产为负（负债 > 资产）是很多年轻人的常态
- 正视现实是改变的第一步
- 产品会帮你规划从负数到正数的路径

### 💡 可以使用区间

- 如果不确定具体数字，可以填区间，比如"10-15万"
- 先让系统跑起来，后续慢慢精确化

### 💡 定期更新比完美更重要

- 每月更新一次即可
- 趋势比单点数据更有价值
- 看到"净资产增长曲线"才是目标

- 月收入 2 万后，月结余提升到 1 万，财务安全达成时间缩短到 11 年

## 常见问题

### Q: 资产怎么估值？

**A**:
- 现金类：查看银行卡、支付宝、微信余额
- 投资类：查看股票/基金账户市值（不是成本）
- 房产：链家/贝壳在线估价，或咨询中介
- 车辆：瓜子/优信在线估价

### Q: 负债只算本金吗？

**A**:
- 是的，只算剩余本金
- 房贷/车贷：查看银行 APP 的「剩余本金」
- 信用卡：查看「账单金额」（未还部分）
- 不要把利息算进去，利息会随时间变化

### Q: 月支出怎么算最简单？

**A**:
- **方法 1**（推荐）：支付宝/微信 → 账单 → 月度账单 → 查看总支出
- **方法 2**：月收入 - 月储蓄 = 月支出
- **方法 3**：记录 1 周支出 × 4.3 = 月支出

### Q: 填错了怎么办？

**A**:
- 随时可以修改！
- 产品和文档都支持随时更新
- 第一次填写的数字只是起点，不是终点
