## 📊 数据分析工具

### 成本趋势分析
```bash
# 计算日均支出趋势
cat dining-cost.log | awk -F: '{
    dates[$1] = $2
} END {
    for (date in dates) {
        print date ": " dates[date]
    }
}' | sort
```

### 节省金额计算
```bash
# 月节省金额计算
original_cost=480  # 原计划日均支出
actual_cost=$(cat dining-cost.log | awk -F: '{sum += $2} END {print sum/NR}')
daily_save=$((original_cost - actual_cost))
monthly_save=$((daily_save * 30))
echo "月节省金额: ¥$monthly_save"
```

### 预警机制
```bash
# 自动检测超标日
cat dining-cost.log | awk -F: '$2 > 300 {
    print "红色预警: " $1 " 支出 ¥" $2
}' | tail -5
```
