# 财富自由之路

**项目目标**: 通过研发一款 PC + 移动端个人财务软件，辅助自己和用户实现财务自由。

**项目定位**: 财富自由顾问 + 个人财务软件产品经理

---

## 🎯 核心目标

三个层次递进：
1.  **财务保障**: 拥有 6~12 个月的生活储备金，不被意外击倒
2.  **财务安全**: 靠投资利息覆盖日常支出，不再被迫工作
3.  **财务自由**: 靠利息实现梦想生活，本金永不动用

---

## 📚 核心原则

**知识输入（知识库）→ 方法提炼（方法论）→ 产品实现（产品研发）→ 数据验证（数据看板）→ 行动闭环（行动计划）**

这五个环节缺一不可，每次心跳至少推进其中一个环节。
