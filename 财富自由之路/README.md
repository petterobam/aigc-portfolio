## 🤖 AI 协作指南

### 读取顺序建议
1. **首次对话** → 先读 `HEARTBEAT.md`（了解角色和信念），再读 `tasks/current.md`（了解当前任务），再读 `MEMORY.md`（了解历史和数据）
2. **执行任务时** → 根据任务类型定位到对应目录，优先读该目录的 `README.md`
3. **更新数据时** → 更新 `数据看板/财务追踪/current-state.md`，重要变更同步到 `MEMORY.md`
4. **完成任务后** → 更新 `tasks/current.md` 中的任务状态

### 写入规范
- **心跳记录** → 写入 `工作日志/latest.md`，旧记录移入 `工作日志/归档记录/`
- **新方法论** → 写入 `方法论/` 对应子目录，更新 `MEMORY.md` 中的方法论列表
- **新知识** → 写入 `知识库/` 对应子目录
- **行动计划** → 写入 `行动计划/` 对应时间维度子目录
- **归档内容** → 移入 `.archive/` 对应子目录，不要删除历史数据

### 命名约定
- 心跳记录: `心跳#NNN-简述.md` 或 `heartbeat-YYYYMMDD-HHMM.md`
- 财务快照: `YYYY-MM-DD-财务快照.md`
- 行动清单: `YYYYMMDD-行动清单.md`
- 方法论: 中文描述性名称，如 `财务三阶段理论.md`
