## 🔍 思想家研究

### 河上肇（1879-1946）

**生平**：
- 1879年生于熊本县
- 1904年毕业于东京帝国大学法学院
- 1907年创办《社会主义评论》
- 1917年出版《贫乏物語》，引发轰动
- 1920年出版《资本论入门》
- 1924年加入日本共产党
- 1933年被捕，1937年出狱
- 1946年逝世

**思想**：
- 马克思主义经济学：将《资本论》系统引入日本
- 社会主义思想：分析日本资本主义矛盾
- 社会批判：批判贫富差距和社会不公

**著作**：
- 《资本论入门》（1920）：日本最重要的马克思主义经济学入门书
- 《贫乏物語》（1917）：社会批判经典
- 《社会主义評論》（1919）：社会主义理论期刊

**评价**：
- 日本马克思主义经济学的奠基人
- 社会主义思想在日本的早期传播者
- 日本左翼运动的理论指导者

### 丸山真男（1914-1996）

**生平**：
- 1914年生于大阪
- 1937年毕业于东京帝国大学法学院
- 1944年被征召入伍
- 1945年回国，开始学术生涯
- 1950年任东京大学法学部教授
- 1996年逝世

**思想**：
- 政治思想史：日本政治思想的脉络分析
- 现代政治理论：民主与权力的关系
- 日本的思想：日本人的精神结构分析
- 民主理论：大众社会与民主的危机

**著作**：
- 《日本政治思想史研究》（1952）：政治思想史经典
- 《现代政治的思想与行动》（1956）：政治理论集
- 《日本的思想》（1961）：文化批判
- 《日本のナショナリズム》（1958）：民族主义批判

**评价**：
- 日本政治思想研究的开创者
- 民主理论的重要贡献者
- 战后日本思想界的领军人物

### 柄谷行人（1941-）

**生平**：
- 1941年生于兵库县
- 1965年毕业于东京大学经济学部
- 1968年开始文学批评活动
- 1980年代转向哲学研究
- 2000年代提出"世界共和国"理论
- 现为关西学院大学教授

**思想**：
- 文学批评：日本近代文学的起源分析
- 哲学批判：解构西方哲学传统
- 马克思主义：重新阐释《资本论》
- 世界共和国：超民族国家的政治构想

**著作**：
- 《日本近代文学の起源》（1980）：文学批评经典
- 《マルクスその可能性の中心》（1985）：马克思主义研究
- 《世界共和国へ》（2006）：政治构想
- 《哲学の起源》（2012）：哲学批判

**评价**：
- 当代日本最重要的思想家之一
- 跨学科研究的先驱
- 国际影响力巨大的理论家
