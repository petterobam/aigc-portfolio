## 🔍 研究指引

### 政治体制演变

| 时期 | 时间 | 政治体制 | 特征 |
|------|------|---------|------|
| **明治维新时期** | 1868-1889 | 绝对君主制 | 天皇专制、藩阀政治 |
| **明治宪法时期** | 1889-1918 | 立宪君主制 | 帝国议会、政党内阁萌芽 |
| **大正民主时期** | 1918-1932 | 议会政治 | 政党政治、普选运动 |
| **法西斯时期** | 1932-1945 | 军部独裁 | 军部专政、政党消亡 |
| **战后民主化** | 1945-1955 | 多党制 | 占领改革、政党重组 |
| **55年体制** | 1955-1993 | 自民党一党优势 | 保革对立、长期政权 |
| **联合政权时期** | 1993-2009 | 联合政权 | 政党重组、政权更迭 |
| **民主党政权** | 2009-2012 | 民主党政权 | 政权交替、政策转向 |
| **安倍政权** | 2012-2020 | 长期政权 | 右翼化、修宪运动 |

### 关键政治事件

- **1868年**：明治维新，江户幕府倒台，建立天皇制国家
- **1889年**：大日本帝国宪法颁布，建立立宪君主制
- **1890年**：第一届帝国议会召开
- **1918年**：原敬组阁，首次政党内阁
- **1925年**：普选法颁布，成年男性获得选举权
- **1931年**：满洲事变，军部政治抬头
- **1936年**：二·二六事件，军部独裁确立
- **1945年**：日本投降，GHQ占领开始
- **1946年**：日本国宪法颁布，象征天皇制
- **1955年**：保守合同，自由民主党成立
- **1960年**：60年安保斗争，岸信介内阁总辞职
- **1970年**：70年安保斗争
- **1993年**：38年自民党政权终结，细川护熙组阁
- **2009年**：民主党大胜，鸠山由纪夫组阁
- **2012年**：自民党大胜，安倍晋三组阁
- **2014年**：集体自卫权法案通过
- **2020年**：安倍辞职，菅义伟组阁

### 政党政治发展

#### 保守政党

- **自由党（1890s-1940s）**：战前主要保守政党
- **自由民主党（1955-）**：战后保守政党，长期执政

#### 革新政党

- **日本社会党（1945-1996）**：主要左翼政党，1996年解散
- **日本共产党（1922-）**：现存最老的社会主义政党
- **民主党（1996-2016）**：中左政党，2009-2012年执政

#### 其他政党

- **公明党（1964-）**：宗教政党（创价学会），长期与自民党联合执政
- **日本维新会（2012-）**：右翼保守政党
- **立宪民主党（2017-）**：中左政党

## 📊 政治制度分析

### 选举制度

- **众议院**：比例代表+小选区并行制，4年任期
- **参议院**：比例代表+选举区制，6年任期（每3年改选一半）
- **地方议会**：中选举区制，4年任期

### 权力结构

- **天皇**：象征性国家元首，无实权
- **内阁**：行政权力中心，首相由国会议员中选举产生
- **国会**：两院制，众议院优先
- **司法**：独立司法权，最高法院拥有违宪审查权

### 官僚体系

- **官僚**：政策制定的专业化，官僚主导的传统
- **政治家**：政治主导化改革，削弱官僚权力
- **省厅**：行政机构，政策执行
