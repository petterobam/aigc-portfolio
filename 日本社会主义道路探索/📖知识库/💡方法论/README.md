## 🔍 核心分析方法

### 历史唯物主义分析

**基本原理**：

1. **生产力决定生产关系**：生产方式决定社会结构
2. **经济基础决定上层建筑**：经济制度决定政治、文化、意识形态
3. **阶级斗争是历史动力**：不同阶级之间的矛盾推动历史发展
4. **社会存在决定社会意识**：社会存在决定人们的思想意识

**应用步骤**：

1. **分析生产方式**：研究社会的生产力水平、生产关系结构
2. **分析阶级结构**：识别社会阶级、阶级关系、阶级矛盾
3. **分析意识形态**：研究主流意识形态、文化传统、价值观念
4. **分析矛盾运动**：识别主要矛盾、次要矛盾、矛盾的转化

### 阶级分析方法

**基本框架**：

| 阶级 | 生产关系 | 收入来源 | 政治地位 |
|------|---------|---------|---------|
| **资产阶级** | 拥有生产资料 | 资本收入 | 统治阶级 |
| **工人阶级** | 出卖劳动力 | 工资收入 | 被统治阶级 |
| **中间阶级** | 小生产者/专业技术人员 | 工资+小资本收入 | 中间阶层 |
| **农民阶级** | 拥有少量土地 | 农业收入 | 边缘阶层 |
| **边缘阶级** | 非正规雇佣 | 不稳定收入 | 底层阶层 |

**分析步骤**：

1. **划分阶级**：根据生产关系划分阶级
2. **分析阶级关系**：研究阶级之间的矛盾和合作
3. **分析阶级意识**：研究阶级的自我意识和团结程度
4. **分析阶级斗争**：研究阶级之间的冲突和妥协

### 矛盾分析方法

**基本原理**：

1. **矛盾普遍性**：矛盾存在于一切事物的发展过程中
2. **矛盾特殊性**：不同事物的矛盾有其特殊性
3. **主要矛盾和次要矛盾**：在复杂事物的发展过程中，有许多矛盾存在，其中必有一种是主要矛盾
4. **矛盾的主要方面和次要方面**：在每一对矛盾中，矛盾的双方力量是不平衡的

**应用步骤**：

1. **识别矛盾**：找出社会中的主要矛盾
2. **分析矛盾性质**：判断矛盾的对抗性或非对抗性
3. **分析矛盾主要方面**：判断矛盾的主要方面
4. **分析矛盾转化**：预测矛盾的转化方向

### 比较研究方法

**基本类型**：

| 类型 | 方法 | 目的 |
|------|------|------|
| **横向比较** | 同一时期不同国家比较 | 识别差异和共同点 |
| **纵向比较** | 同一国家不同时期比较 | 识别变化和发展 |
| **类型比较** | 不同类型国家比较 | 识别类型特征 |
| **混合比较** | 横向+纵向比较 | 综合分析 |

**应用步骤**：

1. **确定比较对象**：选择比较的国家或时期
2. **确定比较维度**：选择比较的维度（经济、政治、文化等）
3. **收集比较数据**：收集相关数据和资料
4. **进行比较分析**：进行对比分析，识别异同
5. **解释差异原因**：解释差异的原因
