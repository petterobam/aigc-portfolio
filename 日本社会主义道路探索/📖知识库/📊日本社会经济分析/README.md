## 📊 核心指标

### 劳动力市场

- **就业率/失业率**：总体就业率、青年失业率、长期失业率
- **就业形态**：正规雇佣/非正规雇佣比例、派遣工比例、兼职比例
- **工资水平**：平均工资、工资增长率、工资分布
- **工作时间**：平均工作时间、过度劳动率
- **劳动生产率**：每小时劳动生产率、劳动生产率增长率

### 收入分配

- **基尼系数**：收入基尼系数、财富基尼系数
- **收入分布**：五分位分配、十等分分配
- **贫困率**：相对贫困率、儿童贫困率、老年贫困率
- **可支配收入**：人均可支配收入、家庭可支配收入

### 社会保障

- **社会保障负担**：社会保障费占GDP比例
- **养老金**：公共养老金覆盖率、替代率
- **医疗保险**：医疗保险覆盖率、医疗支出占GDP比例
- **社会救助**：生活保护受给率、生活保护金额

### 企业结构

- **企业规模分布**：大企业/中小企业比例
- **公司治理**：股东结构、董事会构成
- **企业盈亏**：企业盈亏率、破产企业数
- **投资水平**：固定资本形成、设备投资

## 🔍 分析框架

### 阶级分析框架

| 阶级 | 定义 | 特征 | 占比（估算） |
|------|------|------|-------------|
| **资产阶级** | 拥有生产资料、支配经济权力 | 企业主、资本家、高级管理者 | <1% |
| **工人阶级** | 出卖劳动力、无生产资料 | 正规雇佣工人、产业工人 | ~20% |
| **中间阶级** | 专业技术、小生产者 | 专业技术人员、小企业主 | ~60% |
| **农民阶级** | 农业生产者 | 农民、农业劳动者 | <1% |
| **边缘阶级** | 非正规雇佣、贫困者 | 非正规雇佣、日工、Working Poor | ~18% |

### 结构矛盾分析

1. **正规/非正规雇佣分化**：
   - 正规雇佣：长期雇佣、福利完善、工资稳定
   - 非正规雇佣：短期雇佣、福利缺失、工资不稳定
   - 矛盾：同一阶级内部的两极分化

2. **企业内部等级制**：
   - 终身雇佣制度瓦解
   - 工资差距扩大
   - 非正规雇佣歧视

3. **代际流动性下降**：
   - 父母阶层与子女阶层的相关性上升
   - 教育机会不平等
   - 阶层固化

4. **格差固定化**：
   - 收入差距持续扩大
   - 财富集中
   - 社会流动停滞
