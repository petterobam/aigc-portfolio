## 📖 使用说明

### 材料分类

- **一手史料**：原始文献、统计数据、史料档案
- **二手研究**：学术研究、历史研究、理论分析
- **三手资料**：综述、评论、新闻报道

### 材料标注

所有材料必须标注：
- **作者**：作者姓名
- **出处**：出版信息、期刊名称、网址
- **日期**：出版日期、发布日期
- **类型**：一手/二手/三手材料

### 文件命名

- 使用清晰的文件名
- 包含核心关键词
- 使用中英文混合命名
- 避免过长的文件名

### 目录结构

- 每个子目录有自己的README.md
- README.md包含：
  - 目录内容说明
  - 核心文献清单
  - 研究指引
  - 待补充材料清单
