## 🔍 研究指引

### 研究分期

日本社会主义运动史可分为以下时期：

| 时期 | 时间 | 核心事件 | 特征 |
|------|------|---------|------|
| **萌芽期** | 1890s-1910s | 社会主义协会成立、幸德秋水活动 | 思想引进、初步组织 |
| **分裂与压迫期** | 1910s-1920s | 大逆事件、一战影响 | 激进化、地下化 |
| **战前合法期** | 1920s-1930s | 日本共产党成立、社会党成立 | 短暂合法化、左翼分裂 |
| **战时镇压期** | 1930s-1945 | 15年战争、思想警察 | 全面镇压、转入地下 |
| **战后重建期** | 1945-1950s | 解放、日共合法化、旧社会党成立 | 复活、分裂、冷战对立 |
| **55年体制期** | 1955-1980s | 55年体制、安保斗争、学生运动 | 党派分化、社会运动高潮 |
| **泡沫与失落期** | 1980s-2000s | 泡沫破裂、左翼衰退 | 政治保守化、左翼边缘化 |
| **当代期** | 2000s- | 安倍政权、右翼化 | 左翼重构、寻找新路径 |

### 关键事件

- **1898年**：社会主义协会成立
- **1901年**：日本社会民主党成立（立即解散）
- **1910年**：大逆事件（幸德秋水等12人处决）
- **1922年**：日本共产党成立
- **1932年**：大镇压，日共转入地下
- **1936年**：二·二六事件后全面镇压左翼
- **1945年**：解放，日共合法化
- **1946年**：日本社会党成立
- **1955年**：左右社会党统一
- **1960年**：60年安保斗争
- **1970年**：70年安保斗争
- **1972年**：联合赤军浅间山庄事件
- **1980s**：泡沫经济，左翼衰退
- **1990s**：泡沫破裂，社会党瓦解
- **2000s**：民主党成立，左翼重构
- **2012年**：安倍第二次政权，右翼化加速

### 关键人物

- **早期先驱**：幸德秋水、片山潜、堺利彦、大杉荣
- **共产党创始人**：片山潜、德田球一、野坂参三
- **社会党领导人**：铃木茂三郎、浅沼稻次郎、成田知巳
- **工会领袖**：高野实、太田薰
- **学生运动领袖**：山口二郎、中野孝次
- **当代左翼**：志位和夫（日共）、福岛瑞穗（社民党）
