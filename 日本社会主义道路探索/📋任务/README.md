# 📋 任务管理

本目录管理所有研究任务的创建、跟踪和归档。

## 任务状态说明

| 状态 | 含义 |
|------|------|
| 🔴 待开始 | 任务已创建但尚未启动 |
| 🟡 进行中 | 任务正在推进 |
| 🟢 已完成 | 任务已完成，待归档 |
| ⚪ 已归档 | 任务已归档至 `归档/` 目录 |

## 任务命名规范

`TASK-{编号}-{简要描述}.md`

## 任务文件结构

每个任务文件应包含：
1. 任务目标
2. 具体步骤（带检查框）
3. 参考资料
4. 执行记录
5. 产出链接
