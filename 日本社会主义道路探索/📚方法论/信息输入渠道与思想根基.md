### 2. 宗教思想研究 —— 跨文明比较的思想参照

**路径**：`~/.openclaw/workspace/宗教思想研究/`

宗教思想研究项目提供了另一个维度的参照系。研究日本社会主义道路时，以下方面值得参考：

- **日本佛教与社会变革**：日本佛教（净土宗、禅宗、日莲宗）对日本社会意识的深层影响，特别是"此岸/彼岸"思维对革命热情的消解或促进

- **已有的读书笔记**：`✍️读书笔记/` 目录中已积累大量深度研读笔记，可交叉引用

**使用方式**：
- 当分析日本文化传统对社会转型的影响时，去宗教思想研究目录翻阅相关材料
- 当需要理解日本人的精神世界时，参考佛教、神道教相关研究
- 借鉴其方法论体系（文本细读、比较分析、现象学方法），应用于社会科学研究

## 🔍 实时信息获取工具

### 3. 多搜索引擎 —— 连接真实世界的窗口

**技能路径**：`~/.openclaw/workspace/skills/multi-search-engine/`
**能力**：17 个搜索引擎（8 个国内 + 9 个国际），无需 API Key

研究不能只靠书本和记忆，必须用搜索引擎获取最新的实际信息。以下是核心使用场景：

| 场景 | 推荐引擎 | 示例搜索 |
|------|---------|---------|
| **日本社会新闻** | Google / Bing INT | `Japan inequality 2024`、`日本 格差社会 最新` |
| **日本统计数据** | Google | `site:stat.go.jp 非正規雇用 統計`（日本总务省） |
| **学术论文检索** | Google | `site:jstage.jst.go.jp 日本社会主義運動`（J-STAGE 学术平台） |
| **日本劳动数据** | Google | `site:mhlw.go.jp 賃金構造基本統計`（厚生劳动省） |
| **马克思主义文献** | Google / DuckDuckGo | `site:marxists.org japan socialist` |
| **中文社会评论** | 百度 / 微信搜索 | `日本社会矛盾 阶级分化` |
| **日本选举数据** | Google | `日本 衆議院選挙 結果 共産党 得票率` |
| **经济不平等数据** | WolframAlpha | `Japan gini coefficient` |
| **日本社会运动新闻** | Google / Brave | `Japan labor movement protest 2024` |

**搜索引擎快速参考**：

| 引擎 | URL 模板 |
|------|---------|
| 百度 | `https://www.baidu.com/s?wd={keyword}` |
| 微信 | `https://wx.sogou.com/weixin?type=2&query={keyword}` |
| 头条 | `https://so.toutiao.com/search?keyword={keyword}` |
| Google | `https://www.google.com/search?q={keyword}` |
| DuckDuckGo | `https://duckduckgo.com/html/?q={keyword}` |
| Brave | `https://search.brave.com/search?q={keyword}` |
| WolframAlpha | `https://www.wolframalpha.com/input?i={keyword}` |

**高级搜索技巧**：
- `site:xxx.com keyword` → 站内搜索
- `filetype:pdf keyword` → 搜索 PDF
- `"exact phrase"` → 精确匹配
- `keyword -exclude` → 排除词
- `tbs=qdr:m` → Google 时间过滤（过去一个月）

**使用原则**：
- **先搜后写**：每次写专题研究之前，先用搜索引擎确认关键事实和最新数据
- **交叉验证**：重要事实至少用两个不同来源验证
- **标注来源**：所有从搜索中获得的信息必须在笔记中标注 URL 和获取日期
- **中日双语搜索**：日本社会相关话题，用日语搜索往往能获得更一手的信息
- **避免信息茧房**：同时使用不同立场的信息源（官方数据 + 社会运动组织 + 学术研究 + 新闻报道）

## ⚠️ 绝不凭空造物

这是最重要的研究纪律。

**你有三个信息来源，必须善用：**

1. **经典文本**（毛选 215 篇 + 宗教思想研究积累）→ 提供思想方法和分析框架
2. **搜索引擎**（17 个引擎）→ 提供实时数据、新闻事实、学术文献
3. **已有研究成果**（本工作目录中的积累）→ 避免重复劳动，在已有成果上递进

**禁止行为**：
- ❌ 不查资料就编写"日本某年发生了某事"
- ❌ 不读原文就声称"毛泽东说过某句话"
- ❌ 不看数据就断言"日本基尼系数是多少"
- ❌ 不做搜索就推演"日本社会主义转型的路径"

**正确做法**：
- ✅ 先用搜索引擎查实事实，再写入研究笔记
- ✅ 先读毛选原文，再借鉴其分析方法
- ✅ 先翻阅宗教思想研究的已有成果，再做文化分析
- ✅ 标注每一条信息的来源（文件路径 / URL / 书名页码）
- ✅ 对不确定的信息标注 `[待验证]`，后续用搜索确认
