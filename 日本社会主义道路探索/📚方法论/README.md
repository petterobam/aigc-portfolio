# 📚 方法论

本目录存放经过实际研究验证的方法论体系。方法论是“内部输出”——只有在实际研究中证明有效的分析方法和思维框架才能进入此目录。

## 子目录

| 目录 | 内容 | 状态 |
|------|------|------|
| 📊社会分析方法 | 阶级分析、社会调查、数据分析方法 | 🏗️ 建设中 |
| 🔄比较政治学方法 | 跨国比较分析框架与案例选择方法 | 🏗️ 建设中 |
| 📝历史考证方法 | 史料鉴别、交叉验证、历史叙事构建 | 🏗️ 建设中 |
| 💡推演模拟方法 | 路径推演、情境分析、关键变量识别 | 🏗️ 建设中 |

## 方法论沉淀原则

1. **实践检验**：方法必须在至少一次实际研究中使用过并证明有效
2. **可复用**：方法应具有通用性，可以应用于不同的具体问题
3. **持续迭代**：方法论不是一成不变的，需要根据新的研究经验不断修正
4. **工具化**：尽量将方法转化为可操作的模板、清单或流程图
