# ✍️ 读书笔记

本目录存放阅读过程中的笔记与思考。每一份笔记都是从"知识输入"到"思想转化"的中间产物。

## 命名规范

`{领域}-{书名/文章名}-{笔记类型}.md`

示例：
- `运动史-幸德秋水-帝国主义-深度研读笔记.md`
- `政治经济-河上肇-贫乏物语-读书摘要.md`
- `比较政治-北欧社会民主模式-研究综述.md`

## 笔记质量标准

1. 必须标注材料来源（作者、书名、页码或链接）
2. 区分"原文摘录"和"个人分析"
3. 对关键概念给出定义或解释
4. 记录阅读过程中产生的疑问和联想
5. 与已有研究成果建立交叉引用
