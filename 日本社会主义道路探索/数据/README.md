# 数据

本目录存放研究中使用的原始数据与统计资料。

## 数据类型

| 类型 | 说明 | 来源 |
|------|------|------|
| 经济数据 | GDP、基尼系数、收入分布 | 日本总务省统计局、世界银行 |
| 劳动数据 | 就业结构、工资水平、非正规雇佣比例 | 日本厚生劳动省 |
| 社会数据 | 人口结构、贫困率、社会保障 | 日本总务省、OECD |
| 政治数据 | 选举结果、政党支持率 | 日本总务省选举部 |

## 数据管理原则

- 所有数据必须标注来源和获取日期
- 原始数据不做修改，分析结果另存
- 定期更新关键指标数据
