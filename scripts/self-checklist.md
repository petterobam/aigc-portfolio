# 写作自查清单

> 创建时间：2026-03-26 14:45
> 用途：自查技巧优化 - 创作时的实时检查

---

## 🎯 目标

创作时的实时自查，及时发现重复段落，提高写作质量。

---

## 📋 三个检查阶段

### 阶段1：段落级别自查（每写完一段）

**目标**：确保段落内部没有问题，段落之间不重复。

**检查清单**：
- [ ] **内容检查**
  - [ ] 这段话的主题明确吗？
  - [ ] 这段话提供了新信息吗？
  - [ ] 这段话推进了情节吗？

- [ ] **重复检查**
  - [ ] 这段话在前面出现过吗？（字数 > 30 的相似段落）
  - [ ] 这段话和前面哪段话最相似？（相似度 > 80%）
  - [ ] 这段话有重复词汇吗？（出现 > 2 次）

- [ ] **表达检查**
  - [ ] 这段话的表达方式多样化吗？
  - [ ] 这段话有可以替换的同义词吗？
  - [ ] 这段话的句式结构有变化吗？

- [ ] **逻辑检查**
  - [ ] 这段话的逻辑清晰吗？
  - [ ] 这段话和上段话有逻辑递进吗？
  - [ ] 这段话可以和其他段落合并吗？

---

### 阶段2：章节级别自查（每写完一章）

**目标**：确保章节内部连贯，章节之间不重复。

**检查清单**：
- [ ] **章节内容检查**
  - [ ] 这一章有重复段落吗？（字数 > 50 的完全重复）
  - [ ] 这一章和前面哪章最相似？（相似度 > 70%）
  - [ ] 这一章有推进情节吗？
  - [ ] 这一章的核心冲突清晰吗？

- [ ] **章节结构检查**
  - [ ] 这一章的段落之间有逻辑递进吗？
  - [ ] 这一章的信息推进清晰吗？
  - [ ] 这一章的情绪推进合理吗？
  - [ ] 这一章可以精简吗？

- [ ] **章节对比检查**
  - [ ] 这一章和上一章有回溯吗？
  - [ ] 这一章和前面哪章有内容重复？
  - [ ] 这一章有重复使用的句式吗？
  - [ ] 这一章有重复使用的词汇吗？

---

### 阶段3：全文级别自查（每写完全文）

**目标**：确保全文连贯，没有重复段落。

**检查清单**：
- [ ] **全文重复检查**
  - [ ] 全文有重复段落吗？（字数 > 30 的完全重复）
  - [ ] 全文有相似段落吗？（相似度 > 80%）
  - [ ] 全文有重复使用的句式吗？
  - [ ] 全文有重复使用的词汇吗？（出现 > 5 次）

- [ ] **全文逻辑检查**
  - [ ] 全文的情节推进清晰吗？
  - [ ] 全文的信息推进清晰吗？
  - [ ] 全文的情绪推进合理吗？
  - [ ] 全文的段落之间有逻辑递进吗？

- [ ] **全文质量检查**
  - [ ] 全文的核心冲突清晰吗？
  - [ ] 全文的主线明确吗？
  - [ ] 全文的节奏合理吗？
  - [ ] 全文的结尾满意吗？

## 📌 使用建议

1. **严格执行**：按照三个阶段执行自查，不要跳过任何阶段
2. **及时修改**：发现问题及时修改，不要拖延
3. **工具辅助**：使用辅助工具提高检查效率
4. **记录反思**：记录自查结果，反思问题原因
5. **持续优化**：根据自查结果不断优化写作技巧
