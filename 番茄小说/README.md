## 番茄小说 vs 番茄短篇故事集

### 目录区别

- **番茄小说**: 长篇小说项目
- **番茄短篇故事集**: 短篇小说项目

### 特点对比

| 特点 | 番茄小说 | 番茄短篇故事集 |
|------|---------|---------------|
| 故事类型 | 长篇小说 | 短篇小说 |
| 字数范围 | 数万字以上 | 1-3万字 |
| 章节数 | 数十个以上 | 3-6章 |
| 发布周期 | 长期连载 | 一次性发布 |
| 项目管理 | 独立项目 | 统一管理 |

### 使用建议

- **创作长篇**: 使用 `番茄小说/` 目录
- **创作短篇**: 使用 `番茄短篇故事集/` 目录

## 小说创作流程

### 1. 创作长篇小说

**步骤**:
1. 在 `番茄小说/` 目录下创建新项目目录
2. 使用 `skills/novel-writer/` 技能进行创作
3. 按照 Novel Writer 七步方法论创作

**示例**:
```bash
# 创建新项目
mkdir 番茄小说/新小说名

# 使用 Novel Writer 技能
cd 番茄小说/新小说名
python3 ../../skills/novel-writer/scripts/novel-writer.py
```

---

### 2. 创作短篇小说

**步骤**:
1. 在 `番茄短篇故事集/` 目录下创建新故事项目
2. 使用 `skills/novel-writer/` 技能进行创作
3. 按照 Novel Writer 七步方法论创作

**示例**:
```bash
# 创建新故事项目
cd 番茄短篇故事集
python3 scripts/create-story-project.py

# 使用 Novel Writer 技能
cd novel-writer-projects/<故事ID>号故事-<故事名>/
python3 ../../skills/novel-writer/scripts/novel-writer.py
```

## Novel Writer 七步方法论

### 步骤

1. **Constitution (创作宪法)**: 定义创作原则和目标
2. **Specification (规格设定)**: 设定故事的详细规格
3. **Clarification (明确说明)**: 明确关键概念和设定
4. **Plan (创作规划)**: 制定详细的创作计划
5. **Tasks (任务分解)**: 分解具体的创作任务
6. **Execute (执行创作)**: 执行具体的创作任务
7. **Analyze (综合验证)**: 验证创作结果并优化

### 使用方法

参考 `skills/novel-writer/` 技能的 `SKILL.md` 文件。

## 发布流程

### 长篇小说发布

长篇小说需要长期连载，发布流程相对复杂。

1. **完成大纲和前几章**
2. **创建发布计划**
3. **按计划发布章节**
4. **收集读者反馈**
5. **调整后续章节**

### 短篇小说发布

短篇小说一次性发布，发布流程相对简单。

1. **完成故事创作**
2. **运行质量检查**
3. **创建发布包**
4. **发布到番茄小说平台**
5. **收集数据**

详细流程请参考 `番茄短篇故事集/` 目录。
