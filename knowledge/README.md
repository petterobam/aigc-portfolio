# 知识库

## 概述
这个知识库用于系统地组织和管理重要信息，包括技术知识、工作流程、人物信息和经验教训。

## 结构

```
knowledge/
├── tech/        # 技术知识
│   ├── 记忆优化/
│   ├── 工具箱/
│   └── ...
├── work/        # 工作流程
│   ├── 番茄小说运营/
│   └── ...
├── people/      # 关键人物信息
└── lessons/     # 踩过的坑与经验教训
```

## 使用指南

### 技术知识 (tech/)
- 存储技术文档、API 参考
- 保存代码示例和最佳实践
- 整理学习笔记

### 工作流程 (work/)
- 记录标准化工作流程
- 保存 SOP 和操作程序
- 整理项目管理流程

### 关键人物信息 (people/)
- 记录重要联系人信息
- 保存人脉关系网络
- **注意隐私保护**

### 经验教训 (lessons/)
- 记录踩坑经历和解决方案
- 保存经验教训和反思
- 整理避坑指南

## 命名规范
- 文件名使用 kebab-case: `my-note.md`
- 目录名使用小写字母: `tech/`
- 添加时间戳：`2026-03-31-lesson-name.md`

## 搜索
使用 OpenClaw 的记忆搜索功能快速查找知识：
- 搜索技术文档：`memory_search 技术文档`
- 搜索经验教训：`memory_search 踩坑`
- 搜索工作流程：`memory_search 工作流程`
