## 常用命令

```bash
# 查看所有定时任务
openclaw cron list

# 查看任务详情
openclaw cron show <task-id>

# 启用任务
openclaw cron enable <task-id>

# 禁用任务
openclaw cron disable <task-id>

# 删除任务
openclaw cron delete <task-id>

# 查看任务日志
openclaw cron logs <task-id>
```
