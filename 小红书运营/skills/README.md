# Skills 目录

这里是 AI 自动化能力的定义，每个 skill 代表一个可执行的自动化任务。

## 可用 Skills

| Skill | 功能 | 触发词 |
|-------|------|--------|
| record_topic | 记录选题 | 「记录选题」「有个想法」 |
| develop_topic | 选题深化 | 「深化选题」「展开这个想法」 |
| generate_title | 生成标题 | 「生成标题」「小红书标题」 |
| review_data | 数据复盘 | 「数据复盘」「记录数据」 |

## 如何使用

直接说触发词，AI 会自动执行对应的 skill。

例如：
- 「记录选题：夏天压力肌怎么护肤不油腻」
- 「深化选题：早C晚A翻车经历」
- 「生成标题」
- 「数据复盘：这篇阅读5000点赞200」

## 添加新 Skill

当你发现某个重复性工作可以自动化时：
1. 创建新的 .skill 文件
2. 定义触发词和执行流程
3. 在 CLAUDE.md 中更新技能列表
