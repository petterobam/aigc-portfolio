# 发布前强制检查清单

> **用途**：确保每次发布都符合最佳实践，避免低级错误
> **执行时机**：每次发布前必须逐项检查，不可跳过

---

## 一、发布时间检查 ⚠️ 最重要

- [ ] **当前时间是否在晚8-9点之间？**
  - ❌ 凌晨0-6点：绝对禁止发布（流量低谷期）
  - ❌ 上午9-12点：不推荐（工作时间）
  - ✅ 晚8-9点：最佳发布时段
  - ⚠️ 其他时间：需有明确理由

**如果不在最佳时段**：
- 选项A：保存草稿，等到晚8点再发布
- 选项B：使用定时发布功能（如有）
- 选项C：记录原因，下次避免

---

## 二、内容质量检查

### 2.1 标题检查
- [ ] 标题字数 ≤ 20字（含符号）
- [ ] 包含数字或具体利益点（如"3分钟"、"5个技巧"）
- [ ] 包含身份标签或情绪词（如"早八人"、"救命"）
- [ ] 无错别字、无敏感词

### 2.2 正文检查
- [ ] 开头3秒有钩子（痛点/悬念/利益）
- [ ] 正文分段清晰，使用emoji分隔
- [ ] 包含实用干货（可收藏的内容）
- [ ] 结尾有明确的CTA（引导评论/关注）
- [ ] 标签数量5-8个，包含核心词+场景词+人群词

### 2.3 封面检查
- [ ] 图片清晰度足够（无模糊/压缩过度）
- [ ] 封面文字可读（字体大小合适，对比度足够）
- [ ] 图片风格与账号人设一致
- [ ] 已保存到手机相册（如需手机发布）

---

## 三、发布设置检查

- [ ] 发布类型选择正确（图文/视频）
- [ ] 封面图已设置为第一张图
- [ ] 位置信息已关闭（保护隐私）
- [ ] 可见范围设置为"公开"

---

## 四、发布后必做动作

### 黄金2小时内（关键！）
- [ ] 转发到私域群聊（如有）
- [ ] 用小号或请朋友点赞+收藏
- [ ] 在评论区发布置顶评论（引导互动）
- [ ] 回复每一条评论（提升CES评分）

### 24小时内
- [ ] 记录数据（小眼睛、点赞、收藏、评论）
- [ ] 更新数据追踪看板
- [ ] 分析评论区反馈

---

## 五、常见错误清单

| 错误类型 | 后果 | 预防措施 |
|---------|------|---------|
| 凌晨发布 | 流量极低，浪费内容 | 强制检查发布时间 |
| 标题超过20字 | 被截断，影响CTR | 发布前数字数 |
| 无CTA引导 | 互动率低，难破池 | 每篇必须设计互动点 |
| 发布后不管 | 错过黄金2小时 | 设置闹钟提醒 |
| 标签太少/太多 | 影响推荐精准度 | 固定5-8个标签 |
