## 四、内容策略

### 标题公式
- 数字 + 情绪词 + 悬念
- "谁懂啊" + 共情场景
- "原来是因为这个" + 好奇心缺口
- 真实测评 + 具体时长（3天实测、7天记录）

### 内容结构
1. **钩子**：真实经历/痛点场景
2. **价值**：科学解释/实测数据
3. **行动**：具体方法/产品推荐
4. **互动**：评论区引导

### 配图风格
- 自然光自拍，不精修
- 护肤品平铺/实拍
- 文字卡片：极简、莫兰迪色系
- 图解：线条简洁、信息清晰

## 六、数据监控

### 每日检查
- 粉丝数、新增粉丝
- 总赞藏、新增赞藏
- 3篇笔记的增量数据

### 每篇检查点
- 发布后24h：初版数据
- 发布后72h：稳定数据
- 发布后7天：长尾数据

### 关键指标
- 收藏率 = 收藏数 / 阅读数（目标 >10%）
- 互动率 = (赞+藏+评) / 阅读数
- 涨粉转化率 = 新增粉丝 / 阅读数
