## 🎯 使用场景指南

### 场景1：我要发布第一篇笔记
1. 先看：一键发布清单.md (心理建设+完整流程)
2. 参考：运营总览.md (了解账号定位)

### 场景2：我要发布日常笔记
1. 准备内容：内容生产流水线.md (Stage 1-3)
2. 发布检查：发布前强制检查清单.md
3. 发布执行：内容发布自动化检查清单.md

### 场景3：我要做周复盘
1. 查看数据：每日检查清单.md → 每周检查部分
2. 回顾流水线：内容生产流水线.md → Stage 6复盘归档

### 场景4：我要写文案
1. 直接看：内容生产流水线.md → Stage 2内容策划
2. 参考：爆款文案公式、标题生成速查
