## 📋 发布前检查项

### 一、内容质量检查

| 检查项 | 标准 | 状态 |
|-------|------|------|
| 标题包含数字 | 如"3分钟""5个""100次" | ⬜ |
| 标题包含情绪词 | 如"悟了""踩雷""真相" | ⬜ |
| 开头有社死场景 | 具体细节，引发共鸣 | ⬜ |
| 内容有情绪起伏 | 崩溃→尝试→顿悟→真香 | ⬜ |
| 包含具体数字 | 时间/数量/价格 | ⬜ |
| 有Before/After对比 | 视觉冲击力 | ⬜ |
| 结尾有A/B/C选择题 | 降低评论门槛 | ⬜ |
| 有关注引导 | "关注我，看..." | ⬜ |
| 有下期预告 | 增强系列粘性 | ⬜ |

## 🚀 发布执行流程

### Step 1: 准备阶段（发布前30分钟）

```
⬜ 打开小红书创作者中心
⬜ 确认封面图已准备好
⬜ 复制正文内容到剪贴板
⬜ 确认标签列表
⬜ 检查网络连接
```

### Step 2: 发布阶段（20:30准时）

```
⬜ 点击"发布笔记"
⬜ 上传封面图
⬜ 粘贴正文内容
⬜ 添加标签（8-10个）
⬜ 选择话题（如有）
⬜ 设置发布时间（如定时发布）
⬜ 点击"发布"
```

### Step 3: 发布后检查（发布后5分钟）

```
⬜ 确认笔记已成功发布
⬜ 检查封面图显示正常
⬜ 检查正文排版正常
⬜ 检查标签显示正常
⬜ 截图保存发布记录
```

### Step 4: 数据记录（发布后24h/48h/72h）

```
⬜ 记录浏览量
⬜ 记录点赞数
⬜ 记录收藏数
⬜ 记录评论数
⬜ 更新内容索引表
⬜ 更新数据看板
```

## ⚠️ 常见错误清单

### 绝对不能犯的错误

| 错误 | 后果 | 预防措施 |
|-----|------|---------|
| 凌晨发布 | 流量损失50%+ | 设置20:30闹钟 |
| 无CTA设计 | 评论数=0 | 必须包含A/B/C选择题 |
| 封面无钩子 | CTR<3% | 检查文字钩子 |
| 无情绪设计 | 点赞数=0 | 检查情绪起伏 |
| 发布时间随意 | 错过黄金时段 | 严格执行20:30 |

### 需要避免的问题

| 问题 | 表现 | 解决方案 |
|-----|------|---------|
| 内容同质化 | 用户疲劳 | 轮换护肤/彩妆选题 |
| 热点失效 | 流量下降 | 24小时内响应热点 |
| 发布疲劳 | 质量下降 | 提前储备3篇内容 |
| 数据焦虑 | 频繁刷新 | 设置固定查看时间 |
