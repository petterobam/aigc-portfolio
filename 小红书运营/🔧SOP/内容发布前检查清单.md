# 小红书内容发布前检查清单

> **目的**：确保每次发布的内容质量，避免发布后发现文字乱码、格式错误等问题
> **使用时机**：每次发布前必须逐项检查

---

## ✅ 必检项目（全部通过才能发布）

### 1. 内容完整性检查

- [ ] **标题完整**：没有截断、乱码
- [ ] **正文完整**：所有段落都显示正确
- [ ] **表情符号正常**：✨💫等表情显示为符号，不是方框或问号
- [ ] **数字格式正确**：如"80亿+"、"127%"显示正常
- [ ] **标点符号正常**：中文标点（，。？！）没有变成英文标点

### 2. 图片质量检查

- [ ] **封面图清晰**：没有模糊、变形
- [ ] **封面文字可读**：文字与背景对比度足够，没有乱码
- [ ] **图片尺寸正确**：符合小红书推荐尺寸（3:4竖屏）
- [ ] **图片格式正确**：JPG/PNG格式，没有损坏

### 3. 话题标签检查

- [ ] **标签数量适中**：5-10个话题标签
- [ ] **标签相关性**：所有标签都与内容相关
- [ ] **热门标签包含**：至少1-2个当前热门标签
- [ ] **长尾标签包含**：1-2个精准细分标签

### 4. CTA互动设计检查

- [ ] **有明确的互动引导**：不是陈述句结尾
- [ ] **互动门槛低**：用户容易参与（如选择题、简单问题）
- [ ] **与内容相关**：互动问题与正文主题一致

### 5. 发布时间检查

- [ ] **时间合适**：避开深夜低谷期（00:00-06:00）
- [ ] **目标人群在线**：考虑目标用户的时间习惯
- [ ] **与上次发布间隔**：至少6小时以上

---

## 🔍 可检项目（建议检查）

### 6. 内容优化检查

- [ ] **标题有数字**：包含时间、步骤、数据等数字
- [ ] **标题有情绪词**：如"疯了"、"悟了"、"哭了"
- [ ] **正文有分段**：不是一大段文字，有小标题或分段
- [ ] **有人设植入**：包含个人经历、身份标签

### 7. 趋势热点检查

- [ ] **蹭了热点**：内容与当前热点相关
- [ ] **热点时效性**：热点还在热度期内
- [ ] **热点相关性**：不是硬蹭，有逻辑连接

---

## 🚫 禁止发布的情况

出现以下任何一种情况，**禁止发布**，必须先修复：

1. **文字乱码**：任何地方出现方框、问号、乱码
2. **图片损坏**：图片无法加载或显示异常
3. **标题截断**：标题在预览中被截断，意思不完整
4. **表情异常**：表情符号显示为方框或其他异常符号
5. **格式混乱**：段落错乱、排版混乱
6. **链接失效**：包含的链接无法打开
7. **敏感词未处理**：包含平台违禁词或敏感内容

- [ ] 我已经逐项检查了所有"必检项目"
- [ ] 没有出现任何"禁止发布"的情况
- [ ] 内容我已经预览过，确认可以正常显示
- [ ] 图片我已经查看过，确认没有乱码
- [ ] 我对这篇内容满意，愿意让它代表我

## 🔧 常见问题快速修复

| 问题 | 可能原因 | 快速修复 |
|------|---------|---------|
| 文字乱码 | 编码问题 | 复制到纯文本编辑器，重新复制 |
| 表情变方框 | 表情不兼容 | 替换为通用表情或文字描述 |
| 图片模糊 | 压缩过度 | 使用原始清晰图片 |
| 标题截断 | 字数过多 | 缩短到15字以内 |
| 标签无效 | 标签错误 | 检查#号前后是否有空格 |
