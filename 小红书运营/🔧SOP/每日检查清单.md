# 每日运营检查清单

> **用途**: 确保每日运营动作不遗漏
> **使用时间**: 每天固定时间执行
> **预计耗时**: 10-15分钟

---

## 上午检查（09:00-09:15）

### 数据检查（5分钟）

- [ ] 打开小红书APP，查看昨日发布笔记数据
- [ ] 记录：小眼睛 / 点赞 / 收藏 / 评论
- [ ] 更新数据追踪看板
- [ ] 检查是否有新评论需要回复

**数据记录位置**: `📊数据看板/数据追踪看板.md`

### 内容检查（5分钟）

- [ ] 检查待发布文件夹，确认今日发布包是否就绪
- [ ] 检查发布时间是否设定为20:00-21:00
- [ ] 确认封面图已上传到手机相册

---

## 发布前检查（19:50-20:00）

### 发布包检查（5分钟）

- [ ] 打开今日发布包文档
- [ ] 确认标题已复制到剪贴板
- [ ] 确认正文已复制到剪贴板
- [ ] 确认封面图已在手机相册
- [ ] 确认置顶评论内容已准备

### 环境检查（2分钟）

- [ ] 网络连接正常
- [ ] 手机电量充足
- [ ] 当前时间在20:00-21:00之间

---

## 发布后检查（发布后5分钟内）

### 立即执行（3分钟）

- [ ] 发布成功，确认笔记可见
- [ ] 立即发布置顶评论
- [ ] 截图记录初始数据（小眼睛、点赞、收藏、评论）

### 数据记录（2分钟）

- [ ] 更新数据追踪看板
- [ ] 记录发布时间
- [ ] 标记发布状态为"已发布"

---

## 晚间检查（22:00-22:10）

### 数据追踪（5分钟）

- [ ] 查看今日发布笔记的2小时数据
- [ ] 记录：小眼睛 / 点赞 / 收藏 / 评论
- [ ] 对比预期数据，标记异常

### 互动处理（5分钟）

- [ ] 回复所有用户评论
- [ ] 点赞有价值的评论
- [ ] 在相关笔记下互动（养号）

---

## 每周检查（周日）

### 周复盘检查（30分钟）

- [ ] 统计本周发布数量
- [ ] 统计本周总浏览/点赞/收藏
- [ ] 识别本周最佳表现笔记
- [ ] 识别本周最差表现笔记
- [ ] 分析成功/失败原因
- [ ] 撰写周复盘报告

### 下周规划（30分钟）

- [ ] 确定下周3篇选题
- [ ] 检查内容储备（至少3篇发布包）
- [ ] 更新本周发布计划
- [ ] 设定下周数据目标

---

## 紧急处理清单

### 如果发布失败

- [ ] 检查网络连接
- [ ] 检查图片是否符合规范
- [ ] 检查文案是否有敏感词
- [ ] 尝试重新发布
- [ ] 如仍失败，记录问题，改日再发

### 如果数据异常

- [ ] 检查是否被限流（小眼睛增长停滞）
- [ ] 检查是否有违规提示
- [ ] 对比同类笔记数据
- [ ] 记录异常，复盘时分析

### 如果有负面评论

- [ ] 保持冷静，不要争吵
- [ ] 判断是否为恶意攻击
- [ ] 恶意攻击：举报+删除
- [ ] 合理批评：感谢+改进
- [ ] 记录反馈，优化内容
