## 三、分析框架

### 3.1 爆款内容分析维度

#### 标题分析
- [ ] 是否包含数字
- [ ] 是否包含情绪词
- [ ] 是否包含身份标签
- [ ] 是否制造悬念/反差
- [ ] 是否承诺利益
- [ ] 标题长度（字数）

#### 文案分析
- [ ] 开头3秒钩子
- [ ] 痛点引入方式
- [ ] 内容结构（总分总/对比/步骤）
- [ ] 情绪表达密度
- [ ] emoji使用频率
- [ ] 互动引导位置
- [ ] 结尾转化设计

#### 视觉分析
- [ ] 封面类型（对比/真人/产品/文字）
- [ ] 封面色调
- [ ] 是否使用拼图
- [ ] 文字在封面上的使用
- [ ] 配图数量
- [ ] 图片风格（真实/精致/滤镜）

#### 数据表现分析
- [ ] 收藏率 > 10%（高价值内容）
- [ ] 评论率 > 5%（高互动内容）
- [ ] 涨粉率（转粉效果）

### 3.2 分类标签体系

**按内容类型：**
- 科普知识类
- 产品测评类
- 真实体验类
- 教程步骤类
- 故事分享类
- 合集清单类

**按爆款因素：**
- 标题党型
- 干货价值型
- 情绪共鸣型
- 人设故事型
- 视觉吸引型

**按可借鉴程度：**
- 可直接复用
- 可改编借鉴
- 可参考思路
- 仅作了解

## 四、方法论提炼

### 4.1 每日提炼（30分钟）

**从当日100篇中提炼：**

1. **今日爆款公式**（1-2个）
   - 什么类型的标题今天表现好
   - 什么话题今天热度高

2. **今日热词**（5-10个）
   - 新出现的网络热词
   - 热门的表情符号组合

3. **今日亮点技巧**（3-5个）
   - 值得学习的文案技巧
   - 创新的内容形式

4. **今日差异化机会**（1-2个）
   - 什么内容没人做但应该有
   - 什么角度可以做得更好

### 4.2 每周总结（1小时）

**从700篇中提炼：**

1. **本周内容趋势**
   - 什么类型内容在上升
   - 什么类型内容在下降

2. **本周爆款密码**
   - 高收藏内容的共同特征
   - 高互动内容的共同特征

3. **本周优化建议**
   - 我们下周应该调整什么
   - 新的选题方向

### 4.3 每月复盘（2小时）

**从3000篇中提炼：**

1. **月度行业洞察**
   - 平台算法变化迹象
   - 用户偏好变化趋势

2. **方法论升级**
   - 更新标题公式库
   - 更新文案模板库
   - 更新视觉风格指南
