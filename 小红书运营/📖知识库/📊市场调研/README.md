## 🔄 调研文件处理流程

### 1. 新建调研文件
- 发现新的市场趋势或研究机会时
- 创建新的调研文件保存原始数据
- 使用标准命名格式

### 2. 整理到知识库
- 从调研文件中提炼核心观点
- 整理成可复用的理论知识
- 保存到对应的📖知识库子目录

### 3. 应用到方法论
- 从知识库理论中提炼实战方法
- 验证方法有效性
- 固化到📚方法论目录

### 4. 验证闭环
- 用📊数据看板验证方法效果
- 有效→固化，无效→重新研究
- 形成认知闭环

## 📊 与其他目录的关系
