## 💥 2025小红书爆款公式更新

### CES评分系统（最新算法）
```
CES = 点赞(1分) + 收藏(1分) + 评论(4分) + 转发(4分) + 关注(8分)
```

**关键发现**：
- **评论和关注权重最高**（各4分和8分）
- 点赞和收藏权重相等（各1分）
- 策略重点：引导评论和关注

### 爆款笔记数据基准
- **爆款点击率**：**5.8%+**
- **初级流量池**：约200曝光
- **破池标准**：CES评分高于阈值 → 进入更大流量池

### 用户决策链路
```
封面吸引 → 标题点击 → 前3秒留存 → 深度互动 → 转化
```

**关键指标**：
- **3秒流失率** - 越低越好
- **平均观看时长** - 越长越好

### 美妆博主商业化数据（2025）
- **品牌合作人门槛**：粉丝≥1000 + 实名认证
- **单条报价**：约为粉丝数的1-2%
- **案例**：10万粉账号单条约1000-2000元
