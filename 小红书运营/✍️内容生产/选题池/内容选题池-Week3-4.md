## 📝 内容公式模板

### 公式1: 社死经历 + 解决方案
```
标题: [痛点场景]｜[结果]的[方法]
例: 顶着花脸开会｜被同事问是不是哭了，这套定妆法救了我

结构:
1. 社死场景（情绪钩子）
2. 焦虑升级（共鸣加深）
3. 转折点（发现方法）
4. 解决方案（干货价值）
5. 效果验证（真实记录）
6. 互动提问（评论引导）
```

### 公式2: 失败复盘 + 洞察分享
```
标题: [负面数据]｜但我发现了[真相/方法]
例: 发了8篇笔记0点赞｜但我发现了一个真相

结构:
1. 坦诚暴露问题
2. 数据分析过程
3. 核心洞察发现
4. 行动计划改变
5. 给读者的价值
6. 互动邀请
```

### 公式3: 养成记录 + 进步分享
```
标题: [时间跨度]｜[技能]我终于[结果]
例: 画眼线画了100次｜第101次终于悟了

结构:
1. 起点状态（笨拙/不会）
2. 练习过程（具体次数/时间）
3. 顿悟时刻（关键点）
4. 方法分享（干货）
5. 现在状态（对比）
6. 鼓励互动
```
