## 🔄 工作流程

### 1. 选题阶段
- 📍 位置：选题池/
- 📝 动作：创建/更新选题规划
- ✅ 完成标志：选题确定

### 2. 撰写阶段
- 📍 位置：文案稿/
- 📝 动作：撰写文案终稿
- ✅ 完成标志：文案定稿

### 3. 准备阶段
- 📍 位置：发布包/
- 📝 动作：创建发布包(文案+封面+标签)
- ✅ 完成标志：发布包就绪

### 4. 发布阶段
- 📍 位置：移动到 📤已发布/
- 📝 动作：执行发布
- ✅ 完成标志：发布记录创建

## 🎨 快速操作

### 创建新文案稿
```bash
# 在文案稿目录创建新文件
touch "✍️内容生产/文案稿/第{X}篇-{主题}-文案终稿.md"
```

### 创建发布包
```bash
# 从文案稿创建发布包
cp "✍️内容生产/文案稿/第{X}篇-*.md" "✍️内容生产/发布包/第{X}篇-{主题}-发布包.md"
```

### 移动到已发布
```bash
# 发布后移动记录
mv "✍️内容生产/发布包/第{X}篇-*.md" "📤已发布/2026-03/"
```
