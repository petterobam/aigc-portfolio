## 二、Week 2 核心策略

### 2.1 内容公式升级

**Week 1 公式（只有干货）：**
```
有效内容 = 实用价值（收藏）
```

**Week 2 公式（完整版）：**
```
有效内容 = 实用价值（收藏）+ 情绪价值（点赞）+ 互动设计（评论）
```

### 2.2 三大核心假设

| 假设 | 测试内容 | 验证标准 |
|-----|---------|---------|
| **假设1** | 情绪共鸣优化 → 点赞数>0 | 至少1篇笔记有点赞 |
| **假设2** | A/B/C选择题CTA → 评论数>3 | 单篇评论突破3条 |
| **假设3** | 晚8-9点发布 → 初始曝光>300 | 首小时曝光提升 |

### 2.3 情绪共鸣设计框架

**情绪曲线设计：**
```
开头：社死场景（崩溃/尴尬/焦虑）
  ↓
中间：尝试过程（失败/摸索/顿悟）
  ↓
结尾：成功反转（狂喜/真香/逆袭）
```

**具体技巧：**
- 用具体数字（"画了100次"比"练了很久"更有冲击力）
- 用场景细节（"早八狂奔到地铁站"比"早上很赶"更真实）
- 用情绪词汇（"崩溃""狂喜""真香"比"难过""开心"更有张力）
