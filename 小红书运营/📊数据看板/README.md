## 🔄 数据管理流程

### 每日数据追踪
1. **发布后1h**：记录初始数据
2. **发布后24h**：记录首日数据
3. **每日检查**：更新数据追踪看板

### 每周数据复盘
1. **周日晚上**：统计周数据
2. **撰写复盘**：WeekX-复盘报告.md
3. **制定策略**：WeekX+1-执行计划.md

### 深度分析
1. **每月一次**：深度数据分析
2. **爆款研究**：提取爆款规律
3. **策略调整**：基于数据优化内容

## 🎨 快速操作

### 更新今日数据
```bash
# 打开数据追踪看板
open "📊数据看板/📈核心数据/数据追踪看板.md"
```

### 查看本周复盘
```bash
# 查看最新周复盘
ls -lt "📊数据看板/📊周复盘/" | head -2
```

### 创建新复盘
```bash
# 创建周复盘模板
cp "📊数据看板/📊周复盘/Week1-深度复盘报告.md" "📊数据看板/📊周复盘/WeekX-复盘报告.md"
```
