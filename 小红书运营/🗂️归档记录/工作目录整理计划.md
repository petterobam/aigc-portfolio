## 四、命名规范统一

### 4.1 文件命名规则

```
[类型]-[描述]-[日期].md

类型前缀：
- 数据看板: 📊
- 发布计划: 📝
- 内容生产: ✍️
- 方法论: 📚
- SOP: 🔧
- 归档: 🗂️

示例：
- 发布清单-3月17日.md
- Week1-深度复盘-2026-03-16.md
- 情绪共鸣优化指南.md
```

### 4.2 目录命名规则

- 使用连字符`-`而非下划线`_`
- 使用两位数字前缀排序（如`01-内容生产`）
- 使用emoji前缀增加辨识度
