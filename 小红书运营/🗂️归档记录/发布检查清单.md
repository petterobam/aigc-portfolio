# 小红书笔记发布检查清单

> **用途**：降低发布阻力，确保每次发布都有完整记录
> **使用方式**：发布前逐项检查，发布后填写数据

---

## 一、发布前检查（5分钟）

### 1. 内容检查
- [ ] 文案已最终确认（无错别字、排版清晰）
- [ ] 封面图已生成/准备好
- [ ] 配图数量符合平台建议（图文：3-6张，视频：1条）
- [ ] 标题包含关键词（便于搜索）
- [ ] 正文有明确的CTA（引导评论/收藏/关注）

### 2. 发布设置
- [ ] 选择合适的话题标签（3-5个核心标签+2-3个长尾标签）
- [ ] 定位设置（可选，根据内容决定）
- [ ] 发布时间选择（参考：早7-9点，午12-14点，晚20-22点）

### 3. 互动预设
- [ ] 置顶评论内容已准备好
- [ ] 预设回复话术已准备（针对常见问题）

## 五、快速发布模板

### 模板A：干货教程类
```
标题：[身份]｜[数字][解决方案][结果]
正文：
1. 痛点共鸣（1句）
2. 方法总览（1句）
3. 详细步骤（3-5步）
4. 关键tips（2-3条）
5. CTA（引导互动）
标签：#核心标签 #长尾标签
```

### 模板B：真实分享类
```
标题：[场景]｜[情绪][结果]
正文：
1. 场景描述（2-3句）
2. 真实经历（3-5句）
3. 解决方案/结果（2-3句）
4. 互动提问（1句）
标签：#核心标签 #长尾标签
```

## 六、常见问题速查

| 问题 | 解决方案 |
|-----|---------|
| 发布后没流量 | 正常冷启动，等24小时再看 |
| 评论被吞 | 检查是否有敏感词，重新发布 |
| 图片模糊 | 检查原图分辨率，重新上传高清图 |
| 标签不显示 | 检查标签格式，#后不能有空格 |
