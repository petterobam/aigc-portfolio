## 🔑 核心记忆口诀详解

> **"由内到外多到少，分区上妆大法好，油多粉少要记牢，点按弹飞井有条"**

| 口诀分句 | 含义解读 | 适用场景 |
|:---|:---|:---|
| **由内到外多到少** | 从面部中央向外侧涂抹，用量逐渐减少 | 全脸上妆 |
| **分区上妆大法好** | 不同区域（T区、脸颊等）采用不同手法 | 混合肌/油皮 |
| **油多粉少要记牢** | 出油多的部位少上粉底，避免厚重 | T区易出油 |
| **点按弹飞井有条** | 用美妆蛋点按、弹飞的手法，让底妆服帖 | 工具使用技巧 |

## 💡 对养成系账号的启示

### 1. 专业度≠距离感
- 硬核内容也可以做出爆款
- 关键是把专业内容"翻译"成用户能懂的语言

### 2. 记忆点设计
- 口诀/顺口溜让内容更易传播
- 用户能记住才能分享

### 3. 长视频也能爆
- 7分钟在短视频平台算"长"
- 但干货足够就能留住用户

### 4. 信任建立公式
```
身份背书 + 原理解释 + 实操演示 = 专业可信
```

## 📝 可借鉴的创作公式

### 标题公式
```
"我保证/我赌你/我敢说" + 超级lative + 品类 + 教程
```

**示例**：
- "我保证，这是你看过最硬核的底妆教程"（原案例）
- "我赌你，这是最简单的新手眼妆教程"
- "我敢说，这是油皮最友好的早八底妆法"

### 内容结构
| 阶段 | 时长 | 内容 | 目的 |
|:---|:---|:---|:---|
| **身份锚定** | 3秒 | "4年化妆师告诉你..." | 建立信任 |
| **痛点共鸣** | 10秒 | "是不是经常卡粉？" | 抓住注意力 |
| **原理解析** | 30秒 | 用简单实验/图示解释为什么 | 建立专业感 |
| **口诀记忆** | 5秒 | 给出朗朗上口的记忆点 | 降低记忆成本 |
| **分步演示** | 主体 | 跟着口诀一步步做 | 交付价值 |
| **效果对比** | 结尾 | Before/After视觉冲击 | 证明效果 |
