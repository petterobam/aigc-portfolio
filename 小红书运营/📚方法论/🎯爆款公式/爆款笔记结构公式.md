## 六大爆款结构公式

### 公式1：AIDA结构（经典营销模型）

| 步骤 | 说明 | 美妆案例 |
|:---|:---|:---|
| **A**ttention 注意 | 抓人标题/视觉冲击 | "不是我说，这真的不是补水狂魔嘛" |
| **I**nterest 兴趣 | 详细描述引发好奇 | 背靠大厂、成分权威 |
| **D**esire 欲望 | 突出独家优势 | 膜布独特卖点，人无我有 |
| **A**ction 行动 | 明确引导购买 | "干燥肌肤的宝宝们快安排起来" |

**适用场景**：产品种草、好物推荐

---

### 公式2：4U结构（种草类首选）

| 要素 | 说明 | 案例 |
|:---|:---|:---|
| **U**rgent 紧迫性 | 时间/场景压力 | "让牙刷干干净净，没有蛀牙" |
| **U**nique 独特性 | 人无我有的卖点 | 精致小巧，超级适合女生 |
| **U**ltra-specific 超具体 | 细节参数 | 刷毛硬度、手柄触感实测 |
| **U**seful 有用性 | 实用价值 | 多种刷牙模式+充电方式 |

**适用场景**：产品测评、好物分享

---

### 公式3：问题解决型结构（干货分享）⭐ 我最适合

```
痛点场景 → 错误示范 → 科学原理 → 实操步骤 → 效果对比
```

**完整案例**：《油痘肌自救指南》

1. **开篇**：反复长痘5年的痛苦经历
2. **展示**：错误护肤导致的烂脸照片
3. **原理**：皮肤科医生论文解释致痘原理
4. **步骤**：分步演示"早C晚A"护肤流程
5. **对比**：28天前后皮肤检测报告

**适用场景**：教程类、干货分享、经验总结

### 公式4：清单合集型结构（产品推荐）

```
主题引入 → 分类标准 → 产品清单 → 核心参数 → 使用场景
```

**完整案例**：《100元内平价彩妆红榜》

- **切入**：学生党月生活费1500
- **标准**：单价≤30元且好评率＞95%
- **分类**：底妆、眼影、口红等
- **参数**：色号、克重等关键信息
- **场景**：不同场合妆容效果对比

**适用场景**：好物合集、产品推荐、红黑榜

---

### 公式5：对比测评型结构

**横向对比**（同类产品）：
```
产品A vs 产品B vs 产品C
↓
多维度评分（成分/价格/效果/肤感）
↓
明确结论 + 选购建议
```

**纵向对比**（使用前后）：
```
使用前状态
↓
使用过程记录
↓
使用后效果
↓
真实感受总结
```

**适用场景**：产品测评、成分分析、效果验证

---

### 公式6：情绪类结构（情感营销）

```
戳痛点 → 给方案 → 做呼吁
```

**结构要点**：
1. **戳痛点**：描述焦虑/困扰引发共鸣
2. **给方案**：提供实用工具/解决方案
3. **做呼吁**：强化行动意愿

**适用场景**：美妆、健康、情感赛道

## 正文万能模板（可直接套用）

### 开头：痛点共鸣公式

> **模板**："你是不是也……（痛点）？我之前……（经历），直到发现……（解决方案）！"

**案例**：
> "你是不是每次化妆都卡粉？我之前用啥粉底都起皮，直到闺蜜安利了这个方法，妆感直接妈生皮！"

**我的应用**：
> "你是不是熬夜后第二天脸又黄又垮？我之前见客户前必敷面膜急救，直到发现这3分钟急救法，素颜也敢直接上！"

---

### 中间：干货分层 + 视觉化

**结构原则**：
- 用序号/小标题拆分："3步搞定""5个技巧"
- 每300字插入1张图（对比图、流程图）
- emoji分隔，每段不超过3行

**示例**：
```
去黑头3步曲：

1️⃣ 热毛巾敷5分钟打开毛孔
   （配图：热毛巾敷脸）

2️⃣ 厚涂清洁泥膜10分钟
   （配图：产品+涂抹过程）

3️⃣ 细胞夹轻轻刮掉
   （配图：前后对比）
```

---

### 结尾：互动引导 + 价值升华

**互动引导模板**：
> "你们还有什么问题？评论区告诉我～"
> "这个方法你们试过吗？欢迎交流！"
> "有同样困扰的姐妹举个手🙋‍♀️"

**价值升华模板**：
> "护肤不是一蹴而就，坚持才能看到改变。一起加油！"
> "希望每个早八人都能多睡10分钟，还能美美出门～"
