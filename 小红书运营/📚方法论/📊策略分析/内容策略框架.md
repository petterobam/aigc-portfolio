## 一、核心洞察（基于数据验证）

### 1.1 小红书CES评分公式（2025年最新）

**CES = 点赞(1分) + 收藏(4分) + 评论(4分) + 转发(10分)**

**关键洞察**：
- 收藏和评论权重相同（4分），远高于点赞（1分）
- 转发权重最高（10分），平台重视传播扩散
- **Week 1数据解读**：收藏26但点赞0 → 内容有实用价值但缺乏情绪共鸣

**优化方向**：
```
实用价值（收藏）+ 情绪价值（点赞）+ 互动设计（评论）+ 金句/洞察（转发）= 高CES内容
```

### 1.2 已验证的有效模式

| 模式 | 代表内容 | 数据表现 | CES分析 | 核心公式 |
|-----|---------|---------|---------|---------|
| 趋势+教程 | 帕斯蒂尔风骨相妆 | 259浏览, 14收藏 | 收藏高(4×14) | 热点词+痛点+步骤化 |
| 真实复盘 | 8篇0点赞复盘 | 156浏览, 16点赞 | 点赞+收藏双高 | 反差+真相+人设 |
| 韩系趋势 | 韩系水光肌 | 156浏览, 8收藏 | 收藏主导 | 趋势词+场景+效果 |

### 1.3 已验证的无效模式

| 模式 | 代表内容 | 数据表现 | CES分析 | 问题诊断 |
|-----|---------|---------|---------|---------|
| 小众热点 | 邪修眼妆(3篇) | 36-56浏览 | 各项均低 | 热点太小众 |
| 纯教程 | 无粉底气血感妆 | 11浏览 | 各项均低 | 无情绪+无热点 |
| 重复发布 | 同类眼妆3篇 | 递减效应 | 无新增CES | 用户疲劳 |

## 三、标题公式库（已验证版）

### 3.1 高CTR公式（按效果排序）

#### 公式1：数字+痛点+效果承诺 ⭐⭐⭐⭐⭐
```
模板：[数字][痛点]｜[效果承诺]
案例：早八人5分钟换头术｜被同事问是不是素颜
案例：油皮12小时不脱妆｜这3个步骤我悟了
```

#### 公式2：反差+真相+悬念 ⭐⭐⭐⭐⭐
```
模板：[反差现象]，[真相/发现]
案例：发了8篇笔记0点赞，我发现了这个真相
案例：踩雷12瓶后，这5个真的值得回购
```

#### 公式3：趋势词+场景+身份 ⭐⭐⭐⭐
```
模板：[趋势词]｜[身份][场景][效果]
案例：韩系水光肌｜实习生3分钟get玻璃唇
案例：帕斯蒂尔风｜早八人5分钟骨相妆
```

#### 公式4：时间+对比+成长 ⭐⭐⭐⭐
```
模板：[身份][时间]｜从[状态A]到[状态B]
案例：实习生3个月对比｜从化妆小白到现在的变化
案例：练习生日记｜3年训练教会我的5件事
```

### 3.2 标题自检清单

每篇标题发布前检查：
- [ ] 包含具体数字（时间/步骤/数量）
- [ ] 包含情绪词或趋势词
- [ ] 包含场景或身份标签
- [ ] 有悬念或效果承诺
- [ ] 不超过20字（手机端显示完整）

## 四、正文结构模板

### 4.1 趋势教程类结构

```
【开头】情绪钩子（3行内）
- 痛点共鸣："姐妹们，我发现了一个..."
- 效果预告："现在5分钟搞定，同事问我..."

【中段】干货交付
- 概念解释（什么是XX）
- 步骤拆解（Step 1-4）
- 关键tips（3-5个）

【结尾】人设强化+CTA
- 真实效果反馈
- 互动问题："你们想看我出视频版吗？"
- 养成系标签："我一个个试，然后反馈真实效果"
```

### 4.2 真实复盘类结构

```
【开头】数据冲击
- 先上数据："发布X篇，总浏览X，点赞X"
- 反差制造："是的，X篇笔记，0个赞"

【中段】洞察分享
- 发现的问题
- 复盘的过程
- 学到的教训

【结尾】价值升华+共鸣
- 给读者的建议
- 人设强化
- 互动引导
```

### 4.3 系列内容结构（Week复盘）

```
【固定开头】
Week X 复盘｜[本周核心变化]

【数据板块】
- 本周发布：X篇
- 总浏览：XXX
- 关键突破：[具体数据]

【内容分析】
- 表现最好的内容：[标题] + 成功因素
- 表现最差的内容：[标题] + 失败原因

【下周计划】
- 继续做的：[具体动作]
- 停止做的：[具体动作]
- 新尝试的：[具体动作]

【养成系互动】
- "你们想看我在Week X尝试什么？"
```

## 五、互动设计规范（基于CES公式优化）

### 5.1 CES导向的CTA设计

每篇内容结尾必须包含多层CTA，最大化CES得分：

```
【收藏引导】💡 有用的话点个收藏（4分）
【点赞引导】❤️ 有共鸣的话点个赞（1分）
【转发引导】🔄 值得分享的话转发给需要的人（10分）
【关注引导】⭐ 关注我，下周发布XX内容
【互动引导】💬 评论区告诉我...
```

### 5.2 高CES互动模板

| 模板类型 | CES目标 | 示例 |
|---------|--------|------|
| **选择题+开放题** | 评论4分 | "A.正在经历 B.已经突破 C.来学经验\n评论区告诉我你的笔记数据？" |
| **分层引导** | 多维度 | "💡收藏=4分，❤️点赞=1分\n你的每一个互动都对我很重要" |
| **承诺回复** | 评论4分 | "我会在评论区回复每一条！" |
| **下周预告** | 关注转化 | "下周同一时间发布Week 3复盘，看CES公式实战效果" |
| **帮助引导** | 转发10分 | "🔄转发给正在0点赞的小伙伴，我们一起研究" |

### 5.3 互动率提升技巧

1. **制造轻微争议**（引发评论）
   - "有人说XX不好用，但我用了3瓶"
   - "这个方法可能不适合所有人，但..."

2. **提供独家感**（促进关注+转发）
   - "这个方法我只告诉粉丝"
   - "评论区前10个姐妹，我私信发清单"

3. **建立养成系参与感**（提升粘性）
   - "你们投票，我下周试哪个"
   - "听劝！你们说的我都记下了"

4. **CES透明化**（新颖互动方式）
   - 直接告诉用户CES权重
   - 用"4分""10分"等数字引导行为
