## 一、CES评分机制（核心算法）

### CES评分公式
```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```

| 互动行为 | 权重 | 说明 |
|---------|------|------|
| 关注 | 8分 | **最高权重**，反映用户长期价值(LTV) |
| 评论 | 4分 | 体现内容讨论价值 |
| 转发 | 4分 | 社交传播力指标 |
| 点赞 | 1分 | 基础互动 |
| 收藏 | 1分 | 内容实用价值 |

### 2025年CES新规要点
- **时间衰减系数**：互动数据需乘以 λ=0.1~0.3 的衰减系数，**发布后2小时内互动权重最高**
- **高价值评论**：≥8字且含具体场景描述的评论，权重是普通评论的**3倍**
- **反作弊机制**：秒赞率>50%判定刷量，CES直接**扣1.5分**

## 二、五级流量池层级（2025版）

| 流量层级 | 曝光量级 | 晋级标准/考核指标 |
|---------|---------|----------------|
| **初始池（新手村）** | 50-500次 | CTR≥8%，初始互动率>3%，无违规 |
| **成长池（千人池）** | 1,000-5,000次 | 视频完播率>40%，图文滑动超3屏，CES达标 |
| **潜力池（万人池）** | 3,000-10,000次 | 互动率>5%，用户停留时长>30秒 |
| **爆发池（十万池）** | 1万-10万次 | 收藏率>10%，停留>48秒(图文)/完播率>75%(视频)，触发人工审核 |
| **全域池（百万池）** | 10万+次 | 内容稀缺性评分≥8，进入顶级推荐 |

> **关键门槛**：初始流量池需达到 **CES≥8分** 才能晋级下一层级

## 三、2025年算法核心升级

### 1. 标签匹配机制优化
- **三维兴趣模型**：基于LSTM神经网络，结合用户近30天行为数据（权重62%）
- **AIGC识别技术**：自动标记128个兴趣标签，新增「内容稀缺性评分」
- **LBS+时间双轴模型**：夜间21-23点本地内容曝光量提升37%

### 2. 流量分配新趋势
| 升级点 | 具体变化 |
|-------|---------|
| 长尾扶持 | **50%+流量分配给1000粉丝以下创作者** |
| 遗忘算法 | 用户频繁浏览某垂类时，主动增加其他品类推荐 |
| 原创权重 | 原创性权重提升至**35%-60%** |
| 搜索流量 | 占比逼近**50%**，成为核心流量入口 |

### 3. 内容质量分（CQS）体系
- AI生成内容需人工修改**30%以上**并标注，否则限流50%
- 每100字需包含1个有效信息点
- 首图需在**0.8秒内**传递核心信息

## 五、对我账号的启示

### 当前状态诊断
- 已发布10篇笔记，总曝光~700
- 平均单篇曝光70次，处于**初始池（50-500次）**
- CES评分估算：点赞16 + 收藏31 = 47分（10篇平均4.7分/篇）
- **问题**：单篇CES<8分，难以突破初始池

### 突破策略

#### 1. 提升CES评分（目标：单篇≥8分）
```
当前平均：点赞1.6 + 收藏3.1 = 4.7分
目标配置：点赞5 + 收藏5 + 评论2 = 19分
```

**具体动作**：
- 引导评论：A/B/C选择题设计，目标每篇≥3条评论（+12分）
- 引导关注：文案结尾明确关注理由（关注权重8分）
- 提升收藏：增加"可收藏"的实用信息密度

#### 2. 优化黄金2小时
- 发布后30分钟内主动回复所有评论
- 置顶评论补充干货，延长互动时间
- 分享到私域群聊（如果有）

#### 3. 搜索流量布局
- 标题前10字包含核心关键词
- 正文布局"地域+人群+场景"长尾词
- 标签使用精准搜索词

## 七、关键认知更新

### 1. 为什么我的笔记停留在初始池？

- 用户停留时长>30秒
