## 📝 内容矩阵规划

### 内容配比（721原则）
- **70%**：油皮问题解决（底妆、护肤、定妆）
- **20%**：油皮产品测评（空瓶记、开箱）
- **10%**：个人成长（调味剂，增加人设温度）

### 固定栏目

#### 1. "油皮救星"系列（主栏目）
**更新频率**：每周2-3篇
**内容方向**：
- 油皮底妆持妆攻略
- 春季/夏季油皮护理
- T区控油技巧
- 油皮防晒不油配方

**标题模板**：
```
油皮｜[季节/场景] [问题] [解决方案]
例：油皮｜春季底妆持妆12小时不脱妆
例：油皮｜T区控油3个技巧，下午3点不脱妆
```

#### 2. "空瓶记"系列（测评栏目）
**更新频率**：每周1篇
**内容方向**：
- 用完产品的真实评价
- 油皮适合/不适合的产品清单
- 平价油皮好物推荐

**标题模板**：
```
油皮空瓶记｜[品牌] [产品] 用完了，值吗？
例：油皮空瓶记｜兰蔻粉水用完3瓶，油皮会后悔吗？
```

#### 3. "100次才悟了"系列（踩坑栏目）
**更新频率**：每两周1篇
**内容方向**：
- 化妆踩坑经验总结
- 错误方法vs正确方法对比
- 新手避坑指南

**标题模板**：
```
[主题]｜画了100次才悟出这[X]个技巧
例：单眼皮眼妆｜画了100次才悟出这3步
```

#### 4. "油皮Q&A"系列（互动栏目）
**更新频率**：根据评论区问题
**内容方向**：
- 回复评论区高频问题
- 粉丝投稿问题解答
- 油皮疑惑解答

**标题模板**：
```
油皮Q&A｜[粉丝提问]
例：油皮Q&A｜为什么下午3点必脱妆？
```
