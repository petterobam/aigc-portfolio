## 🔄 工作流程

### 周计划制定流程
```
周日复盘 → 制定下周计划 → 创建周计划文件 → 确定选题 → 分配日期
```

### 内容发布流程
```
选题确定 → 文案撰写 → 创建发布包 → 移入待发布 → 按计划发布 → 移入已发布
```

## 💡 发布计划原则

### 发布频率
- **基础频率**：每周3-4篇
- **高频测试**：每周5-7篇(数据好的时候)
- **低频维护**：每周2-3篇(内容不足时)

### 发布时间
- **最佳时间**：20:00-22:00
- **次优时间**：08:00-09:00
- **周末时间**：10:00-12:00 或 20:00-22:00

### 内容分布
- **教程类**：每周1-2篇
- **好物分享**：每周1篇
- **经验分享**：每周1篇
- **复盘内容**：每周日1篇
