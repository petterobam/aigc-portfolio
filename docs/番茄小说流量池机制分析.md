## 二、番茄小说推荐机制假设

### 2.1 核心评分指标（推测）

基于番茄小说平台特点，推测核心评分指标为：

```
小说评分 = CTR×0.3 + 完读率×0.4 + 互动率×0.2 + 收藏率×0.1
```

| 指标 | 权重（推测） | 说明 |
|------|------------|------|
| **完读率** | 40% | **最高权重**，反映内容质量 |
| **CTR（点击率）** | 30% | 标题吸引力 |
| **互动率** | 20% | 评论、点赞、分享 |
| **收藏率** | 10% | 追更潜力 |

### 2.2 完读率计算（推测）

```
完读率 = 阅读完成人数 ÷ 进入阅读人数 × 100%
```

| 完读率区间 | 质量 | 说明 |
|-----------|------|------|
| 0-20% | 差 | 开篇无法吸引读者 |
| 20-50% | 中 | 开篇有吸引力，中段流失 |
| 50-70% | 良 | 内容质量不错，大部分读者读完 |
| 70-90% | 优 | 内容质量高，读者粘性强 |
| 90%+ | 超优 | 内容质量极高，读者追更 |

### 2.3 互动率计算（推测）

```
互动率 = (点赞 + 评论 + 收藏 + 分享) ÷ 阅读人数 × 100%
```

| 互动率区间 | 质量 | 说明 |
|-----------|------|------|
| 0-1% | 差 | 读者无互动 |
| 1-3% | 中 | 读者少量互动 |
| 3-5% | 良 | 读者互动积极 |
| 5-10% | 优 | 读者互动活跃 |
| 10%+ | 超优 | 读者互动极活跃 |

| **兴趣标签优化** | 基于读者历史阅读行为，精准推荐相似内容 |

| **签约作品加权** | 已签约作品获得更多推荐流量 |

**流量池分布（推测）**：
| 流量池 | 作品数 | 占比 | 说明 |
|-------|-------|------|------|
| 初始池（0-10阅读） | 8篇 | 80% | 未能突破初始池 |
| 推荐池（10-100阅读） | 1篇 | 10% | 《重生之我被逼顶替学籍》45阅读 |

**关键问题**：
- 80%作品（8篇）未能突破初始池
- 完读率可能不足（推测）
- 互动率较低（验证：10篇作品总互动率约2.9%）

### 5.2 突破策略

#### 策略1：提升CTR（点击率）
```

目标CTR：≥5%

- 优化封面（如有）

```

#### 策略2：提升完读率（最重要）
```

具体动作：
```

#### 策略3：提升互动率
```

```

## 七、关键认知更新

### 1. 为什么大部分作品停留在初始池？

**推测原因**：
- CTR低于5%（标题不够吸引人）
- 完读率低于30%（开篇3章质量差）
- 互动率低于1%（爽点密度不足）

**验证方法**：
- 观察阅读数据（如果有）
- 分析评论区反馈（如有）
- A/B测试不同标题和开篇

### 2. 如何突破到推荐池（10-100阅读）？

**推测条件**：
- CTR≥5%
- 完读率≥30%
- 互动率≥1%
- 字数≥6000字

### 3. 如何突破到爆款池（100-500阅读）？

**推测条件**：
- CTR≥8%
- 完读率≥50%
- 互动率≥3%
- 收藏率≥2%

### 4. 番茄小说 vs 小红书的流量池差异

**相同点**：
- 都有多级流量池
- 都有冷启动期
- 都有长尾期

**不同点**：
- 小红书注重互动（CES评分），番茄小说注重完读率
- 小红书推荐时间短（2小时），番茄小说推荐时间长（24小时）
- 小红书有明确的发布时间优化，番茄小说推测影响较小

## 九、待验证假设

### 9.1 假设1：签约作品流量加权

**假设**：已签约作品比未签约作品获得更多推荐流量

**验证方法**：
- 对比已签约作品（9篇）和未签约作品（1篇）的流量
- 如已签约作品平均流量 > 未签约作品，假设成立

### 9.2 假设2：作者等级流量加权

**假设**：Lv.2作者比Lv.1作者作品获得更多推荐流量

**验证方法**：
- 当前账号为Lv.2，观察作品流量
- 如平均流量 > 新作者（Lv.1）平均水平，假设成立

### 9.3 假设3：系列化作品追更率更高

**假设**：系列化作品比单篇作品的追更率更高

**验证方法**：
- 创建系列化作品（如"职场弹幕系统"系列）
- 追踪系列化作品的追更率
- 如追更率 > 单篇作品，假设成立

### 9.4 假设4：完读率是最关键指标

**假设**：完读率是决定流量池晋级的最关键指标

**验证方法**：
- 优化开篇3章，提升完读率
- 观察流量池晋级情况
